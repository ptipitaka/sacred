ඉධ පන, භික්ඛවේ, භික්ඛු සම්බහුලා සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිත්වා පරිමාණා අප්පටිච්ඡාදේත්වා …පේ…  අපරිමාණා අප්පටිච්ඡාදේත්වා …පේ…  ඒකනාමා අප්පටිච්ඡාදේත්වා …පේ…  නානානාමා අප්පටිච්ඡාදේත්වා …පේ…  සභාගා අප්පටිච්ඡාදේත්වා …පේ…  විසභාගා අප්පටිච්ඡාදේත්වා …පේ…  වවත්ථිතා අප්පටිච්ඡාදේත්වා …පේ…  සම්භින්නා අප්පටිච්ඡාදේත්වා විබ්භමති …පේ…  (යථා හේට්ඨා තථා විත්ථාරේතබ්බං).

</Paragraph>
