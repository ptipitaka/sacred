තේන ඛෝ පන සමයේන අඤ්ඤතරෝ භික්ඛු ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපන්නෝ හෝති ද්වේමාසප්පටිච්ඡන්නායෝ. තස්ස ඒතදහෝසි—  “අහං ඛෝ ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිං ද්වේමාසප්පටිච්ඡන්නායෝ. යන්නූනාහං සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචේය්‍යන්”ති. සෝ සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචි. තස්ස සංඝෝ ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං අදාසි. තස්ස පරිවසන්තස්ස ලජ්ජීධම්මෝ ඕක්කමි—  “අහං ඛෝ ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිං ද්වේමාසප්පටිච්ඡන්නායෝ. තස්ස මේ ඒතදහෝසි—  ‘අහං ඛෝ ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිං ද්වේමාසප්පටිච්ඡන්නායෝ. යන්නූනාහං සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචේය්‍යන්’ති. සෝහං සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචිං. තස්ස මේ සංඝෝ ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං අදාසි. තස්ස මේ පරිවසන්තස්ස ලජ්ජීධම්මෝ ඕක්කමි—  ‘යන්නූනාහං සංඝං ඉතරිස්සාපි ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචේය්‍යන්’”ති.
</Paragraph>

<Paragraph number="628" book="cula">
සෝ භික්ඛූනං ආරෝචේසි—  “අහං, ආවුසෝ, ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිං ද්වේමාසප්පටිච්ඡන්නායෝ. තස්ස මේ ඒතදහෝසි—  ‘අහං ඛෝ ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිං ද්වේමාසප්පටිච්ඡන්නායෝ. යන්නූනාහං සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචේය්‍යන්’ති. සෝහං සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචිං. තස්ස මේ සංඝෝ ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං අදාසි. තස්ස මේ පරිවසන්තස්ස ලජ්ජීධම්මෝ ඕක්කමි—  ‘අහං ඛෝ ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිං ද්වේමාසප්පටිච්ඡන්නායෝ. තස්ස මේ ඒතදහෝසි—  “අහං ඛෝ ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිං ද්වේමාසප්පටිච්ඡන්නායෝ. යන්නූනාහං සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචේය්‍යන්”’ති. සෝහං සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචිං. තස්ස මේ සංඝෝ ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං අදාසි. තස්ස මේ පරිවසන්තස්ස ලජ්ජීධම්මෝ ඕක්කමි—  ‘යන්නූනාහං සංඝං ඉතරිස්සාපි ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචේය්‍යන්’ති. කථං නු ඛෝ මයා පටිපජ්ජිතබ්බන්”ති? භගවතෝ ඒතමත්ථං ආරෝචේසුං. “තේන හි, භික්ඛවේ, සංඝෝ තස්ස භික්ඛුනෝ ඉතරිස්සාපි ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං දේතු. ඒවඤ්ච පන, භික්ඛවේ, දාතබ්බෝ—
</Paragraph>

තේන, භික්ඛවේ, භික්ඛුනා සංඝං උපසඞ්කමිත්වා, ඒකංසං උත්තරාසඞ්ගං කරිත්වා, වුඩ්ඪානං භික්ඛූනං පාදේ වන්දිත්වා, උක්කුටිකං නිසීදිත්වා, අඤ්ජලිං පග්ගහේත්වා, ඒවමස්ස වචනීයෝ—  ‘අහං, භන්තේ, ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිං ද්වේමාසප්පටිච්ඡන්නායෝ. තස්ස මේ ඒතදහෝසි—  “අහං ඛෝ ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිං ද්වේමාසප්පටිච්ඡන්නායෝ. යන්නූනාහං සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචේය්‍යන්”ති. සෝහං සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචිං. තස්ස මේ සංඝෝ ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං අදාසි. තස්ස මේ පරිවසන්තස්ස ලජ්ජීධම්මෝ ඕක්කමි—  “අහං ඛෝ ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිං ද්වේමාසප්පටිච්ඡන්නායෝ. තස්ස මේ ඒතදහෝසි—  අහං ඛෝ ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිං ද්වේමාසප්පටිච්ඡන්නායෝ. යන්නූනාහං සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචේය්‍යන්”ති. සෝහං සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචිං. තස්ස මේ සංඝෝ ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං අදාසි. තස්ස මේ පරිවසන්තස්ස ලජ්ජීධම්මෝ ඕක්කමි—  “යන්නූනාහං සංඝං ඉතරිස්සාපි ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචේය්‍යන්”ති. සෝහං, භන්තේ, සංඝං ඉතරිස්සාපි ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචාමී’ති.

දුතියම්පි යාචිතබ්බෝ. තතියම්පි යාචිතබ්බෝ. බ්‍යත්තේන භික්ඛුනා පටිබලේන සංඝෝ ඤාපේතබ්බෝ—

‘සුණාතු මේ, භන්තේ, සංඝෝ. අයං ඉත්ථන්නාමෝ භික්ඛු ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජි ද්වේමාසප්පටිච්ඡන්නායෝ. තස්ස ඒතදහෝසි—  “අහං ඛෝ ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිං ද්වේමාසප්පටිච්ඡන්නායෝ. යන්නූනාහං සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචේය්‍යන්”ති. සෝ සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචි. තස්ස සංඝෝ ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං අදාසි. තස්ස පරිවසන්තස්ස ලජ්ජීධම්මෝ ඕක්කමි—  “අහං ඛෝ ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිං ද්වේමාසප්පටිච්ඡන්නායෝ. තස්ස මේ ඒතදහෝසි—  අහං ඛෝ ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිං ද්වේමාසප්පටිච්ඡන්නායෝ. යන්නූනාහං සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචේය්‍යන්”ති. සෝහං සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචිං. තස්ස මේ සංඝෝ ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං අදාසි. තස්ස මේ පරිවසන්තස්ස ලජ්ජීධම්මෝ ඕක්කමි—  “යන්නූනාහං සංඝං ඉතරිස්සාපි ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචේය්‍යන්”ති. සෝ සංඝං ඉතරිස්සාපි ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචති. යදි සංඝස්ස පත්තකල්ලං, සංඝෝ ඉත්ථන්නාමස්ස භික්ඛුනෝ ඉතරිස්සාපි ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං දදේය්‍ය. ඒසා ඤත්ති.

සුණාතු මේ, භන්තේ, සංඝෝ. අයං ඉත්ථන්නාමෝ භික්ඛු ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජි ද්වේමාසප්පටිච්ඡන්නායෝ. තස්ස ඒතදහෝසි—  “අහං ඛෝ ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිං ද්වේමාසප්පටිච්ඡන්නායෝ. යන්නූනාහං සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචේය්‍යන්”ති. සෝ සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචි. තස්ස සංඝෝ ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං අදාසි. තස්ස පරිවසන්තස්ස ලජ්ජීධම්මෝ ඕක්කමි—  “අහං ඛෝ ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිං ද්වේමාසප්පටිච්ඡන්නායෝ. තස්ස මේ ඒතදහෝසි—  අහං ඛෝ ද්වේ සංඝාදිසේසා ආපත්තියෝ ආපජ්ජිං ද්වේමාසප්පටිච්ඡන්නායෝ. යන්නූනාහං සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචේය්‍යන්”ති. සෝහං සංඝං ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචිං. තස්ස මේ සංඝෝ ඒකිස්සා ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං අදාසි. තස්ස මේ පරිවසන්තස්ස ලජ්ජීධම්මෝ ඕක්කමි—  “යන්නූනාහං සංඝං ඉතරිස්සාපි ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචේය්‍යන්”ති. සෝ සංඝං ඉතරිස්සාපි ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං යාචති. සංඝෝ ඉත්ථන්නාමස්ස භික්ඛුනෝ ඉතරිස්සාපි ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසං දේති. යස්සායස්මතෝ ඛමති ඉත්ථන්නාමස්ස භික්ඛුනෝ ඉතරිස්සාපි ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසස්ස දානං, සෝ තුණ්හස්ස; යස්ස නක්ඛමති, සෝ භාසේය්‍ය.

දින්නෝ සංඝේන ඉත්ථන්නාමස්ස භික්ඛුනෝ ඉතරිස්සාපි ආපත්තියා ද්වේමාසප්පටිච්ඡන්නාය ද්වේමාසපරිවාසෝ. ඛමති සංඝස්ස, තස්මා තුණ්හී, ඒවමේතං ධාරයාමී’ති. තේන, භික්ඛවේ, භික්ඛුනා තදුපාදාය ද්වේ මාසා පරිවසිතබ්බා.
