<Paragraph number="1169" book="para">
තේන ඛෝ පන සමයේන අඤ්ඤතරා ඉත්ථී නවරත්තං කම්බලං පාරුතා හෝති. අඤ්ඤතරෝ භික්ඛු සාරත්තෝ තං ඉත්ථිං ඒතදවෝච—  “ලෝහිතං ඛෝ තේ, භගිනී”ති. සා න පටිවිජානි. “ආමාය්‍ය, නවරත්තෝ කම්බලෝ”ති. තස්ස කුක්කුච්චං අහෝසි “භගවතා සික්ඛාපදං පඤ්ඤත්තං, කච්චි නු ඛෝ අහං සංඝාදිසේසං ආපත්තිං ආපන්නෝ”ති? භගවතෝ ඒතමත්ථං ආරෝචේසි. “අනාපත්ති, භික්ඛු, සංඝාදිසේසස්ස; ආපත්ති දුක්කටස්සා”ති. (1)
</Paragraph>

<Paragraph number="1170" book="para">
තේන ඛෝ පන සමයේන අඤ්ඤතරා ඉත්ථී ඛරකම්බලං පාරුතා හෝති. අඤ්ඤතරෝ භික්ඛු සාරත්තෝ තං ඉත්ථිං ඒතදවෝච—  “කක්කසලෝමං ඛෝ තේ, භගිනී”ති. සා න පටිවිජානි. “ආමාය්‍ය, ඛරකම්බලෝ”ති. තස්ස කුක්කුච්චං අහෝසි …පේ…  “අනාපත්ති, භික්ඛු, සංඝාදිසේසස්ස; ආපත්ති දුක්කටස්සා”ති. (2)
</Paragraph>

<Paragraph number="1171" book="para">
තේන ඛෝ පන සමයේන අඤ්ඤතරා ඉත්ථී නවාවුතං කම්බලං පාරුතා හෝති. අඤ්ඤතරෝ භික්ඛු සාරත්තෝ තං ඉත්ථිං ඒතදවෝච—  “ආකිණ්ණලෝමං ඛෝ තේ, භගිනී”ති. සා න පටිවිජානි. “ආමාය්‍ය, නවාවුතෝ කම්බලෝ”ති. තස්ස කුක්කුච්චං අහෝසි …පේ…  “අනාපත්ති, භික්ඛු, සංඝාදිසේසස්ස; ආපත්ති දුක්කටස්සා”ති. (3)
</Paragraph>

<Paragraph number="1172" book="para">
තේන ඛෝ පන සමයේන අඤ්ඤතරා ඉත්ථී ඛරකම්බලකං පාරුතා හෝති. අඤ්ඤතරෝ භික්ඛු සාරත්තෝ තං ඉත්ථිං ඒතදවෝච—  “ඛරලෝමං ඛෝ තේ, භගිනී”ති. සා න පටිවිජානි. “ආමාය්‍ය, ඛරකම්බලකෝ”ති. තස්ස කුක්කුච්චං අහෝසි …පේ…  “අනාපත්ති, භික්ඛු, සංඝාදිසේසස්ස; ආපත්ති දුක්කටස්සා”ති. (4)
</Paragraph>

<Paragraph number="1173" book="para">
තේන ඛෝ පන සමයේන අඤ්ඤතරා ඉත්ථී පාවාරං පාරුතා හෝති. අඤ්ඤතරෝ භික්ඛු සාරත්තෝ තං ඉත්ථිං ඒතදවෝච—  “දීඝලෝමං ඛෝ තේ, භගිනී”ති. සා න පටිවිජානි. “ආමාය්‍ය, පාවාරෝ”ති. තස්ස කුක්කුච්චං අහෝසි …පේ…  “අනාපත්ති, භික්ඛු, සංඝාදිසේසස්ස; ආපත්ති දුක්කටස්සා”ති. (5)
</Paragraph>

</Division>
<Division number="289" book="para">

<Paragraph number="1174" book="para">
තේන ඛෝ පන සමයේන අඤ්ඤතරා ඉත්ථී ඛේත්තං වපාපේත්වා ආගච්ඡති. අඤ්ඤතරෝ භික්ඛු සාරත්තෝ තං ඉත්ථිං ඒතදවෝච—  “වාපිතං ඛෝ තේ, භගිනී”ති? සා න පටිවිජානි. “ආමාය්‍ය, නෝ ච ඛෝ පටිවුත්තන්”ති. තස්ස කුක්කුච්චං අහෝසි …පේ…  “අනාපත්ති, භික්ඛු, සංඝාදිසේසස්ස; ආපත්ති දුක්කටස්සා”ති. (6)
</Paragraph>

<Paragraph number="1175" book="para">
තේන ඛෝ පන සමයේන අඤ්ඤතරෝ භික්ඛු පරිබ්බාජිකං පටිපථේ පස්සිත්වා සාරත්තෝ තං පරිබ්බාජිකං ඒතදවෝච—  “කච්චි, භගිනි, මග්ගෝ සංසීදතී”ති? සා න පටිවිජානි. “ආම, භික්ඛු, පටිපජ්ජිස්සසී”ති. තස්ස කුක්කුච්චං අහෝසි …පේ…  “අනාපත්ති, භික්ඛු, සංඝාදිසේසස්ස; ආපත්ති ථුල්ලච්චයස්සා”ති. (7)
</Paragraph>

<Paragraph number="1176" book="para">
තේන ඛෝ පන සමයේන අඤ්ඤතරෝ භික්ඛු සාරත්තෝ අඤ්ඤතරං ඉත්ථිං ඒතදවෝච—  “සද්ධාසි ත්වං, භගිනි. අපි ච යං සාමිකස්ස දේසි තං නාම්හාකං දේසී”ති. “කිං, භන්තේ”ති? “මේථුනධම්මන්”ති. තස්ස කුක්කුච්චං අහෝසි …පේ…  “ආපත්තිං ත්වං, භික්ඛු, ආපන්නෝ සංඝාදිසේසන්”ති. (8)
</Paragraph>

<Paragraph number="1177" book="para">
තේන ඛෝ පන සමයේන අඤ්ඤතරෝ භික්ඛු සාරත්තෝ අඤ්ඤතරං ඉත්ථිං ඒතදවෝච—  “සද්ධාසි ත්වං, භගිනි. අපි ච යං අග්ගදානං තං නාම්හාකං දේසී”ති. “කිං, භන්තේ, අග්ගදානන්”ති? “මේථුනධම්මන්”ති. තස්ස කුක්කුච්චං අහෝසි …පේ…  “ආපත්තිං ත්වං, භික්ඛු, ආපන්නෝ සංඝාදිසේසන්”ති. (9)
</Paragraph>

<Paragraph number="1178" book="para">
තේන ඛෝ පන සමයේන අඤ්ඤතරා ඉත්ථී කම්මං කරෝති. අඤ්ඤතරෝ භික්ඛු සාරත්තෝ තං ඉත්ථිං ඒතදවෝච—  “තිට්ඨ, භගිනි, අහං කරිස්සාමී”ති …පේ…  නිසීද, භගිනි, අහං කරිස්සාමී”ති …පේ…  නිපජ්ජ, භගිනි, අහං කරිස්සාමී”ති. සා න පටිවිජානි. තස්ස කුක්කුච්චං අහෝසි …පේ…  “අනාපත්ති, භික්ඛු, සංඝාදිසේසස්ස; ආපත්ති දුක්කටස්සා”ති. (10–12)

</Paragraph>
