<Paragraph number="1730" book="vi-maha">
ඉධ පන, භික්ඛවේ, වස්සංවුට්ඨානං භික්ඛූනං අනුප්පන්නේ චීවරේ සංඝෝ භිජ්ජති. තත්ථ මනුස්සා ඒකස්මිං පක්ඛේ උදකං දේන්ති, ඒකස්මිං පක්ඛේ චීවරං දේන්ති—  සංඝස්ස දේමාති. සංඝස්සේවේතං.
</Paragraph>

<Paragraph number="1731" book="vi-maha">
ඉධ පන, භික්ඛවේ, වස්සංවුට්ඨානං භික්ඛූනං අනුප්පන්නේ චීවරේ සංඝෝ භිජ්ජති. තත්ථ මනුස්සා ඒකස්මිං පක්ඛේ උදකං දේන්ති, තස්මිංයේව පක්ඛේ චීවරං දේන්ති—  සංඝස්ස දේමාති. සංඝස්සේවේතං.
</Paragraph>

<Paragraph number="1732" book="vi-maha">
ඉධ පන, භික්ඛවේ, වස්සංවුට්ඨානං භික්ඛූනං අනුප්පන්නේ චීවරේ සංඝෝ භිජ්ජති. තත්ථ මනුස්සා ඒකස්මිං පක්ඛේ උදකං දේන්ති, ඒකස්මිං පක්ඛේ චීවරං දේන්ති—  පක්ඛස්ස දේමාති. පක්ඛස්සේවේතං.
</Paragraph>

<Paragraph number="1733" book="vi-maha">
ඉධ පන, භික්ඛවේ, වස්සංවුට්ඨානං භික්ඛූනං අනුප්පන්නේ චීවරේ සංඝෝ භිජ්ජති. තත්ථ මනුස්සා ඒකස්මිං පක්ඛේ උදකං දේන්ති, තස්මිංයේව පක්ඛේ චීවරං දේන්ති—  පක්ඛස්ස දේමාති. පක්ඛස්සේවේතං.
</Paragraph>

<Paragraph number="1734" book="vi-maha">
ඉධ පන, භික්ඛවේ, වස්සංවුට්ඨානං භික්ඛූනං උප්පන්නේ චීවරේ අභාජිතේ සංඝෝ භිජ්ජති. සබ්බේසං සමකං භාජේතබ්බන්”ති.
</Paragraph>
