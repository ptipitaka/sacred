තේන ඛෝ පන සමයේන සාකේතේ සේට්ඨිභරියාය සත්තවස්සිකෝ සීසාබාධෝ හෝති. බහූ මහන්තා මහන්තා දිසාපාමෝක්ඛා වේජ්ජා ආගන්ත්වා නාසක්ඛිංසු අරෝගං කාතුං. බහුං හිරඤ්ඤං ආදාය අගමංසු. අථ ඛෝ ජීවකෝ කෝමාරභච්චෝ සාකේතං පවිසිත්වා මනුස්සේ පුච්ඡි—  “කෝ, භණේ, ගිලානෝ, කං තිකිච්ඡාමී”ති? “ඒතිස්සා, ආචරිය, සේට්ඨිභරියාය සත්තවස්සිකෝ සීසාබාධෝ; ගච්ඡ, ආචරිය, සේට්ඨිභරියං තිකිච්ඡාහී”ති. අථ ඛෝ ජීවකෝ කෝමාරභච්චෝ යේන සේට්ඨිස්ස ගහපතිස්ස නිවේසනං තේනුපසඞ්කමි; උපසඞ්කමිත්වා දෝවාරිකං ආණාපේසි—  “ගච්ඡ, භණේ දෝවාරික, සේට්ඨිභරියාය පාවද—  ‘වේජ්ජෝ, අය්‍යේ, ආගතෝ, සෝ තං දට්ඨුකාමෝ’”ති. “ඒවං, ආචරියා”ති ඛෝ සෝ දෝවාරිකෝ ජීවකස්ස කෝමාරභච්චස්ස පටිස්සුත්වා යේන සේට්ඨිභරියා තේනුපසඞ්කමි, උපසඞ්කමිත්වා සේට්ඨිභරියං ඒතදවෝච—  “වේජ්ජෝ, අය්‍යේ, ආගතෝ; සෝ තං දට්ඨුකාමෝ”ති. “කීදිසෝ, භණේ දෝවාරික, වේජ්ජෝ”ති? “දහරකෝ, අය්‍යේ”ති. “අලං, භණේ දෝවාරික, කිං මේ දහරකෝ වේජ්ජෝ කරිස්සති? බහූ මහන්තා මහන්තා දිසාපාමෝක්ඛා වේජ්ජා ආගන්ත්වා නාසක්ඛිංසු අරෝගං කාතුං. බහුං හිරඤ්ඤං ආදාය අගමංසූ”ති. අථ ඛෝ සෝ දෝවාරිකෝ යේන ජීවකෝ කෝමාරභච්චෝ තේනුපසඞ්කමි; උපසඞ්කමිත්වා ජීවකං කෝමාරභච්චං ඒතදවෝච—  “සේට්ඨිභරියා, ආචරිය, ඒවමාහ—  ‘අලං, භණේ දෝවාරික, කිං මේ දහරකෝ වේජ්ජෝ කරිස්සති? බහූ මහන්තා මහන්තා දිසාපාමෝක්ඛා වේජ්ජා ආගන්ත්වා නාසක්ඛිංසු අරෝගං කාතුං. බහුං හිරඤ්ඤං ආදාය අගමංසූ’”ති. “ගච්ඡ, භණේ දෝවාරික, සේට්ඨිභරියාය පාවද—  ‘වේජ්ජෝ, අය්‍යේ, ඒවමාහ—  මා කිර, අය්‍යේ, පුරේ කිඤ්චි අදාසි. යදා අරෝගා අහෝසි තදා යං ඉච්ඡේය්‍යාසි තං දජ්ජේය්‍යාසී’”ති. “ඒවං, ආචරියා”ති ඛෝ සෝ දෝවාරිකෝ ජීවකස්ස කෝමාරභච්චස්ස පටිස්සුත්වා යේන සේට්ඨිභරියා තේනුපසඞ්කමි; උපසඞ්කමිත්වා සේට්ඨිභරියං ඒතදවෝච—  “වේජ්ජෝ, අය්‍යේ, ඒවමාහ—  ‘මා කිර, අය්‍යේ, පුරේ කිඤ්චි අදාසි. යදා අරෝගා අහෝසි තදා යං ඉච්ඡේය්‍යාසි තං දජ්ජේය්‍යාසී’”ති. “තේන හි, භණේ දෝවාරික, වේජ්ජෝ ආගච්ඡතූ”ති. “ඒවං, අය්‍යේ”ති ඛෝ සෝ දෝවාරිකෝ සේට්ඨිභරියාය පටිස්සුත්වා යේන ජීවකෝ කෝමාරභච්චෝ තේනුපසඞ්කමි, උපසඞ්කමිත්වා ජීවකං කෝමාරභච්චං ඒතදවෝච—  “සේට්ඨිභරියා තං, ආචරිය, පක්කෝසතී”ති.
</Paragraph>

අථ ඛෝ ජීවකෝ කෝමාරභච්චෝ යේන සේට්ඨිභරියා තේනුපසඞ්කමි, උපසඞ්කමිත්වා සේට්ඨිභරියාය විකාරං සල්ලක්ඛේත්වා සේට්ඨිභරියං ඒතදවෝච—  “පසතේන, අය්‍යේ, සප්පිනා අත්ථෝ”ති. අථ ඛෝ සේට්ඨිභරියා ජීවකස්ස කෝමාරභච්චස්ස පසතං සප්පිං දාපේසි. අථ ඛෝ ජීවකෝ කෝමාරභච්චෝ තං පසතං සප්පිං නානාභේසජ්ජේහි නිප්පචිත්වා සේට්ඨිභරියං මඤ්චකේ උත්තානං නිපාතේත්වා නත්ථුතෝ අදාසි. අථ ඛෝ තං සප්පි නත්ථුතෝ දින්නං මුඛතෝ උග්ගඤ්ඡි. අථ ඛෝ සේට්ඨිභරියා පටිග්ගහේ නිට්ඨුභිත්වා දාසිං ආණාපේසි—  “හන්ද, ජේ, ඉමං සප්පිං පිචුනා ගණ්හාහී”ති. අථ ඛෝ ජීවකස්ස කෝමාරභච්චස්ස ඒතදහෝසි—  “අච්ඡරියං යාව ලූඛායං ඝරණී, යත්‍ර හි නාම ඉමං ඡඩ්ඩනීයධම්මං සප්පිං පිචුනා ගාහාපේස්සති. බහුකානි ච මේ මහග්ඝානි භේසජ්ජානි උපගතානි. කිම්පි මායං කිඤ්චි දේය්‍යධම්මං දස්සතී”ති. අථ ඛෝ සේට්ඨිභරියා ජීවකස්ස කෝමාරභච්චස්ස විකාරං සල්ලක්ඛේත්වා ජීවකං කෝමාරභච්චං ඒතදවෝච—  “කිස්ස ත්වං, ආචරිය, විමනෝසී”ති? ඉධ මේ ඒතදහෝසි—  “අච්ඡරියං යාව ලූඛායං ඝරණී, යත්‍ර හි නාම ඉමං ඡඩ්ඩනීයධම්මං සප්පිං පිචුනා ගාහාපේස්සති. බහුකානි ච මේ මහග්ඝානි භේසජ්ජානි උපගතානි. කිම්පි මායං කිඤ්චි දේය්‍යධම්මං දස්සතී”ති. “මයං ඛෝ, ආචරිය, ආගාරිකා නාම උපජානාමේතස්ස සංයමස්ස. වරමේතං සප්පි දාසානං වා කම්මකරානං වා පාදබ්භඤ්ජනං වා පදීපකරණේ වා ආසිත්තං. මා ඛෝ ත්වං, ආචරිය, විමනෝ අහෝසි. න තේ දේය්‍යධම්මෝ හායිස්සතී”ති.
</Paragraph>

අථ ඛෝ ජීවකෝ කෝමාරභච්චෝ සේට්ඨිභරියාය සත්තවස්සිකං සීසාබාධං ඒකේනේව නත්ථුකම්මේන අපකඩ්ඪි. අථ ඛෝ සේට්ඨිභරියා අරෝගා සමානා ජීවකස්ස කෝමාරභච්චස්ස චත්තාරි සහස්සානි පාදාසි. පුත්තෝ—  “මාතා මේ අරෝගා ඨිතා”ති චත්තාරි සහස්සානි පාදාසි. සුණිසා—  “සස්සු මේ අරෝගා ඨිතා”ති චත්තාරි සහස්සානි පාදාසි. සේට්ඨි ගහපති—  “භරියා මේ අරෝගා ඨිතා”ති චත්තාරි සහස්සානි පාදාසි දාසඤ්ච දාසිඤ්ච අස්සරථඤ්ච.
</Paragraph>

අථ ඛෝ ජීවකෝ කෝමාරභච්චෝ තානි සෝළසසහස්සානි ආදාය දාසඤ්ච දාසිඤ්ච අස්සරථඤ්ච යේන රාජගහං තේන පක්කාමි. අනුපුබ්බේන යේන රාජගහං යේන අභයෝ රාජකුමාරෝ තේනුපසඞ්කමි, උපසඞ්කමිත්වා අභයං රාජකුමාරං ඒතදවෝච—  “ඉදං මේ, දේව, පඨමකම්මං සෝළසසහස්සානි දාසෝ ච දාසී ච අස්සරථෝ ච. පටිග්ගණ්හාතු මේ දේවෝ පෝසාවනිකන්”ති. “අලං, භණේ ජීවක; තුය්හමේව හෝතු. අම්හාකඤ්ඤේව අන්තේපුරේ නිවේසනං මාපේහී”ති. “ඒවං, දේවා”ති ඛෝ ජීවකෝ කෝමාරභච්චෝ අභයස්ස රාජකුමාරස්ස පටිස්සුත්වා අභයස්ස රාජකුමාරස්ස අන්තේපුරේ නිවේසනං මාපේසි.
