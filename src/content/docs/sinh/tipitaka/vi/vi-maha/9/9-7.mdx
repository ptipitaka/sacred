<Paragraph number="1814" book="vi-maha">
ඉධ පන, භික්ඛවේ, භික්ඛුස්ස න හෝති ආපත්ති දට්ඨබ්බා. තමේනං චෝදේති සංඝෝ වා සම්බහුලා වා ඒකපුග්ගලෝ වා—  ‘ආපත්තිං ත්වං, ආවුසෝ, ආපන්නෝ, පස්සසේතං ආපත්තින්’ති? සෝ ඒවං වදේති—  ‘නත්ථි මේ, ආවුසෝ, ආපත්ති, යමහං පස්සේය්‍යන්’ති. තං සංඝෝ ආපත්තියා අදස්සනේ උක්ඛිපති—  අධම්මකම්මං.
</Paragraph>

<Paragraph number="1815" book="vi-maha">
ඉධ පන, භික්ඛවේ, භික්ඛුස්ස න හෝති ආපත්ති පටිකාතබ්බා. තමේනං චෝදේති සංඝෝ වා සම්බහුලා වා ඒකපුග්ගලෝ වා—  ‘ආපත්තිං ත්වං, ආවුසෝ, ආපන්නෝ, පටිකරෝහි තං ආපත්තින්’ති. සෝ ඒවං වදේති—  ‘නත්ථි මේ, ආවුසෝ, ආපත්ති, යමහං පටිකරේය්‍යන්’ති. තං සංඝෝ ආපත්තියා අප්පටිකම්මේ උක්ඛිපති—  අධම්මකම්මං.
</Paragraph>

<Paragraph number="1816" book="vi-maha">
ඉධ පන, භික්ඛවේ, භික්ඛුස්ස න හෝති පාපිකා දිට්ඨි පටිනිස්සජ්ජේතා. තමේනං චෝදේති සංඝෝ වා සම්බහුලා වා ඒකපුග්ගලෝ වා—  ‘පාපිකා තේ, ආවුසෝ, දිට්ඨි, පටිනිස්සජ්ජේතං පාපිකං දිට්ඨින්’ති. සෝ ඒවං වදේති—  ‘නත්ථි මේ, ආවුසෝ, පාපිකා දිට්ඨි, යමහං පටිනිස්සජ්ජේය්‍යන්’ති. තං සංඝෝ පාපිකාය දිට්ඨියා අප්පටිනිස්සග්ගේ උක්ඛිපති—  අධම්මකම්මං.
</Paragraph>

<Paragraph number="1817" book="vi-maha">
ඉධ පන, භික්ඛවේ, භික්ඛුස්ස න හෝති ආපත්ති දට්ඨබ්බා, න හෝති ආපත්ති පටිකාතබ්බා. තමේනං චෝදේති සංඝෝ වා සම්බහුලා වා ඒකපුග්ගලෝ වා—  ‘ආපත්තිං ත්වං, ආවුසෝ, ආපන්නෝ, පස්සසේතං ආපත්තිං? පටිකරෝහි තං ආපත්තින්’ති. සෝ ඒවං වදේති—  ‘නත්ථි මේ, ආවුසෝ, ආපත්ති, යමහං පස්සේය්‍යං. නත්ථි මේ, ආවුසෝ, ආපත්ති, යමහං පටිකරේය්‍යන්’ති. තං සංඝෝ අදස්සනේ වා අප්පටිකම්මේ වා උක්ඛිපති—  අධම්මකම්මං.
</Paragraph>

<Paragraph number="1818" book="vi-maha">
ඉධ පන, භික්ඛවේ, භික්ඛුස්ස න හෝති ආපත්ති දට්ඨබ්බා, න හෝති පාපිකා දිට්ඨි පටිනිස්සජ්ජේතා. තමේනං චෝදේති සංඝෝ වා සම්බහුලා වා ඒකපුග්ගලෝ වා—  ‘ආපත්තිං ත්වං, ආවුසෝ, ආපන්නෝ, පස්සසේතං ආපත්තිං? පාපිකා තේ දිට්ඨි, පටිනිස්සජ්ජේතං පාපිකං දිට්ඨින්’ති. සෝ ඒවං වදේති—  ‘නත්ථි මේ, ආවුසෝ, ආපත්ති, යමහං පස්සේය්‍යං; නත්ථි මේ, ආවුසෝ, පාපිකා දිට්ඨි, යමහං පටිනිස්සජ්ජේය්‍යන්’ති. තං සංඝෝ අදස්සනේ වා අප්පටිනිස්සග්ගේ වා උක්ඛිපති—  අධම්මකම්මං.
</Paragraph>

<Paragraph number="1819" book="vi-maha">
ඉධ පන, භික්ඛවේ, භික්ඛුස්ස න හෝති ආපත්ති පටිකාතබ්බා, න හෝති පාපිකා දිට්ඨි පටිනිස්සජ්ජේතා. තමේනං චෝදේති සංඝෝ වා සම්බහුලා වා ඒකපුග්ගලෝ වා—  ‘ආපත්තිං ත්වං, ආවුසෝ, ආපන්නෝ, පටිකරෝහි තං ආපත්තිං; පාපිකා තේ දිට්ඨි, පටිනිස්සජ්ජේතං පාපිකං දිට්ඨින්’ති. සෝ ඒවං වදේති—  ‘නත්ථි මේ, ආවුසෝ, ආපත්ති, යමහං පටිකරේය්‍යං. නත්ථි මේ, ආවුසෝ, පාපිකා දිට්ඨි, යමහං පටිනිස්සජ්ජේය්‍යන්’ති. තං සංඝෝ අප්පටිකම්මේ වා අප්පටිනිස්සග්ගේ වා උක්ඛිපති—  අධම්මකම්මං.
</Paragraph>

<Paragraph number="1820" book="vi-maha">
ඉධ පන, භික්ඛවේ, භික්ඛුස්ස න හෝති ආපත්ති දට්ඨබ්බා, න හෝති ආපත්ති පටිකාතබ්බා, න හෝති පාපිකා දිට්ඨි පටිනිස්සජ්ජේතා. තමේනං චෝදේති සංඝෝ වා සම්බහුලා වා ඒකපුග්ගලෝ වා—  ‘ආපත්තිං ත්වං, ආවුසෝ, ආපන්නෝ, පස්සසේතං ආපත්තිං? පටිකරෝහි තං ආපත්තිං; පාපිකා තේ දිට්ඨි, පටිනිස්සජ්ජේතං පාපිකං දිට්ඨින්’ති. සෝ ඒවං වදේති—  ‘නත්ථි මේ, ආවුසෝ, ආපත්ති, යමහං පස්සේය්‍යං. නත්ථි මේ, ආවුසෝ, ආපත්ති, යමහං පටිකරේය්‍යං. නත්ථි මේ, ආවුසෝ, පාපිකා දිට්ඨි, යමහං පටිනිස්සජ්ජේය්‍යන්’ති. තං සංඝෝ අදස්සනේ වා අප්පටිකම්මේ වා අප්පටිනිස්සග්ගේ වා උක්ඛිපති—  අධම්මකම්මං.
</Paragraph>

<Paragraph number="1821" book="vi-maha">
ඉධ පන, භික්ඛවේ, භික්ඛුස්ස හෝති ආපත්ති දට්ඨබ්බා. තමේනං චෝදේති. සංඝෝ වා සම්බහුලා වා ඒකපුග්ගලෝ වා—  ‘ආපත්තිං ත්වං, ආවුසෝ, ආපන්නෝ, පස්සසේතං ආපත්තින්’ති? සෝ ඒවං වදේති—  ‘ආමාවුසෝ, පස්සාමී’ති. තං සංඝෝ ආපත්තියා අදස්සනේ උක්ඛිපති—  අධම්මකම්මං.
</Paragraph>

ඉධ පන, භික්ඛවේ, භික්ඛුස්ස හෝති ආපත්ති පටිකාතබ්බා. තමේනං චෝදේති සංඝෝ වා සම්බහුලා වා ඒකපුග්ගලෝ වා—  ‘ආපත්තිං ත්වං, ආවුසෝ, ආපන්නෝ, පටිකරෝහි තං ආපත්තින්’ති. සෝ ඒවං වදේති—  ‘ආමාවුසෝ, පටිකරිස්සාමී’ති. තං සංඝෝ ආපත්තියා අප්පටිකම්මේ උක්ඛිපති—  අධම්මකම්මං.
</Paragraph>

<Paragraph number="1823" book="vi-maha">
ඉධ පන, භික්ඛවේ, භික්ඛුස්ස හෝති පාපිකා දිට්ඨි පටිනිස්සජ්ජේතා. තමේනං චෝදේති සංඝෝ වා සම්බහුලා වා ඒකපුග්ගලෝ වා—  ‘පාපිකා තේ, ආවුසෝ, දිට්ඨි; පටිනිස්සජ්ජේතං පාපිකං දිට්ඨින්’ති. සෝ ඒවං වදේති—  ‘ආමාවුසෝ, පටිනිස්සජ්ජිස්සාමී’ති. තං සංඝෝ පාපිකාය දිට්ඨියා අප්පටිනිස්සග්ගේ උක්ඛිපති—  අධම්මකම්මං.
</Paragraph>

<Paragraph number="1824" book="vi-maha">
ඉධ පන, භික්ඛවේ, භික්ඛුස්ස හෝති ආපත්ති දට්ඨබ්බා, හෝති ආපත්ති පටිකාතබ්බා …පේ…  හෝති ආපත්ති දට්ඨබ්බා, හෝති පාපිකා දිට්ඨි පටිනිස්සජ්ජේතා …පේ…  හෝති ආපත්ති පටිකාතබ්බා, හෝති පාපිකා දිට්ඨි පටිනිස්සජ්ජේතා …පේ…  හෝති ආපත්ති දට්ඨබ්බා, හෝති ආපත්ති පටිකාතබ්බා, හෝති පාපිකා දිට්ඨි පටිනිස්සජ්ජේතා. තමේනං චෝදේති සංඝෝ වා සම්බහුලා වා ඒකපුග්ගලෝ වා—  ‘ආපත්තිං ත්වං, ආවුසෝ, ආපන්නෝ, පස්සසේතං ආපත්තිං? පටිකරෝහි තං ආපත්තිං; පාපිකා තේ දිට්ඨි, පටිනිස්සජ්ජේතං පාපිකං දිට්ඨින්’ති. සෝ ඒවං වදේති—  ‘ආමාවුසෝ, පස්සාමි, ආම පටිකරිස්සාමි, ආම පටිනිස්සජ්ජිස්සාමී’ති. තං සංඝෝ අදස්සනේ වා අප්පටිකම්මේ වා අප්පටිනිස්සග්ගේ වා උක්ඛිපති—  අධම්මකම්මං.
</Paragraph>

<Paragraph number="1825" book="vi-maha">
ඉධ පන, භික්ඛවේ, භික්ඛුස්ස හෝති ආපත්ති දට්ඨබ්බා. තමේනං චෝදේති සංඝෝ වා සම්බහුලා වා ඒකපුග්ගලෝ වා—  ‘ආපත්තිං ත්වං, ආවුසෝ, ආපන්නෝ, පස්සසේතං ආපත්තින්’ති? සෝ ඒවං වදේති—  ‘නත්ථි මේ, ආවුසෝ, ආපත්ති, යමහං පස්සේය්‍යන්’ති. තං සංඝෝ ආපත්තියා අදස්සනේ උක්ඛිපති—  ධම්මකම්මං.

<Paragraph number="1826" book="vi-maha">
ඉධ පන, භික්ඛවේ, භික්ඛුස්ස හෝති ආපත්ති පටිකාතබ්බා. තමේනං චෝදේති සංඝෝ වා සම්බහුලා වා ඒකපුග්ගලෝ වා—  ‘ආපත්තිං ත්වං, ආවුසෝ, ආපන්නෝ, පටිකරෝහි තං ආපත්තින්’ති. සෝ ඒවං වදේති—  ‘නත්ථි මේ, ආවුසෝ, ආපත්ති, යමහං පටිකරේය්‍යන්’ති. තං සංඝෝ ආපත්තියා අප්පටිකම්මේ උක්ඛිපති—  ධම්මකම්මං.

<Paragraph number="1827" book="vi-maha">
ඉධ පන, භික්ඛවේ, භික්ඛුස්ස හෝති පාපිකා දිට්ඨි පටිනිස්සජ්ජේතා. තමේනං චෝදේති සංඝෝ වා සම්බහුලා වා ඒකපුග්ගලෝ වා—  ‘පාපිකා තේ, ආවුසෝ, දිට්ඨි, පටිනිස්සජ්ජේතං පාපිකං දිට්ඨින්’ති. සෝ ඒවං වදේති—  ‘නත්ථි මේ, ආවුසෝ, පාපිකා දිට්ඨි, යමහං පටිනිස්සජ්ජේය්‍යන්’ති. තං සංඝෝ පාපිකාය දිට්ඨියා අප්පටිනිස්සග්ගේ උක්ඛිපති—  ධම්මකම්මං.
</Paragraph>

ඉධ පන, භික්ඛවේ, භික්ඛුස්ස හෝති ආපත්ති දට්ඨබ්බා, හෝති ආපත්ති පටිකාතබ්බා …පේ…  හෝති ආපත්ති දට්ඨබ්බා, හෝති පාපිකා දිට්ඨි පටිනිස්සජ්ජේතා …පේ…  හෝති ආපත්ති පටිකාතබ්බා, හෝති පාපිකා දිට්ඨි පටිනිස්සජ්ජේතා …පේ…  හෝති ආපත්ති දට්ඨබ්බා, හෝති ආපත්ති පටිකාතබ්බා, හෝති පාපිකා දිට්ඨි පටිනිස්සජ්ජේතා. තමේනං චෝදේති සංඝෝ වා සම්බහුලා වා ඒකපුග්ගලෝ වා—  ‘ආපත්තිං ත්වං, ආවුසෝ, ආපන්නෝ, පස්සසේතං ආපත්තිං? පටිකරෝහි තං ආපත්තිං. පාපිකා තේ දිට්ඨි, පටිනිස්සජ්ජේතං පාපිකං දිට්ඨින්’ති. සෝ ඒවං වදේති—  ‘නත්ථි මේ, ආවුසෝ, ආපත්ති, යමහං පස්සේය්‍යං. නත්ථි මේ, ආවුසෝ, ආපත්ති යමහං පටිකරේය්‍යං. නත්ථි මේ, ආවුසෝ, පාපිකා දිට්ඨි, යමහං පටිනිස්සජ්ජේය්‍යන්’ති. තං සංඝෝ අදස්සනේ වා අප්පටිකම්මේ වා අප්පටිනිස්සග්ගේ වා උක්ඛිපති—  ධම්මකම්මන්”ති.
