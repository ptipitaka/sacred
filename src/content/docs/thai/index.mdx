ยินดีต้อนรับสู่ **SACRED** แพลตฟอร์มความร่วมมือที่อุทิศเพื่อรักษาความบริสุทธิ์ของพระธรรมอันศักดิ์สิทธิ์ จากการสืบทอดผ่าน**ระบบสังคายนา** เพื่อเป็น**มาตรฐานและแหล่งอ้างอิง**สำหรับการศึกษาและการสืบทอดสู่คนรุ่นหลัง

จากประวัติศาสตร์แห่งการสืบทอดอันยาวนาน ผ่านกระบวนจดจำโดยพระอรหันต์เถระ การจารึกบนใบลาน จนถึงยุคเทคโนโลยีการพิมพ์ บัดนี้โลกได้ก้าวเข้าสู่ยุคดิจิทัลและปัญญาประดิษฐ์เพื่อขยายการเข้าถึงพระธรรมคำสอนให้สะดวกและแพร่หลายยิ่งขึ้น **พร้อมทั้งรักษาความบริสุทธิ์และเที่ยงตรงของพระธรรม เพื่อการธำรงไว้แห่งพระสัทธรรมที่มั่นคงสืบไป**

### พระไตรปิฎก ฉัฏฐสังคายนา

**การสังคายนาพระธรรมวินัยครั้งล่าสุด** จัดขึ้นในปี พ.ศ. 2497 (ค.ศ. 1954) ณ กรุงย่างกุ้ง ประเทศเมียนมา โดยรวบรวมพระสงฆ์กว่า 2,500 รูปจากประเทศเถรวาท เพื่อรักษาพระไตรปิฎกภาษาบาลี

  - [**မြန်မာ** (เมียนมา)](h/mymr/tipitaka/vi/para/) - อักษรเมียนมา
  - [**සිංහල** (ศรีลังกา)](h/sinh/tipitaka/vi/para/) - อักษรสิงหล  
  - [**ไทย** (ประเทศไทย)](h/thai/tipitaka/vi/para/) - อักษรไทย
  - [**Roman**](h/romn/tipitaka/vi/para/) - อักษรโรมัน
  - [**देवनागरी** (อินเดีย)](h/deva/tipitaka/vi/para/) - อักษรเทวนาครี
  - [**ᩃ᩶ᩣᨶ᩶ᩣ** (ล้านนา)](h/lana/tipitaka/vi/para/) - อักษรธรรมล้านนา
  - [**ລາວ** (ลาว)](h/laoo/tipitaka/vi/para/) - อักษรธรรมลาว
  - [**ខ្មែរ** (กัมพูชา)](h/khmr/tipitaka/vi/para/) - อักษรขอม
