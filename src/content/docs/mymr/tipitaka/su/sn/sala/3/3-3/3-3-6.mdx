“ပဉ္စိမာနိ, ဘိက္ခဝေ, မာတုဂါမဿ ဗလာနိ. ကတမာနိ ပဉ္စ? ရူပဗလံ, ဘောဂဗလံ, ဉာတိဗလံ, ပုတ္တဗလံ, သီလဗလံ. ရူပဗလေန စ, ဘိက္ခဝေ, မာတုဂါမော သမန္နာဂတော ဟောတိ, န စ သီလဗလေန, နာသေန္တေဝ နံ, ကုလေ န ဝါသေန္တိ. ရူပဗလေန စ, ဘိက္ခဝေ, မာတုဂါမော သမန္နာဂတော ဟောတိ, ဘောဂဗလေန စ, န စ သီလဗလေန, နာသေန္တေဝ နံ, ကုလေ န ဝါသေန္တိ. ရူပဗလေန စ, ဘိက္ခဝေ, မာတုဂါမော သမန္နာဂတော ဟောတိ, ဘောဂဗလေန စ, ဉာတိဗလေန စ, န စ သီလဗလေန, နာသေန္တေဝ နံ, ကုလေ န ဝါသေန္တိ. ရူပဗလေန စ, ဘိက္ခဝေ, မာတုဂါမော သမန္နာဂတော ဟောတိ, ဘောဂဗလေန စ, ဉာတိဗလေန စ, ပုတ္တဗလေန စ, န စ သီလဗလေန, နာသေန္တေဝ နံ, ကုလေ န ဝါသေန္တိ. သီလဗလေန စ, ဘိက္ခဝေ, မာတုဂါမော သမန္နာဂတော ဟောတိ, န စ ရူပဗလေန, ဝါသေန္တေဝ နံ, ကုလေ န နာသေန္တိ. သီလဗလေန စ, ဘိက္ခဝေ, မာတုဂါမော သမန္နာဂတော ဟောတိ, န စ ဘောဂဗလေန, ဝါသေန္တေဝ နံ, ကုလေ န နာသေန္တိ. သီလဗလေန စ, ဘိက္ခဝေ, မာတုဂါမော သမန္နာဂတော ဟောတိ, န စ ဉာတိဗလေန, ဝါသေန္တေဝ နံ, ကုလေ န နာသေန္တိ. သီလဗလေန စ, ဘိက္ခဝေ, မာတုဂါမော သမန္နာဂတော ဟောတိ, န စ ပုတ္တဗလေန, ဝါသေန္တေဝ နံ, ကုလေ န နာသေန္တိ. ဣမာနိ ခေါ, ဘိက္ခဝေ, ပဉ္စ မာတုဂါမဿ ဗလာနီ”တိ.
