ဧကံ သမယံ ဘဂဝါ နာဠန္ဒာယံ ဝိဟရတိ ပါဝါရိကမ္ဗဝနေ. အထ ခေါ, ဥပါလိ ဂဟပတိ, ယေန ဘဂဝါ တေနုပသင်္ကမိ …ပေ…  ဧကမန္တံ နိသိန္နော ခေါ, ဥပါလိ ဂဟပတိ, ဘဂဝန္တံ ဧတဒဝေါစ—  “ကော နု ခေါ, ဘန္တေ, ဟေတု, ကော ပစ္စယော ယေန မိဓေကစ္စေ သတ္တာ ဒိဋ္ဌေဝ ဓမ္မေ နော ပရိနိဗ္ဗာယန္တိ? ကော ပန, ဘန္တေ, ဟေတု, ကော ပစ္စယော ယေန မိဓေကစ္စေ သတ္တာ ဒိဋ္ဌေဝ ဓမ္မေ ပရိနိဗ္ဗာယန္တီ”တိ? (ယထာ ပုရိမသုတ္တန္တံ, ဧဝံ ဝိတ္ထာရေတဗ္ဗံ.) “အယံ ခေါ, ဂဟပတိ, ဟေတု အယံ ပစ္စယော ယေန မိဓေကစ္စေ သတ္တာ ဒိဋ္ဌေဝ ဓမ္မေ ပရိနိဗ္ဗာယန္တီ”တိ.

</Paragraph>
