“ပဉ္စိမာနိ, ဘိက္ခဝေ, မာတုဂါမဿ ဗလာနိ. ကတမာနိ ပဉ္စ? ရူပဗလံ, ဘောဂဗလံ, ဉာတိဗလံ, ပုတ္တဗလံ, သီလဗလံ. န, ဘိက္ခဝေ, မာတုဂါမော ရူပဗလဟေတု ဝါ ဘောဂဗလဟေတု ဝါ ဉာတိဗလဟေတု ဝါ ပုတ္တဗလဟေတု ဝါ ကာယဿ ဘေဒါ ပရံ မရဏာ သုဂတိံ သဂ္ဂံ လောကံ ဥပပဇ္ဇတိ. သီလဗလဟေတု ခေါ, ဘိက္ခဝေ, မာတုဂါမော ကာယဿ ဘေဒါ ပရံ မရဏာ သုဂတိံ သဂ္ဂံ လောကံ ဥပပဇ္ဇတိ. ဣမာနိ ခေါ, ဘိက္ခဝေ, ပဉ္စ မာတုဂါမဿ ဗလာနီ”တိ.
