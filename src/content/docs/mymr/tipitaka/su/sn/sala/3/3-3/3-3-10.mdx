<Paragraph number="1318" book="sala">
“ပဉ္စဟိ, ဘိက္ခဝေ, ဝဍ္ဎီဟိ ဝဍ္ဎမာနာ အရိယသာဝိကာ အရိယာယ ဝဍ္ဎိယာ ဝဍ္ဎတိ သာရာဒါယိနီ စ ဟောတိ ဝရာဒါယိနီ စ ကာယဿ. ကတမေဟိ ပဉ္စဟိ? သဒ္ဓါယ ဝဍ္ဎတိ, သီလေန ဝဍ္ဎတိ, သုတေန ဝဍ္ဎတိ, စာဂေန ဝဍ္ဎတိ, ပညာယ ဝဍ္ဎတိ—  ဣမေဟိ ခေါ, ဘိက္ခဝေ, ပဉ္စဟိ ဝဍ္ဎီဟိ ဝဍ္ဎမာနာ အရိယသာဝိကာ အရိယာယ ဝဍ္ဎိယာ ဝဍ္ဎတိ, သာရာဒါယိနီ စ ဟောတိ, ဝရာဒါယိနီ စ ကာယဿာတိ.
</Paragraph>

<Paragraph number="1319" type="verses" book="sala">
_သဒ္ဓါယ သီလေန စ ယာဓ ဝဍ္ဎတိ,_  
_ပညာယ စာဂေန သုတေန စူဘယံ;_  
_သာ တာဒိသီ သီလဝတီ ဥပါသိကာ,_  
_အာဒီယတိ သာရမိဓေဝ အတ္တနော”တိ._  


</Paragraph>
</Division>
