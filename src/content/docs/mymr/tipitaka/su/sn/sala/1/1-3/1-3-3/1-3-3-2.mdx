ဧကံ သမယံ ဘဂဝါ ဝဇ္ဇီသု ဝိဟရတိ ဟတ္ထိဂါမေ. အထ ခေါ ဥဂ္ဂေါ ဂဟပတိ ဟတ္ထိဂါမကော ယေန ဘဂဝါ တေနုပသင်္ကမိ; ဥပသင်္ကမိတွာ ဧကမန္တံ နိသီဒိ. ဧကမန္တံ နိသိန္နော ခေါ ဥဂ္ဂေါ ဂဟပတိ ဟတ္ထိဂါမကော ဘဂဝန္တံ ဧတဒဝေါစ—  “ကော နု ခေါ, ဘန္တေ, ဟေတု ကော ပစ္စယော ယေန မိဓေကစ္စေ သတ္တာ ဒိဋ္ဌေဝ ဓမ္မေ နော ပရိနိဗ္ဗာယန္တိ? ကော ပန, ဘန္တေ, ဟေတု ကော ပစ္စယော ယေန မိဓေကစ္စေ သတ္တာ ဒိဋ္ဌေဝ ဓမ္မေ ပရိနိဗ္ဗာယန္တီ”တိ? (ယထာ ပုရိမသုတ္တန္တံ, ဧဝံ ဝိတ္ထာရေတဗ္ဗံ.) “အယံ ခေါ, ဂဟပတိ, ဟေတု အယံ ပစ္စယော ယေန မိဓေကစ္စေ သတ္တာ ဒိဋ္ဌေဝ ဓမ္မေ ပရိနိဗ္ဗာယန္တီ”တိ.
