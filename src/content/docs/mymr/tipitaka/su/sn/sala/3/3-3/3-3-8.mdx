<Paragraph number="1313" book="sn">
“ပဉ္စိမာနိ, ဘိက္ခဝေ, ဌာနာနိ ဒုလ္လဘာနိ အကတပုညေန မာတုဂါမေန. ကတမာနိ ပဉ္စ? ပတိရူပေ ကုလေ ဇာယေယျန္တိ—  ဣဒံ, ဘိက္ခဝေ, ပဌမံ ဌာနံ ဒုလ္လဘံ အကတပုညေန မာတုဂါမေန. ပတိရူပေ ကုလေ ဇာယိတွာ ပတိရူပံ ကုလံ ဂစ္ဆေယျန္တိ—  ဣဒံ, ဘိက္ခဝေ, ဒုတိယံ ဌာနံ ဒုလ္လဘံ အကတပုညေန မာတုဂါမေန. ပတိရူပေ ကုလေ ဇာယိတွာ, ပတိရူပံ ကုလံ ဂန္တွာ, အသပတ္တိ အဂါရံ အဇ္ဈာဝသေယျန္တိ—  ဣဒံ, ဘိက္ခဝေ, တတိယံ ဌာနံ ဒုလ္လဘံ အကတပုညေန မာတုဂါမေန. ပတိရူပေ ကုလေ ဇာယိတွာ, ပတိရူပံ ကုလံ ဂန္တွာ, အသပတ္တိ အဂါရံ အဇ္ဈာဝသန္တီ ပုတ္တဝတီ အဿန္တိ—  ဣဒံ, ဘိက္ခဝေ, စတုတ္ထံ ဌာနံ ဒုလ္လဘံ အကတပုညေန မာတုဂါမေန. ပတိရူပေ ကုလေ ဇာယိတွာ, ပတိရူပံ ကုလံ ဂန္တွာ, အသပတ္တိ အဂါရံ အဇ္ဈာဝသန္တီ ပုတ္တဝတီ သမာနာ သာမိကံ အဘိဘုယျ ဝတ္တေယျန္တိ—  ဣဒံ, ဘိက္ခဝေ, ပဉ္စမံ ဌာနံ ဒုလ္လဘံ အကတပုညေန မာတုဂါမေန. ဣမာနိ ခေါ, ဘိက္ခဝေ, ပဉ္စ ဌာနာနိ ဒုလ္လဘာနိ အကတပုညေန မာတုဂါမေနာတိ.
</Paragraph>

<Paragraph number="1314" book="sn">
ပဉ္စိမာနိ, ဘိက္ခဝေ, ဌာနာနိ သုလဘာနိ ကတပုညေန မာတုဂါမေန. ကတမာနိ ပဉ္စ? ပတိရူပေ ကုလေ ဇာယေယျန္တိ—  ဣဒံ, ဘိက္ခဝေ, ပဌမံ ဌာနံ သုလဘံ ကတပုညေန မာတုဂါမေန. ပတိရူပေ ကုလေ ဇာယိတွာ ပတိရူပံ ကုလံ ဂစ္ဆေယျန္တိ—  ဣဒံ, ဘိက္ခဝေ, ဒုတိယံ ဌာနံ သုလဘံ ကတပုညေန မာတုဂါမေန. ပတိရူပေ ကုလေ ဇာယိတွာ ပတိရူပံ ကုလံ ဂန္တွာ အသပတ္တိ အဂါရံ အဇ္ဈာဝသေယျန္တိ—  ဣဒံ, ဘိက္ခဝေ, တတိယံ ဌာနံ သုလဘံ ကတပုညေန မာတုဂါမေန. ပတိရူပေ ကုလေ ဇာယိတွာ ပတိရူပံ ကုလံ ဂန္တွာ အသပတ္တိ အဂါရံ အဇ္ဈာဝသန္တီ ပုတ္တဝတီ အဿန္တိ—  ဣဒံ, ဘိက္ခဝေ, စတုတ္ထံ ဌာနံ သုလဘံ ကတပုညေန မာတုဂါမေန. ပတိရူပေ ကုလေ ဇာယိတွာ ပတိရူပံ ကုလံ ဂန္တွာ အသပတ္တိ အဂါရံ အဇ္ဈာဝသန္တီ ပုတ္တဝတီ သမာနာ သာမိကံ အဘိဘုယျ ဝတ္တေယျန္တိ—  ဣဒံ, ဘိက္ခဝေ, ပဉ္စမံ ဌာနံ သုလဘံ ကတပုညေန မာတုဂါမေန. ဣမာနိ ခေါ, ဘိက္ခဝေ, ပဉ္စ ဌာနာနိ သုလဘာနိ ကတပုညေန မာတုဂါမေနာ”တိ.

</Paragraph>
</Division>
