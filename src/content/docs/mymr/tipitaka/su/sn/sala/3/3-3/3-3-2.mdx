“ပဉ္စိမာနိ, ဘိက္ခဝေ, မာတုဂါမဿ ဗလာနိ. ကတမာနိ ပဉ္စ? ရူပဗလံ, ဘောဂဗလံ, ဉာတိဗလံ, ပုတ္တဗလံ, သီလဗလံ—  ဣမာနိ ခေါ, ဘိက္ခဝေ, ပဉ္စ မာတုဂါမဿ ဗလာနိ. ဣမေဟိ ခေါ, ဘိက္ခဝေ, ပဉ္စဟိ ဗလေဟိ သမန္နာဂတော မာတုဂါမော သာမိကံ ပသယှ အဂါရံ အဇ္ဈာဝသတီ”တိ.
