“ပဉ္စဟိ, ဘိက္ခဝေ, ဓမ္မေဟိ သမန္နာဂတော မာတုဂါမော ဝိသာရဒေါ အဂါရံ အဇ္ဈာဝသတိ. ကတမေဟိ ပဉ္စဟိ? ပါဏာတိပါတာ ပဋိဝိရတော စ ဟောတိ, အဒိန္နာဒါနာ ပဋိဝိရတော စ ဟောတိ, ကာမေသုမိစ္ဆာစာရာ ပဋိဝိရတော စ ဟောတိ, မုသာဝါဒါ ပဋိဝိရတော စ ဟောတိ, သုရာမေရယမဇ္ဇပ္ပမာဒဋ္ဌာနာ ပဋိဝိရတော စ ဟောတိ—  ဣမေဟိ ခေါ, ဘိက္ခဝေ, ပဉ္စဟိ ဓမ္မေဟိ သမန္နာဂတော မာတုဂါမော ဝိသာရဒေါ အဂါရံ အဇ္ဈာဝသတီ”တိ.
