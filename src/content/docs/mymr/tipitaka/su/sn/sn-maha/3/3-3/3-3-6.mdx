<Paragraph number="881" book="sn-maha">
ဧကံ သမယံ အာယသ္မာ စ သာရိပုတ္တော အာယသ္မာ စ မဟာမောဂ္ဂလ္လာနော အာယသ္မာ စ အနုရုဒ္ဓေါ သာကေတေ ဝိဟရန္တိ ကဏ္ဍကီဝနေ. အထ ခေါ အာယသ္မာ စ သာရိပုတ္တော အာယသ္မာ စ မဟာမောဂ္ဂလ္လာနော သာယနှသမယံ ပဋိသလ္လာနာ ဝုဋ္ဌိတာ ယေနာယသ္မာ အနုရုဒ္ဓေါ တေနုပသင်္ကမိံသု; ဥပသင်္ကမိတွာ အာယသ္မတာ အနုရုဒ္ဓေန သဒ္ဓိံ သမ္မောဒိံသု. သမ္မောဒနီယံ ကထံ သာရဏီယံ ဝီတိသာရေတွာ ဧကမန္တံ နိသီဒိံသု. ဧကမန္တံ နိသိန္နော ခေါ အာယသ္မာ သာရိပုတ္တော အာယသ္မန္တံ အနုရုဒ္ဓံ ဧတဒဝေါစ—  “‘သေခေါ, သေခေါ’တိ, အာဝုသော အနုရုဒ္ဓ, ဝုစ္စတိ. ကိတ္တာဝတာ နု ခေါ, အာဝုသော, သေခေါ ဟောတီ”တိ? “စတုန္နံ ခေါ, အာဝုသော, သတိပဋ္ဌာနာနံ ပဒေသံ ဘာဝိတတ္တာ သေခေါ ဟောတိ.
</Paragraph>

<Paragraph number="882" book="sn-maha">
ကတမေသံ စတုန္နံ? ဣဓာဝုသော, ဘိက္ခု ကာယေ ကာယာနုပဿီ ဝိဟရတိ အာတာပီ သမ္ပဇာနော သတိမာ, ဝိနေယျ လောကေ အဘိဇ္ဈာဒေါမနဿံ; ဝေဒနာသု …ပေ…  စိတ္တေ …ပေ…  ဓမ္မေသု ဓမ္မာနုပဿီ ဝိဟရတိ အာတာပီ သမ္ပဇာနော သတိမာ, ဝိနေယျ လောကေ အဘိဇ္ဈာဒေါမနဿံ. ဣမေသံ ခေါ, အာဝုသော, စတုန္နံ သတိပဋ္ဌာနာနံ ပဒေသံ ဘာဝိတတ္တာ သေခေါ ဟောတီ”တိ.

</Paragraph>
</Division>
