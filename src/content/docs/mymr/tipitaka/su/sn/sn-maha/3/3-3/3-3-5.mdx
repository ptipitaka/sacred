<Paragraph number="876" book="sn">
ဧဝံ မေ သုတံ—  ဧကံ သမယံ ဘဂဝါ သာဝတ္ထိယံ ဝိဟရတိ ဇေတဝနေ အနာထပိဏ္ဍိကဿ အာရာမေ. အထ ခေါ အညတရော ဗြာဟ္မဏော ယေန ဘဂဝါ တေနုပသင်္ကမိ; ဥပသင်္ကမိတွာ ဘဂဝတာ သဒ္ဓိံ သမ္မောဒိ. သမ္မောဒနီယံ ကထံ သာရဏီယံ ဝီတိသာရေတွာ ဧကမန္တံ နိသီဒိ. ဧကမန္တံ နိသိန္နော ခေါ သော ဗြာဟ္မဏော ဘဂဝန္တံ ဧတဒဝေါစ—  “ကော နု ခေါ, ဘော ဂေါတမ, ဟေတု, ကော ပစ္စယော ယေန တထာဂတေ ပရိနိဗ္ဗုတေ သဒ္ဓမ္မော န စိရဋ္ဌိတိကော ဟောတိ? ကော ပန, ဘော ဂေါတမ, ဟေတု, ကော ပစ္စယော ယေန တထာဂတေ ပရိနိဗ္ဗုတေ သဒ္ဓမ္မော စိရဋ္ဌိတိကော ဟောတီ”တိ?
</Paragraph>

<Paragraph number="877" book="sn">
“စတုန္နံ ခေါ, ဗြာဟ္မဏ, သတိပဋ္ဌာနာနံ အဘာဝိတတ္တာ အဗဟုလီကတတ္တာ တထာဂတေ ပရိနိဗ္ဗုတေ သဒ္ဓမ္မော န စိရဋ္ဌိတိကော ဟောတိ. စတုန္နဉ္စ ခေါ, ဗြာဟ္မဏ, သတိပဋ္ဌာနာနံ ဘာဝိတတ္တာ ဗဟုလီကတတ္တာ တထာဂတေ ပရိနိဗ္ဗုတေ သဒ္ဓမ္မော စိရဋ္ဌိတိကော ဟောတိ.
</Paragraph>

<Paragraph number="878" book="sn">
ကတမေသံ စတုန္နံ? ဣဓ, ဗြာဟ္မဏ, ဘိက္ခု ကာယေ ကာယာနုပဿီ ဝိဟရတိ အာတာပီ သမ္ပဇာနော သတိမာ, ဝိနေယျ လောကေ အဘိဇ္ဈာဒေါမနဿံ; ဝေဒနာသု …ပေ…  စိတ္တေ …ပေ…  ဓမ္မေသု ဓမ္မာနုပဿီ ဝိဟရတိ အာတာပီ သမ္ပဇာနော သတိမာ, ဝိနေယျ လောကေ အဘိဇ္ဈာဒေါမနဿံ. ဣမေသံ ခေါ, ဗြာဟ္မဏ, စတုန္နံ သတိပဋ္ဌာနာနံ အဘာဝိတတ္တာ အဗဟုလီကတတ္တာ တထာဂတေ ပရိနိဗ္ဗုတေ သဒ္ဓမ္မော န စိရဋ္ဌိတိကော ဟောတိ. ဣမေသဉ္စ ခေါ, ဗြာဟ္မဏ, စတုန္နံ သတိပဋ္ဌာနာနံ ဘာဝိတတ္တာ ဗဟုလီကတတ္တာ တထာဂတေ ပရိနိဗ္ဗုတေ သဒ္ဓမ္မော စိရဋ္ဌိတိကော ဟောတီ”တိ.
</Paragraph>

<Paragraph number="879" book="sn">
ဧဝံ ဝုတ္တေ, သော ဗြာဟ္မဏော ဘဂဝန္တံ ဧတဒဝေါစ—  “အဘိက္ကန္တံ, ဘော ဂေါတမ …ပေ…  ဥပါသကံ မံ ဘဝံ ဂေါတမော ဓာရေတု အဇ္ဇတဂ္ဂေ ပါဏုပေတံ သရဏံ ဂတန်”တိ.

</Paragraph>
</Division>
