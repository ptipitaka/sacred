<Paragraph number="887" book="sn-maha">
တံယေဝ နိဒါနံ. ဧကမန္တံ နိသိန္နော ခေါ အာယသ္မာ သာရိပုတ္တော အာယသ္မန္တံ အနုရုဒ္ဓံ ဧတဒဝေါစ—  “ကတမေသံ, အာဝုသော အနုရုဒ္ဓ, ဓမ္မာနံ ဘာဝိတတ္တာ ဗဟုလီကတတ္တာ မဟာဘိညတံ ပတ္တော”တိ? “စတုန္နံ, အာဝုသော, သတိပဋ္ဌာနာနံ ဘာဝိတတ္တာ ဗဟုလီကတတ္တာ မဟာဘိညတံ ပတ္တော.
</Paragraph>

<Paragraph number="888" book="sn-maha">
ကတမေသံ စတုန္နံ? ဣဓာဟံ, အာဝုသော, ကာယေ ကာယာနုပဿီ ဝိဟရာမိ အာတာပီ သမ္ပဇာနော သတိမာ, ဝိနေယျ လောကေ အဘိဇ္ဈာဒေါမနဿံ; ဝေဒနာသု …ပေ…  စိတ္တေ …ပေ…  ဓမ္မေသု ဓမ္မာနုပဿီ ဝိဟရာမိ အာတာပီ သမ္ပဇာနော သတိမာ, ဝိနေယျ လောကေ အဘိဇ္ဈာဒေါမနဿံ. ဣမေသံ ခွါဟံ, အာဝုသော, စတုန္နံ သတိပဋ္ဌာနာနံ ဘာဝိတတ္တာ ဗဟုလီကတတ္တာ မဟာဘိညတံ ပတ္တော. ဣမေသဉ္စ ပနာဟံ, အာဝုသော, စတုန္နံ သတိပဋ္ဌာနာနံ ဘာဝိတတ္တာ ဗဟုလီကတတ္တာ သဟဿံ လောကံ အဘိဇာနာမီ”တိ.

</Paragraph>
</Division>
