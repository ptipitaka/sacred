တံယေဝ နိဒါနံ. တေန ခေါ ပန သမယေန မာနဒိန္နော ဂဟပတိ အာဗာဓိကော ဟောတိ ဒုက္ခိတော ဗာဠှဂိလာနော. အထ ခေါ မာနဒိန္နော ဂဟပတိ အညတရံ ပုရိသံ အာမန္တေသိ—  “ဧဟိ တွံ, အမ္ဘော ပုရိသ …ပေ…  န မေ, ဘန္တေ, ခမနီယံ န ယာပနီယံ. ဗာဠှာ မေ ဒုက္ခာ ဝေဒနာ အဘိက္ကမန္တိ, နော ပဋိက္ကမန္တိ; အဘိက္ကမောသာနံ ပညာယတိ, နော ပဋိက္ကမောတိ. ဧဝရူပါယ စာဟံ, ဘန္တေ, ဒုက္ခာယ ဝေဒနာယ ဖုဋ္ဌော သမာနော ကာယေ ကာယာနုပဿီ ဝိဟရာမိ အာတာပီ သမ္ပဇာနော သတိမာ, ဝိနေယျ လောကေ အဘိဇ္ဈာဒေါမနဿံ; ဝေဒနာသု …ပေ…  စိတ္တေ …ပေ…  ဓမ္မေသု ဓမ္မာနုပဿီ ဝိဟရာမိ အာတာပီ သမ္ပဇာနော သတိမာ, ဝိနေယျ လောကေ အဘိဇ္ဈာဒေါမနဿံ. ယာနိ စိမာနိ, ဘန္တေ, ဘဂဝတာ ပဉ္စောရမ္ဘာဂိယာနိ သံယောဇနာနိ ဒေသိတာနိ, နာဟံ, ဘန္တေ, တေသံ ကိဉ္စိ အတ္တနိ အပ္ပဟီနံ သမနုပဿာမီ”တိ. “လာဘာ တေ, ဂဟပတိ, သုလဒ္ဓံ တေ, ဂဟပတိ. အနာဂါမိဖလံ တယာ, ဂဟပတိ, ဗျာကတန်”တိ.
