<Paragraph number="777" book="sn">
သာဝတ္ထိယံ ဝိဟရတိ. “ဓာတုသောဝ, ဘိက္ခဝေ, သတ္တာ သံသန္ဒန္တိ သမေန္တိ. မိစ္ဆာဒိဋ္ဌိကာ မိစ္ဆာဒိဋ္ဌိကေဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တိ; မိစ္ဆာသင်္ကပ္ပါ …ပေ…  မိစ္ဆာဝါစာ…  မိစ္ဆာကမ္မန္တာ…  မိစ္ဆာအာဇီဝါ…  မိစ္ဆာဝါယာမာ…  မိစ္ဆာသတိနော…  မိစ္ဆာသမာဓိနော မိစ္ဆာသမာဓီဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တိ.
</Paragraph>

<Paragraph number="778" book="sn">
သမ္မာဒိဋ္ဌိကာ သမ္မာဒိဋ္ဌိကေဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တိ; သမ္မာသင်္ကပ္ပါ …ပေ…  သမ္မာဝါစာ…  သမ္မာကမ္မန္တာ…  သမ္မာအာဇီဝါ…  သမ္မာဝါယာမာ…  သမ္မာသတိနော…  သမ္မာသမာဓိနော သမ္မာသမာဓီဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တီ”တိ.
