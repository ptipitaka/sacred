သာဝတ္ထိယံ ဝိဟရတိ. “ဓာတုသောဝ, ဘိက္ခဝေ, သတ္တာ သံသန္ဒန္တိ သမေန္တိ. ပါဏာတိပါတိနော ပါဏာတိပါတီဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တိ; အဒိန္နာဒါယိနော …ပေ…  ကာမေသုမိစ္ဆာစာရိနော…  မုသာဝါဒိနော…  ပိသုဏဝါစာ…  ဖရုသဝါစာ…  သမ္ဖပ္ပလာပိနော သမ္ဖပ္ပလာပီဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တိ; အဘိဇ္ဈာလုနော အဘိဇ္ဈာလူဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တိ; ဗျာပန္နစိတ္တာ ဗျာပန္နစိတ္တေဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တိ; မိစ္ဆာဒိဋ္ဌိကာ မိစ္ဆာဒိဋ္ဌိကေဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တိ.
</Paragraph>
