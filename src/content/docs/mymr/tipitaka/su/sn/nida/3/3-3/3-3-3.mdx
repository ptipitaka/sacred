သာဝတ္ထိယံ ဝိဟရတိ. “ဓာတုသောဝ, ဘိက္ခဝေ, သတ္တာ သံသန္ဒန္တိ သမေန္တိ. ပါဏာတိပါတိနော ပါဏာတိပါတီဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တိ; အဒိန္နာဒါယိနော အဒိန္နာဒါယီဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တိ; ကာမေသုမိစ္ဆာစာရိနော ကာမေသုမိစ္ဆာစာရီဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တိ; မုသာဝါဒိနော မုသာဝါဒီဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တိ; သုရာမေရယမဇ္ဇပ္ပမာဒဋ္ဌာယိနော သုရာမေရယမဇ္ဇပ္ပမာဒဋ္ဌာယီဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တိ.
</Paragraph>

<Paragraph number="769" book="sn">
ပါဏာတိပါတာ ပဋိဝိရတာ ပါဏာတိပါတာ ပဋိဝိရတေဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တိ; အဒိန္နာဒါနာ ပဋိဝိရတာ အဒိန္နာဒါနာ ပဋိဝိရတေဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တိ; ကာမေသုမိစ္ဆာစာရာ ပဋိဝိရတာ ကာမေသုမိစ္ဆာစာရာ ပဋိဝိရတေဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တိ; မုသာဝါဒါ ပဋိဝိရတာ မုသာဝါဒါ ပဋိဝိရတေဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တိ; သုရာမေရယမဇ္ဇပ္ပမာဒဋ္ဌာနာ ပဋိဝိရတာ သုရာမေရယမဇ္ဇပ္ပမာဒဋ္ဌာနာ ပဋိဝိရတေဟိ သဒ္ဓိံ သံသန္ဒန္တိ သမေန္တီ”တိ.

</Paragraph>
