<Paragraph number="465" book="it">
ဝုတ္တဉှေတံ ဘဂဝတာ ဝုတ္တမရဟတာတိ မေ သုတံ—
</Paragraph>

<Paragraph number="466" book="it">
“တိဿော ဣမာ, ဘိက္ခဝေ, နိဿရဏိယာ ဓာတုယော. ကတမာ တိဿော? ကာမာနမေတံ နိဿရဏံ ယဒိဒံ နေက္ခမ္မံ, ရူပါနမေတံ နိဿရဏံ ယဒိဒံ အာရုပ္ပံ, ယံ ခေါ ပန ကိဉ္စိ ဘူတံ သင်္ခတံ ပဋိစ္စသမုပ္ပန္နံ နိရောဓော တဿ နိဿရဏံ—  ဣမာ ခေါ, ဘိက္ခဝေ, တိဿော နိဿရဏိယာ ဓာတုယော”တိ. ဧတမတ္ထံ ဘဂဝါ အဝေါစ. တတ္ထေတံ ဣတိ ဝုစ္စတိ—
</Paragraph>

<Paragraph number="467" type="verses" book="it">
_“ကာမနိဿရဏံ ဉတွာ,_  
_ရူပါနဉ္စ အတိက္ကမံ;_  
_သဗ္ဗသင်္ခါရသမထံ,_  
_ဖုသံ အာတာပိ သဗ္ဗဒါ._  

</Paragraph>

<Paragraph number="468" type="verses" book="it">
_သ ဝေ သမ္မဒ္ဒသော ဘိက္ခု,_  
_ယတော တတ္ထ ဝိမုစ္စတိ;_  
_အဘိညာဝေါသိတော သန္တော,_  
_သ ဝေ ယောဂါတိဂေါ မုနီ”တိ._  

</Paragraph>

<Paragraph number="469" book="it">
အယမ္ပိ အတ္ထော ဝုတ္တော ဘဂဝတာ, ဣတိ မေ သုတန္တိ.

</Paragraph>
</Division>

တတိယံ.
