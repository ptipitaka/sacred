ဝုတ္တဉှေတံ ဘဂဝတာ ဝုတ္တမရဟတာတိ မေ သုတံ—
</Paragraph>

<Paragraph number="511" book="it">
“ဘိဒုရာယံ, ဘိက္ခဝေ, ကာယော, ဝိညာဏံ ဝိရာဂဓမ္မံ, သဗ္ဗေ ဥပဓီ အနိစ္စာ ဒုက္ခာ ဝိပရိဏာမဓမ္မာ”တိ. ဧတမတ္ထံ ဘဂဝါ အဝေါစ. တတ္ထေတံ ဣတိ ဝုစ္စတိ—
</Paragraph>

<Paragraph number="512" type="verses" book="it">
_“ကာယဉ္စ ဘိဒုရံ ဉတွာ,_  
_ဝိညာဏဉ္စ ဝိရာဂုနံ;_  
ဥပဓီသု ဘယံ ဒိသွာ,_  
_ဇာတိမရဏမစ္စဂါ;_  
_သမ္ပတွာ ပရမံ သန္တိံ,_  
_ကာလံ ကင်္ခတိ ဘာဝိတတ္တော”တိ._  

</Paragraph>

<Paragraph number="513" book="it">
အယမ္ပိ အတ္ထော ဝုတ္တော ဘဂဝတာ, ဣတိ မေ သုတန္တိ.

</Paragraph>
</Division>
