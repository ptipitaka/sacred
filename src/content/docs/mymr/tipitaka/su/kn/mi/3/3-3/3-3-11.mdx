<Paragraph number="296" book="mi">
“ဘန္တေ နာဂသေန, ကိံလက္ခဏာ စေတနာ”တိ? “စေတယိတလက္ခဏာ, မဟာရာဇ, စေတနာ အဘိသင်္ခရဏလက္ခဏာ စာ”တိ.
</Paragraph>

<Paragraph number="297" book="mi">
“ဩပမ္မံ ကရောဟီ”တိ. “ယထာ, မဟာရာဇ, ကောစိဒေဝ ပုရိသော ဝိသံ အဘိသင်္ခရိတွာ အတ္တနာ စ ပိဝေယျ, ပရေ စ ပါယေယျ, သော အတ္တနာပိ ဒုက္ခိတော ဘဝေယျ, ပရေပိ ဒုက္ခိတာ ဘဝေယျုံ; ဧဝမေဝ ခေါ, မဟာရာဇ, ဣဓေကစ္စော ပုဂ္ဂလော အကုသလံ ကမ္မံ စေတနာယ စေတယိတွာ ကာယဿ ဘေဒါ ပရံ မရဏာ အပါယံ ဒုဂ္ဂတိံ ဝိနိပါတံ နိရယံ ဥပပဇ္ဇေယျ. ယေပိ တဿ အနုသိက္ခန္တိ, တေပိ ကာယဿ ဘေဒါ ပရံ မရဏာ အပါယံ ဒုဂ္ဂတိံ ဝိနိပါတံ နိရယံ ဥပပဇ္ဇန္တိ.
</Paragraph>

<Paragraph number="298" book="mi">
ယထာ ဝါ ပန, မဟာရာဇ, ကောစိဒေဝ ပုရိသော သပ္ပိနဝနီတတေလမဓုဖာဏိတံ ဧကဇ္ဈံ အဘိသင်္ခရိတွာ အတ္တနာ စ ပိဝေယျ, ပရေ စ ပါယေယျ, သော အတ္တနာ သုခိတော ဘဝေယျ, ပရေပိ သုခိတာ ဘဝေယျုံ; ဧဝမေဝ ခေါ, မဟာရာဇ, ဣဓေကစ္စော ပုဂ္ဂလော ကုသလံ ကမ္မံ စေတနာယ စေတယိတွာ ကာယဿ ဘေဒါ ပရံ မရဏာ သုဂတိံ သဂ္ဂံ လောကံ ဥပပဇ္ဇတိ. ယေပိ တဿ အနုသိက္ခန္တိ, တေပိ ကာယဿ ဘေဒါ ပရံ မရဏာ သုဂတိံ သဂ္ဂံ လောကံ ဥပပဇ္ဇန္တိ. ဧဝံ ခေါ, မဟာရာဇ, စေတယိတလက္ခဏာ စေတနာ အဘိသင်္ခရဏလက္ခဏာ စာ”တိ.
</Paragraph>
