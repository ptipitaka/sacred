<Paragraph number="237" book="mi">
ရာဇာ အာဟ—  “ဘန္တေ နာဂသေန, ယံ ပနေတံ ဗြူသိ ‘ပုရိမာ ကောဋိ န ပညာယတီ’တိ, ကတမာ စ သာ ပုရိမာ ကောဋီ”တိ? “ယော ခေါ, မဟာရာဇ, အတီတော အဒ္ဓါ, ဧသာ ပုရိမာ ကောဋီ”တိ. “ဘန္တေ နာဂသေန, ယံ ပနေတံ ဗြူသိ ‘ပုရိမာ ကောဋိ န ပညာယတီ’တိ, ကိံ ပန, ဘန္တေ, သဗ္ဗာပိ ပုရိမာ ကောဋိ န ပညာယတီ”တိ? “ကာစိ, မဟာရာဇ, ပညာယတိ, ကာစိ န ပညာယတီ”တိ. “ကတမာ, ဘန္တေ, ပညာယတိ, ကတမာ န ပညာယတီ”တိ? “ဣတော ပုဗ္ဗေ, မဟာရာဇ, သဗ္ဗေန သဗ္ဗံ သဗ္ဗထာ သဗ္ဗံ အဝိဇ္ဇာ နာဟောသီတိ ဧသာ ပုရိမာ ကောဋိ န ပညာယတိ, ယံ အဟုတွာ သမ္ဘောတိ, ဟုတွာ ပဋိဝိဂစ္ဆတိ, ဧသာ ပုရိမာ ကောဋိ ပညာယတီ”တိ.
</Paragraph>

<Paragraph number="238" book="mi">
“ဘန္တေ နာဂသေန, ယံ အဟုတွာ သမ္ဘောတိ, ဟုတွာ ပဋိဝိဂစ္ဆတိ, နနု တံ ဥဘတော ဆိန္နံ အတ္ထံ ဂစ္ဆတီ”တိ? “ယဒိ, မဟာရာဇ, ဥဘတော ဆိန္နံ အတ္ထံ ဂစ္ဆတိ, ဥဘတော ဆိန္နာ သက္ကာ ဝဍ္ဎေတုန်”တိ? “အာမ သာပိ သက္ကာ ဝဍ္ဎေတုန်”တိ. “နာဟံ, ဘန္တေ, ဧတံ ပုစ္ဆာမိ ကောဋိတော သက္ကာ ဝဍ္ဎေတုန်”တိ? “အာမ သက္ကာ ဝဍ္ဎေတုန်”တိ.
</Paragraph>

<Paragraph number="239" book="mi">
“ဩပမ္မံ ကရောဟီ”တိ. ထေရော တဿ ရုက္ခူပမံ အကာသ်, “ခန္ဓာ စ ကေဝလဿ ဒုက္ခက္ခန္ဓဿ ဗီဇာနီ”တိ.
</Paragraph>

<Paragraph number="240" book="mi">
“ကလ္လောသိ, ဘန္တေ နာဂသေနာ”တိ.

</Paragraph>
