<Paragraph number="305" book="mi">
“ဘန္တေ နာဂသေန, ကိံလက္ခဏော ဝိတက္ကော”တိ? “အပ္ပနာလက္ခဏော, မဟာရာဇ, ဝိတက္ကော”တိ.
</Paragraph>

<Paragraph number="306" book="mi">
“ဩပမ္မံ ကရောဟီ”တိ. “ယထာ, မဟာရာဇ, ဝဍ္ဎကီ သုပရိကမ္မကတံ ဒါရုံ သန္ဓိသ္မိံ အပ္ပေတိ; ဧဝမေဝ ခေါ, မဟာရာဇ, အပ္ပနာလက္ခဏော ဝိတက္ကော”တိ.
</Paragraph>

<Paragraph number="307" book="mi">
“ကလ္လောသိ, ဘန္တေ နာဂသေနာ”တိ.

</Paragraph>
</Division>

ဝိတက္ကလက္ခဏပဉှော တေရသမော.
</Paragraph>
