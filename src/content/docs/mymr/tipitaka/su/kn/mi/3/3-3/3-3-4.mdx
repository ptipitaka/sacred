ရာဇာ အာဟ—  “ဘန္တေ နာဂသေန, အတ္ထိ ကေစိ သင်္ခါရာ, ယေ ဇာယန္တီ”တိ? “အာမ, မဟာရာဇ, အတ္ထိ သင်္ခါရာ, ယေ ဇာယန္တီ”တိ. “ကတမေ တေ, ဘန္တေ”တိ? “စက္ခုသ္မိဉ္စ ခေါ, မဟာရာဇ, သတိ ရူပေသု စ စက္ခုဝိညာဏံ ဟောတိ, စက္ခုဝိညာဏေ သတိ စက္ခုသမ္ဖဿော ဟောတိ, စက္ခုသမ္ဖဿေ သတိ ဝေဒနာ ဟောတိ, ဝေဒနာယ သတိ တဏှာ ဟောတိ, တဏှာယ သတိ ဥပါဒါနံ ဟောတိ, ဥပါဒါနေ သတိ ဘဝေါ ဟောတိ, ဘဝေ သတိ ဇာတိ ဟောတိ, ဇာတိယာ သတိ ဇရာမရဏံ သောကပရိဒေဝဒုက္ခဒေါမနဿုပါယာသာ သမ္ဘဝန္တိ, ဧဝမေတဿ ကေဝလဿ ဒုက္ခက္ခန္ဓဿ သမုဒယော ဟောတိ. စက္ခုသ္မိဉ္စ ခေါ, မဟာရာဇ, အသတိ ရူပေသု စ အသတိ စက္ခုဝိညာဏံ န ဟောတိ, စက္ခုဝိညာဏေ အသတိ စက္ခုသမ္ဖဿော န ဟောတိ, စက္ခုသမ္ဖဿေ အသတိ ဝေဒနာ န ဟောတိ, ဝေဒနာယ အသတိ တဏှာ န ဟောတိ, တဏှာယ အသတိ ဥပါဒါနံ န ဟောတိ, ဥပါဒါနေ အသတိ ဘဝေါ န ဟောတိ, ဘဝေ အသတိ ဇာတိ န ဟောတိ, ဇာတိယာ အသတိ ဇရာမရဏံ သောကပရိဒေဝဒုက္ခဒေါမနဿုပါယာသာ န ဟောန္တိ, ဧဝမေတဿ ကေဝလဿ ဒုက္ခက္ခန္ဓဿ နိရောဓော ဟောတီ”တိ.
</Paragraph>
