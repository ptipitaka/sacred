<Paragraph number="309" book="mi">
“ဘန္တေ နာဂသေန, ကိံလက္ခဏော ဝိစာရော”တိ? “အနုမဇ္ဇနလက္ခဏော, မဟာရာဇ, ဝိစာရော”တိ.
</Paragraph>

<Paragraph number="310" book="mi">
“ဩပမ္မံ ကရောဟီ”တိ. “ယထာ, မဟာရာဇ, ကံသထာလံ အာကောဋိတံ ပစ္ဆာ အနုရဝတိ အနုသန္ဒဟတိ, ယထာ, မဟာရာဇ, အာကောဋနာ, ဧဝံ ဝိတက္ကော ဒဋ္ဌဗ္ဗော. ယထာ အနုရဝနာ, ဧဝံ ဝိစာရော ဒဋ္ဌဗ္ဗော”တိ.
</Paragraph>

<Paragraph number="311" book="mi">
“ကလ္လောသိ, ဘန္တေ နာဂသေနာ”တိ.

</Paragraph>
</Division>

<Paragraph number="312" type="center" book="mi">
ဝိစာရလက္ခဏပဉှော စုဒ္ဒသမော.



</Paragraph>

<Paragraph number="313" type="center" book="mi">
ဝိစာရဝဂ္ဂေါ တတိယော.



</Paragraph>

<Paragraph number="314" type="center" book="mi">
ဣမသ္မိံ ဝဂ္ဂေ စုဒ္ဒသ ပဉှာ.
</Paragraph>
