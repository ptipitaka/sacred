<Paragraph number="1834" book="ps">
သာဝတ္ထိနိဒါနံ. “သေယျထာပိ, ဘိက္ခဝေ, ယေ ကေစိ ဗလကရဏီယာ ကမ္မန္တာ ကရီယန္တိ, သဗ္ဗေ တေ ပထဝိံ နိဿာယ ပထဝိယံ ပတိဋ္ဌာယ ဧဝမေတေ ဗလကရဏီယာ ကမ္မန္တာ ကရီယန္တိ; ဧဝမေဝံ, ဘိက္ခဝေ, ဘိက္ခု သီလံ နိဿာယ သီလေ ပတိဋ္ဌာယ အရိယံ အဋ္ဌင်္ဂိကံ မဂ္ဂံ ဘာဝေတိ, အရိယံ အဋ္ဌင်္ဂိကံ မဂ္ဂံ ဗဟုလီကရောတိ.


</Paragraph>

<Paragraph number="1835" book="ps">
ကထဉ္စ, ဘိက္ခဝေ, ဘိက္ခု သီလံ နိဿာယ သီလေ ပတိဋ္ဌာယ အရိယံ အဋ္ဌင်္ဂိကံ မဂ္ဂံ ဘာဝေတိ, အရိယံ အဋ္ဌင်္ဂိကံ မဂ္ဂံ ဗဟုလီကရောတိ? ဣဓ, ဘိက္ခဝေ, ဘိက္ခု သမ္မာဒိဋ္ဌိံ ဘာဝေတိ ဝိဝေကနိဿိတံ ဝိရာဂနိဿိတံ နိရောဓနိဿိတံ ဝေါသဂ္ဂပရိဏာမိံ. သမ္မာသင်္ကပ္ပံ …ပေ…  သမ္မာဝါစံ…  သမ္မာကမ္မန္တံ…  သမ္မာအာဇီဝံ…  သမ္မာဝါယာမံ…  သမ္မာသတိံ…  သမ္မာသမာဓိံ ဘာဝေတိ ဝိဝေကနိဿိတံ ဝိရာဂနိဿိတံ နိရောဓနိဿိတံ ဝေါသဂ္ဂပရိဏာမိံ. ဧဝံ ခေါ, ဘိက္ခဝေ, ဘိက္ခု သီလံ နိဿာယ သီလေ ပတိဋ္ဌာယ အရိယံ အဋ္ဌင်္ဂိကံ မဂ္ဂံ ဘာဝေတိ, အရိယံ အဋ္ဌင်္ဂိကံ မဂ္ဂံ ဗဟုလီကရောတိ.


</Paragraph>

</Division>
<Division number="23" book="ps" e="ch" v="26" p="315">

<Paragraph number="1836" book="ps">
သေယျထာပိ, ဘိက္ခဝေ, ယေ ကေစိ ဗီဇဂါမဘူတဂါမာ ဝုဒ္ဓိံ ဝိရုဠှိံ ဝေပုလ္လံ အာပဇ္ဇန္တိ, သဗ္ဗေ တေ ပထဝိံ နိဿာယ ပထဝိယံ ပတိဋ္ဌာယ ဧဝမေတေ ဗီဇဂါမဘူတဂါမာ ဝုဒ္ဓိံ ဝိရုဠှိံ ဝေပုလ္လံ အာပဇ္ဇန္တိ; ဧဝမေဝံ ခေါ, ဘိက္ခဝေ, ဘိက္ခု သီလံ နိဿာယ သီလေ ပတိဋ္ဌာယ အရိယံ အဋ္ဌင်္ဂိကံ မဂ္ဂံ ဘာဝေန္တော အရိယံ အဋ္ဌင်္ဂိကံ မဂ္ဂံ ဗဟုလီကရောန္တော ဝုဒ္ဓိံ ဝိရုဠှိံ ဝေပုလ္လံ ပါပုဏာတိ ဓမ္မေသု.


</Paragraph>

<Paragraph number="1837" book="ps">
ကထဉ္စ, ဘိက္ခဝေ, ဘိက္ခု သီလံ နိဿာယ သီလေ ပတိဋ္ဌာယ အရိယံ အဋ္ဌင်္ဂိကံ မဂ္ဂံ ဘာဝေန္တော အရိယံ အဋ္ဌင်္ဂိကံ မဂ္ဂံ ဗဟုလီကရောန္တော ဝုဒ္ဓိံ ဝိရုဠှိံ ဝေပုလ္လံ ပါပုဏာတိ ဓမ္မေသု? ဣဓ, ဘိက္ခဝေ, ဘိက္ခု သမ္မာဒိဋ္ဌိံ ဘာဝေတိ ဝိဝေကနိဿိတံ ဝိရာဂနိဿိတံ နိရောဓနိဿိတံ ဝေါသဂ္ဂပရိဏာမိံ. သမ္မာသင်္ကပ္ပံ ဘာဝေတိ …ပေ…  သမ္မာဝါစံ ဘာဝေတိ…  သမ္မာကမ္မန္တံ ဘာဝေတိ…  သမ္မာအာဇီဝံ ဘာဝေတိ…  သမ္မာဝါယာမံ ဘာဝေတိ…  သမ္မာသတိံ ဘာဝေတိ…  သမ္မာသမာဓိံ ဘာဝေတိ ဝိဝေကနိဿိတံ ဝိရာဂနိဿိတံ နိရောဓနိဿိတံ ဝေါသဂ္ဂပရိဏာမိံ. ဧဝံ ခေါ, ဘိက္ခဝေ, ဘိက္ခု သီလံ နိဿာယ သီလေ ပတိဋ္ဌာယ အရိယံ အဋ္ဌင်္ဂိကံ မဂ္ဂံ ဘာဝေန္တော အရိယံ အဋ္ဌင်္ဂိကံ မဂ္ဂံ ဗဟုလီကရောန္တော ဝုဒ္ဓိံ ဝိရုဠှိံ ဝေပုလ္လံ ပါပုဏာတိ ဓမ္မေသူ”တိ.

</Paragraph>
