<Paragraph number="766" type="verses" book="ja1">
_“ကာယံ ဗလာကာ သိခိနီ,_  
_စောရီ လင်္ဃိပိတာမဟာ;_  
_ဩရံ ဗလာကေ အာဂစ္ဆ,_  
_စဏ္ဍော မေ ဝါယသော သခါ”._  

</Paragraph>

</Division>
<Division number="71" book="ja1">

<Paragraph number="767" type="verses" book="ja1">
_“နာဟံ ဗလာကာ သိခိနီ,_  
_အဟံ လောလောသ္မိ ဝါယသော;_  
_အကတွာ ဝစနံ တုယှံ,_  
_ပဿ လူနောသ္မိ အာဂတော”._  

</Paragraph>

</Division>
<Division number="72" book="ja1">

<Paragraph number="768" type="verses" book="ja1">
_“ပုနပါပဇ္ဇသီ သမ္မ,_  
_သီလဉှိ တဝ တာဒိသံ;_  
_န ဟိ မာနုသကာ ဘောဂါ,_  
_သုဘုဉ္ဇာ ဟောန္တိ ပက္ခိနာ”တိ._  


</Paragraph>
</Division>

</Paragraph>
