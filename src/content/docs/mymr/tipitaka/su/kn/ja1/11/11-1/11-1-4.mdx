<Paragraph number="2146" type="verses" book="ja1">
_“ဧကာ နိသိန္နာ သုစိ သညတူရူ,_  
_ပါသာဒမာရုယှ အနိန္ဒိတင်္ဂီ;_  
_ယာစာမိ တံ ကိန္နရနေတ္တစက္ခု,_  
ဣမေကရတ္တိံ ဥဘယော ဝသေမ”._  

</Paragraph>

</Division>
<Division number="38" book="ja1" e="ch" v="22" p="231">

<Paragraph number="2147" type="verses" book="ja1">
_“ဩကိဏ္ဏန္တရပရိခံ,_  
_ဒဠှမဋ္ဋာလကောဋ္ဌကံ;_  
_ရက္ခိတံ ခဂ္ဂဟတ္ထေဟိ,_  
_ဒုပ္ပဝေသမိဒံ ပုရံ._  

</Paragraph>

</Division>
<Division number="39" book="ja1" e="ch" v="22" p="231">

<Paragraph number="2148" type="verses" book="ja1">
_ဒဟရဿ ယုဝိနော စာပိ,_  
_အာဂမော စ န ဝိဇ္ဇတိ;_  
_အထ ကေန နု ဝဏ္ဏေန,_  
_သင်္ဂမံ ဣစ္ဆသေ မယာ”._  

</Paragraph>

</Division>
<Division number="40" book="ja1" e="ch" v="22" p="231">

<Paragraph number="2149" type="verses" book="ja1">
_“ယက္ခောဟမသ္မိ ကလျာဏိ,_  
_အာဂတောသ္မိ တဝန္တိကေ;_  
_တွံ မံ နန္ဒယ ဘဒ္ဒန္တေ,_  
_ပုဏ္ဏကံသံ ဒဒါမိ တေ”._  

</Paragraph>

</Division>
<Division number="41" book="ja1" e="ch" v="22" p="231">

<Paragraph number="2150" type="verses" book="ja1">
_“ဒေဝံ ဝ ယက္ခံ အထ ဝါ မနုဿံ,_  
_န ပတ္ထယေ ဥဒယမတိစ္စ အညံ;_  
_ဂစ္ဆေဝ တွံ ယက္ခ မဟာနုဘာဝ,_  
_မာ စဿု ဂန္တွာ ပုနရာဝဇိတ္ထ”._  

</Paragraph>

</Division>
<Division number="42" book="ja1" e="ch" v="22" p="231">

<Paragraph number="2151" type="verses" book="ja1">
_“ယာ သာ ရတိ ဥတ္တမာ ကာမဘောဂိနံ,_  
_ယံ ဟေတု သတ္တာ ဝိသမံ စရန္တိ;_  
_မာ တံ ရတိံ ဇီယိ တုဝံ သုစိမှိတေ,_  
_ဒဒါမိ တေ ရူပိယံ ကံသပူရံ”._  

</Paragraph>

</Division>
<Division number="43" book="ja1" e="ch" v="22" p="232">

<Paragraph number="2152" type="verses" book="ja1">
_“နာရိံ နရော နိဇ္ဈပယံ ဓနေန,_  
ဥက္ကံသတီ ယတ္ထ ကရောတိ ဆန္ဒံ;_  
_ဝိပစ္စနီကော တဝ ဒေဝဓမ္မော,_  
_ပစ္စက္ခတော ထောကတရေန ဧသိ”._  

</Paragraph>

</Division>
<Division number="44" book="ja1" e="ch" v="22" p="232">

<Paragraph number="2153" type="verses" book="ja1">
_“အာယု စ ဝဏ္ဏော စ မနုဿလောကေ,_  
_နိဟီယတိ မနုဇာနံ သုဂတ္တေ;_  
_တေနေဝ ဝဏ္ဏေန ဓနမ္ပိ တုယှံ,_  
_နိဟီယတိ ဇိဏ္ဏတရာသိ အဇ္ဇ._  

</Paragraph>

</Division>
<Division number="45" book="ja1" e="ch" v="22" p="232">

<Paragraph number="2154" type="verses" book="ja1">
_ဧဝံ မေ ပေက္ခမာနဿ,_  
_ရာဇပုတ္တိ ယသဿိနိ;_  
_ဟာယတေဝ တဝ ဝဏ္ဏော,_  
_အဟောရတ္တာနမစ္စယေ”._  

</Paragraph>

</Division>
<Division number="46" book="ja1" e="ch" v="22" p="232">

<Paragraph number="2155" type="verses" book="ja1">
_“ဣမိနာဝ တွံ ဝယသာ,_  
_ရာဇပုတ္တိ သုမေဓသေ;_  
_ဗြဟ္မစရိယံ စရေယျာသိ,_  
_ဘိယျော ဝဏ္ဏဝတီ သိယာ”._  

</Paragraph>

</Division>
<Division number="47" book="ja1" e="ch" v="22" p="232">

<Paragraph number="2156" type="verses" book="ja1">
_“ဒေဝါ န ဇီရန္တိ ယထာ မနုဿာ,_  
_ဂတ္တေသု တေသံ ဝလိယော န ဟောန္တိ;_  
_ပုစ္ဆာမိ တံ ယက္ခ မဟာနုဘာဝ,_  
_ကထံ နု ဒေဝါန သရီရဒေဟော”._  

</Paragraph>

</Division>
<Division number="48" book="ja1" e="ch" v="22" p="232">

<Paragraph number="2157" type="verses" book="ja1">
_“ဒေဝါ န ဇီရန္တိ ယထာ မနုဿာ,_  
_ဂတ္တေသု တေသံ ဝလိယော န ဟောန္တိ;_  
_သုဝေ သုဝေ ဘိယျတရောဝ တေသံ,_  
_ဒိဗ္ဗော စ ဝဏ္ဏော ဝိပုလာ စ ဘောဂါ”._  

</Paragraph>

</Division>
<Division number="49" book="ja1" e="ch" v="22" p="232">

<Paragraph number="2158" type="verses" book="ja1">
_“ကိံသူဓ ဘီတာ ဇနတာ အနေကာ,_  
_မဂ္ဂေါ စ နေကာယတနံ ပဝုတ္တော;_  
_ပုစ္ဆာမိ တံ ယက္ခ မဟာနုဘာဝ,_  
_ကတ္ထဋ္ဌိတော ပရလောကံ န ဘာယေ”._  

</Paragraph>

</Division>
<Division number="50" book="ja1" e="ch" v="22" p="233">

<Paragraph number="2159" type="verses" book="ja1">
_“ဝါစံ မနဉ္စ ပဏိဓာယ သမ္မာ,_  
_ကာယေန ပါပါနိ အကုဗ္ဗမာနော;_  
_ဗဟုန္နပါနံ ဃရမာဝသန္တော,_  
_သဒ္ဓေါ မုဒူ သံဝိဘာဂီ ဝဒညူ;_  
_သင်္ဂါဟကော သခိလော သဏှဝါစော,_  
_ဧတ္ထဋ္ဌိတော ပရလောကံ န ဘာယေ”._  

</Paragraph>

</Division>
<Division number="51" book="ja1" e="ch" v="22" p="233">

<Paragraph number="2160" type="verses" book="ja1">
_“အနုသာသသိ မံ ယက္ခ,_  
_ယထာ မာတာ ယထာ ပိတာ;_  
ဥဠာရဝဏ္ဏံ ပုစ္ဆာမိ,_  
_ကော နု တွမသိ သုဗြဟာ”._  

</Paragraph>

</Division>
<Division number="52" book="ja1" e="ch" v="22" p="233">

<Paragraph number="2161" type="verses" book="ja1">
_“ဥဒယောဟမသ္မိ ကလျာဏိ,_  
_သင်္ဂရတ္တာ ဣဓာဂတော;_  
_အာမန္တ ခေါ တံ ဂစ္ဆာမိ,_  
_မုတ္တောသ္မိ တဝ သင်္ဂရာ”._  

</Paragraph>

</Division>
<Division number="53" book="ja1" e="ch" v="22" p="233">

<Paragraph number="2162" type="verses" book="ja1">
_“သစေ ခေါ တွံ ဥဒယောသိ,_  
_သင်္ဂရတ္တာ ဣဓာဂတော;_  
_အနုသာသ မံ ရာဇပုတ္တ,_  
_ယထာဿ ပုန သင်္ဂမော”._  

</Paragraph>

</Division>
<Division number="54" book="ja1" e="ch" v="22" p="233">

<Paragraph number="2163" type="verses" book="ja1">
_“အတိပတတိ ဝယော ခဏော တထေဝ,_  
_ဌာနံ နတ္ထိ ဓုဝံ စဝန္တိ သတ္တာ;_  
_ပရိဇိယျတိ အဒ္ဓုဝံ သရီရံ,_  
ဥဒယေ မာ ပမာဒ စရဿု ဓမ္မံ._  

</Paragraph>

</Division>
<Division number="55" book="ja1" e="ch" v="22" p="233">

<Paragraph number="2164" type="verses" book="ja1">
_ကသိဏာ ပထဝီ ဓနဿ ပူရာ,_  
_ဧကဿေဝ သိယာ အနညဓေယျာ;_  
_တဉ္စာပိ ဇဟတိ အဝီတရာဂေါ,_  
ဥဒယေ မာ ပမာဒ စရဿု ဓမ္မံ._  

</Paragraph>

</Division>
<Division number="56" book="ja1" e="ch" v="22" p="233">

<Paragraph number="2165" type="verses" book="ja1">
_မာတာ စ ပိတာ စ ဘာတရော စ,_  
_ဘရိယာ ယာပိ ဓနေန ဟောတိ ကီတာ;_  
_တေ စာပိ ဇဟန္တိ အညမညံ,_  
ဥဒယေ မာ ပမာဒ စရဿု ဓမ္မံ._  

</Paragraph>

</Division>
<Division number="57" book="ja1" e="ch" v="22" p="208">

<Paragraph number="2166" book="ja1">
_ကာယော ပရဘောဇနန္တိ ဉတွာ,_  
_သံသာရေ သုဂတိဉ္စ ဒုဂ္ဂတိဉ္စ;_  
ဣတ္တရဝါသောတိ ဇာနိယာန,_  
ဥဒယေ မာ ပမာဒ စရဿု ဓမ္မံ”._  

</Paragraph>

</Division>
<Division number="58" book="ja1" e="ch" v="22" p="248">

<Paragraph number="2167" type="verses" book="ja1">
_“သာဓု ဘာသတိယံ ယက္ခော,_  
_အပ္ပံ မစ္စာန ဇီဝိတံ;_  
_ကသိရဉ္စ ပရိတ္တဉ္စ,_  
_တဉ္စ ဒုက္ခေန သံယုတံ;_  
_သာဟံ ဧကာ ပဗ္ဗဇိဿာမိ,_  
_ဟိတွာ ကာသိံ သုရုန္ဓနန်”တိ._  


</Paragraph>
</Division>
