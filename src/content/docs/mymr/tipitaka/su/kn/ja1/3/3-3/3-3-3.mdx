<Paragraph number="762" type="verses" book="ja1">
_“ကော နု ဥဒ္ဓိတဘတ္တောဝ,_  
_ပူရဟတ္ထောဝ ဗြာဟ္မဏော;_  
_ကဟံ နု ဘိက္ခံ အစရိ,_  
_ကံ သဒ္ဓံ ဥပသင်္ကမိ”._  

</Paragraph>

</Division>
<Division number="68" book="ja1">

<Paragraph number="763" type="verses" book="ja1">
_“အဟံ ကပိသ္မိ ဒုမ္မေဓော,_  
_အနာမာသာနိ အာမသိံ;_  
_တွံ မံ မောစယ ဘဒ္ဒံ တေ,_  
_မုတ္တော ဂစ္ဆေယျ ပဗ္ဗတံ”._  

</Paragraph>

</Division>
<Division number="69" book="ja1">

<Paragraph number="764" type="verses" book="ja1">
_“ကစ္ဆပါ ကဿပါ ဟောန္တိ,_  
_ကောဏ္ဍညာ ဟောန္တိ မက္ကဋာ;_  
_မုဉ္စ ကဿပ ကောဏ္ဍညံ,_  
_ကတံ မေထုနကံ တယာ”တိ._  


</Paragraph>
</Division>
