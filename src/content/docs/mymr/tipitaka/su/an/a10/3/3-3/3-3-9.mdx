“ဒသယိမေ, ဘိက္ခဝေ, ဓမ္မာ ဘာဝိတာ ဗဟုလီကတာ ဧကန္တနိဗ္ဗိဒါယ ဝိရာဂါယ နိရောဓာယ ဥပသမာယ အဘိညာယ သမ္ဗောဓာယ နိဗ္ဗာနာယ သံဝတ္တန္တိ, နာညတြ သုဂတဝိနယာ. ကတမေ ဒသ? သမ္မာဒိဋ္ဌိ …ပေ…  သမ္မာဝိမုတ္တိ—  ဣမေ ခေါ, ဘိက္ခဝေ, ဒသ ဓမ္မာ ဘာဝိတာ ဗဟုလီကတာ ဧကန္တနိဗ္ဗိဒါယ ဝိရာဂါယ နိရောဓာယ ဥပသမာယ အဘိညာယ သမ္ဗောဓာယ နိဗ္ဗာနာယ သံဝတ္တန္တိ, နာညတြ သုဂတဝိနယာ”တိ.
