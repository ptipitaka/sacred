“ဒသယိမေ, ဘိက္ခဝေ, ဓမ္မာ ပရိသုဒ္ဓါ ပရိယောဒါတာ, နာညတြ သုဂတဝိနယာ. ကတမေ ဒသ? သမ္မာဒိဋ္ဌိ, သမ္မာသင်္ကပ္ပေါ, သမ္မာဝါစာ, သမ္မာကမ္မန္တော, သမ္မာအာဇီဝေါ, သမ္မာဝါယာမော, သမ္မာသတိ, သမ္မာသမာဓိ, သမ္မာဉာဏံ, သမ္မာဝိမုတ္တိ—  ဣမေ ခေါ, ဘိက္ခဝေ, ဒသ ဓမ္မာ ပရိသုဒ္ဓါ ပရိယောဒါတာ, နာညတြ သုဂတဝိနယာ”တိ.

</Paragraph>
