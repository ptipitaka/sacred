“စတ္တာရောမေ, ဘိက္ခဝေ, အစ္ဆရိယာ အဗ္ဘုတာ ဓမ္မာ အာနန္ဒေ. ကတမေ စတ္တာရော? သစေ, ဘိက္ခဝေ, ဘိက္ခုပရိသာ အာနန္ဒံ ဒဿနာယ ဥပသင်္ကမတိ, ဒဿနေနပိ သာ အတ္တမနာ ဟောတိ. တတြ စေ အာနန္ဒော ဓမ္မံ ဘာသတိ, ဘာသိတေနပိ သာ အတ္တမနာ ဟောတိ. အတိတ္တာဝ, ဘိက္ခဝေ, ဘိက္ခုပရိသာ ဟောတိ, အထ အာနန္ဒော တုဏှီ ဘဝတိ.
</Paragraph>

<Paragraph number="821" book="a4">

</Paragraph>

<Paragraph number="822" book="a4">
