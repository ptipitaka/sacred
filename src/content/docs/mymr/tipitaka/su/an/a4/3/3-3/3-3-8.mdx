<Paragraph number="815" book="a4">
“တထာဂတဿ, ဘိက္ခဝေ, အရဟတော သမ္မာသမ္ဗုဒ္ဓဿ ပါတုဘာဝါ စတ္တာရော အစ္ဆရိယာ အဗ္ဘုတာ ဓမ္မာ ပါတုဘဝန္တိ. ကတမေ စတ္တာရော? အာလယာရာမာ, ဘိက္ခဝေ, ပဇာ အာလယရတာ အာလယသမ္မုဒိတာ; သာ တထာဂတေန အနာလယေ ဓမ္မေ ဒေသိယမာနေ သုဿူသတိ သောတံ ဩဒဟတိ အညာ စိတ္တံ ဥပဋ္ဌပေတိ. တထာဂတဿ, ဘိက္ခဝေ, အရဟတော သမ္မာသမ္ဗုဒ္ဓဿ ပါတုဘာဝါ အယံ ပဌမော အစ္ဆရိယော အဗ္ဘုတော ဓမ္မော ပါတုဘဝတိ.
</Paragraph>

<Paragraph number="816" book="a4">
မာနာရာမာ, ဘိက္ခဝေ, ပဇာ မာနရတာ မာနသမ္မုဒိတာ. သာ တထာဂတေန မာနဝိနယေ ဓမ္မေ ဒေသိယမာနေ သုဿူသတိ သောတံ ဩဒဟတိ အညာ စိတ္တံ ဥပဋ္ဌပေတိ. တထာဂတဿ, ဘိက္ခဝေ, အရဟတော သမ္မာသမ္ဗုဒ္ဓဿ ပါတုဘာဝါ အယံ ဒုတိယော အစ္ဆရိယော အဗ္ဘုတော ဓမ္မော ပါတုဘဝတိ.
</Paragraph>

<Paragraph number="817" book="a4">
အနုပသမာရာမာ, ဘိက္ခဝေ, ပဇာ အနုပသမရတာ အနုပသမသမ္မုဒိတာ. သာ တထာဂတေန ဩပသမိကေ ဓမ္မေ ဒေသိယမာနေ သုဿူသတိ သောတံ ဩဒဟတိ အညာ စိတ္တံ ဥပဋ္ဌပေတိ. တထာဂတဿ, ဘိက္ခဝေ, အရဟတော သမ္မာသမ္ဗုဒ္ဓဿ ပါတုဘာဝါ အယံ တတိယော အစ္ဆရိယော အဗ္ဘုတော ဓမ္မော ပါတုဘဝတိ.
</Paragraph>

<Paragraph number="818" book="a4">
အဝိဇ္ဇာဂတာ, ဘိက္ခဝေ, ပဇာ အဏ္ဍဘူတာ ပရိယောနဒ္ဓါ. သာ တထာဂတေန အဝိဇ္ဇာဝိနယေ ဓမ္မေ ဒေသိယမာနေ သုဿူသတိ သောတံ ဩဒဟတိ အညာ စိတ္တံ ဥပဋ္ဌပေတိ. တထာဂတဿ, ဘိက္ခဝေ, အရဟတော သမ္မာသမ္ဗုဒ္ဓဿ ပါတုဘာဝါ အယံ စတုတ္ထော အစ္ဆရိယော အဗ္ဘုတော ဓမ္မော ပါတုဘဝတိ. တထာဂတဿ, ဘိက္ခဝေ, အရဟတော သမ္မာသမ္ဗုဒ္ဓဿ ပါတုဘာဝါ ဣမေ စတ္တာရော အစ္ဆရိယာ အဗ္ဘုတာ ဓမ္မာ ပါတုဘဝန္တီ”တိ.

</Paragraph>
