“ပဉ္စိမာနိ, ဘိက္ခဝေ, သမဏဒုက္ခာနိ. ကတမာနိ ပဉ္စ? ဣဓ, ဘိက္ခဝေ, ဘိက္ခု အသန္တုဋ္ဌော ဟောတိ ဣတရီတရေန စီဝရေန, အသန္တုဋ္ဌော ဟောတိ ဣတရီတရေန ပိဏ္ဍပါတေန, အသန္တုဋ္ဌော ဟောတိ ဣတရီတရေန သေနာသနေန, အသန္တုဋ္ဌော ဟောတိ ဣတရီတရေန ဂိလာနပ္ပစ္စယဘေသဇ္ဇပရိက္ခာရေန, အနဘိရတော စ ဗြဟ္မစရိယံ စရတိ. ဣမာနိ ခေါ, ဘိက္ခဝေ, ပဉ္စ သမဏဒုက္ခာနိ.
</Paragraph>

<Paragraph number="723" book="a5">
ပဉ္စိမာနိ, ဘိက္ခဝေ, သမဏသုခါနိ. ကတမာနိ ပဉ္စ? ဣဓ, ဘိက္ခဝေ, ဘိက္ခု သန္တုဋ္ဌော ဟောတိ ဣတရီတရေန စီဝရေန, သန္တုဋ္ဌော ဟောတိ ဣတရီတရေန ပိဏ္ဍပါတေန, သန္တုဋ္ဌော ဟောတိ ဣတရီတရေန သေနာသနေန, သန္တုဋ္ဌော ဟောတိ ဣတရီတရေန ဂိလာနပ္ပစ္စယဘေသဇ္ဇပရိက္ခာရေန, အဘိရတော စ ဗြဟ္မစရိယံ စရတိ. ဣမာနိ ခေါ, ဘိက္ခဝေ, ပဉ္စ သမဏသုခါနီ”တိ.
