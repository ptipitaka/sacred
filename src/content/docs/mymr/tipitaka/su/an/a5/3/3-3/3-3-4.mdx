“ပဉ္စဟိ, ဘိက္ခဝေ, ဓမ္မေဟိ သမန္နာဂတော ဂိလာနုပဋ္ဌာကော နာလံ ဂိလာနံ ဥပဋ္ဌာတုံ. ကတမေဟိ ပဉ္စဟိ? နပ္ပဋိဗလော ဟောတိ ဘေသဇ္ဇံ သံဝိဓာတုံ; သပ္ပါယာသပ္ပါယံ န ဇာနာတိ, အသပ္ပါယံ ဥပနာမေတိ, သပ္ပါယံ အပနာမေတိ; အာမိသန္တရော ဂိလာနံ ဥပဋ္ဌာတိ, နော မေတ္တစိတ္တော; ဇေဂုစ္ဆီ ဟောတိ ဥစ္စာရံ ဝါ ပဿာဝံ ဝါ ဝန္တံ ဝါ ခေဠံ ဝါ နီဟရိတုံ; နပ္ပဋိဗလော ဟောတိ ဂိလာနံ ကာလေန ကာလံ ဓမ္မိယာ ကထာယ သန္ဒဿေတုံ သမာဒပေတုံ သမုတ္တေဇေတုံ သမ္ပဟံသေတုံ. ဣမေဟိ ခေါ, ဘိက္ခဝေ, ပဉ္စဟိ ဓမ္မေဟိ သမန္နာဂတော ဂိလာနုပဋ္ဌာကော နာလံ ဂိလာနံ ဥပဋ္ဌာတုံ.

ပဉ္စဟိ, ဘိက္ခဝေ, ဓမ္မေဟိ သမန္နာဂတော ဂိလာနုပဋ္ဌာကော အလံ ဂိလာနံ ဥပဋ္ဌာတုံ. ကတမေဟိ ပဉ္စဟိ? ပဋိဗလော ဟောတိ ဘေသဇ္ဇံ သံဝိဓာတုံ; သပ္ပါယာသပ္ပါယံ ဇာနာတိ, အသပ္ပါယံ အပနာမေတိ, သပ္ပါယံ ဥပနာမေတိ; မေတ္တစိတ္တော ဂိလာနံ ဥပဋ္ဌာတိ, နော အာမိသန္တရော; အဇေဂုစ္ဆီ ဟောတိ ဥစ္စာရံ ဝါ ပဿာဝံ ဝါ ဝန္တံ ဝါ ခေဠံ ဝါ နီဟရိတုံ; ပဋိဗလော ဟောတိ ဂိလာနံ ကာလေန ကာလံ ဓမ္မိယာ ကထာယ သန္ဒဿေတုံ သမာဒပေတုံ သမုတ္တေဇေတုံ သမ္ပဟံသေတုံ. ဣမေဟိ ခေါ, ဘိက္ခဝေ, ပဉ္စဟိ ဓမ္မေဟိ သမန္နာဂတော ဂိလာနုပဋ္ဌာကော အလံ ဂိလာနံ ဥပဋ္ဌာတုန်”တိ.
