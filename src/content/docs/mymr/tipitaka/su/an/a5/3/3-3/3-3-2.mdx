<Paragraph number="704" book="a5">
“ယော ဟိ ကောစိ, ဘိက္ခဝေ, ဘိက္ခု ဝါ ဘိက္ခုနီ ဝါ ပဉ္စ ဓမ္မေ ဘာဝေတိ ပဉ္စ ဓမ္မေ ဗဟုလီကရောတိ, တဿ ဒွိန္နံ ဖလာနံ အညတရံ ဖလံ ပါဋိကင်္ခံ—  ဒိဋ္ဌေဝ ဓမ္မေ အညာ, သတိ ဝါ ဥပါဒိသေသေ အနာဂါမိတာ.
</Paragraph>

<Paragraph number="705" book="a5">
ကတမေ ပဉ္စ? ဣဓ, ဘိက္ခဝေ, ဘိက္ခုနော အဇ္ဈတ္တညေဝ သတိ သူပဋ္ဌိတာ ဟောတိ ဓမ္မာနံ ဥဒယတ္ထဂါမိနိယာ ပညာယ, အသုဘာနုပဿီ ကာယေ ဝိဟရတိ, အာဟာရေ ပဋိကူလသညီ, သဗ္ဗလောကေ အနဘိရတသညီ, သဗ္ဗသင်္ခါရေသု အနိစ္စာနုပဿီ. ယော ဟိ ကောစိ, ဘိက္ခဝေ, ဘိက္ခု ဝါ ဘိက္ခုနီ ဝါ ဣမေ ပဉ္စ ဓမ္မေ ဘာဝေတိ ဣမေ ပဉ္စ ဓမ္မေ ဗဟုလီကရောတိ, တဿ ဒွိန္နံ ဖလာနံ အညတရံ ဖလံ ပါဋိကင်္ခံ—  ဒိဋ္ဌေဝ ဓမ္မေ အညာ, သတိ ဝါ ဥပါဒိသေသေ အနာဂါမိတာ”တိ.

</Paragraph>
</Division>
