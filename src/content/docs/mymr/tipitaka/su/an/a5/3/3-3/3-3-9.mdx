“ပဉ္စိမေ, ဘိက္ခဝေ, အာပါယိကာ နေရယိကာ ပရိကုပ္ပါ အတေကိစ္ဆာ. ကတမေ ပဉ္စ? မာတာ ဇီဝိတာ ဝေါရောပိတာ ဟောတိ, ပိတာ ဇီဝိတာ ဝေါရောပိတော ဟောတိ, အရဟံ ဇီဝိတာ ဝေါရောပိတော ဟောတိ, တထာဂတဿ ဒုဋ္ဌေန စိတ္တေန လောဟိတံ ဥပ္ပါဒိတံ ဟောတိ, သံဃော ဘိန္နော ဟောတိ. ဣမေ ခေါ, ဘိက္ခဝေ, ပဉ္စ အာပါယိကာ နေရယိကာ ပရိကုပ္ပါ အတေကိစ္ဆာ”တိ.
