“ပဉ္စိမေ, ဘိက္ခဝေ, ဖာသုဝိဟာရာ. ကတမေ ပဉ္စ? ဣဓ, ဘိက္ခဝေ, ဘိက္ခုနော မေတ္တံ ကာယကမ္မံ ပစ္စုပဋ္ဌိတံ ဟောတိ သဗြဟ္မစာရီသု အာဝိ စေဝ ရဟော စ, မေတ္တံ ဝစီကမ္မံ…  မေတ္တံ မနောကမ္မံ ပစ္စုပဋ္ဌိတံ ဟောတိ သဗြဟ္မစာရီသု အာဝိ စေဝ ရဟော စ. ယာနိ တာနိ သီလာနိ အခဏ္ဍာနိ အစ္ဆိဒ္ဒါနိ အသဗလာနိ အကမ္မာသာနိ ဘုဇိဿာနိ ဝိညုပ္ပသတ္ထာနိ အပရာမဋ္ဌာနိ သမာဓိသံဝတ္တနိကာနိ, တထာရူပေဟိ သီလေဟိ သီလသာမညဂတော ဝိဟရတိ သဗြဟ္မစာရီဟိ အာဝိ စေဝ ရဟော စ. ယာယံ ဒိဋ္ဌိ အရိယာ နိယျာနိကာ နိယျာတိ တက္ကရဿ သမ္မာ ဒုက္ခက္ခယာယ, တထာရူပါယ ဒိဋ္ဌိယာ ဒိဋ္ဌိသာမညဂတော ဝိဟရတိ သဗြဟ္မစာရီဟိ အာဝိ စေဝ ရဟော စ. ဣမေ ခေါ, ဘိက္ခဝေ, ပဉ္စ ဖာသုဝိဟာရာ”တိ.
