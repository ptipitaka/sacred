<Paragraph number="1005" book="a3">
“တီဏိမာနိ, ဘိက္ခဝေ, ပဋိစ္ဆန္နာနိ အာဝဟန္တိ, နော ဝိဝဋာနိ. ကတမာနိ တီဏိ? မာတုဂါမော, ဘိက္ခဝေ, ပဋိစ္ဆန္နော အာဝဟတိ, နော ဝိဝဋော; ဗြာဟ္မဏာနံ, ဘိက္ခဝေ, မန္တာ ပဋိစ္ဆန္နာ အာဝဟန္တိ, နော ဝိဝဋာ; မိစ္ဆာဒိဋ္ဌိ, ဘိက္ခဝေ, ပဋိစ္ဆန္နာ အာဝဟတိ, နော ဝိဝဋာ. ဣမာနိ ခေါ, ဘိက္ခဝေ, တီဏိ ပဋိစ္ဆန္နာနိ အာဝဟန္တိ, နော ဝိဝဋာနိ.
</Paragraph>

<Paragraph number="1006" book="a3">
တီဏိမာနိ, ဘိက္ခဝေ, ဝိဝဋာနိ ဝိရောစန္တိ, နော ပဋိစ္ဆန္နာနိ. ကတမာနိ တီဏိ? စန္ဒမဏ္ဍလံ, ဘိက္ခဝေ, ဝိဝဋံ ဝိရောစတိ, နော ပဋိစ္ဆန္နံ; သူရိယမဏ္ဍလံ, ဘိက္ခဝေ, ဝိဝဋံ ဝိရောစတိ, နော ပဋိစ္ဆန္နံ; တထာဂတပ္ပဝေဒိတော ဓမ္မဝိနယော, ဘိက္ခဝေ, ဝိဝဋော ဝိရောစတိ, နော ပဋိစ္ဆန္နော. ဣမာနိ ခေါ, ဘိက္ခဝေ, တီဏိ ဝိဝဋာနိ ဝိရောစန္တိ, နော ပဋိစ္ဆန္နာနီ”တိ.

</Paragraph>
