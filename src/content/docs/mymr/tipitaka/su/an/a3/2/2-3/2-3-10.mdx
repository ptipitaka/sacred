<Paragraph number="712" book="a3">
အထ ခေါ အာယသ္မာ အာနန္ဒော ယေန ဘဂဝါ တေနုပသင်္ကမိ; ဥပသင်္ကမိတွာ ဘဂဝန္တံ အဘိဝါဒေတွာ ဧကမန္တံ နိသီဒိ. ဧကမန္တံ နိသိန္နော ခေါ အာယသ္မာ အာနန္ဒော ဘဂဝန္တံ ဧတဒဝေါစ—  “သမ္မုခါ မေတံ, ဘန္တေ, ဘဂဝတော သုတံ သမ္မုခါ ပဋိဂ္ဂဟိတံ—  ‘ဘဂဝတော, အာနန္ဒ, သိခိဿ အဘိဘူ နာမ သာဝကော ဗြဟ္မလောကေ ဌိတော သဟဿိလောကဓာတုံ သရေန ဝိညာပေသီ’တိ. ဘဂဝါ ပန, ဘန္တေ, အရဟံ သမ္မာသမ္ဗုဒ္ဓေါ ကီဝတကံ ပဟောတိ သရေန ဝိညာပေတုန်”တိ? “သာဝကော သော, အာနန္ဒ, အပ္ပမေယျာ တထာဂတာ”တိ.
</Paragraph>

<Paragraph number="713" book="a3">
ဒုတိယမ္ပိ ခေါ အာယသ္မာ အာနန္ဒော ဘဂဝန္တံ ဧတဒဝေါစ—  “သမ္မုခါ မေတံ, ဘန္တေ, ဘဂဝတော သုတံ သမ္မုခါ ပဋိဂ္ဂဟိတံ—  ‘ဘဂဝတော, အာနန္ဒ, သိခိဿ အဘိဘူ နာမ သာဝကော ဗြဟ္မလောကေ ဌိတော သဟဿိလောကဓာတုံ သရေန ဝိညာပေသီ’တိ. ဘဂဝါ ပန, ဘန္တေ, အရဟံ သမ္မာသမ္ဗုဒ္ဓေါ ကီဝတကံ ပဟောတိ သရေန ဝိညာပေတုန်”တိ? “သာဝကော သော, အာနန္ဒ, အပ္ပမေယျာ တထာဂတာ”တိ.
</Paragraph>

<Paragraph number="714" book="a3">
တတိယမ္ပိ ခေါ အာယသ္မာ အာနန္ဒော ဘဂဝန္တံ ဧတဒဝေါစ—  “သမ္မုခါ မေတံ, ဘန္တေ, ဘဂဝတော သုတံ သမ္မုခါ ပဋိဂ္ဂဟိတံ—  ‘ဘဂဝတော, အာနန္ဒ, သိခိဿ အဘိဘူ နာမ သာဝကော ဗြဟ္မလောကေ ဌိတော သဟဿိလောကဓာတုံ သရေန ဝိညာပေသီ’တိ. ဘဂဝါ ပန, ဘန္တေ, အရဟံ သမ္မာသမ္ဗုဒ္ဓေါ ကီဝတကံ ပဟောတိ သရေန ဝိညာပေတုန်”တိ? “သုတာ တေ, အာနန္ဒ, သဟဿီ စူဠနိကာ လောကဓာတူ”တိ? “ဧတဿ, ဘဂဝါ, ကာလော; ဧတဿ, သုဂတ, ကာလော, ယံ ဘဂဝါ ဘာသေယျ. ဘဂဝတော သုတွာ ဘိက္ခူ ဓာရေဿန္တီ”တိ. “တေနဟာနန္ဒ, သုဏာဟိ သာဓုကံ မနသိ ကရောဟိ, ဘာသိဿာမီ”တိ. “ဧဝံ, ဘန္တေ”တိ ခေါ အာယသ္မာ အာနန္ဒော ဘဂဝတော ပစ္စဿောသိ. ဘဂဝါ ဧတဒဝေါစ—
</Paragraph>

<Paragraph number="715" book="a3">
“ယာဝတာ, အာနန္ဒ, စန္ဒိမသူရိယာ ပရိဟရန္တိ, ဒိသာ ဘန္တိ ဝိရောစနာ, တာဝ သဟဿဓာ လောကော. တသ္မိံ သဟဿဓာ လောကေ သဟဿံ စန္ဒာနံ, သဟဿံ သူရိယာနံ, သဟဿံ သိနေရုပဗ္ဗတရာဇာနံ, သဟဿံ ဇမ္ဗုဒီပါနံ, သဟဿံ အပရဂေါယာနာနံ, သဟဿံ ဥတ္တရကုရူနံ, သဟဿံ ပုဗ္ဗဝိဒေဟာနံ, စတ္တာရိ မဟာသမုဒ္ဒသဟဿာနိ, စတ္တာရိ မဟာရာဇသဟဿာနိ, သဟဿံ စာတုမဟာရာဇိကာနံ, သဟဿံ တာဝတိံသာနံ, သဟဿံ ယာမာနံ, သဟဿံ တုသိတာနံ, သဟဿံ နိမ္မာနရတီနံ, သဟဿံ ပရနိမ္မိတဝသဝတ္တီနံ, သဟဿံ ဗြဟ္မလောကာနံ—  အယံ ဝုစ္စတာနန္ဒ, သဟဿီ စူဠနိကာ လောကဓာတု.
</Paragraph>

<Paragraph number="716" book="a3">
ယာဝတာနန္ဒ, သဟဿီ စူဠနိကာ လောကဓာတု တာဝ သဟဿဓာ လောကော. အယံ ဝုစ္စတာနန္ဒ, ဒွိသဟဿီ မဇ္ဈိမိကာ လောကဓာတု.
</Paragraph>

<Paragraph number="717" book="a3">
ယာဝတာနန္ဒ, ဒွိသဟဿီ မဇ္ဈိမိကာ လောကဓာတု တာဝ သဟဿဓာ လောကော. အယံ ဝုစ္စတာနန္ဒ, တိသဟဿီ မဟာသဟဿီ လောကဓာတု.
</Paragraph>

<Paragraph number="718" book="a3">
အာကင်္ခမာနော, အာနန္ဒ, တထာဂတော တိသဟဿိမဟာသဟဿိလောကဓာတုံ သရေန ဝိညာပေယျ, ယာဝတာ ပန အာကင်္ခေယျာ”တိ.
</Paragraph>

<Paragraph number="719" book="a3">
“ယထာ ကထံ ပန, ဘန္တေ, ဘဂဝါ တိသဟဿိမဟာသဟဿိလောကဓာတုံ သရေန ဝိညာပေယျ, ယာဝတာ ပန အာကင်္ခေယျာ”တိ? “ဣဓာနန္ဒ, တထာဂတော တိသဟဿိမဟာသဟဿိလောကဓာတုံ ဩဘာသေန ဖရေယျ. ယဒါ တေ သတ္တာ တံ အာလောကံ သဉ္ဇာနေယျုံ, အထ တထာဂတော ဃောသံ ကရေယျ သဒ္ဒမနုဿာဝေယျ. ဧဝံ ခေါ, အာနန္ဒ, တထာဂတော တိသဟဿိမဟာသဟဿိလောကဓာတုံ သရေန ဝိညာပေယျ, ယာဝတာ ပန အာကင်္ခေယျာ”တိ.
</Paragraph>

<Paragraph number="720" book="a3">
ဧဝံ ဝုတ္တေ, အာယသ္မာ အာနန္ဒော အာယသ္မန္တံ ဥဒါယိံ ဧတဒဝေါစ—  “လာဘာ ဝတ မေ, သုလဒ္ဓံ ဝတ မေ, ယဿ မေ သတ္ထာ ဧဝံမဟိဒ္ဓိကော ဧဝံမဟာနုဘာဝေါ”တိ. ဧဝံ ဝုတ္တေ, အာယသ္မာ ဥဒါယီ အာယသ္မန္တံ အာနန္ဒံ ဧတဒဝေါစ—  “ကိံ တုယှေတ္ထ, အာဝုသော အာနန္ဒ, ယဒိ တေ သတ္ထာ ဧဝံမဟိဒ္ဓိကော ဧဝံမဟာနုဘာဝေါ”တိ? ဧဝံ ဝုတ္တေ, ဘဂဝါ အာယသ္မန္တံ ဥဒါယိံ ဧတဒဝေါစ—  “မာ ဟေဝံ, ဥဒါယိ, မာ ဟေဝံ, ဥဒါယိ. သစေ, ဥဒါယိ, အာနန္ဒော အဝီတရာဂေါ ကာလံ ကရေယျ, တေန စိတ္တပ္ပသာဒေန သတ္တက္ခတ္တုံ ဒေဝေသု ဒေဝရဇ္ဇံ ကာရေယျ, သတ္တက္ခတ္တုံ ဣမသ္မိံယေဝ ဇမ္ဗုဒီပေ မဟာရဇ္ဇံ ကာရေယျ. အပိ စ, ဥဒါယိ, အာနန္ဒော ဒိဋ္ဌေဝ ဓမ္မေ ပရိနိဗ္ဗာယိဿတီ”တိ.

</Paragraph>
