<Paragraph number="982" book="a3">
ဧကံ သမယံ ဘဂဝါ သာဝတ္ထိယံ ဝိဟရတိ ဇေတဝနေ အနာထပိဏ္ဍိကဿ အာရာမေ. အထ ခေါ ဟတ္ထကော ဒေဝပုတ္တော အဘိက္ကန္တာယ ရတ္တိယာ အဘိက္ကန္တဝဏ္ဏော ကေဝလကပ္ပံ ဇေတဝနံ ဩဘာသေတွာ ယေန ဘဂဝါ တေနုပသင်္ကမိ; ဥပသင်္ကမိတွာ—  “ဘဂဝတော ပုရတော ဌဿာမီ”တိ ဩသီဒတိမေဝ သံသီဒတိမေဝ, န သက္ကောတိ သဏ္ဌာတုံ. သေယျထာပိ နာမ သပ္ပိ ဝါ တေလံ ဝါ ဝါလုကာယ အာသိတ္တံ ဩသီဒတိမေဝ သံသီဒတိမေဝ, န သဏ္ဌာတိ; ဧဝမေဝံ ဟတ္ထကော ဒေဝပုတ္တော—  “ဘဂဝတော ပုရတော ဌဿာမီ”တိ ဩသီဒတိမေဝ သံသီဒတိမေဝ, န သက္ကောတိ သဏ္ဌာတုံ.
</Paragraph>

<Paragraph number="983" book="a3">
အထ ခေါ ဘဂဝါ ဟတ္ထကံ ဒေဝပုတ္တံ ဧတဒဝေါစ—  “ဩဠာရိကံ, ဟတ္ထက, အတ္တဘာဝံ အဘိနိမ္မိနာဟီ”တိ. “ဧဝံ, ဘန္တေ”တိ, ခေါ ဟတ္ထကော ဒေဝပုတ္တော ဘဂဝတော ပဋိဿုတွာ ဩဠာရိကံ အတ္တဘာဝံ အဘိနိမ္မိနိတွာ ဘဂဝန္တံ အဘိဝါဒေတွာ ဧကမန္တံ အဋ္ဌာသိ. ဧကမန္တံ ဌိတံ ခေါ ဟတ္ထကံ ဒေဝပုတ္တံ ဘဂဝါ ဧတဒဝေါစ—
</Paragraph>

<Paragraph number="984" book="a3">
“ယေ တေ, ဟတ္ထက, ဓမ္မာ ပုဗ္ဗေ မနုဿဘူတဿ ပဝတ္တိနော အဟေသုံ, အပိ နု တေ တေ ဓမ္မာ ဧတရဟိ ပဝတ္တိနော”တိ? “ယေ စ မေ, ဘန္တေ, ဓမ္မာ ပုဗ္ဗေ မနုဿဘူတဿ ပဝတ္တိနော အဟေသုံ, တေ စ မေ ဓမ္မာ ဧတရဟိ ပဝတ္တိနော; ယေ စ မေ, ဘန္တေ, ဓမ္မာ ပုဗ္ဗေ မနုဿဘူတဿ နပ္ပဝတ္တိနော အဟေသုံ, တေ စ မေ ဓမ္မာ ဧတရဟိ ပဝတ္တိနော. သေယျထာပိ, ဘန္တေ, ဘဂဝါ ဧတရဟိ အာကိဏ္ဏော ဝိဟရတိ ဘိက္ခူဟိ ဘိက္ခုနီဟိ ဥပါသကေဟိ ဥပါသိကာဟိ ရာဇူဟိ ရာဇမဟာမတ္တေဟိ တိတ္ထိယေဟိ တိတ္ထိယသာဝကေဟိ; ဧဝမေဝံ ခေါ အဟံ, ဘန္တေ, အာကိဏ္ဏော ဝိဟရာမိ ဒေဝပုတ္တေဟိ. ဒူရတောပိ, ဘန္တေ, ဒေဝပုတ္တာ အာဂစ္ဆန္တိ ဟတ္ထကဿ ဒေဝပုတ္တဿ သန္တိကေ ‘ဓမ္မံ သောဿာမာ’တိ. တိဏ္ဏာဟံ, ဘန္တေ, ဓမ္မာနံ အတိတ္တော အပ္ပဋိဝါနော ကာလင်္ကတော. ကတမေသံ တိဏ္ဏံ? ဘဂဝတော အဟံ, ဘန္တေ, ဒဿနဿ အတိတ္တော အပ္ပဋိဝါနော ကာလင်္ကတော; သဒ္ဓမ္မသဝနဿာဟံ, ဘန္တေ, အတိတ္တော အပ္ပဋိဝါနော ကာလင်္ကတော; သံဃဿာဟံ, ဘန္တေ, ဥပဋ္ဌာနဿ အတိတ္တော အပ္ပဋိဝါနော ကာလင်္ကတော. ဣမေသံ ခေါ အဟံ, ဘန္တေ, တိဏ္ဏံ ဓမ္မာနံ အတိတ္တော အပ္ပဋိဝါနော ကာလင်္ကတောတိ.
</Paragraph>

<Paragraph number="985" type="verses" book="a3">
_နာဟံ ဘဂဝတော ဒဿနဿ,_  
_တိတ္တိမဇ္ဈဂါ ကုဒါစနံ;_  
_သံဃဿ ဥပဋ္ဌာနဿ,_  
_သဒ္ဓမ္မသဝနဿ စ._  

</Paragraph>

<Paragraph number="986" type="verses" book="a3">
_အဓိသီလံ သိက္ခမာနော,_  
_သဒ္ဓမ္မသဝနေ ရတော;_  
_တိဏ္ဏံ ဓမ္မာနံ အတိတ္တော,_  
_ဟတ္ထကော အဝိဟံ ဂတော”တိ._  


</Paragraph>
