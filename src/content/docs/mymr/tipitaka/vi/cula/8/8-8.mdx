တေန ခေါ ပန သမယေန သမ္ဗဟုလာ ဘိက္ခူ အဇ္ဈောကာသေ စီဝရကမ္မံ ကရောန္တိ. ဆဗ္ဗဂ္ဂိယာ ဘိက္ခူ ပဋိဝါတေ အင်္ဂဏေ သေနာသနံ ပပ္ဖေါဋေသုံ. ဘိက္ခူ ရဇေန ဩကိရိံသု. ယေ တေ ဘိက္ခူ အပ္ပိစ္ဆာ …ပေ…  တေ ဥဇ္ဈာယန္တိ ခိယျန္တိ ဝိပါစေန္တိ—  “ကထဉှိ နာမ ဆဗ္ဗဂ္ဂိယာ ဘိက္ခူ ပဋိဝါတေ အင်္ဂဏေ သေနာသနံ ပပ္ဖေါဋေဿန္တိ. ဘိက္ခူ ရဇေန ဩကိရိံသူ”တိ.
</Paragraph>

<Paragraph number="1694" book="cula">
အထ ခေါ တေ ဘိက္ခူ ဘဂဝတော ဧတမတ္ထံ အာရောစေသုံ …ပေ…  “သစ္စံ ကိရ, ဘိက္ခဝေ, ဆဗ္ဗဂ္ဂိယာ ဘိက္ခူ ပဋိဝါတေ အင်္ဂဏေ သေနာသနံ ပပ္ဖေါဋေန္တိ, ဘိက္ခူ ရဇေန ဩကိရိံသူ”တိ? “သစ္စံ, ဘဂဝါ”တိ …ပေ…  ဝိဂရဟိတွာ …ပေ…  ဓမ္မိံ ကထံ ကတွာ ဘိက္ခူ အာမန္တေသိ—
</Paragraph>

<Division number="370" book="cula" e="ch" v="4" p="387">

<Paragraph number="1695" book="cula">
“တေန ဟိ, ဘိက္ခဝေ, ဘိက္ခူနံ သေနာသနဝတ္တံ ပညပေဿာမိ ယထာ ဘိက္ခူဟိ သေနာသနေ သမ္မာ ဝတ္တိတဗ္ဗံ. ယသ္မိံ ဝိဟာရေ ဝိဟရတိ, သစေ သော ဝိဟာရော ဥက္လာပေါ ဟောတိ, သစေ ဥဿဟတိ, သောဓေတဗ္ဗော. ဝိဟာရံ သောဓေန္တေန ပဌမံ ပတ္တစီဝရံ နီဟရိတွာ ဧကမန္တံ နိက္ခိပိတဗ္ဗံ; နိသီဒနပစ္စတ္ထရဏံ နီဟရိတွာ ဧကမန္တံ နိက္ခိပိတဗ္ဗံ; ဘိသိဗိဗ္ဗောဟနံ နီဟရိတွာ ဧကမန္တံ နိက္ခိပိတဗ္ဗံ; မဉ္စော နီစံ ကတွာ သာဓုကံ အပ္ပဋိဃံသန္တေန, အသင်္ဃဋ္ဋေန္တေန ကဝါဋပိဋ္ဌံ, နီဟရိတွာ ဧကမန္တံ နိက္ခိပိတဗ္ဗော; ပီဌံ နီစံ ကတွာ သာဓုကံ အပ္ပဋိဃံသန္တေန, အသင်္ဃဋ္ဋေန္တေန ကဝါဋပိဋ္ဌံ, နီဟရိတွာ ဧကမန္တံ နိက္ခိပိတဗ္ဗံ; မဉ္စပဋိပါဒကာ နီဟရိတွာ ဧကမန္တံ နိက္ခိပိတဗ္ဗာ; ခေဠမလ္လကော နီဟရိတွာ ဧကမန္တံ နိက္ခိပိတဗ္ဗော; အပဿေနဖလကံ နီဟရိတွာ ဧကမန္တံ နိက္ခိပိတဗ္ဗံ; ဘုမ္မတ္ထရဏံ ယထာပညတ္တံ သလ္လက္ခေတွာ နီဟရိတွာ ဧကမန္တံ နိက္ခိပိတဗ္ဗံ. သစေ ဝိဟာရေ သန္တာနကံ ဟောတိ, ဥလ္လောကာ ပဌမံ ဩဟာရေတဗ္ဗံ, အာလောကသန္ဓိကဏ္ဏဘာဂါ ပမဇ္ဇိတဗ္ဗာ. သစေ ဂေရုကပရိကမ္မကတာ ဘိတ္တိ ကဏ္ဏကိတာ ဟောတိ, စောဠကံ တေမေတွာ ပီဠေတွာ ပမဇ္ဇိတဗ္ဗာ. သစေ ကာဠဝဏ္ဏကတာ ဘူမိ ကဏ္ဏကိတာ ဟောတိ, စောဠကံ တေမေတွာ ပီဠေတွာ ပမဇ္ဇိတဗ္ဗာ. သစေ အကတာ ဟောတိ ဘူမိ, ဥဒကေန ပရိပ္ဖေါသိတွာ ပရိပ္ဖေါသိတွာ သမ္မဇ္ဇိတဗ္ဗာ—  မာ ဝိဟာရော ရဇေန ဥဟညီတိ. သင်္ကာရံ ဝိစိနိတွာ ဧကမန္တံ ဆဍ္ဍေတဗ္ဗံ.
</Paragraph>

<Paragraph number="1696" book="cula">
န ဘိက္ခုသာမန္တာ သေနာသနံ ပပ္ဖေါဋေတဗ္ဗံ. န ဝိဟာရသာမန္တာ သေနာသနံ ပပ္ဖေါဋေတဗ္ဗံ. န ပါနီယသာမန္တာ သေနာသနံ ပပ္ဖေါဋေတဗ္ဗံ. န ပရိဘောဇနီယသာမန္တာ သေနာသနံ ပပ္ဖေါဋေတဗ္ဗံ. န ပဋိဝါတေ အင်္ဂဏေ သေနာသနံ ပပ္ဖေါဋေတဗ္ဗံ. အဓောဝါတေ သေနာသနံ ပပ္ဖေါဋေတဗ္ဗံ.
</Paragraph>

<Paragraph number="1697" book="cula">
ဘုမ္မတ္ထရဏံ ဧကမန္တံ ဩတာပေတွာ သောဓေတွာ ပပ္ဖေါဋေတွာ အတိဟရိတွာ ယထာပညတ္တံ ပညပေတဗ္ဗံ. မဉ္စပဋိပါဒကာ ဧကမန္တံ ဩတာပေတွာ ပမဇ္ဇိတွာ အတိဟရိတွာ ယထာဌာနေ ဌပေတဗ္ဗာ. မဉ္စော ဧကမန္တံ ဩတာပေတွာ သောဓေတွာ ပပ္ဖေါဋေတွာ နီစံ ကတွာ သာဓုကံ အပ္ပဋိဃံသန္တေန, အသင်္ဃဋ္ဋေန္တေန ကဝါဋပိဋ္ဌံ, အတိဟရိတွာ ယထာပညတ္တံ ပညပေတဗ္ဗော. ပီဌံ ဧကမန္တံ ဩတာပေတွာ သောဓေတွာ ပပ္ဖေါဋေတွာ နီစံ ကတွာ သာဓုကံ အပ္ပဋိဃံသန္တေန, အသင်္ဃဋ္ဋေန္တေန ကဝါဋပိဋ္ဌံ, အတိဟရိတွာ ယထာပညတ္တံ ပညပေတဗ္ဗံ. ဘိသိဗိဗ္ဗောဟနံ ဧကမန္တံ ဩတာပေတွာ သောဓေတွာ ပပ္ဖေါဋေတွာ အတိဟရိတွာ ယထာပညတ္တံ ပညပေတဗ္ဗံ. နိသီဒနပစ္စတ္ထရဏံ ဧကမန္တံ ဩတာပေတွာ သောဓေတွာ ပပ္ဖေါဋေတွာ အတိဟရိတွာ ယထာပညတ္တံ ပညပေတဗ္ဗံ. ခေဠမလ္လကော ဧကမန္တံ ဩတာပေတွာ ပမဇ္ဇိတွာ အတိဟရိတွာ ယထာဌာနေ ဌပေတဗ္ဗော. အပဿေနဖလကံ ဧကမန္တံ ဩတာပေတွာ ပမဇ္ဇိတွာ အတိဟရိတွာ ယထာဌာနေ ဌပေတဗ္ဗံ. ပတ္တစီဝရံ နိက္ခိပိတဗ္ဗံ. ပတ္တံ နိက္ခိပန္တေန ဧကေန ဟတ္ထေန ပတ္တံ ဂဟေတွာ ဧကေန ဟတ္ထေန ဟေဋ္ဌာမဉ္စံ ဝါ ဟေဋ္ဌာပီဌံ ဝါ ပရာမသိတွာ ပတ္တော နိက္ခိပိတဗ္ဗော. န စ အနန္တရဟိတာယ ဘူမိယာ ပတ္တော နိက္ခိပိတဗ္ဗော. စီဝရံ နိက္ခိပန္တေန ဧကေန ဟတ္ထေန စီဝရံ ဂဟေတွာ ဧကေန ဟတ္ထေန စီဝရဝံသံ ဝါ စီဝရရဇ္ဇုံ ဝါ ပမဇ္ဇိတွာ ပါရတော အန္တံ ဩရတော ဘောဂံ ကတွာ စီဝရံ နိက္ခိပိတဗ္ဗံ.
</Paragraph>

<Paragraph number="1698" book="cula">
သစေ ပုရတ္ထိမာ သရဇာ ဝါတာ ဝါယန္တိ, ပုရတ္ထိမာ ဝါတပါနာ ထကေတဗ္ဗာ. သစေ ပစ္ဆိမာ သရဇာ ဝါတာ ဝါယန္တိ, ပစ္ဆိမာ ဝါတပါနာ ထကေတဗ္ဗာ. သစေ ဥတ္တရာ သရဇာ ဝါတာ ဝါယန္တိ ဥတ္တရာ ဝါတပါနာ ထကေတဗ္ဗာ. သစေ ဒက္ခိဏာ သရဇာ ဝါတာ ဝါယန္တိ, ဒက္ခိဏာ ဝါတပါနာ ထကေတဗ္ဗာ. သစေ သီတကာလော ဟောတိ, ဒိဝါ ဝါတပါနာ ဝိဝရိတဗ္ဗာ, ရတ္တိံ ထကေတဗ္ဗာ. သစေ ဥဏှကာလော ဟောတိ, ဒိဝါ ဝါတပါနာ ထကေတဗ္ဗာ, ရတ္တိံ ဝိဝရိတဗ္ဗာ.
</Paragraph>

<Paragraph number="1699" book="cula">
သစေ ပရိဝေဏံ ဥက္လာပံ ဟောတိ, ပရိဝေဏံ သမ္မဇ္ဇိတဗ္ဗံ. သစေ ကောဋ္ဌကော ဥက္လာပေါ ဟောတိ, ကောဋ္ဌကော သမ္မဇ္ဇိတဗ္ဗော. သစေ ဥပဋ္ဌာနသာလာ ဥက္လာပါ ဟောတိ, ဥပဋ္ဌာနသာလာ သမ္မဇ္ဇိတဗ္ဗာ. သစေ အဂ္ဂိသာလာ ဥက္လာပါ ဟောတိ, အဂ္ဂိသာလာ သမ္မဇ္ဇိတဗ္ဗာ. သစေ ဝစ္စကုဋိ ဥက္လာပါ ဟောတိ, ဝစ္စကုဋိ သမ္မဇ္ဇိတဗ္ဗာ. သစေ ပါနီယံ န ဟောတိ, ပါနီယံ ဥပဋ္ဌာပေတဗ္ဗံ. သစေ ပရိဘောဇနီယံ န ဟောတိ, ပရိဘောဇနီယံ ဥပဋ္ဌာပေတဗ္ဗံ. သစေ အာစမနကုမ္ဘိယာ ဥဒကံ န ဟောတိ, အာစမနကုမ္ဘိယာ ဥဒကံ အာသိဉ္စိတဗ္ဗံ.
</Paragraph>

<Paragraph number="1700" book="cula">
သစေ ဝုဍ္ဎေန သဒ္ဓိံ ဧကဝိဟာရေ ဝိဟရတိ, န ဝုဍ္ဎံ အနာပုစ္ဆာ ဥဒ္ဒေသော ဒါတဗ္ဗော, န ပရိပုစ္ဆာ ဒါတဗ္ဗာ, န သဇ္ဈာယော ကာတဗ္ဗော, န ဓမ္မော ဘာသိတဗ္ဗော, န ပဒီပေါ ကာတဗ္ဗော, န ပဒီပေါ ဝိဇ္ဈာပေတဗ္ဗော, န ဝါတပါနာ ဝိဝရိတဗ္ဗာ, န ဝါတပါနာ ထကေတဗ္ဗာ. သစေ ဝုဍ္ဎေန သဒ္ဓိံ ဧကစင်္ကမေ စင်္ကမတိ, ယေန ဝုဍ္ဎော တေန ပရိဝတ္တိတဗ္ဗံ, န စ ဝုဍ္ဎော သင်္ဃာဋိကဏ္ဏေန ဃဋ္ဋေတဗ္ဗော. ဣဒံ ခေါ, ဘိက္ခဝေ, ဘိက္ခူနံ သေနာသနဝတ္တံ ယထာ ဘိက္ခူဟိ သေနာသနေ သမ္မာ ဝတ္တိတဗ္ဗန်”တိ.
</Paragraph>
