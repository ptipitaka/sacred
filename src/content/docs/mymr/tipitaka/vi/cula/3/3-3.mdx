<Paragraph number="673" book="cula">
တေန ခေါ ပန သမယေန အညတရော ဘိက္ခု ပရိဝသန္တော ဝိဗ္ဘမိ. သော ပုန ပစ္စာဂန္တွာ ဘိက္ခူ ဥပသမ္ပဒံ ယာစိ. ဘဂဝတော ဧတမတ္ထံ အာရောစေသုံ.
</Paragraph>

<Paragraph number="674" book="cula">
“ဣဓ ပန, ဘိက္ခဝေ, ဘိက္ခု ပရိဝသန္တော ဝိဗ္ဘမတိ. ဝိဗ္ဘန္တကဿ, ဘိက္ခဝေ, ပရိဝါသော န ရုဟတိ. သော စေ ပုန ဥပသမ္ပဇ္ဇတိ, တဿ တဒေဝ ပုရိမံ ပရိဝါသဒါနံ. ယော ပရိဝါသော ဒိန္နော သုဒိန္နော, ယော ပရိဝုတ္ထော သုပရိဝုတ္ထော, အဝသေသော ပရိဝသိတဗ္ဗော. (1)
</Paragraph>

<Paragraph number="675" book="cula">
ဣဓ ပန, ဘိက္ခဝေ, ဘိက္ခု ပရိဝသန္တော သာမဏေရော ဟောတိ. သာမဏေရဿ, ဘိက္ခဝေ, ပရိဝါသော န ရုဟတိ. သော စေ ပုန ဥပသမ္ပဇ္ဇတိ, တဿ တဒေဝ ပုရိမံ ပရိဝါသဒါနံ. ယော ပရိဝါသော ဒိန္နော သုဒိန္နော, ယော ပရိဝုတ္ထော သုပရိဝုတ္ထော, အဝသေသော ပရိဝသိတဗ္ဗော. (2)
</Paragraph>

<Paragraph number="676" book="cula">
ဣဓ ပန, ဘိက္ခဝေ, ဘိက္ခု ပရိဝသန္တော ဥမ္မတ္တကော ဟောတိ. ဥမ္မတ္တကဿ, ဘိက္ခဝေ, ပရိဝါသော န ရုဟတိ. သော စေ ပုန အနုမ္မတ္တကော ဟောတိ, တဿ တဒေဝ ပုရိမံ ပရိဝါသဒါနံ. ယော ပရိဝါသော ဒိန္နော သုဒိန္နော, ယော ပရိဝုတ္ထော သုပရိဝုတ္ထော, အဝသေသော ပရိဝသိတဗ္ဗော. (3)
</Paragraph>

<Paragraph number="677" book="cula">
ဣဓ ပန, ဘိက္ခဝေ, ဘိက္ခု ပရိဝသန္တော ခိတ္တစိတ္တော ဟောတိ. ခိတ္တစိတ္တဿ, ဘိက္ခဝေ, ပရိဝါသော န ရုဟတိ. သော စေ ပုန အခိတ္တစိတ္တော ဟောတိ, တဿ တဒေဝ ပုရိမံ ပရိဝါသဒါနံ. ယော ပရိဝါသော ဒိန္နော သုဒိန္နော, ယော ပရိဝုတ္ထော သုပရိဝုတ္ထော, အဝသေသော ပရိဝသိတဗ္ဗော. (4)
</Paragraph>

<Paragraph number="678" book="cula">
ဣဓ ပန, ဘိက္ခဝေ, ဘိက္ခု ပရိဝသန္တော ဝေဒနာဋ္ဋော ဟောတိ. ဝေဒနာဋ္ဋဿ, ဘိက္ခဝေ, ပရိဝါသော န ရုဟတိ. သော စေ ပုန အဝေဒနာဋ္ဋော ဟောတိ, တဿ တဒေဝ ပုရိမံ ပရိဝါသဒါနံ. ယော ပရိဝါသော ဒိန္နော သုဒိန္နော, ယော ပရိဝုတ္ထော သုပရိဝုတ္ထော, အဝသေသော ပရိဝသိတဗ္ဗော. (5)
</Paragraph>

<Paragraph number="679" book="cula">
ဣဓ ပန, ဘိက္ခဝေ, ဘိက္ခု ပရိဝသန္တော, အာပတ္တိယာ အဒဿနေ, ဥက္ခိပိယျတိ. ဥက္ခိတ္တကဿ, ဘိက္ခဝေ, ပရိဝါသော န ရုဟတိ. သော စေ ပုန ဩသာရိယျတိ, တဿ တဒေဝ ပုရိမံ ပရိဝါသဒါနံ. ယော ပရိဝါသော ဒိန္နော သုဒိန္နော, ယော ပရိဝုတ္ထော သုပရိဝုတ္ထော, အဝသေသော ပရိဝသိတဗ္ဗော. (6)
</Paragraph>

<Paragraph number="680" book="cula">
ဣဓ ပန, ဘိက္ခဝေ, ဘိက္ခု ပရိဝသန္တော, အာပတ္တိယာ အပ္ပဋိကမ္မေ, ဥက္ခိပိယျတိ. ဥက္ခိတ္တကဿ, ဘိက္ခဝေ, ပရိဝါသော န ရုဟတိ. သော စေ ပုန ဩသာရိယျတိ, တဿ တဒေဝ ပုရိမံ ပရိဝါသဒါနံ. ယော ပရိဝါသော ဒိန္နော သုဒိန္နော, ယော ပရိဝုတ္ထော သုပရိဝုတ္ထော, အဝသေသော ပရိဝသိတဗ္ဗော. (7)
</Paragraph>

<Paragraph number="681" book="cula">
ဣဓ ပန, ဘိက္ခဝေ, ဘိက္ခု ပရိဝသန္တော, ပါပိကာယ ဒိဋ္ဌိယာ, အပ္ပဋိနိဿဂ္ဂေ, ဥက္ခိပိယျတိ. ဥက္ခိတ္တကဿ, ဘိက္ခဝေ, ပရိဝါသော န ရုဟတိ. သော စေ ပုန ဩသာရိယျတိ, တဿ တဒေဝ ပုရိမံ ပရိဝါသဒါနံ. ယော ပရိဝါသော ဒိန္နော သုဒိန္နော, ယော ပရိဝုတ္ထော သုပရိဝုတ္ထော, အဝသေသော ပရိဝသိတဗ္ဗော. (8)
</Paragraph>

<Paragraph number="683" book="cula">

(16)

ဣဓ ပန, ဘိက္ခဝေ, ဘိက္ခု အဗ္ဘာနာရဟော သာမဏေရော ဟောတိ …ပေ…  ဥမ္မတ္တကော ဟောတိ …ပေ…  ခိတ္တစိတ္တော ဟောတိ …ပေ…  ဝေဒနာဋ္ဋော ဟောတိ …ပေ…  အာပတ္တိယာ အဒဿနေ, ဥက္ခိပိယျတိ …ပေ…  အာပတ္တိယာ အပ္ပဋိကမ္မေ, ဥက္ခိပိယျတိ …ပေ…  ပါပိကာယ ဒိဋ္ဌိယာ, အပ္ပဋိနိဿဂ္ဂေ, ဥက္ခိပိယျတိ. ဥက္ခိတ္တကဿ, ဘိက္ခဝေ, အဗ္ဘာနံ န ရုဟတိ. သော စေ ပုန ဩသာရိယျတိ, တဿ တဒေဝ ပုရိမံ ပရိဝါသဒါနံ. ယော ပရိဝါသော ဒိန္နော သုဒိန္နော, ယော ပရိဝုတ္ထော သုပရိဝုတ္ထော; ယံ မာနတ္တံ ဒိန္နံ သုဒိန္နံ, ယံ မာနတ္တံ စိဏ္ဏံ သုစိဏ္ဏံ. သော ဘိက္ခု အဗ္ဘေတဗ္ဗော.
