<Paragraph number="61" book="cula">
တေန ခေါ ပန သမယေန အာယသ္မာ သေယျသကော ဗာလော ဟောတိ အဗျတ္တော အာပတ္တိဗဟုလော အနပဒါနော; ဂိဟိသံသဋ္ဌော ဝိဟရတိ အနနုလောမိကေဟိ ဂိဟိသံသဂ္ဂေဟိ; အပိဿု ဘိက္ခူ ပကတာ ပရိဝါသံ ဒေန္တာ မူလာယ ပဋိကဿန္တာ မာနတ္တံ ဒေန္တာ အဗ္ဘေန္တာ. ယေ တေ ဘိက္ခူ အပ္ပိစ္ဆာ …ပေ…  တေ ဥဇ္ဈာယန္တိ ခိယျန္တိ ဝိပါစေန္တိ—  “ကထဉှိ နာမ အာယသ္မာ သေယျသကော ဗာလော ဘဝိဿတိ အဗျတ္တော အာပတ္တိဗဟုလော အနပဒါနော; ဂိဟိသံသဋ္ဌော ဝိဟရိဿတိ အနနုလောမိကေဟိ ဂိဟိသံသဂ္ဂေဟိ; အပိဿု ဘိက္ခူ ပကတာ ပရိဝါသံ ဒေန္တာ မူလာယ ပဋိကဿန္တာ မာနတ္တံ ဒေန္တာ အဗ္ဘေန္တာ”တိ. အထ ခေါ တေ ဘိက္ခူ ဘဂဝတော ဧတမတ္ထံ အာရောစေသုံ.


</Paragraph>

<Paragraph number="62" book="cula">
အထ ခေါ ဘဂဝါ ဧတသ္မိံ နိဒါနေ ဧတသ္မိံ ပကရဏေ ဘိက္ခုသံဃံ သန္နိပါတာပေတွာ ဘိက္ခူ ပဋိပုစ္ဆိ—  “သစ္စံ ကိရ, ဘိက္ခဝေ, သေယျသကော ဘိက္ခု ဗာလော အဗျတ္တော အာပတ္တိဗဟုလော အနပဒါနော; ဂိဟိသံသဋ္ဌော ဝိဟရတိ အနနုလောမိကေဟိ ဂိဟိသံသဂ္ဂေဟိ; အပိဿု ဘိက္ခူ ပကတာ ပရိဝါသံ ဒေန္တာ မူလာယ ပဋိကဿန္တာ မာနတ္တံ ဒေန္တာ အဗ္ဘေန္တာ”တိ? “သစ္စံ, ဘဂဝါ”တိ. ဝိဂရဟိ ဗုဒ္ဓေါ ဘဂဝါ—  “အနနုစ္ဆဝိကံ, ဘိက္ခဝေ, တဿ မောဃပုရိသဿ အနနုလောမိကံ အပ္ပတိရူပံ အဿာမဏကံ အကပ္ပိယံ အကရဏီယံ. ကထဉှိ နာမ သော, ဘိက္ခဝေ, မောဃပုရိသော ဗာလော ဘဝိဿတိ အဗျတ္တော အာပတ္တိဗဟုလော အနပဒါနော; ဂိဟိသံသဋ္ဌော ဝိဟရိဿတိ အနနုလောမိကေဟိ ဂိဟိသံသဂ္ဂေဟိ; အပိဿု ဘိက္ခူ ပကတာ ပရိဝါသံ ဒေန္တာ မူလာယ ပဋိကဿန္တာ မာနတ္တံ ဒေန္တာ အဗ္ဘေန္တာ. နေတံ, ဘိက္ခဝေ, အပ္ပသန္နာနံ ဝါ ပသာဒါယ …ပေ…  ဝိဂရဟိတွာ …ပေ…  ဓမ္မိံ ကထံ ကတွာ ဘိက္ခူ အာမန္တေသိ—  “တေန ဟိ, ဘိက္ခဝေ, သံဃော သေယျသကဿ ဘိက္ခုနော နိယဿကမ္မံ ကရောတု—  နိဿာယ တေ ဝတ္ထဗ္ဗန္တိ. ဧဝဉ္စ ပန, ဘိက္ခဝေ, ကာတဗ္ဗံ. ပဌမံ သေယျသကော ဘိက္ခု စောဒေတဗ္ဗော, စောဒေတွာ သာရေတဗ္ဗော, သာရေတွာ အာပတ္တိံ အာရောပေတဗ္ဗော, အာပတ္တိံ အာရောပေတွာ ဗျတ္တေန ဘိက္ခုနာ ပဋိဗလေန သံဃော ဉာပေတဗ္ဗော—


</Paragraph>

</Division>
<Division number="12" book="cula" e="ch" v="4" p="14">

<Paragraph number="63" book="cula">
‘သုဏာတု မေ, ဘန္တေ, သံဃော. အယံ သေယျသကော ဘိက္ခု ဗာလော အဗျတ္တော အာပတ္တိဗဟုလော အနပဒါနော; ဂိဟိသံသဋ္ဌော ဝိဟရတိ အနနုလောမိကေဟိ ဂိဟိသံသဂ္ဂေဟိ; အပိဿု ဘိက္ခူ ပကတာ ပရိဝါသံ ဒေန္တာ မူလာယ ပဋိကဿန္တာ မာနတ္တံ ဒေန္တာ အဗ္ဘေန္တာ. ယဒိ သံဃဿ ပတ္တကလ္လံ, သံဃော သေယျသကဿ ဘိက္ခုနော နိယဿကမ္မံ ကရေယျ—  နိဿာယ တေ ဝတ္ထဗ္ဗန္တိ. ဧသာ ဉတ္တိ.


</Paragraph>

<Paragraph number="64" book="cula">
သုဏာတု မေ, ဘန္တေ, သံဃော. အယံ သေယျသကော ဘိက္ခု ဗာလော အဗျတ္တော အာပတ္တိဗဟုလော အနပဒါနော; ဂိဟိသံသဋ္ဌော ဝိဟရတိ အနနုလောမိကေဟိ ဂိဟိသံသဂ္ဂေဟိ; အပိဿု ဘိက္ခူ ပကတာ ပရိဝါသံ ဒေန္တာ မူလာယ ပဋိကဿန္တာ မာနတ္တံ ဒေန္တာ အဗ္ဘေန္တာ. သံဃော သေယျသကဿ ဘိက္ခုနော နိယဿကမ္မံ ကရောတိ—  နိဿာယ တေ ဝတ္ထဗ္ဗန္တိ. ယဿာယသ္မတော ခမတိ သေယျသကဿ ဘိက္ခုနော နိယဿဿ ကမ္မဿ ကရဏံ—  နိဿာယ တေ ဝတ္ထဗ္ဗန္တိ, သော တုဏှဿ; ယဿ နက္ခမတိ, သော ဘာသေယျ.


</Paragraph>

<Paragraph number="65" book="cula">
ဒုတိယမ္ပိ ဧတမတ္ထံ ဝဒါမိ …ပေ…  တတိယမ္ပိ ဧတမတ္ထံ ဝဒါမိ—  သုဏာတု မေ, ဘန္တေ, သံဃော. အယံ သေယျသကော ဘိက္ခု ဗာလော အဗျတ္တော အာပတ္တိဗဟုလော အနပဒါနော; ဂိဟိသံသဋ္ဌော ဝိဟရတိ အနနုလောမိကေဟိ ဂိဟိသံသဂ္ဂေဟိ; အပိဿု ဘိက္ခူ ပကတာ ပရိဝါသံ ဒေန္တာ မူလာယ ပဋိကဿန္တာ မာနတ္တံ ဒေန္တာ အဗ္ဘေန္တာ. သံဃော သေယျသကဿ ဘိက္ခုနော နိယဿကမ္မံ ကရောတိ—  နိဿာယ တေ ဝတ္ထဗ္ဗန္တိ. ယဿာယသ္မတော ခမတိ သေယျသကဿ ဘိက္ခုနော နိယဿဿ ကမ္မဿ ကရဏံ—  နိဿာယ တေ ဝတ္ထဗ္ဗန္တိ, သော တုဏှဿ; ယဿ နက္ခမတိ, သော ဘာသေယျ.


</Paragraph>

<Paragraph number="66" book="cula">
ကတံ သံဃေန သေယျသကဿ ဘိက္ခုနော နိယဿကမ္မံ—  နိဿာယ တေ ဝတ္ထဗ္ဗန္တိ. ခမတိ သံဃဿ, တသ္မာ တုဏှီ, ဧဝမေတံ ဓာရယာမီ’”တိ.

</Paragraph>
</Division>
