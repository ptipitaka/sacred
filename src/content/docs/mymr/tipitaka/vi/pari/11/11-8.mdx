သမ္မုခါဝိနယောတိ ဝါ သတိဝိနယောတိ ဝါ—  ဣမေ ဓမ္မာ သံသဋ္ဌာ ဥဒါဟု ဝိသံသဋ္ဌာ? လဗ္ဘာ စ ပနိမေသံ ဓမ္မာနံ ဝိနိဗ္ဘုဇိတွာ ဝိနိဗ္ဘုဇိတွာ နာနာကရဏံ ပညာပေတုံ? သမ္မုခါဝိနယောတိ ဝါ အမူဠှဝိနယောတိ ဝါ …ပေ…  သမ္မုခါဝိနယောတိ ဝါ ပဋိညာတကရဏန္တိ ဝါ…  သမ္မုခါဝိနယောတိ ဝါ ယေဘုယျသိကာတိ ဝါ…  သမ္မုခါဝိနယောတိ ဝါ တဿပါပိယသိကာတိ ဝါ…  သမ္မုခါဝိနယောတိ ဝါ တိဏဝတ္ထာရကောတိ ဝါ—  ဣမေ ဓမ္မာ သံသဋ္ဌာ, ဥဒါဟု ဝိသံသဋ္ဌာ? လဗ္ဘာ စ ပနိမေသံ ဓမ္မာနံ ဝိနိဗ္ဘုဇိတွာ ဝိနိဗ္ဘုဇိတွာ နာနာကရဏံ ပညာပေတုံ?
</Paragraph>

<Paragraph number="1737" book="pari">
သမ္မုခါဝိနယောတိ ဝါ သတိဝိနယောတိ ဝါ—  ဣမေ ဓမ္မာ သံသဋ္ဌာ, နော ဝိသံသဋ္ဌာ; န စ လဗ္ဘာ ဣမေသံ ဓမ္မာနံ ဝိနိဗ္ဘုဇိတွာ ဝိနိဗ္ဘုဇိတွာ နာနာကရဏံ ပညာပေတုံ. သမ္မုခါဝိနယောတိ ဝါ အမူဠှဝိနယောတိ ဝါ …ပေ…  သမ္မုခါဝိနယောတိ ဝါ ပဋိညာတကရဏန္တိ ဝါ…  သမ္မုခါဝိနယောတိ ဝါ ယေဘုယျသိကာတိ ဝါ…  သမ္မုခါဝိနယောတိ ဝါ တဿပါပိယသိကာတိ ဝါ…  သမ္မုခါဝိနယောတိ ဝါ တိဏဝတ္ထာရကောတိ ဝါ—  ဣမေ ဓမ္မာ သံသဋ္ဌာ, နော ဝိသံသဋ္ဌာ; န စ လဗ္ဘာ ဣမေသံ ဓမ္မာနံ ဝိနိဗ္ဘုဇိတွာ ဝိနိဗ္ဘုဇိတွာ နာနာကရဏံ ပညာပေတုံ.
</Paragraph>
</Division>
