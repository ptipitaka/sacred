<Paragraph number="2411" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ. ဒုမ္မင်္ကူနံ ပုဂ္ဂလာနံ နိဂ္ဂဟာယ, ပေသလာနံ ဘိက္ခူနံ ဖာသုဝိဟာရာယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ.
</Paragraph>

<Paragraph number="2412" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ. ဒိဋ္ဌဓမ္မိကာနံ အာသဝါနံ သံဝရာယ, သမ္ပရာယိကာနံ အာသဝါနံ ပဋိဃာတာယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ.
</Paragraph>

<Paragraph number="2413" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ. ဒိဋ္ဌဓမ္မိကာနံ ဝေရာနံ သံဝရာယ, သမ္ပရာယိကာနံ ဝေရာနံ ပဋိဃာတာယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ.
</Paragraph>

<Paragraph number="2414" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ. ဒိဋ္ဌဓမ္မိကာနံ ဝဇ္ဇာနံ သံဝရာယ, သမ္ပရာယိကာနံ ဝဇ္ဇာနံ ပဋိဃာတာယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ.
</Paragraph>

<Paragraph number="2415" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ. ဒိဋ္ဌဓမ္မိကာနံ ဘယာနံ သံဝရာယ, သမ္ပရာယိကာနံ ဘယာနံ ပဋိဃာတာယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ.
</Paragraph>

<Paragraph number="2416" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ. ဒိဋ္ဌဓမ္မိကာနံ အကုသလာနံ ဓမ္မာနံ သံဝရာယ, သမ္ပရာယိကာနံ အကုသလာနံ ဓမ္မာနံ ပဋိဃာတာယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ.
</Paragraph>

<Paragraph number="2417" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ. ဂိဟီနံ အနုကမ္ပာယ, ပါပိစ္ဆာနံ ပက္ခုပစ္ဆေဒါယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ.
</Paragraph>

<Paragraph number="2418" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ. အပ္ပသန္နာနံ ပသာဒါယ, ပသန္နာနံ ဘိယျောဘာဝါယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ.
</Paragraph>

<Paragraph number="2419" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ. သဒ္ဓမ္မဋ္ဌိတိယာ, ဝိနယာနုဂ္ဂဟာယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ သိက္ခာပဒံ ပညတ္တံ.
