<Paragraph number="2423" book="pari">
…ပေ…  အပညတ္တေ ပညတ္တံ, ပညတ္တေ အနုပညတ္တံ …ပေ…  သမ္မုခါဝိနယော ပညတ္တော …ပေ…  သတိဝိနယော ပညတ္တော …ပေ…  အမူဠှဝိနယော ပညတ္တော …ပေ…  ပဋိညာတကရဏံ ပညတ္တံ …ပေ…  ယေဘုယျသိကာ ပညတ္တာ …ပေ…  တဿပါပိယသိကာ ပညတ္တာ …ပေ…  တိဏဝတ္ထာရကော ပညတ္တော သံဃသုဋ္ဌုတာယ, သံဃဖာသုတာယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော.
</Paragraph>

<Paragraph number="2424" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော. ဒုမ္မင်္ကူနံ ပုဂ္ဂလာနံ နိဂ္ဂဟာယ, ပေသလာနံ ဘိက္ခူနံ ဖာသုဝိဟာရာယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော.
</Paragraph>

<Paragraph number="2425" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော. ဒိဋ္ဌဓမ္မိကာနံ အာသဝါနံ သံဝရာယ, သမ္ပရာယိကာနံ အာသဝါနံ ပဋိဃာတာယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော.
</Paragraph>

<Paragraph number="2426" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော. ဒိဋ္ဌဓမ္မိကာနံ ဝေရာနံ သံဝရာယ, သမ္ပရာယိကာနံ ဝေရာနံ ပဋိဃာတာယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော.
</Paragraph>

<Paragraph number="2427" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော. ဒိဋ္ဌဓမ္မိကာနံ ဝဇ္ဇာနံ သံဝရာယ, သမ္ပရာယိကာနံ ဝဇ္ဇာနံ ပဋိဃာတာယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော.
</Paragraph>

<Paragraph number="2428" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော. ဒိဋ္ဌဓမ္မိကာနံ ဘယာနံ သံဝရာယ, သမ္ပရာယိကာနံ ဘယာနံ ပဋိဃာတာယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော.
</Paragraph>

<Paragraph number="2429" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော. ဒိဋ္ဌဓမ္မိကာနံ အကုသလာနံ ဓမ္မာနံ သံဝရာယ, သမ္ပရာယိကာနံ အကုသလာနံ ဓမ္မာနံ ပဋိဃာတာယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော.
</Paragraph>

<Paragraph number="2430" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော. ဂိဟီနံ အနုကမ္ပာယ, ပါပိစ္ဆာနံ ပက္ခုပစ္ဆေဒါယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော.
</Paragraph>

<Paragraph number="2431" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော. အပ္ပသန္နာနံ ပသာဒါယ, ပသန္နာနံ ဘိယျောဘာဝါယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော.
</Paragraph>

<Paragraph number="2432" book="pari">
ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော. သဒ္ဓမ္မဋ္ဌိတိယာ, ဝိနယာနုဂ္ဂဟာယ—  ဣမေ ဒွေ အတ္ထဝသေ ပဋိစ္စ တထာဂတေန သာဝကာနံ တိဏဝတ္ထာရကော ပညတ္တော.

</Paragraph>
