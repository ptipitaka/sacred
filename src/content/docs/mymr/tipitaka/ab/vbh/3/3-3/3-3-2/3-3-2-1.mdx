သတ္တရသ ဓာတုယော န ဟေတူ. ဓမ္မဓာတု သိယာ ဟေတု, သိယာ န ဟေတု. သောဠသ ဓာတုယော အဟေတုကာ. ဒွေ ဓာတုယော သိယာ သဟေတုကာ, သိယာ အဟေတုကာ. သောဠသ ဓာတုယော ဟေတုဝိပ္ပယုတ္တာ. ဒွေ ဓာတုယော သိယာ ဟေတုသမ္ပယုတ္တာ, သိယာ ဟေတုဝိပ္ပယုတ္တာ. သောဠသ ဓာတုယော န ဝတ္တဗ္ဗာ—  “ဟေတု စေဝ သဟေတုကာ စာ”တိပိ, “သဟေတုကာ စေဝ န စ ဟေတူ”တိပိ. မနောဝိညာဏဓာတု န ဝတ္တဗ္ဗာ—  “ဟေတု စေဝ သဟေတုကာ စာ”တိ, သိယာ သဟေတုကာ စေဝ န စ ဟေတု, သိယာ န ဝတ္တဗ္ဗာ—  “သဟေတုကာ စေဝ န စ ဟေတူ”တိ. ဓမ္မဓာတု သိယာ ဟေတု စေဝ သဟေတုကာ စ, သိယာ သဟေတုကာ စေဝ န စ ဟေတု, သိယာ န ဝတ္တဗ္ဗာ—  “ဟေတု စေဝ သဟေတုကာ စာ”တိပိ, “သဟေတုကာ စေဝ န စ ဟေတူ”တိပိ. သောဠသ ဓာတုယော န ဝတ္တဗ္ဗာ—  “ဟေတူ စေဝ ဟေတုသမ္ပယုတ္တာ စာ”တိပိ, “ဟေတုသမ္ပယုတ္တာ စေဝ န စ ဟေတူ”တိပိ. မနောဝိညာဏဓာတု န ဝတ္တဗ္ဗာ—  “ဟေတု စေဝ ဟေတုသမ္ပယုတ္တာ စာ”တိ, သိယာ ဟေတုသမ္ပယုတ္တာ စေဝ န စ ဟေတု, သိယာ န ဝတ္တဗ္ဗာ—  “ဟေတုသမ္ပယုတ္တာ စေဝ န စ ဟေတူ”တိ. ဓမ္မဓာတု သိယာ ဟေတု စေဝ ဟေတုသမ္ပယုတ္တာ စ, သိယာ ဟေတုသမ္ပယုတ္တာ စေဝ န စ ဟေတု, သိယာ န ဝတ္တဗ္ဗာ—  “ဟေတု စေဝ ဟေတုသမ္ပယုတ္တာ စာ”တိပိ, “ဟေတုသမ္ပယုတ္တာ စေဝ န စ ဟေတူ”တိပိ. သောဠသ ဓာတုယော န ဟေတုအဟေတုကာ. မနောဝိညာဏဓာတု သိယာ န ဟေတုသဟေတုကာ, သိယာ န ဟေတုအဟေတုကာ. ဓမ္မဓာတု သိယာ န ဟေတုသဟေတုကာ, သိယာ န ဟေတုအဟေတုကာ, သိယာ န ဝတ္တဗ္ဗာ—  “န ဟေတုသဟေတုကာ”တိပိ, “န ဟေတုအဟေတုကာ”တိပိ.
</Paragraph>
