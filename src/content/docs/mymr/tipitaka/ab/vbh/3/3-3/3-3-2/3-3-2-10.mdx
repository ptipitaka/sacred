<Paragraph number="701" book="vbh">
ဒသ ဓာတုယော အနာရမ္မဏာ. သတ္တ ဓာတုယော သာရမ္မဏာ. ဓမ္မဓာတု သိယာ သာရမ္မဏာ, သိယာ အနာရမ္မဏာ. သတ္တ ဓာတုယော စိတ္တာ. ဧကာဒသ ဓာတုယော နော စိတ္တာ. သတ္တရသ ဓာတုယော အစေတသိကာ. ဓမ္မဓာတု သိယာ စေတသိကာ, သိယာ အစေတသိကာ. ဒသ ဓာတုယော စိတ္တဝိပ္ပယုတ္တာ. ဓမ္မဓာတု သိယာ စိတ္တသမ္ပယုတ္တာ, သိယာ စိတ္တဝိပ္ပယုတ္တာ. သတ္တ ဓာတုယော န ဝတ္တဗ္ဗာ—  “စိတ္တေန သမ္ပယုတ္တာ”တိပိ, “စိတ္တေန ဝိပ္ပယုတ္တာ”တိပိ. ဒသ ဓာတုယော စိတ္တဝိသံသဋ္ဌာ. ဓမ္မဓာတု သိယာ စိတ္တသံသဋ္ဌာ, သိယာ စိတ္တဝိသံသဋ္ဌာ. သတ္တ ဓာတုယော န ဝတ္တဗ္ဗာ—  “စိတ္တေန သံသဋ္ဌာ”တိပိ, “စိတ္တေန ဝိသံသဋ္ဌာ”တိပိ.
</Paragraph>

<Paragraph number="702" book="vbh">
ဒွါဒသ ဓာတုယော နော စိတ္တသမုဋ္ဌာနာ. ဆ ဓာတုယော သိယာ စိတ္တသမုဋ္ဌာနာ, သိယာ နော စိတ္တသမုဋ္ဌာနာ. သတ္တရသ ဓာတုယော နော စိတ္တသဟဘုနော. ဓမ္မဓာတု သိယာ စိတ္တသဟဘူ, သိယာ နော စိတ္တသဟဘူ. သတ္တရသ ဓာတုယော နော စိတ္တာနုပရိဝတ္တိနော. ဓမ္မဓာတု သိယာ စိတ္တာနုပရိဝတ္တီ, သိယာ နော စိတ္တာနုပရိဝတ္တီ. သတ္တရသ ဓာတုယော နော စိတ္တသံသဋ္ဌသမုဋ္ဌာနာ. ဓမ္မဓာတု သိယာ စိတ္တသံသဋ္ဌသမုဋ္ဌာနာ, သိယာ နော စိတ္တသံသဋ္ဌသမုဋ္ဌာနာ. သတ္တရသ ဓာတုယော နော စိတ္တသံသဋ္ဌသမုဋ္ဌာနသဟဘုနော. ဓမ္မဓာတု သိယာ စိတ္တသံသဋ္ဌသမုဋ္ဌာနသဟဘူ, သိယာ နော စိတ္တသံသဋ္ဌသမုဋ္ဌာနသဟဘူ. သတ္တရသ ဓာတုယော နော စိတ္တသံသဋ္ဌသမုဋ္ဌာနာနုပရိဝတ္တိနော. ဓမ္မဓာတု သိယာ စိတ္တသံသဋ္ဌသမုဋ္ဌာနာနုပရိဝတ္တီ, သိယာ နော စိတ္တသံသဋ္ဌသမုဋ္ဌာနာနုပရိဝတ္တီ.
</Paragraph>

<Paragraph number="703" type="center" book="vbh">
ဒွါဒသ ဓာတုယော အဇ္ဈတ္တိကာ. ဆ ဓာတုယော ဗာဟိရာ.
</Paragraph>

<Paragraph number="704" book="vbh">
နဝ ဓာတုယော ဥပါဒါ. အဋ္ဌ ဓာတုယော နော ဥပါဒါ. ဓမ္မဓာတု သိယာ ဥပါဒါ, သိယာ နော ဥပါဒါ. ဒသ ဓာတုယော ဥပါဒိန္နာ. သဒ္ဒဓာတု အနုပါဒိန္နာ. သတ္တဓာတုယော သိယာ ဥပါဒိန္နာ, သိယာ အနုပါဒိန္နာ.
</Paragraph>
