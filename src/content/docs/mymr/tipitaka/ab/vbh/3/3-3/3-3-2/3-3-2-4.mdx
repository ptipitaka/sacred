သတ္တရသ ဓာတုယော နော သံယောဇနာ. ဓမ္မဓာတု သိယာ သံယောဇနံ, သိယာ နော သံယောဇနံ. သောဠသ ဓာတုယော သံယောဇနိယာ. ဒွေ ဓာတုယော သိယာ သံယောဇနိယာ, သိယာ အသံယောဇနိယာ. သောဠသ ဓာတုယော သံယောဇနဝိပ္ပယုတ္တာ. ဒွေ ဓာတုယော သိယာ သံယောဇနသမ္ပယုတ္တာ, သိယာ သံယောဇနဝိပ္ပယုတ္တာ. သောဠသ ဓာတုယော န ဝတ္တဗ္ဗာ—  “သံယောဇနာ စေဝ သံယောဇနိယာ စာ”တိ, သံယောဇနိယာ စေဝ နော စ သံယောဇနာ. မနောဝိညာဏဓာတု န ဝတ္တဗ္ဗာ—  “သံယောဇနဉ္စေဝ သံယောဇနိယာ စာ”တိ, သိယာ သံယောဇနိယာ စေဝ နော စ သံယောဇနံ, သိယာ န ဝတ္တဗ္ဗာ—  “သံယောဇနိယာ စေဝ နော စ သံယောဇနန်”တိ. ဓမ္မဓာတု သိယာ သံယောဇနဉ္စေဝ သံယောဇနိယာ စ, သိယာ သံယောဇနိယာ စေဝ နော စ သံယောဇနံ, သိယာ န ဝတ္တဗ္ဗာ—  “သံယောဇနဉ္စေဝ သံယောဇနိယာ စာ”တိပိ, “သံယောဇနိယာ စေဝ နော စ သံယောဇနန်”တိပိ. သောဠသ ဓာတုယော န ဝတ္တဗ္ဗာ—  “သံယောဇနာ စေဝ သံယောဇနသမ္ပယုတ္တာ စာ”တိပိ, “သံယောဇနသမ္ပယုတ္တာ စေဝ နော စ သံယောဇနာ”တိပိ. မနောဝိညာဏဓာတု န ဝတ္တဗ္ဗာ—  “သံယောဇနဉ္စေဝ သံယောဇနသမ္ပယုတ္တာ စာ”တိ, သိယာ သံယောဇနသမ္ပယုတ္တာ စေဝ နော စ သံယောဇနံ, သိယာ န ဝတ္တဗ္ဗာ—  “သံယောဇနသမ္ပယုတ္တာ စေဝ နော စ သံယောဇနန်”တိ. ဓမ္မဓာတု သိယာ သံယောဇနဉ္စေဝ သံယောဇနသမ္ပယုတ္တာ စ, သိယာ သံယောဇနသမ္ပယုတ္တာ စေဝ နော စ သံယောဇနံ, သိယာ န ဝတ္တဗ္ဗာ—  “သံယောဇနဉ္စေဝ သံယောဇနသမ္ပယုတ္တာ စာ”တိပိ, “သံယောဇနသမ္ပယုတ္တာ စေဝ နော စ သံယောဇနန်”တိပိ. သောဠသ ဓာတုယော သံယောဇနဝိပ္ပယုတ္တသံယောဇနိယာ. ဒွေ ဓာတုယော သိယာ သံယောဇနဝိပ္ပယုတ္တသံယောဇနိယာ, သိယာ သံယောဇနဝိပ္ပယုတ္တအသံယောဇနိယာ, သိယာ န ဝတ္တဗ္ဗာ—  “သံယောဇနဝိပ္ပယုတ္တသံယောဇနိယာ”တိပိ, “သံယောဇနဝိပ္ပယုတ္တအသံယောဇနိယာ”တိပိ.
</Paragraph>
