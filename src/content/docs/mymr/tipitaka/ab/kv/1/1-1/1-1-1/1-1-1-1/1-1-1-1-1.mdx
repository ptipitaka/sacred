๐ ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာတိ? အာမန္တာ. ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာတိ? န ဟေဝံ ဝတ္တဗ္ဗေ.
</Paragraph>

<Paragraph number="3" book="kv">
အာဇာနာဟိ နိဂ္ဂဟံ. ဟဉ္စိ ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေန, တေန ဝတ ရေ ဝတ္တဗ္ဗေ—  “ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ”တိ. ယံ တတ္ထ ဝဒေသိ—  “ဝတ္တဗ္ဗေ ခေါ—  ‘ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေန’, နော စ ဝတ္တဗ္ဗေ—  ‘ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ’”တိ မိစ္ဆာ.
</Paragraph>

<Paragraph number="4" book="kv">
နော စေ ပန ဝတ္တဗ္ဗေ—  “ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ”တိ, နော စ ဝတ ရေ ဝတ္တဗ္ဗေ—  “ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ”တိ. ယံ တတ္ထ ဝဒေသိ—  “ဝတ္တဗ္ဗေ ခေါ—  ‘ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေန’, နော စ ဝတ္တဗ္ဗေ—  ‘ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ’”တိ မိစ္ဆာ.

× ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာတိ? အာမန္တာ. ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာတိ? န ဟေဝံ ဝတ္တဗ္ဗေ.
</Paragraph>

<Paragraph number="7" book="kv">
အာဇာနာဟိ ပဋိကမ္မံ. ဟဉ္စိ ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေန, တေန ဝတ ရေ ဝတ္တဗ္ဗေ—  “ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ”တိ. ယံ တတ္ထ ဝဒေသိ—  “ဝတ္တဗ္ဗေ ခေါ—  ‘ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေန’, နော စ ဝတ္တဗ္ဗေ—  ‘ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ’”တိ မိစ္ဆာ.
</Paragraph>

<Paragraph number="8" book="kv">
နော စေ ပန ဝတ္တဗ္ဗေ—  “ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ”တိ, နော စ ဝတ ရေ ဝတ္တဗ္ဗေ—  “ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ”တိ. ယံ တတ္ထ ဝဒေသိ—  “ဝတ္တဗ္ဗေ ခေါ—  ‘ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေန’, နော စ ဝတ္တဗ္ဗေ—  ‘ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ’”တိ မိစ္ဆာ.

</Paragraph>

ပဋိကမ္မစတုက္ကံ.

<Division number="3" book="kv">

<Paragraph number="10" book="kv">
× တွဉ္စေ ပန မညသိ—  “ဝတ္တဗ္ဗေ ခေါ—  ‘ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေန’, နော စ ဝတ္တဗ္ဗေ—  ‘ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ’”တိ, တေန တဝ တတ္ထ ဟေတာယ ပဋိညာယ ဟေဝံ ပဋိဇာနန္တံ ဟေဝံ နိဂ္ဂဟေတဗ္ဗေ. အထ တံ နိဂ္ဂဏှာမ. သုနိဂ္ဂဟိတော စ ဟောသိ.
</Paragraph>

<Paragraph number="11" book="kv">
ဟဉ္စိ ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေန, တေန ဝတ ရေ ဝတ္တဗ္ဗေ—  “ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ”တိ. ယံ တတ္ထ ဝဒေသိ—  “ဝတ္တဗ္ဗေ ခေါ—  ‘ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေန’, နော စ ဝတ္တဗ္ဗေ—  ‘ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ’”တိ မိစ္ဆာ.
</Paragraph>

<Paragraph number="12" book="kv">
နော စေ ပန ဝတ္တဗ္ဗေ—  “ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ”တိ, နော စ ဝတ ရေ ဝတ္တဗ္ဗေ—  “ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ”တိ. ယံ တတ္ထ ဝဒေသိ—  “ဝတ္တဗ္ဗေ ခေါ—  ‘ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေန’, နော စ ဝတ္တဗ္ဗေ—  ‘ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော နုပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ’”တိ ဣဒံ တေ မိစ္ဆာ.

</Paragraph>

<Paragraph number="14" book="kv">
× ဧသေ စေ ဒုန္နိဂ္ဂဟိတေ ဟေဝမေဝံ တတ္ထ ဒက္ခ. ဝတ္တဗ္ဗေ ခေါ—  “ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေန”, နော စ ဝတ္တဗ္ဗေ—  “ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ”တိ. နော စ မယံ တယာ တတ္ထ ဟေတာယ ပဋိညာယ ဟေဝံ ပဋိဇာနန္တာ ဟေဝံ နိဂ္ဂဟေတဗ္ဗာ. အထ မံ နိဂ္ဂဏှာသိ. ဒုန္နိဂ္ဂဟိတာ စ ဟောမ.
</Paragraph>

<Paragraph number="15" book="kv">
ဟဉ္စိ ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေန, တေန ဝတ ရေ ဝတ္တဗ္ဗေ—  “ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ”တိ. ယံ တတ္ထ ဝဒေသိ—  “ဝတ္တဗ္ဗေ ခေါ—  ‘ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေန’, နော စ ဝတ္တဗ္ဗေ—  ‘ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ’”တိ မိစ္ဆာ.
</Paragraph>

<Paragraph number="16" book="kv">
နော စေ ပန ဝတ္တဗ္ဗေ—  “ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ”တိ, နော စ ဝတ ရေ ဝတ္တဗ္ဗေ—  “ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ”တိ. ယံ တတ္ထ ဝဒေသိ—  “ဝတ္တဗ္ဗေ ခေါ—  ‘ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေန’, နော စ ဝတ္တဗ္ဗေ—  ‘ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ’”တိ ဣဒံ တေ မိစ္ဆာ.

<Paragraph number="18" book="kv">
× န ဟေဝံ နိဂ္ဂဟေတဗ္ဗေ. တေန ဟိ ယံ နိဂ္ဂဏှာသိ—  “ဟဉ္စိ ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေန, တေန ဝတ ရေ ဝတ္တဗ္ဗေ—  ‘ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ’တိ. ယံ တတ္ထ ဝဒေသိ—  ‘ဝတ္တဗ္ဗေ ခေါ—  ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေန, နော စ ဝတ္တဗ္ဗေ—  ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ’”တိ မိစ္ဆာ.
</Paragraph>

<Paragraph number="19" book="kv">
နော စေ ပန ဝတ္တဗ္ဗေ—  “ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ”တိ, နော စ ဝတ ရေ ဝတ္တဗ္ဗေ—  “ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ”တိ. ယံ တတ္ထ ဝဒေသိ—  “ဝတ္တဗ္ဗေ ခေါ—  ‘ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေန’, နော စ ဝတ္တဗ္ဗေ—  ‘ယော သစ္စိကဋ္ဌော ပရမတ္ထော, တတော သော ပုဂ္ဂလော ဥပလဗ္ဘတိ သစ္စိကဋ္ဌပရမတ္ထေနာ’”တိ ဣဒံ တေ မိစ္ဆာ. တေန ဟိ ယေ ကတေ နိဂ္ဂဟေ သေ နိဂ္ဂဟေ ဒုက္ကဋေ. သုကတေ ပဋိကမ္မေ. သုကတာ ပဋိပါဒနာတိ.
