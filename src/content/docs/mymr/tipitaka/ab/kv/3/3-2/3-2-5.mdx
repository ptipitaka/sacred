<Paragraph number="2111" book="kv">
๐ သတ္တက္ခတ္တုပရမော ပုဂ္ဂလော သတ္တက္ခတ္တုပရမတာနိယတောတိ? အာမန္တာ. အဘဗ္ဗော အန္တရာ ဓမ္မံ အဘိသမေတုန္တိ? န ဟေဝံ ဝတ္တဗ္ဗေ …ပေ… . အဘဗ္ဗော အန္တရာ ဓမ္မံ အဘိသမေတုန္တိ? အာမန္တာ. မာတာ ဇီဝိတာ ဝေါရောပိတာ…  ပိတာ ဇီဝိတာ ဝေါရောပိတော…  အရဟာ ဇီဝိတာ ဝေါရောပိတော…  ဒုဋ္ဌေန စိတ္တေန တထာဂတဿ လောဟိတံ ဥပ္ပါဒိတံ…  သံဃော ဘိန္နောတိ? န ဟေဝံ ဝတ္တဗ္ဗေ …ပေ… .

<Paragraph number="2112" book="kv">
๐ သတ္တက္ခတ္တုပရမော ပုဂ္ဂလော သတ္တက္ခတ္တုပရမတာနိယတောတိ? အာမန္တာ. အတ္ထိ သော နိယမော ယေန နိယမေန သတ္တက္ခတ္တုပရမော ပုဂ္ဂလော သတ္တက္ခတ္တုပရမတာနိယတောတိ? န ဟေဝံ ဝတ္တဗ္ဗေ …ပေ…  အတ္ထိ တေ သတိပဋ္ဌာနာ …ပေ…  သမ္မပ္ပဓာနာ…  ဣဒ္ဓိပါဒါ…  ဣန္ဒြိယာ…  ဗလာ…  ဗောဇ္ဈင်္ဂါ ယေဟိ ဗောဇ္ဈင်္ဂေဟိ သတ္တက္ခတ္တုပရမော ပုဂ္ဂလော သတ္တက္ခတ္တုပရမတာနိယတောတိ? န ဟေဝံ ဝတ္တဗ္ဗေ …ပေ… .
</Paragraph>

</Division>

<Paragraph number="2113" book="kv">
๐ နတ္ထိ သော နိယမော ယေန နိယမေန သတ္တက္ခတ္တုပရမော ပုဂ္ဂလော သတ္တက္ခတ္တုပရမတာနိယတောတိ? အာမန္တာ. ဟဉ္စိ နတ္ထိ သော နိယမော ယေန နိယမေန သတ္တက္ခတ္တုပရမော ပုဂ္ဂလော သတ္တက္ခတ္တုပရမတာနိယတော, နော စ ဝတ ရေ ဝတ္တဗ္ဗေ—  “သတ္တက္ခတ္တုပရမော ပုဂ္ဂလော သတ္တက္ခတ္တုပရမတာနိယတော”တိ.
</Paragraph>

<Paragraph number="2114" book="kv">
๐ နတ္ထိ တေ သတိပဋ္ဌာနာ …ပေ…  ဗောဇ္ဈင်္ဂါ ယေဟိ ဗောဇ္ဈင်္ဂေဟိ သတ္တက္ခတ္တုပရမော ပုဂ္ဂလော သတ္တက္ခတ္တုပရမတာနိယတောတိ? အာမန္တာ. ဟဉ္စိ နတ္ထိ တေ ဗောဇ္ဈင်္ဂါ ယေဟိ ဗောဇ္ဈင်္ဂေဟိ သတ္တက္ခတ္တုပရမော ပုဂ္ဂလော သတ္တက္ခတ္တုပရမတာနိယတော, နော စ ဝတ ရေ ဝတ္တဗ္ဗေ—  “သတ္တက္ခတ္တုပရမော ပုဂ္ဂလော သတ္တက္ခတ္တုပရမတာနိယတော”တိ.
