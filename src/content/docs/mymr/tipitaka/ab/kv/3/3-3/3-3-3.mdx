๐ အနန္တရာပယုတ္တော ပုဂ္ဂလော အဘဗ္ဗော သမ္မတ္တနိယာမံ ဩက္ကမိတုန္တိ? အာမန္တာ. မာတာ ဇီဝိတာ ဝေါရောပိတာ…  ပိတာ ဇီဝိတာ ဝေါရောပိတော…  အရဟာ ဇီဝိတာ ဝေါရောပိတော…  ဒုဋ္ဌေန စိတ္တေန တထာဂတဿ လောဟိတံ ဥပ္ပါဒိတံ…  သံဃော ဘိန္နောတိ? န ဟေဝံ ဝတ္တဗ္ဗေ …ပေ… .
</Paragraph>

<Paragraph number="2153" book="kv">
๐ အနန္တရာပယုတ္တော ပုဂ္ဂလော တံ ကမ္မံ ပဋိသံဟရိတွာ ကုက္ကုစ္စံ ပဋိဝိနောဒေတွာ ဝိပ္ပဋိသာရိယံ ပဋိဝိနေတွာ အဘဗ္ဗော သမ္မတ္တနိယာမံ ဩက္ကမိတုန္တိ? အာမန္တာ. မာတာ ဇီဝိတာ ဝေါရောပိတာ…  ပိတာ ဇီဝိတာ ဝေါရောပိတော …ပေ…  သံဃော ဘိန္နောတိ? န ဟေဝံ ဝတ္တဗ္ဗေ …ပေ… .
</Paragraph>

<Paragraph number="2154" book="kv">
๐ အနန္တရာပယုတ္တော ပုဂ္ဂလော တံ ကမ္မံ ပဋိသံဟရိတွာ ကုက္ကုစ္စံ ပဋိဝိနောဒေတွာ ဝိပ္ပဋိသာရိယံ ပဋိဝိနေတွာ အဘဗ္ဗော သမ္မတ္တနိယာမံ ဩက္ကမိတုန္တိ? အာမန္တာ. နနု တံ ကမ္မံ ပဋိသံဟဋံ ကုက္ကုစ္စံ ပဋိဝိနောဒိတံ ဝိပ္ပဋိသာရိယံ ပဋိဝိနီတန္တိ? အာမန္တာ. ဟဉ္စိ တံ ကမ္မံ ပဋိသံဟဋံ ကုက္ကုစ္စံ ပဋိဝိနောဒိတံ ဝိပ္ပဋိသာရိယံ ပဋိဝိနီတံ, နော စ ဝတ ရေ ဝတ္တဗ္ဗေ—  “အနန္တရာပယုတ္တော ပုဂ္ဂလော တံ ကမ္မံ ပဋိသံဟရိတွာ ကုက္ကုစ္စံ ပဋိဝိနောဒေတွာ ဝိပ္ပဋိသာရိယံ ပဋိဝိနေတွာ အဘဗ္ဗော သမ္မတ္တနိယာမံ ဩက္ကမိတုန်”တိ.
