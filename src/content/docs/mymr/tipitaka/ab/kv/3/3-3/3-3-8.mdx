<Paragraph number="2180" book="kv">
๐ အတ္ထိ အသာတရာဂေါတိ? အာမန္တာ. ဒုက္ခာဘိနန္ဒိနော သတ္တာ, အတ္ထိ ကေစိ ဒုက္ခံ ပတ္ထေန္တိ ပိဟေန္တိ ဧသန္တိ ဂဝေသန္တိ ပရိယေသန္တိ, ဒုက္ခံ အဇ္ဈောသာယ တိဋ္ဌန္တီတိ? န ဟေဝံ ဝတ္တဗ္ဗေ …ပေ…  နနု သုခါဘိနန္ဒိနော သတ္တာ, အတ္ထိ ကေစိ သုခံ ပတ္ထေန္တိ ပိဟေန္တိ ဧသန္တိ ဂဝေသန္တိ ပရိယေသန္တိ, သုခံ အဇ္ဈောသာယ တိဋ္ဌန္တီတိ? အာမန္တာ. ဟဉ္စိ သုခါဘိနန္ဒိနော သတ္တာ, အတ္ထိ ကေစိ သုခံ ပတ္ထေန္တိ ပိဟေန္တိ ဧသန္တိ ဂဝေသန္တိ ပရိယေသန္တိ, သုခံ အဇ္ဈောသာယ တိဋ္ဌန္တိ, နော စ ဝတ ရေ ဝတ္တဗ္ဗေ—  “အတ္ထိ အသာတရာဂေါ”တိ.
</Paragraph>

<Paragraph number="2181" book="kv">
๐ အတ္ထိ အသာတရာဂေါတိ? အာမန္တာ. ဒုက္ခာယ ဝေဒနာယ ရာဂါနုသယော အနုသေတိ, သုခါယ ဝေဒနာယ ပဋိဃာနုသယော အနုသေတီတိ? န ဟေဝံ ဝတ္တဗ္ဗေ …ပေ…  နနု သုခါယ ဝေဒနာယ ရာဂါနုသယော အနုသေတိ, ဒုက္ခာယ ဝေဒနာယ ပဋိဃာနုသယော အနုသေတီတိ? အာမန္တာ. ဟဉ္စိ သုခါယ ဝေဒနာယ ရာဂါနုသယော အနုသေတိ, ဒုက္ခာယ ဝေဒနာယ ပဋိဃာနုသယော အနုသေတိ, နော စ ဝတ ရေ ဝတ္တဗ္ဗေ—  “အတ္ထိ အသာတရာဂေါ”တိ.
</Paragraph>

</Division>

<Paragraph number="2182" book="kv">
× န ဝတ္တဗ္ဗံ—  “အတ္ထိ အသာတရာဂေါ”တိ? အာမန္တာ. နနု ဝုတ္တံ ဘဂဝတာ—  “သော ဧဝံ အနုရောဓဝိရောဓံ သမာပန္နော ယံ ကိဉ္စိ ဝေဒနံ ဝေဒယတိ သုခံ ဝါ ဒုက္ခံ ဝါ အဒုက္ခမသုခံ ဝါ, သော တံ ဝေဒနံ အဘိနန္ဒတိ အဘိဝဒတိ အဇ္ဈောသာယ တိဋ္ဌတီ”တိ. အတ္ထေဝ သုတ္တန္တောတိ? အာမန္တာ. တေန ဟိ အတ္ထိ အသာတရာဂေါတိ.

</Paragraph>
