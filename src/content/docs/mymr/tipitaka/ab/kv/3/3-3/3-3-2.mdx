๐ ကပ္ပဋ္ဌော ကုသလံ စိတ္တံ န ပဋိလဘေယျာတိ? အာမန္တာ. ကပ္ပဋ္ဌော ဒါနံ ဒဒေယျာတိ? အာမန္တာ. ဟဉ္စိ ကပ္ပဋ္ဌော ဒါနံ ဒဒေယျ, နော စ ဝတ ရေ ဝတ္တဗ္ဗေ—  “ကပ္ပဋ္ဌော ကုသလံ စိတ္တံ န ပဋိလဘေယျာ”တိ.

<Paragraph number="2147" book="kv">
๐ ကပ္ပဋ္ဌော ကုသလံ စိတ္တံ န ပဋိလဘေယျာတိ? အာမန္တာ. ကပ္ပဋ္ဌော စီဝရံ ဒဒေယျ …ပေ…  ပိဏ္ဍပါတံ ဒဒေယျ …ပေ…  သေနာသနံ ဒဒေယျ …ပေ…  ဂိလာနပစ္စယဘေသဇ္ဇပရိက္ခာရံ ဒဒေယျ…  ခါဒနီယံ ဒဒေယျ…  ဘောဇနီယံ ဒဒေယျ…  ပါနီယံ ဒဒေယျ…  စေတိယံ ဝန္ဒေယျ…  စေတိယေ မာလံ အာရောပေယျ…  ဂန္ဓံ အာရောပေယျ…  ဝိလေပနံ အာရောပေယျ …ပေ…  စေတိယံ အဘိဒက္ခိဏံ ကရေယျာတိ? အာမန္တာ. ဟဉ္စိ ကပ္ပဋ္ဌော စေတိယံ အဘိဒက္ခိဏံ ကရေယျ, နော စ ဝတ ရေ ဝတ္တဗ္ဗေ—  “ကပ္ပဋ္ဌော ကုသလံ စိတ္တံ န ပဋိလဘေယျာ”တိ …ပေ… .
</Paragraph>
