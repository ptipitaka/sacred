<Paragraph number="1111" book="p2">
အာသဝဝိပ္ပယုတ္တံ ဓမ္မံ ပဋိစ္စ အာသဝဝိပ္ပယုတ္တော ဓမ္မော ဥပ္ပဇ္ဇတိ အဓိပတိပစ္စယာ—  အာသဝဝိပ္ပယုတ္တံ ဧကံ ခန္ဓံ ပဋိစ္စ တယော ခန္ဓာ စိတ္တသမုဋ္ဌာနဉ္စ ရူပံ …ပေ…  ဒွေ ခန္ဓေ …ပေ…  ဒေါမနဿသဟဂတံ မောဟံ ပဋိစ္စ စိတ္တသမုဋ္ဌာနံ ရူပံ, ဧကံ မဟာဘူတံ …ပေ…  မဟာဘူတေ ပဋိစ္စ စိတ္တသမုဋ္ဌာနံ ရူပံ ဥပါဒါရူပံ. (1)
</Paragraph>

<Paragraph number="1112" book="p2">
အာသဝဝိပ္ပယုတ္တံ ဓမ္မံ ပဋိစ္စ အာသဝသမ္ပယုတ္တော ဓမ္မော ဥပ္ပဇ္ဇတိ အဓိပတိပစ္စယာ—  ဒေါမနဿသဟဂတံ မောဟံ ပဋိစ္စ သမ္ပယုတ္တကာ ခန္ဓာ. (2)
</Paragraph>

<Paragraph number="1113" book="p2">
အာသဝဝိပ္ပယုတ္တံ ဓမ္မံ ပဋိစ္စ အာသဝသမ္ပယုတ္တော စ အာသဝဝိပ္ပယုတ္တော စ ဓမ္မာ ဥပ္ပဇ္ဇန္တိ အဓိပတိပစ္စယာ—  ဒေါမနဿသဟဂတံ မောဟံ ပဋိစ္စ သမ္ပယုတ္တကာ ခန္ဓာ စိတ္တသမုဋ္ဌာနဉ္စ ရူပံ. (3)
</Paragraph>

</Division>

<Paragraph number="1114" book="p2">
အာသဝသမ္ပယုတ္တဉ္စ အာသဝဝိပ္ပယုတ္တဉ္စ ဓမ္မံ ပဋိစ္စ အာသဝသမ္ပယုတ္တော ဓမ္မော ဥပ္ပဇ္ဇတိ အဓိပတိပစ္စယာ—  ဒေါမနဿသဟဂတံ ဧကံ ခန္ဓဉ္စ မောဟဉ္စ ပဋိစ္စ တယော ခန္ဓာ …ပေ…  ဒွေ ခန္ဓေ …ပေ… . (1)
</Paragraph>

<Paragraph number="1115" book="p2">
အာသဝသမ္ပယုတ္တဉ္စ အာသဝဝိပ္ပယုတ္တဉ္စ ဓမ္မံ ပဋိစ္စ အာသဝဝိပ္ပယုတ္တော ဓမ္မော ဥပ္ပဇ္ဇတိ အဓိပတိပစ္စယာ—  အာသဝသမ္ပယုတ္တေ ခန္ဓေ စ မဟာဘူတေ စ ပဋိစ္စ စိတ္တသမုဋ္ဌာနံ ရူပံ, ဒေါမနဿသဟဂတေ ခန္ဓေ စ မောဟဉ္စ ပဋိစ္စ စိတ္တသမုဋ္ဌာနံ ရူပံ. (2)
</Paragraph>

<Paragraph number="1116" book="p2">
