<Paragraph number="1152" book="p2">
အာသဝဝိပ္ပယုတ္တံ ဓမ္မံ ပစ္စယာ အာသဝဝိပ္ပယုတ္တော ဓမ္မော ဥပ္ပဇ္ဇတိ အာရမ္မဏပစ္စယာ—  အာသဝဝိပ္ပယုတ္တံ ဧကံ ခန္ဓံ ပစ္စယာ တယော ခန္ဓာ …ပေ…  ဒွေ ခန္ဓေ …ပေ…  ပဋိသန္ဓိက္ခဏေ …ပေ…  ဝတ္ထုံ ပစ္စယာ ခန္ဓာ, စက္ခာယတနံ ပစ္စယာ စက္ခုဝိညာဏံ …ပေ…  ကာယာယတနံ ပစ္စယာ ကာယဝိညာဏံ, ဝတ္ထုံ ပစ္စယာ အာသဝဝိပ္ပယုတ္တာ ခန္ဓာ, ဝတ္ထုံ ပစ္စယာ ဒေါမနဿသဟဂတော ဝိစိကိစ္ဆာသဟဂတော ဥဒ္ဓစ္စသဟဂတော မောဟော. (1)
</Paragraph>

<Paragraph number="1153" book="p2">
အာသဝဝိပ္ပယုတ္တံ ဓမ္မံ ပစ္စယာ အာသဝသမ္ပယုတ္တော ဓမ္မော ဥပ္ပဇ္ဇတိ အာရမ္မဏပစ္စယာ—  ဝတ္ထုံ ပစ္စယာ အာသဝသမ္ပယုတ္တကာ ခန္ဓာ, ဒေါမနဿသဟဂတံ ဝိစိကိစ္ဆာသဟဂတံ ဥဒ္ဓစ္စသဟဂတံ မောဟံ ပစ္စယာ သမ္ပယုတ္တကာ ခန္ဓာ. (2)
</Paragraph>

<Paragraph number="1154" book="p2">
အာသဝဝိပ္ပယုတ္တံ ဓမ္မံ ပစ္စယာ အာသဝသမ္ပယုတ္တော စ အာသဝဝိပ္ပယုတ္တော စ ဓမ္မာ ဥပ္ပဇ္ဇန္တိ အာရမ္မဏပစ္စယာ—  ဝတ္ထုံ ပစ္စယာ ဒေါမနဿသဟဂတာ ဝိစိကိစ္ဆာသဟဂတာ ဥဒ္ဓစ္စသဟဂတာ ခန္ဓာ မောဟော စ. (3)
</Paragraph>

<Paragraph number="1155" book="p2">

</Paragraph>

<Paragraph number="1156" book="p2">
အာသဝသမ္ပယုတ္တဉ္စ အာသဝဝိပ္ပယုတ္တဉ္စ ဓမ္မံ ပစ္စယာ အာသဝဝိပ္ပယုတ္တော ဓမ္မော ဥပ္ပဇ္ဇတိ အာရမ္မဏပစ္စယာ—  ဒေါမနဿသဟဂတေ ဝိစိကိစ္ဆာသဟဂတေ ဥဒ္ဓစ္စသဟဂတေ ခန္ဓေ စ ဝတ္ထုဉ္စ ပစ္စယာ ဒေါမနဿသဟဂတော ဝိစိကိစ္ဆာသဟဂတော ဥဒ္ဓစ္စသဟဂတော မောဟော. (2)
</Paragraph>

<Paragraph number="1157" book="p2">

</Paragraph>
