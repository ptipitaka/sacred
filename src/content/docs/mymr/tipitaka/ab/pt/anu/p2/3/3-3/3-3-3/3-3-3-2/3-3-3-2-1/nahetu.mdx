<Paragraph number="1161">
အာသဝဝိပ္ပယုတ္တံ ဓမ္မံ ပစ္စယာ အာသဝဝိပ္ပယုတ္တော ဓမ္မော ဥပ္ပဇ္ဇတိ နဟေတုပစ္စယာ—  အဟေတုကံ အာသဝဝိပ္ပယုတ္တံ ဧကံ ခန္ဓံ ပစ္စယာ တယော ခန္ဓာ စိတ္တသမုဋ္ဌာနဉ္စ ရူပံ …ပေ…  ဒွေ ခန္ဓေ …ပေ…  အဟေတုကပဋိသန္ဓိက္ခဏေ …ပေ…  (ယာဝ အသညသတ္တာ) စက္ခာယတနံ ပစ္စယာ စက္ခုဝိညာဏံ …ပေ…  ကာယာယတနံ ပစ္စယာ ကာယဝိညာဏံ, ဝတ္ထုံ ပစ္စယာ အဟေတုကာ အာသဝဝိပ္ပယုတ္တာ ခန္ဓာ, ဝတ္ထုံ ပစ္စယာ ဝိစိကိစ္ဆာသဟဂတော ဥဒ္ဓစ္စသဟဂတော မောဟော. (1)
</Paragraph>

<Paragraph number="1162">
