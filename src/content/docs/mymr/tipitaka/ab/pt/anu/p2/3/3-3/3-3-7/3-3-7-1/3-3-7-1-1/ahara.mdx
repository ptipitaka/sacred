<Paragraph number="1224">
အာသဝသမ္ပယုတ္တော ဓမ္မော အာသဝသမ္ပယုတ္တဿ ဓမ္မဿ အာဟာရပစ္စယေန ပစ္စယော—  အာသဝသမ္ပယုတ္တာ အာဟာရာ သမ္ပယုတ္တကာနံ ခန္ဓာနံ အာဟာရပစ္စယေန ပစ္စယော. (1)
</Paragraph>

<Paragraph number="1225">
အာသဝသမ္ပယုတ္တော ဓမ္မော အာသဝဝိပ္ပယုတ္တဿ ဓမ္မဿ အာဟာရပစ္စယေန ပစ္စယော—  အာသဝသမ္ပယုတ္တာ အာဟာရာ စိတ္တသမုဋ္ဌာနာနံ ရူပါနံ အာဟာရပစ္စယေန ပစ္စယော; ဒေါမနဿသဟဂတာ ဝိစိကိစ္ဆာသဟဂတာ ဥဒ္ဓစ္စသဟဂတာ အာဟာရာ မောဟဿ စိတ္တသမုဋ္ဌာနာနဉ္စ ရူပါနံ အာဟာရပစ္စယေန ပစ္စယော. (2)
</Paragraph>

<Paragraph number="1226">
အာသဝသမ္ပယုတ္တော ဓမ္မော အာသဝသမ္ပယုတ္တဿ စ အာသဝဝိပ္ပယုတ္တဿ စ ဓမ္မဿ အာဟာရပစ္စယေန ပစ္စယော—  အာသဝသမ္ပယုတ္တာ အာဟာရာ သမ္ပယုတ္တကာနံ ခန္ဓာနံ စိတ္တသမုဋ္ဌာနာနဉ္စ ရူပါနံ အာဟာရပစ္စယေန ပစ္စယော; ဒေါမနဿသဟဂတာ ဝိစိကိစ္ဆာသဟဂတာ ဥဒ္ဓစ္စသဟဂတာ အာဟာရာ သမ္ပယုတ္တကာနံ ခန္ဓာနံ မောဟဿ စ စိတ္တသမုဋ္ဌာနာနဉ္စ ရူပါနံ အာဟာရပစ္စယေန ပစ္စယော. (3)
</Paragraph>

</Division>

<Paragraph number="1227">
အာသဝဝိပ္ပယုတ္တော ဓမ္မော အာသဝဝိပ္ပယုတ္တဿ ဓမ္မဿ အာဟာရပစ္စယေန ပစ္စယော—  အာသဝဝိပ္ပယုတ္တာ အာဟာရာ သမ္ပယုတ္တကာနံ ခန္ဓာနံ စိတ္တသမုဋ္ဌာနာနဉ္စ ရူပါနံ အာဟာရပစ္စယေန ပစ္စယော; ပဋိသန္ဓိက္ခဏေ …ပေ… . <Emphasis>ကဗဠီကာရော အာဟာရော</Emphasis>—  ဣမဿ ကာယဿ အာဟာရပစ္စယေန ပစ္စယော. (1)
