<Division number="97">

<Paragraph number="1229">
အာသဝသမ္ပယုတ္တော ဓမ္မော အာသဝဝိပ္ပယုတ္တဿ ဓမ္မဿ ဝိပ္ပယုတ္တပစ္စယေန ပစ္စယော—  သဟဇာတံ, ပစ္ဆာဇာတံ. <Emphasis>သဟဇာတာ</Emphasis>—  အာသဝသမ္ပယုတ္တာ ခန္ဓာ စိတ္တသမုဋ္ဌာနာနံ ရူပါနံ ဝိပ္ပယုတ္တပစ္စယေန ပစ္စယော. <Emphasis>ပစ္ဆာဇာတာ</Emphasis>—  အာသဝသမ္ပယုတ္တာ ခန္ဓာ ပုရေဇာတဿ ဣမဿ ကာယဿ ဝိပ္ပယုတ္တပစ္စယေန ပစ္စယော. (1)
</Paragraph>

<Paragraph number="1230">
အာသဝဝိပ္ပယုတ္တော ဓမ္မော အာသဝဝိပ္ပယုတ္တဿ ဓမ္မဿ ဝိပ္ပယုတ္တပစ္စယေန ပစ္စယော—  သဟဇာတံ, ပုရေဇာတံ, ပစ္ဆာဇာတံ. (သံခိတ္တံ. ဝိတ္ထာရေတဗ္ဗံ.) (1)
</Paragraph>

</Division>
<Division number="98">

<Paragraph number="1231">
အာသဝဝိပ္ပယုတ္တော ဓမ္မော အာသဝသမ္ပယုတ္တဿ ဓမ္မဿ ဝိပ္ပယုတ္တပစ္စယေန ပစ္စယော. <Emphasis>ပုရေဇာတံ</Emphasis>—  ဝတ္ထု အာသဝသမ္ပယုတ္တကာနံ ခန္ဓာနံ ဝိပ္ပယုတ္တပစ္စယေန ပစ္စယော. (1)
</Paragraph>

<Paragraph number="1232">
အာသဝဝိပ္ပယုတ္တော ဓမ္မော အာသဝသမ္ပယုတ္တဿ စ အာသဝဝိပ္ပယုတ္တဿ စ ဓမ္မဿ ဝိပ္ပယုတ္တပစ္စယေန ပစ္စယော. <Emphasis>ပုရေဇာတံ</Emphasis>—  ဝတ္ထု ဒေါမနဿသဟဂတာနံ ဝိစိကိစ္ဆာသဟဂတာနံ ဥဒ္ဓစ္စသဟဂတာနံ ခန္ဓာနံ မောဟဿ စ ဝိပ္ပယုတ္တပစ္စယေန ပစ္စယော. (2)
</Paragraph>

<Paragraph number="1233">
အာသဝသမ္ပယုတ္တော စ အာသဝဝိပ္ပယုတ္တော စ ဓမ္မာ အာသဝဝိပ္ပယုတ္တဿ ဓမ္မဿ ဝိပ္ပယုတ္တပစ္စယေန ပစ္စယော—  သဟဇာတံ, ပစ္ဆာဇာတံ. <Emphasis>သဟဇာတာ</Emphasis>—  ဒေါမနဿသဟဂတာ ဝိစိကိစ္ဆာသဟဂတာ ဥဒ္ဓစ္စသဟဂတာ ခန္ဓာ စ မောဟော စ စိတ္တသမုဋ္ဌာနာနံ ရူပါနံ ဝိပ္ပယုတ္တပစ္စယေန ပစ္စယော. <Emphasis>ပစ္ဆာဇာတာ</Emphasis>—  ဒေါမနဿသဟဂတာ ဝိစိကိစ္ဆာသဟဂတာ ဥဒ္ဓစ္စသဟဂတာ ခန္ဓာ စ မောဟော စ ပုရေဇာတဿ ဣမဿ ကာယဿ ဝိပ္ပယုတ္တပစ္စယေန ပစ္စယော. (1)
</Paragraph>
