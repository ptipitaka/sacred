အာသဝဝိပ္ပယုတ္တော ဓမ္မော အာသဝဝိပ္ပယုတ္တဿ ဓမ္မဿ ပုရေဇာတပစ္စယေန ပစ္စယော—  အာရမ္မဏပုရေဇာတံ, ဝတ္ထုပုရေဇာတံ. <Emphasis>အာရမ္မဏပုရေဇာတံ</Emphasis>—  စက္ခုံ …ပေ…  ဝတ္ထုံ အနိစ္စတော ဒုက္ခတော အနတ္တတော ဝိပဿတိ; ဒိဗ္ဗေန စက္ခုနာ ရူပံ ပဿတိ, ဒိဗ္ဗာယ သောတဓာတုယာ သဒ္ဒံ သုဏာတိ. ရူပါယတနံ စက္ခုဝိညာဏဿ …ပေ…  ဖောဋ္ဌဗ္ဗာယတနံ ကာယဝိညာဏဿ ပုရေဇာတပစ္စယေန ပစ္စယော. <Emphasis>ဝတ္ထုပုရေဇာတံ</Emphasis>—  စက္ခာယတနံ စက္ခုဝိညာဏဿ …ပေ…  ကာယာယတနံ ကာယဝိညာဏဿ …ပေ…  ဝတ္ထု အာသဝဝိပ္ပယုတ္တာနံ ခန္ဓာနံ မောဟဿ စ ပုရေဇာတပစ္စယေန ပစ္စယော. (1)
</Paragraph>

<Paragraph number="1213" book="p2">
အာသဝဝိပ္ပယုတ္တော ဓမ္မော အာသဝသမ္ပယုတ္တဿ ဓမ္မဿ ပုရေဇာတပစ္စယေန ပစ္စယော—  အာရမ္မဏပုရေဇာတံ, ဝတ္ထုပုရေဇာတံ. <Emphasis>အာရမ္မဏပုရေဇာတံ</Emphasis>—  စက္ခုံ …ပေ…  ဝတ္ထုံ အဿာဒေတိ အဘိနန္ဒတိ, တံ အာရဗ္ဘ ရာဂေါ ဥပ္ပဇ္ဇတိ …ပေ…  ဒေါမနဿံ ဥပ္ပဇ္ဇတိ. <Emphasis>ဝတ္ထုပုရေဇာတံ</Emphasis>—  ဝတ္ထု အာသဝသမ္ပယုတ္တကာနံ ခန္ဓာနံ ပုရေဇာတပစ္စယေန ပစ္စယော. (2)
</Paragraph>

<Paragraph number="1214" book="p2">
အာသဝဝိပ္ပယုတ္တော ဓမ္မော အာသဝသမ္ပယုတ္တဿ စ အာသဝဝိပ္ပယုတ္တဿ စ ဓမ္မဿ ပုရေဇာတပစ္စယေန ပစ္စယော—  အာရမ္မဏပုရေဇာတံ, ဝတ္ထုပုရေဇာတံ. <Emphasis>အာရမ္မဏပုရေဇာတံ</Emphasis>—  စက္ခုံ …ပေ…  ဝတ္ထုံ အာရဗ္ဘ ဒေါမနဿသဟဂတာ ဝိစိကိစ္ဆာသဟဂတာ ဥဒ္ဓစ္စသဟဂတာ ခန္ဓာ စ မောဟော စ ဥပ္ပဇ္ဇန္တိ. <Emphasis>ဝတ္ထုပုရေဇာတံ</Emphasis>—  ဝတ္ထု ဒေါမနဿသဟဂတာနံ ဝိစိကိစ္ဆာသဟဂတာနံ ဥဒ္ဓစ္စသဟဂတာနံ ခန္ဓာနံ မောဟဿ စ ပုရေဇာတပစ္စယေန ပစ္စယော. (3)
</Paragraph>
