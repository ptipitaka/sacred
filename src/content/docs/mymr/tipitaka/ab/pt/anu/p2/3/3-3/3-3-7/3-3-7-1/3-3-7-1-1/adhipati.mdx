<Paragraph number="1190" book="p2">
အာသဝသမ္ပယုတ္တော ဓမ္မော အာသဝဝိပ္ပယုတ္တဿ ဓမ္မဿ အဓိပတိပစ္စယေန ပစ္စယော. <Emphasis>သဟဇာတာဓိပတိ</Emphasis>—  အာသဝသမ္ပယုတ္တာဓိပတိ စိတ္တသမုဋ္ဌာနာနံ ရူပါနံ အဓိပတိပစ္စယေန ပစ္စယော; ဒေါမနဿသဟဂတာဓိပတိ မောဟဿ စိတ္တသမုဋ္ဌာနာနဉ္စ ရူပါနံ အဓိပတိပစ္စယေန ပစ္စယော. (2)
</Paragraph>

အာသဝဝိပ္ပယုတ္တော ဓမ္မော အာသဝဝိပ္ပယုတ္တဿ ဓမ္မဿ အဓိပတိပစ္စယေန ပစ္စယော—  အာရမ္မဏာဓိပတိ, သဟဇာတာဓိပတိ. <Emphasis>အာရမ္မဏာဓိပတိ</Emphasis>—  ဒါနံ ဒတွာ, သီလံ …ပေ…  ဥပေါသထကမ္မံ ကတွာ တံ ဂရုံ ကတွာ ပစ္စဝေက္ခတိ; ပုဗ္ဗေ သုစိဏ္ဏာနိ …ပေ…  ဈာနာ …ပေ…  အရိယာ မဂ္ဂါ ဝုဋ္ဌဟိတွာ မဂ္ဂံ …ပေ…  ဖလံ …ပေ…  နိဗ္ဗာနံ ဂရုံ ကတွာ ပစ္စဝေက္ခန္တိ; နိဗ္ဗာနံ ဂေါတြဘုဿ, ဝေါဒါနဿ, မဂ္ဂဿ, ဖလဿ အဓိပတိပစ္စယေန ပစ္စယော. <Emphasis>သဟဇာတာဓိပတိ</Emphasis>—  အာသဝဝိပ္ပယုတ္တာဓိပတိ သမ္ပယုတ္တကာနံ ခန္ဓာနံ စိတ္တသမုဋ္ဌာနာနဉ္စ ရူပါနံ အဓိပတိပစ္စယေန ပစ္စယော. (1)
</Paragraph>
