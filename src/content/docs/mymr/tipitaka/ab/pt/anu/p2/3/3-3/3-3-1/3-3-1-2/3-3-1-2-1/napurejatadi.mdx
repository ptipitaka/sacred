<Paragraph number="1127">

</Paragraph>

<Paragraph number="1128">

<Paragraph number="1129">
အာသဝဝိပ္ပယုတ္တံ ဓမ္မံ ပဋိစ္စ အာသဝဝိပ္ပယုတ္တော ဓမ္မော ဥပ္ပဇ္ဇတိ နပုရေဇာတပစ္စယာ—  အရူပေ အာသဝဝိပ္ပယုတ္တံ ဧကံ ခန္ဓံ ပဋိစ္စ တယော ခန္ဓာ …ပေ…  ဒွေ ခန္ဓေ …ပေ…  အာသဝဝိပ္ပယုတ္တေ ခန္ဓေ ပဋိစ္စ စိတ္တသမုဋ္ဌာနံ ရူပံ, ဒေါမနဿသဟဂတံ ဝိစိကိစ္ဆာသဟဂတံ ဥဒ္ဓစ္စသဟဂတံ မောဟံ ပဋိစ္စ စိတ္တသမုဋ္ဌာနံ ရူပံ; ပဋိသန္ဓိက္ခဏေ …ပေ… . (ယာဝ အသညသတ္တာ.) (1)
</Paragraph>

<Paragraph number="1130">
