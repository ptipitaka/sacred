ဟေတုယာ တီဏိ, အာရမ္မဏေ စတ္တာရိ, အဓိပတိယာ စတ္တာရိ, အနန္တရေ စတ္တာရိ, သမနန္တရေ စတ္တာရိ, သဟဇာတေ သတ္တ, အညမညေ ဆ, နိဿယေ သတ္တ, ဥပနိဿယေ စတ္တာရိ, ပုရေဇာတေ ပစ္ဆာဇာတေ အာသေဝနေ ဒွေ, ကမ္မေ စတ္တာရိ, ဝိပါကေ စတ္တာရိ, အာဟာရေ ဣန္ဒြိယေ ဈာနေ စတ္တာရိ, မဂ္ဂေ တီဏိ, သမ္ပယုတ္တေ ဒွေ, ဝိပ္ပယုတ္တေ တီဏိ …ပေ…  အဝိဂတေ သတ္တ. (သံခိတ္တံ.)
