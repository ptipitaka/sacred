အာသဝသမ္ပယုတ္တော ဓမ္မော အာသဝသမ္ပယုတ္တဿ ဓမ္မဿ ကမ္မပစ္စယေန ပစ္စယော—  အာသဝသမ္ပယုတ္တကာ စေတနာ သမ္ပယုတ္တကာနံ ခန္ဓာနံ ကမ္မပစ္စယေန ပစ္စယော. (1)

<Paragraph number="1220" book="p2">
အာသဝသမ္ပယုတ္တော ဓမ္မော အာသဝဝိပ္ပယုတ္တဿ ဓမ္မဿ ကမ္မပစ္စယေန ပစ္စယော—  သဟဇာတာ, နာနာက္ခဏိကာ. <Emphasis>သဟဇာတာ</Emphasis>—  အာသဝသမ္ပယုတ္တာ စေတနာ စိတ္တသမုဋ္ဌာနာနံ ရူပါနံ ကမ္မပစ္စယေန ပစ္စယော; ဒေါမနဿသဟဂတာ ဝိစိကိစ္ဆာသဟဂတာ ဥဒ္ဓစ္စသဟဂတာ စေတနာ မောဟဿ စိတ္တသမုဋ္ဌာနာနဉ္စ ရူပါနံ ကမ္မပစ္စယေန ပစ္စယော. <Emphasis>နာနာက္ခဏိကာ</Emphasis>—  အာသဝသမ္ပယုတ္တကာ စေတနာ ဝိပါကာနံ ခန္ဓာနံ ကဋတ္တာ စ ရူပါနံ ကမ္မပစ္စယေန ပစ္စယော. (2)
</Paragraph>

<Paragraph number="1221" book="p2">
အာသဝသမ္ပယုတ္တော ဓမ္မော အာသဝသမ္ပယုတ္တဿ စ အာသဝဝိပ္ပယုတ္တဿ စ ဓမ္မဿ ကမ္မပစ္စယေန ပစ္စယော—  အာသဝသမ္ပယုတ္တာ စေတနာ သမ္ပယုတ္တကာနံ ခန္ဓာနံ စိတ္တသမုဋ္ဌာနာနဉ္စ ရူပါနံ ကမ္မပစ္စယေန ပစ္စယော; ဒေါမနဿသဟဂတာ ဝိစိကိစ္ဆာသဟဂတာ ဥဒ္ဓစ္စသဟဂတာ စေတနာ သမ္ပယုတ္တကာနံ ခန္ဓာနံ မောဟဿ စ စိတ္တသမုဋ္ဌာနာနဉ္စ ရူပါနံ ကမ္မပစ္စယေန ပစ္စယော. (3)
</Paragraph>

<Paragraph number="1222" book="p2">
