အနိဒဿနံ ဓမ္မံ ပစ္စယာ အနိဒဿနော ဓမ္မော ဥပ္ပဇ္ဇတိ ဟေတုပစ္စယာ—  အနိဒဿနံ ဧကံ ခန္ဓံ ပစ္စယာ တယော ခန္ဓာ အနိဒဿနံ စိတ္တသမုဋ္ဌာနဉ္စ ရူပံ …ပေ…  ဒွေ ခန္ဓေ …ပေ…  ပဋိသန္ဓိက္ခဏေ …ပေ…  ခန္ဓေ ပစ္စယာ ဝတ္ထု, ဝတ္ထုံ ပစ္စယာ ခန္ဓာ; ဧကံ မဟာဘူတံ ပစ္စယာ …ပေ…  မဟာဘူတေ ပစ္စယာ အနိဒဿနံ စိတ္တသမုဋ္ဌာနံ ရူပံ ကဋတ္တာရူပံ ဥပါဒါရူပံ, ဝတ္ထုံ ပစ္စယာ အနိဒဿနာ ခန္ဓာ. (ဣတရေပိ ဒွေ ပဉှာ ကာတဗ္ဗာ.)
