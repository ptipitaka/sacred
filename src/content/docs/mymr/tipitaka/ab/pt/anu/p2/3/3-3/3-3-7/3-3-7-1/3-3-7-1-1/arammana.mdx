(ဣဓ အဿာဒနာ နတ္ထိ) ဒိဗ္ဗေန စက္ခုနာ ရူပံ ပဿတိ, ဒိဗ္ဗာယ သောတဓာတုယာ သဒ္ဒံ သုဏာတိ. စေတောပရိယဉာဏေန အာသဝဝိပ္ပယုတ္တစိတ္တသမင်္ဂိဿ စိတ္တံ ဇာနာတိ, အာကာသာနဉ္စာယတနံ ဝိညာဏဉ္စာယတနဿ …ပေ…  အာကိဉ္စညာယတနံ နေဝသညာနာသညာယတနဿ …ပေ…  ရူပါယတနံ စက္ခုဝိညာဏဿ …ပေ…  ဖောဋ္ဌဗ္ဗာယတနံ ကာယဝိညာဏဿ …ပေ…  အာသဝဝိပ္ပယုတ္တာ ခန္ဓာ ဣဒ္ဓိဝိဓဉာဏဿ, စေတောပရိယဉာဏဿ, ပုဗ္ဗေနိဝါသာနုဿတိဉာဏဿ, ယထာကမ္မူပဂဉာဏဿ, အနာဂတံသဉာဏဿ, အာဝဇ္ဇနာယ မောဟဿ အာရမ္မဏပစ္စယေန ပစ္စယော. (1)
</Paragraph>
