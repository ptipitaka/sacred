<Paragraph number="881">
အသညသတ္တာနံ တေသံ တတ္ထ ရူပက္ခန္ဓော ဥပ္ပဇ္ဇိဿတိ, နော စ တေသံ တတ္ထ ဝေဒနာက္ခန္ဓော နိရုဇ္ဈိဿတိ. ပဉ္စဝေါကာရာနံ တေသံ တတ္ထ ရူပက္ခန္ဓော စ ဥပ္ပဇ္ဇိဿတိ ဝေဒနာက္ခန္ဓော စ နိရုဇ္ဈိဿတိ.
</Paragraph>

<Paragraph number="882">
«  ယဿ ဝါ ပန ယတ္ထ ဝေဒနာက္ခန္ဓော နိရုဇ္ဈိဿတိ တဿ တတ္ထ ရူပက္ခန္ဓော ဥပ္ပဇ္ဇိဿတီတိ?
</Paragraph>

<Paragraph number="883">
ပစ္ဆိမဘဝိကာနံ ပဉ္စဝေါကာရံ ဥပပဇ္ဇန္တာနံ အရူပါနံ တေသံ တတ္ထ ဝေဒနာက္ခန္ဓော နိရုဇ္ဈိဿတိ, နော စ တေသံ တတ္ထ ရူပက္ခန္ဓော ဥပ္ပဇ္ဇိဿတိ. ဣတရေသံ ပဉ္စဝေါကာရာနံ တေသံ တတ္ထ ဝေဒနာက္ခန္ဓော စ နိရုဇ္ဈိဿတိ ရူပက္ခန္ဓော စ ဥပ္ပဇ္ဇိဿတိ.
</Paragraph>
