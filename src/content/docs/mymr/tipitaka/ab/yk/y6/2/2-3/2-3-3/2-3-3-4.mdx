<Paragraph number="611">
ကာမာဝစရာနံ ပစ္ဆိမစိတ္တဿ ဥပ္ပါဒက္ခဏေ ယဿ စိတ္တဿ အနန္တရာ ကာမာဝစရာနံ ပစ္ဆိမစိတ္တံ ဥပ္ပဇ္ဇိဿတိ ရူပါဝစရေ အရူပါဝစရေ ပစ္ဆိမဘဝိကာနံ ယေ စ ရူပါဝစရံ အရူပါဝစရံ ဥပပဇ္ဇိတွာ ပရိနိဗ္ဗာယိဿန္တိ တေသံ စဝန္တာနံ တေသံ ကာယသင်္ခါရော နုပ္ပဇ္ဇိဿတိ, နော စ တေသံ ဝစီသင်္ခါရော န နိရုဇ္ဈိဿတိ. သဝိတက္ကသဝိစာရပစ္ဆိမစိတ္တဿ ဘင်္ဂက္ခဏေ အဝိတက္ကအဝိစာရပစ္ဆိမစိတ္တသမင်္ဂီနံ ယဿ စိတ္တဿ အနန္တရာ အဝိတက္ကအဝိစာရံ ပစ္ဆိမစိတ္တံ ဥပ္ပဇ္ဇိဿတိ တေသံ ကာယသင်္ခါရော စ နုပ္ပဇ္ဇိဿတိ ဝစီသင်္ခါရော စ န နိရုဇ္ဈိဿတိ.
</Paragraph>
