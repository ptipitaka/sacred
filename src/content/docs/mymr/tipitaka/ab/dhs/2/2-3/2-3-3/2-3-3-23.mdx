<Paragraph number="2049" book="dhs">
တတ္ထ ကတမာ ဌာနကုသလတာ? ယေ ယေ ဓမ္မာ ယေသံ ယေသံ ဓမ္မာနံ ဟေတူ ပစ္စယာ ဥပ္ပါဒါယ တံ တံ ဌာနန္တိ, ယာ တတ္ထ ပညာ ပဇာနနာ …ပေ…  အမောဟော ဓမ္မဝိစယော သမ္မာဒိဋ္ဌိ—  အယံ ဝုစ္စတိ ဌာနကုသလတာ.
</Paragraph>

</Division>
<Division number="1345" book="dhs">

<Paragraph number="2050" book="dhs">
တတ္ထ ကတမာ အဋ္ဌာနကုသလတာ? ယေ ယေ ဓမ္မာ ယေသံ ယေသံ ဓမ္မာနံ န ဟေတူ န ပစ္စယာ ဥပ္ပါဒါယ တံ တံ အဋ္ဌာနန္တိ, ယာ တတ္ထ ပညာ ပဇာနနာ …ပေ…  အမောဟော ဓမ္မဝိစယော သမ္မာဒိဋ္ဌိ—  အယံ ဝုစ္စတိ အဋ္ဌာနကုသလတာ.
</Paragraph>
</Division>
