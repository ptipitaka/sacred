တတ္ထ ကတမာ ခန္တိ? ယာ ခန္တိ ခမနတာ အဓိဝါသနတာ အစဏ္ဍိက္ကံ အနသုရောပေါ အတ္တမနတာ စိတ္တဿ—  အယံ ဝုစ္စတိ ခန္တိ.

တတ္ထ ကတမံ သောရစ္စံ? ယော ကာယိကော အဝီတိက္ကမော, ဝါစသိကော အဝီတိက္ကမော, ကာယိကဝါစသိကော အဝီတိက္ကမော—  ဣဒံ ဝုစ္စတိ သောရစ္စံ. သဗ္ဗောပိ သီလသံဝရော သောရစ္စံ.
</Paragraph>
