<Paragraph number="1388" book="vi-maha">
ເຕນ ສມເຍນ ພຸທ຺ໂຘ ຠຄວາ ສາວຕ຺ຖິຍໍ ວິຫຣຕິ ເຊຕວເນ ອນາຖປິຓ຺ຑິກສ຺ສ ອາຣາເມ. ເຕນ ໂຂ ປນ ສມເຍນ ຕິໍສມຕ຺ຕາ ປາເວຍ຺ຍກາ ຠິກ຺ຂູ, ສພ຺ເພ ອາຣຎ຺ຎິກາ ສພ຺ເພ ປິຓ຺ຑປາຕິກາ ສພ຺ເພ ປໍສຸກູລິກາ ສພ຺ເພ ເຕຈີວຣິກາ ສາວຕ຺ຖິໍ ອາຄຈ຺ຉນ຺ຕາ ຠຄວນ຺ຕໍ ທສ຺ສນາຍ ອຸປກຏ຺ຐາຍ ວສ຺ສູປນາຍິກາຍ ນາສກ຺ຂິໍສຸ ສາວຕ຺ຖິຍໍ ວສ຺ສູປນາຍິກໍ ສມ຺ຠາເວຕຸໍ; ອນ຺ຕຣາມຄ຺ເຄ ສາເກເຕ ວສ຺ສໍ ອຸປຄຈ຺ຉິໍສຸ. ເຕ ອຸກ຺ກຓ຺ຐິຕຣູປາ ວສ຺ສໍ ວສິໍສຸ—  ອາສນ຺ເນວ ໂນ ຠຄວາ ວິຫຣຕິ ອິໂຕ ຉສຸ ໂຍຊເນສຸ, ນ ຈ ມຍໍ ລຠາມ ຠຄວນ຺ຕໍ ທສ຺ສນາຍາຕິ. ອຖ ໂຂ ເຕ ຠິກ຺ຂູ ວສ຺ສໍວຸຏ຺ຐາ, ເຕມາສຈ຺ຈເຍນ ກຕາຍ ປວາຣຓາຍ, ເທເວ ວສ຺ສນ຺ເຕ, ອຸທກສງ຺ຄເຫ ອຸທກຈິກ຺ຂລ຺ເລ ໂອກປຸຓ຺ເຓຫິ ຈີວເຣຫິ ກິລນ຺ຕຣູປາ ເຍນ ສາວຕ຺ຖິ ເຊຕວນໍ ອນາຖປິຓ຺ຑິກສ຺ສ ອາຣາໂມ, ເຍນ ຠຄວາ ເຕນຸປສງ຺ກມິໍສຸ; ອຸປສງ຺ກມິຕ຺ວາ ຠຄວນ຺ຕໍ ອຠິວາເທຕ຺ວາ ເອກມນ຺ຕໍ ນິສີທິໍສຸ. ອາຈິຓ຺ຓໍ ໂຂ ປເນຕໍ ພຸທ຺ຘານໍ ຠຄວນ຺ຕານໍ ອາຄນ຺ຕຸເກຫິ ຠິກ຺ຂູຫິ ສທ຺ຘິໍ ປຏິສມ຺ໂມທິຕຸໍ.
</Paragraph>

<Paragraph number="1389" book="vi-maha">
ອຖ ໂຂ ຠຄວາ ເຕ ຠິກ຺ຂູ ເອຕທໂວຈ—  “ກຈ຺ຈິ, ຠິກ຺ຂເວ, ຂມນີຍໍ, ກຈ຺ຈິ ຍາປນີຍໍ, ກຈ຺ຈິ ສມຄ຺ຄາ ສມ຺ໂມທມານາ ອວິວທມານາ ຜາສຸກໍ ວສ຺ສໍ ວສິຕ຺ຖ, ນ ຈ ປິຓ຺ຑເກນ ກິລມິຕ຺ຖາ”ຕິ? “ຂມນີຍໍ, ຠຄວາ, ຍາປນີຍໍ, ຠຄວາ; ສມຄ຺ຄາ ຈ ມຍໍ, ຠນ຺ເຕ, ສມ຺ໂມທມານາ ອວິວທມານາ ວສ຺ສໍ ວສິມ຺ຫາ, ນ ຈ ປິຓ຺ຑເກນ ກິລມິມ຺ຫາ. ອິຘ ມຍໍ, ຠນ຺ເຕ, ຕິໍສມຕ຺ຕາ ປາເວຍ຺ຍກາ ຠິກ຺ຂູ ສາວຕ຺ຖິໍ ອາຄຈ຺ຉນ຺ຕາ ຠຄວນ຺ຕໍ ທສ຺ສນາຍ ອຸປກຏ຺ຐາຍ ວສ຺ສູປນາຍິກາຍ ນາສກ຺ຂິມ຺ຫາ ສາວຕ຺ຖິຍໍ ວສ຺ສູປນາຍິກໍ ສມ຺ຠາເວຕຸໍ, ອນ຺ຕຣາມຄ຺ເຄ ສາເກເຕ ວສ຺ສໍ ອຸປຄຈ຺ຉິມ຺ຫາ. ເຕ ມຍໍ, ຠນ຺ເຕ, ອຸກ຺ກຓ຺ຐິຕຣູປາ ວສ຺ສໍ ວສິມ຺ຫາ—  ‘ອາສນ຺ເນວ ໂນ ຠຄວາ ວິຫຣຕິ ອິໂຕ ຉສຸ ໂຍຊເນສຸ, ນ ຈ ມຍໍ ລຠາມ ຠຄວນ຺ຕໍ ທສ຺ສນາຍາ’ຕິ. ອຖ ໂຂ ມຍໍ, ຠນ຺ເຕ, ວສ຺ສໍວຸຏ຺ຐາ, ເຕມາສຈ຺ຈເຍນ ກຕາຍ ປວາຣຓາຍ, ເທເວ ວສ຺ສນ຺ເຕ, ອຸທກສງ຺ຄເຫ ອຸທກຈິກ຺ຂລ຺ເລ ໂອກປຸຓ຺ເຓຫິ ຈີວເຣຫິ ກິລນ຺ຕຣູປາ ອທ຺ຘານໍ ອາຄຕາ”ຕິ.
</Paragraph>

<Paragraph number="1390" book="vi-maha">
ອຖ ໂຂ ຠຄວາ ເອຕສ຺ມິໍ ນິທາເນ ເອຕສ຺ມິໍ ປກຣເຓ ຘມ຺ມິໍ ກຖໍ ກຕ຺ວາ ຠິກ຺ຂູ ອາມນ຺ເຕສິ—  “ອນຸຊານາມິ, ຠິກ຺ຂເວ, ວສ຺ສໍວຸຏ຺ຐານໍ ຠິກ຺ຂູນໍ ກຖິນໍ ອຕ຺ຖຣິຕຸໍ. ອຕ຺ຖຕກຖິນານໍ ໂວ, ຠິກ຺ຂເວ, ປຎ຺ຈ ກປ຺ປິສ຺ສນ຺ຕິ—  ອນາມນ຺ຕຈາໂຣ, ອສມາທານຈາໂຣ, ຄຓໂຠຊນໍ, ຍາວທຕ຺ຖຈີວຣໍ, ໂຍ ຈ ຕຕ຺ຖ ຈີວຣຸປ຺ປາໂທ ໂສ ເນສໍ ຠວິສ຺ສຕີຕິ. ອຕ຺ຖຕກຖິນານໍ ໂວ, ຠິກ຺ຂເວ, ອິມານິ ປຎ຺ຈ ກປ຺ປິສ຺ສນ຺ຕິ. ເອວຎ຺ຈ ປນ, ຠິກ຺ຂເວ, ກຖິນໍ ອຕ຺ຖຣິຕພ຺ພໍ. ພ຺ຍຕ຺ເຕນ ຠິກ຺ຂຸນາ ປຏິພເລນ ສໍໂຆ ຎາເປຕພ຺ໂພ—
</Paragraph>

</Division>
<Division number="307" book="vi-maha">

<Paragraph number="1391" book="vi-maha">
‘ສຸຓາຕຸ ເມ, ຠນ຺ເຕ, ສໍໂຆ. ອິທໍ ສໍຆສ຺ສ ກຖິນທຸສ຺ສໍ ອຸປ຺ປນ຺ນໍ. ຍທິ ສໍຆສ຺ສ ປຕ຺ຕກລ຺ລໍ, ສໍໂຆ ອິມໍ ກຖິນທຸສ຺ສໍ ອິຕ຺ຖນ຺ນາມສ຺ສ ຠິກ຺ຂຸໂນ ທເທຍ຺ຍ ກຖິນໍ ອຕ຺ຖຣິຕຸໍ. ເອສາ ຎຕ຺ຕິ.
</Paragraph>

<Paragraph number="1392" book="vi-maha">
ສຸຓາຕຸ ເມ, ຠນ຺ເຕ, ສໍໂຆ. ອິທໍ ສໍຆສ຺ສ ກຖິນທຸສ຺ສໍ ອຸປ຺ປນ຺ນໍ. ສໍໂຆ ອິມໍ ກຖິນທຸສ຺ສໍ ອິຕ຺ຖນ຺ນາມສ຺ສ ຠິກ຺ຂຸໂນ ເທຕິ ກຖິນໍ ອຕ຺ຖຣິຕຸໍ. ຍສ຺ສາຍສ຺ມໂຕ ຂມຕິ ອິມສ຺ສ ກຖິນທຸສ຺ສສ຺ສ ອິຕ຺ຖນ຺ນາມສ຺ສ ຠິກ຺ຂຸໂນ ທານໍ ກຖິນໍ ອຕ຺ຖຣິຕຸໍ, ໂສ ຕຸຓ຺ຫສ຺ສ; ຍສ຺ສ ນກ຺ຂມຕິ, ໂສ ຠາເສຍ຺ຍ.
</Paragraph>

<Paragraph number="1393" book="vi-maha">
ທິນ຺ນໍ ອິທໍ ສໍເຆນ ກຖິນທຸສ຺ສໍ ອິຕ຺ຖນ຺ນາມສ຺ສ ຠິກ຺ຂຸໂນ ກຖິນໍ ອຕ຺ຖຣິຕຸໍ. ຂມຕິ ສໍຆສ຺ສ, ຕສ຺ມາ ຕຸຓ຺ຫີ, ເອວເມຕໍ ຘາຣຍາມີ’ຕິ.
</Paragraph>

</Division>
<Division number="308" book="vi-maha">

<Paragraph number="1394" book="vi-maha">
ເອວໍ ໂຂ, ຠິກ຺ຂເວ, ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ເອວໍ ອນຕ຺ຖຕໍ. ກຖຎ຺ຈ ປນ, ຠິກ຺ຂເວ, ອນຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ? ນ ອຸລ຺ລິຂິຕມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ໂຘວນມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ຈີວຣວິຈາຣຓມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ເຉທນມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ພນ຺ຘນມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ໂອວຏ຺ຏິຍກຣຓມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ກຓ຺ຑຸສກຣຓມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ທຬ຺ຫີກມ຺ມກຣຓມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ອນຸວາຕກຣຓມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ປຣິຠຓ຺ຑກຣຓມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ໂອວທ຺ເຘຍ຺ຍກຣຓມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ກມ຺ພລມທ຺ທນມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ນິມິຕ຺ຕກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ປຣິກຖາກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ກຸກ຺ກຸກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ສນ຺ນິຘິກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ນິສ຺ສຄ຺ຄິເຍນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ອກປ຺ປກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ອຎ຺ຎຕ຺ຣ ສງ຺ຆາຏິຍາ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ອຎ຺ຎຕ຺ຣ ອຸຕ຺ຕຣາສງ຺ເຄນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ອຎ຺ຎຕ຺ຣ ອນ຺ຕຣວາສເກນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ອຎ຺ຎຕ຺ຣ ປຎ຺ຈເກນ ວາ ອຕິເຣກປຎ຺ຈເກນ ວາ ຕທເຫວ ສຎ຺ຉິນ຺ເນນ ສມຓ຺ຑລີກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ອຎ຺ຎຕ຺ຣ ປຸຄ຺ຄລສ຺ສ ອຕ຺ຖາຣາ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ; ສມ຺ມາ ເຈ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ຕຎ຺ເຈ ນິສ຺ສີມຏ຺ໂຐ ອນຸໂມທຕິ, ເອວມ຺ປິ ອນຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ. ເອວໍ ໂຂ, ຠິກ຺ຂເວ, ອນຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ.
</Paragraph>

</Division>

ກຖຎ຺ຈ, ຠິກ຺ຂເວ, ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ? ອຫເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ອຫຕກປ຺ເປນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ປິໂລຕິກາຍ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ປໍສຸກູເລນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ປາປຓິເກນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ອນິມິຕ຺ຕກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ອປຣິກຖາກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ອກຸກ຺ກຸກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ອສນ຺ນິຘິກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ອນິສ຺ສຄ຺ຄິເຍນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ກປ຺ປກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ສງ຺ຆາຏິຍາ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ອຸຕ຺ຕຣາສງ຺ເຄນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ອນ຺ຕຣວາສເກນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ປຎ຺ຈເກນ ວາ ອຕິເຣກປຎ຺ຈເກນ ວາ ຕທເຫວ ສຎ຺ຉິນ຺ເນນ ສມຓ຺ຑລີກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ປຸຄ຺ຄລສ຺ສ ອຕ຺ຖາຣາ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ; ສມ຺ມາ ເຈ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ຕຎ຺ເຈ ສີມຏ຺ໂຐ ອນຸໂມທຕິ, ເອວມ຺ປິ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ. ເອວໍ ໂຂ, ຠິກ຺ຂເວ, ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ.
</Paragraph>
