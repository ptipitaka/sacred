ອິຘ ປນ, ຠິກ຺ຂເວ, ຠິກ຺ຂຸ ສໍເຆນ ປາປິກາຍ ທິຏ຺ຐິຍາ ອປ຺ປຏິນິສ຺ສຄ຺ເຄ ອຸກ຺ເຂປນີຍກມ຺ມກໂຕ ສມ຺ມາ ວຕ຺ຕຕິ, ໂລມໍ ປາເຕຕິ, ເນຕ຺ຖາຣໍ ວຕ຺ຕຕິ, ປາປິກາຍ ທິຏ຺ຐິຍາ ອປ຺ປຏິນິສ຺ສຄ຺ເຄ ອຸກ຺ເຂປນີຍສ຺ສ ກມ຺ມສ຺ສ ປຏິປ຺ປສ຺ສທ຺ຘິໍ ຍາຈຕິ. ຕຕ຺ຣ ເຈ ຠິກ຺ຂູນໍ ເອວໍ ໂຫຕິ—  ‘ອຍໍ ໂຂ, ອາວຸໂສ, ຠິກ຺ຂຸ ສໍເຆນ ປາປິກາຍ ທິຏ຺ຐິຍາ ອປ຺ປຏິນິສ຺ສຄ຺ເຄ ອຸກ຺ເຂປນີຍກມ຺ມກໂຕ ສມ຺ມາ ວຕ຺ຕຕິ, ໂລມໍ ປາເຕຕິ, ເນຕ຺ຖາຣໍ ວຕ຺ຕຕິ, ປາປິກາຍ ທິຏ຺ຐິຍາ ອປ຺ປຏິນິສ຺ສຄ຺ເຄ ອຸກ຺ເຂປນີຍສ຺ສ ກມ຺ມສ຺ສ ປຏິປ຺ປສ຺ສທ຺ຘິໍ ຍາຈຕິ. ຫນ຺ທສ຺ສ ມຍໍ ປາປິກາຍ ທິຏ຺ຐິຍາ ອປ຺ປຏິນິສ຺ສຄ຺ເຄ ອຸກ຺ເຂປນີຍກມ຺ມໍ ປຏິປ຺ປສ຺ສມ຺ເຠມາ’ຕິ. ເຕ ຕສ຺ສ ປາປິກາຍ ທິຏ຺ຐິຍາ ອປ຺ປຏິນິສ຺ສຄ຺ເຄ ອຸກ຺ເຂປນີຍກມ຺ມໍ ປຏິປ຺ປສ຺ສມ຺ເຠນ຺ຕິ—  ອຘມ຺ເມນ ວຄ຺ຄາ. ໂສ ຕມ຺ຫາ ອາວາສາ ອຎ຺ຎໍ ອາວາສໍ ຄຈ຺ຉຕິ. ຕຕ຺ຖປິ ຠິກ຺ຂູນໍ ເອວໍ ໂຫຕິ—  ‘ອິມສ຺ສ ໂຂ, ອາວຸໂສ, ຠິກ຺ຂຸໂນ ສໍເຆນ ປາປິກາຍ ທິຏ຺ຐິຍາ ອປ຺ປຏິນິສ຺ສຄ຺ເຄ ອຸກ຺ເຂປນີຍກມ຺ມໍ ປຏິປ຺ປສ຺ສທ຺ຘໍ—  ອຘມ຺ເມນ ວຄ຺ເຄຫິ. ຫນ຺ທສ຺ສ ມຍໍ ປາປິກາຍ ທິຏ຺ຐິຍາ ອປ຺ປຏິນິສ຺ສຄ຺ເຄ ອຸກ຺ເຂປນີຍກມ຺ມໍ ປຏິປ຺ປສ຺ສມ຺ເຠມາ’ຕິ. ເຕ ຕສ຺ສ ປາປິກາຍ ທິຏ຺ຐິຍາ ອປ຺ປຏິນິສ຺ສຄ຺ເຄ ອຸກ຺ເຂປນີຍກມ຺ມໍ ປຏິປ຺ປສ຺ສມ຺ເຠນ຺ຕິ…  ອຘມ຺ເມນ ສມຄ຺ຄາ …ເປ…  ຘມ຺ເມນ ວຄ຺ຄາ…  ຘມ຺ມປຕິຣູປເກນ ວຄ຺ຄາ…  ຘມ຺ມປຕິຣູປເກນ ສມຄ຺ຄາ …ເປ… .
