ເຕນ ໂຂ ປນ ສມເຍນ ອຎ຺ຎຕໂຣ ມາຓວໂກ ປິຕຣໍ ຊີວິຕາ ໂວໂຣເປສິ. ໂສ ເຕນ ປາປເກນ ກມ຺ເມນ ອຏ຺ຏີຍຕິ ຫຣາຍຕິ ຊິຄຸຈ຺ຉຕິ. ອຖ ໂຂ ຕສ຺ສ ມາຓວກສ຺ສ ເອຕທໂຫສິ—  “ເກນ ນຸ ໂຂ ອຫໍ ອຸປາເຍນ ອິມສ຺ສ ປາປກສ຺ສ ກມ຺ມສ຺ສ ນິກ຺ຂນ຺ຕິໍ ກເຣຍ຺ຍນ຺”ຕິ. ອຖ ໂຂ ຕສ຺ສ ມາຓວກສ຺ສ ເອຕທໂຫສິ—  “ອິເມ ໂຂ ສມຓາ ສກ຺ຍປຸຕ຺ຕິຍາ ຘມ຺ມຈາຣິໂນ ສມຈາຣິໂນ ພ຺ຣຫ຺ມຈາຣິໂນ ສຈ຺ຈວາທິໂນ ສີລວນ຺ໂຕ ກລ຺ຍາຓຘມ຺ມາ, ສເຈ ໂຂ ອຫໍ ສມເຓສຸ ສກ຺ຍປຸຕ຺ຕິເຍສຸ ປພ຺ພເຊຍ຺ຍໍ, ເອວາຫໍ ອິມສ຺ສ ປາປກສ຺ສ ກມ຺ມສ຺ສ ນິກ຺ຂນ຺ຕິໍ ກເຣຍ຺ຍນ຺”ຕິ. ອຖ ໂຂ ໂສ ມາຓວໂກ ຠິກ຺ຂູ ອຸປສງ຺ກມິຕ຺ວາ ປພ຺ພຊ຺ຊໍ ຍາຈິ. ຠິກ຺ຂູ ອາຍສ຺ມນ຺ຕໍ ອຸປາລິໍ ເອຕທໂວຈຸໍ—  “ປຸພ຺ເພປິ ໂຂ, ອາວຸໂສ ອຸປາລິ, ນາໂຄ ມາຓວກວຓ຺ເຓນ ຠິກ຺ຂູສຸ ປພ຺ພຊິໂຕ. ອິງ຺ຆາວຸໂສ, ອຸປາລິ, ອິມໍ ມາຓວກໍ ອນຸຍຸຎ຺ຊາຫີ”ຕິ. ອຖ ໂຂ ໂສ ມາຓວໂກ ອາຍສ຺ມຕາ ອຸປາລິນາ ອນຸຍຸຎ຺ຊີຍມາໂນ ເອຕມຕ຺ຖໍ ອາໂຣເຈສິ. ອາຍສ຺ມາ ອຸປາລິ ຠິກ຺ຂູນໍ ເອຕມຕ຺ຖໍ ອາໂຣເຈສິ. ຠິກ຺ຂູ ຠຄວໂຕ ເອຕມຕ຺ຖໍ ອາໂຣເຈສຸໍ. “ປິຕຸຆາຕໂກ, ຠິກ຺ຂເວ, ອນຸປສມ຺ປນ຺ໂນ ນ ອຸປສມ຺ປາເທຕພ຺ໂພ, ອຸປສມ຺ປນ຺ໂນ ນາເສຕພ຺ໂພ”ຕິ.
