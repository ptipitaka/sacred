<Paragraph number="916" book="vi-maha">
ເຕນ ໂຂ ປນ ສມເຍນ ສາວຕ຺ຖິຍາ ສໍເຆນ ເອວຣູປາ ກຕິກາ ກຕາ ໂຫຕິ—  “ອນ຺ຕຣາວສ຺ສໍ ນ ປພ຺ພາເຊຕພ຺ພນ຺”ຕິ. ວິສາຂາຍ ມິຄາຣມາຕຸຍາ ນຕ຺ຕາ ຠິກ຺ຂູ ອຸປສງ຺ກມິຕ຺ວາ ປພ຺ພຊ຺ຊໍ ຍາຈິ. ຠິກ຺ຂູ ເອວມາຫໍສຸ—  “ສໍເຆນ ໂຂ, ອາວຸໂສ, ເອວຣູປາ ກຕິກາ ກຕາ ‘ອນ຺ຕຣາວສ຺ສໍ ນ ປພ຺ພາເຊຕພ຺ພນ຺’ຕິ. ອາຄເມຫິ, ອາວຸໂສ, ຍາວ ຠິກ຺ຂູ ວສ຺ສໍ ວສນ຺ຕິ. ວສ຺ສໍວຸຏ຺ຐາ ປພ຺ພາເຊສ຺ສນ຺ຕີ”ຕິ. ອຖ ໂຂ ເຕ ຠິກ຺ຂູ ວສ຺ສໍວຸຏ຺ຐາ ວິສາຂາຍ ມິຄາຣມາຕຸຍາ ນຕ຺ຕາຣໍ ເອຕທໂວຈຸໍ—  “ເອຫິ ທານິ, ອາວຸໂສ, ປພ຺ພຊາຫີ”ຕິ. ໂສ ເອວມາຫ—  “ສຈາຫໍ, ຠນ຺ເຕ, ປພ຺ພຊິໂຕ ອສ຺ສໍ, ອຠິຣເມຍ຺ຍາມຫໍ. ນ ທານາຫໍ, ຠນ຺ເຕ, ປພ຺ພຊິສ຺ສາມີ”ຕິ. ວິສາຂາ ມິຄາຣມາຕາ ອຸຊ຺ຌາຍຕິ ຂິຍ຺ຍຕິ ວິປາເຈຕິ—  “ກຖຎ຺ຫິ ນາມ ອຍ຺ຍາ ເອວຣູປໍ ກຕິກໍ ກຣິສ຺ສນ຺ຕິ—  ‘ນ ອນ຺ຕຣາວສ຺ສໍ ປພ຺ພາເຊຕພ຺ພນ຺’ຕິ. ກໍ ກາລໍ ຘມ຺ໂມ ນ ຈຣິຕພ຺ໂພ”ຕິ?
</Paragraph>

<Paragraph number="917" book="vi-maha">
ອສ຺ໂສສຸໍ ໂຂ ຠິກ຺ຂູ ວິສາຂາຍ ມິຄາຣມາຕຸຍາ ອຸຊ຺ຌາຍນ຺ຕິຍາ ຂິຍ຺ຍນ຺ຕິຍາ ວິປາເຈນ຺ຕິຍາ. ອຖ ໂຂ ເຕ ຠິກ຺ຂູ ຠຄວໂຕ ເອຕມຕ຺ຖໍ ອາໂຣເຈສຸໍ. “ນ, ຠິກ຺ຂເວ, ເອວຣູປາ ກຕິກາ ກາຕພ຺ພາ—  ‘ນ ອນ຺ຕຣາວສ຺ສໍ ປພ຺ພາເຊຕພ຺ພນ຺’ຕິ. ໂຍ ກເຣຍ຺ຍ, ອາປຕ຺ຕິ ທຸກ຺ກຏສ຺ສາ”ຕິ.
</Paragraph>
