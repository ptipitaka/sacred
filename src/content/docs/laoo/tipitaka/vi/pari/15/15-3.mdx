<Emphasis>ກຖໍ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ</Emphasis>? ອຘມ຺ມໍ ອຘມ຺ໂມຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ, ຘມ຺ມໍ ຘມ຺ໂມຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ, ອວິນຍໍ ອວິນໂຍຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ, ວິນຍໍ ວິນໂຍຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ, ອຠາສິຕໍ ອລປິຕໍ ຕຖາຄເຕນ ອຠາສິຕໍ ອລປິຕໍ ຕຖາຄເຕນາຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ, ຠາສິຕໍ ລປິຕໍ ຕຖາຄເຕນ ຠາສິຕໍ ລປິຕໍ ຕຖາຄເຕນາຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ, ອນາຈິຓ຺ຓໍ ຕຖາຄເຕນ ອນາຈິຓ຺ຓໍ ຕຖາຄເຕນາຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ, ອາຈິຓ຺ຓໍ ຕຖາຄເຕນ ອາຈິຓ຺ຓໍ ຕຖາຄເຕນາຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ, ອປຎ຺ຎຕ຺ຕໍ ຕຖາຄເຕນ ອປຎ຺ຎຕ຺ຕໍ ຕຖາຄເຕນາຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ, ປຎ຺ຎຕ຺ຕໍ ຕຖາຄເຕນ ປຎ຺ຎຕ຺ຕໍ ຕຖາຄເຕນາຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ, ອນາປຕ຺ຕິໍ ອນາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ, ອາປຕ຺ຕິໍ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ, ລຫຸກໍ ອາປຕ຺ຕິໍ ລຫຸກາ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ, ຄຣຸກໍ ອາປຕ຺ຕິໍ ຄຣຸກາ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ, ສາວເສສໍ ອາປຕ຺ຕິໍ ສາວເສສາ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ, ອນວເສສໍ ອາປຕ຺ຕິໍ ອນວເສສາ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ, ທຸຏ຺ຐຸລ຺ລໍ ອາປຕ຺ຕິໍ ທຸຏ຺ຐຸລ຺ລາ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ, ອທຸຏ຺ຐຸລ຺ລໍ ອາປຕ຺ຕິໍ ອທຸຏ຺ຐຸລ຺ລາ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ. ເອວໍ ນ ຉນ຺ທາຄຕິໍ ຄຈ຺ຉຕິ.
</Paragraph>

<Paragraph number="1853" book="pari">
<Emphasis>ກຖໍ ນ ໂທສາຄຕິໍ ຄຈ຺ຉຕິ</Emphasis>? ອຘມ຺ມໍ ອຘມ຺ໂມຕິ ທີເປນ຺ໂຕ ນ ໂທສາຄຕິໍ ຄຈ຺ຉຕິ, ຘມ຺ມໍ ຘມ຺ໂມຕິ ທີເປນ຺ໂຕ ນ ໂທສາຄຕິໍ ຄຈ຺ຉຕິ …ເປ…  ທຸຏ຺ຐຸລ຺ລໍ ອາປຕ຺ຕິໍ ທຸຏ຺ຐຸລ຺ລາ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ໂທສາຄຕິໍ ຄຈ຺ຉຕິ, ອທຸຏ຺ຐຸລ຺ລໍ ອາປຕ຺ຕິໍ ອທຸຏ຺ຐຸລ຺ລາ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ໂທສາຄຕິໍ ຄຈ຺ຉຕິ. ເອວໍ ນ ໂທສາຄຕິໍ ຄຈ຺ຉຕິ.
</Paragraph>

</Division>
<Division number="385" book="pari">

<Paragraph number="1854" book="pari">
<Emphasis>ກຖໍ ນ ໂມຫາຄຕິໍ ຄຈ຺ຉຕິ</Emphasis>? ອຘມ຺ມໍ ອຘມ຺ໂມຕິ ທີເປນ຺ໂຕ ນ ໂມຫາຄຕິໍ ຄຈ຺ຉຕິ, ຘມ຺ມໍ ຘມ຺ໂມຕິ ທີເປນ຺ໂຕ ນ ໂມຫາຄຕິໍ ຄຈ຺ຉຕິ …ເປ…  ທຸຏ຺ຐຸລ຺ລໍ ອາປຕ຺ຕິໍ ທຸຏ຺ຐຸລ຺ລາ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ໂມຫາຄຕິໍ ຄຈ຺ຉຕິ, ອທຸຏ຺ຐຸລ຺ລໍ ອາປຕ຺ຕິໍ ອທຸຏ຺ຐຸລ຺ລາ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ໂມຫາຄຕິໍ ຄຈ຺ຉຕິ. ເອວໍ ນ ໂມຫາຄຕິໍ ຄຈ຺ຉຕິ.
</Paragraph>

</Division>
<Division number="386" book="pari">

<Paragraph number="1855" book="pari">
<Emphasis>ກຖໍ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ</Emphasis>? ອຘມ຺ມໍ ອຘມ຺ໂມຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ, ຘມ຺ມໍ ຘມ຺ໂມຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ, ອວິນຍໍ ອວິນໂຍຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ, ວິນຍໍ ວິນໂຍຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ, ອຠາສິຕໍ ອລປິຕໍ ຕຖາຄເຕນ ອຠາສິຕໍ ອລປິຕໍ ຕຖາຄເຕນາຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ, ຠາສິຕໍ ລປິຕໍ ຕຖາຄເຕນ ຠາສິຕໍ ລປິຕໍ ຕຖາຄເຕນາຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ, ອນາຈິຓ຺ຓໍ ຕຖາຄເຕນ ອນາຈິຓ຺ຓໍ ຕຖາຄເຕນາຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ, ອາຈິຓ຺ຓໍ ຕຖາຄເຕນ ອາຈິຓ຺ຓໍ ຕຖາຄເຕນາຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ, ອປຎ຺ຎຕ຺ຕໍ ຕຖາຄເຕນ ອປຎ຺ຎຕ຺ຕໍ ຕຖາຄເຕນາຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ, ປຎ຺ຎຕ຺ຕໍ ຕຖາຄເຕນ ປຎ຺ຎຕ຺ຕໍ ຕຖາຄເຕນາຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ, ອນາປຕ຺ຕິໍ ອນາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ, ອາປຕ຺ຕິໍ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ, ລຫຸກໍ ອາປຕ຺ຕິໍ ລຫຸກາ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ, ຄຣຸກໍ ອາປຕ຺ຕິໍ ຄຣຸກາ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ, ສາວເສສໍ ອາປຕ຺ຕິໍ ສາວເສສາ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ, ອນວເສສໍ ອາປຕ຺ຕິໍ ອນວເສສາ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ, ທຸຏ຺ຐຸລ຺ລໍ ອາປຕ຺ຕິໍ ທຸຏ຺ຐຸລ຺ລາ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ, ອທຸຏ຺ຐຸລ຺ລໍ ອາປຕ຺ຕິໍ ອທຸຏ຺ຐຸລ຺ລາ ອາປຕ຺ຕີຕິ ທີເປນ຺ໂຕ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ. ເອວໍ ນ ຠຍາຄຕິໍ ຄຈ຺ຉຕິ.
</Paragraph>
