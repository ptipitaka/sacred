ກສ຺ສ ກຖິນໍ ອນຕ຺ຖຕໍ? ກສ຺ສ ກຖິນໍ ອຕ຺ຖຕໍ? ກິນ຺ຕິ ກຖິນໍ ອນຕ຺ຖຕໍ? ກິນ຺ຕິ ກຖິນໍ ອຕ຺ຖຕໍ?
</Paragraph>

<Paragraph number="1881" book="pari">
<Emphasis>ກສ຺ສ ກຖິນໍ ອນຕ຺ຖຕນ຺</Emphasis>ຕິ? ທ຺ວິນ຺ນໍ ປຸຄ຺ຄລານໍ ອນຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ—  ອນຕ຺ຖາຣກສ຺ສ ຈ ອນນຸໂມທກສ຺ສ ຈ. ອິເມສໍ ທ຺ວິນ຺ນໍ ປຸຄ຺ຄລານໍ ອນຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ.
</Paragraph>

<Paragraph number="1882" book="pari">
<Emphasis>ກສ຺ສ ກຖິນໍ ອຕ຺ຖຕນ຺</Emphasis>ຕິ? ທ຺ວິນ຺ນໍ ປຸຄ຺ຄລານໍ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ—  ອຕ຺ຖາຣກສ຺ສ ຈ ອນຸໂມທກສ຺ສ ຈ. ອິເມສໍ ທ຺ວິນ຺ນໍ ປຸຄ຺ຄລານໍ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ.
</Paragraph>

<Paragraph number="1883" book="pari">
<Emphasis>ກິນ຺ຕິ ກຖິນໍ ອນຕ຺ຖຕນ຺</Emphasis>ຕິ? ຈຕຸວີສຕິຍາ ອາກາເຣຫິ ອນຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ອຸລ຺ລິຂິຕມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ໂຘວນມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ຈີວຣວິຈາຣຓມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ເຉທນມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ພນ຺ຘນມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ໂອວຏ຺ຏິຍກຣຓມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ກຓ຺ຑຸສກຣຓມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ທຬ຺ຫີກມ຺ມກຣຓມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ອນຸວາຕກຣຓມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ປຣິຠຓ຺ຑກຣຓມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ໂອວທ຺ເຘຍ຺ຍກຣຓມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ກມ຺ພລມທ຺ທນມຕ຺ເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ນິມິຕ຺ຕກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ປຣິກຖາກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ກຸກ຺ກຸກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ສນ຺ນິຘິກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ນິສ຺ສຄ຺ຄິເຍນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ອກປ຺ປກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ອຎ຺ຎຕ຺ຣ ສງ຺ຆາຏິຍາ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ອຎ຺ຎຕ຺ຣ ອຸຕ຺ຕຣາສງ຺ເຄນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ອຎ຺ຎຕ຺ຣ ອນ຺ຕຣວາສເກນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ອຎ຺ຎຕ຺ຣ ປຎ຺ຈເກນ ວາ ອຕິເຣກປຎ຺ຈເກນ ວາ ຕທເຫວ ສຎ຺ຉິນ຺ເນນ ສມຓ຺ຑລີກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ນ ອຎ຺ຎຕ຺ຣ ປຸຄ຺ຄລສ຺ສ ອຕ຺ຖາຣາ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ສມ຺ມາ ເຈ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ຕຎ຺ເຈ ນິສ຺ສີມຏ຺ໂຐ ອນຸໂມທຕິ. ເອວມ຺ປິ ອນຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ.
</Paragraph>

<Paragraph number="1884" book="pari">
ນິມິຕ຺ຕກມ຺ມໍ ນາມ ນິມິຕ຺ຕໍ ກໂຣຕິ—  “ອິມິນາ ທຸສ຺ເສນ ກຖິນໍ ອຕ຺ຖຣິສ຺ສາມີ”ຕິ. ປຣິກຖາ ນາມ ປຣິກຖໍ ກໂຣຕິ—  “ອິມາຍ ປຣິກຖາຍ ກຖິນທຸສ຺ສໍ ນິພ຺ພຕ຺ເຕສ຺ສາມີ”ຕິ. ກຸກ຺ກຸກຕໍ ນາມ ອນາທິຍທານໍ ວຸຈ຺ຈຕິ. ສນ຺ນິຘິ ນາມ ທ຺ເວ ສນ຺ນິຘິໂຍ—  ກຣຓສນ຺ນິຘິ ວາ ນິຈຍສນ຺ນິຘິ ວາ. ນິສ຺ສຄ຺ຄິຍໍ ນາມ ກຍິຣມາເນ ອຣຸຓໍ ອຸຏ຺ຐຫຕິ. ອິເມຫິ ຈຕຸວີສຕິຍາ ອາກາເຣຫິ ອນຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ.
</Paragraph>

<Paragraph number="1885" book="pari">
<Emphasis>ກິນ຺ຕິ ກຖິນໍ ອຕ຺ຖຕນ຺</Emphasis>ຕິ? ສຕ຺ຕຣສຫິ ອາກາເຣຫິ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ. ອຫເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ອຫຕກປ຺ເປນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ປິໂລຕິກາຍ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ປໍສຸກູເລນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ປາປຓິເກນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ອນິມິຕ຺ຕກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ອປຣິກຖາກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ອກຸກ຺ກຸກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ອສນ຺ນິຘິກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ອນິສ຺ສຄ຺ຄິເຍນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ກປ຺ປກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ສງ຺ຆາຏິຍາ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ອຸຕ຺ຕຣາສງ຺ເຄນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ອນ຺ຕຣວາສເກນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ປຎ຺ຈເກນ ວາ ອຕິເຣກປຎ຺ຈເກນ ວາ ຕທເຫວ ສຎ຺ຉິນ຺ເນນ ສມຓ຺ຑລີກເຕນ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ປຸຄ຺ຄລສ຺ສ ອຕ຺ຖາຣາ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ສມ຺ມາ ເຈ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ, ຕຎ຺ເຈ ສີມຏ຺ໂຐ ອນຸໂມທຕິ, ເອວມ຺ປິ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ. ອິເມຫິ ສຕ຺ຕຣສຫິ ອາກາເຣຫິ ອຕ຺ຖຕໍ ໂຫຕິ ກຖິນໍ.
</Paragraph>

<Paragraph number="1886" book="pari">
ສຫ ກຖິນສ຺ສ ອຕ຺ຖາຣາ ກຕິ ຘມ຺ມາ ຊາຍນ຺ຕິ? ສຫ ກຖິນສ຺ສ ອຕ຺ຖາຣາ ປນ຺ນຣສ ຘມ຺ມາ ຊາຍນ຺ຕິ—  ອຏ຺ຐ ມາຕິກາ, ທ຺ເວ ປລິໂພຘາ, ປຎ຺ຈານິສໍສາ. ສຫ ກຖິນສ຺ສ ອຕ຺ຖາຣາ ອິເມ ປນ຺ນຣສ ຘມ຺ມາ ຊາຍນ຺ຕິ.
</Paragraph>
