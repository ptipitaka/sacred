<Paragraph number="213" book="cula">
ເຕນ ສມເຍນ ພຸທ຺ໂຘ ຠຄວາ ໂກສມ຺ພິຍໍ ວິຫຣຕິ ໂຆສິຕາຣາເມ. ເຕນ ໂຂ ປນ ສມເຍນ ອາຍສ຺ມາ ຉນ຺ໂນ ອາປຕ຺ຕິໍ ອາປຊ຺ຊິຕ຺ວາ ນ ອິຈ຺ຉຕິ ອາປຕ຺ຕິໍ ປສ຺ສິຕຸໍ. ເຍ ເຕ ຠິກ຺ຂູ ອປ຺ປິຈ຺ຉາ …ເປ…  ເຕ ອຸຊ຺ຌາຍນ຺ຕິ ຂິຍ຺ຍນ຺ຕິ ວິປາເຈນ຺ຕິ—  “ກຖຎ຺ຫິ ນາມ ອາຍສ຺ມາ ຉນ຺ໂນ ອາປຕ຺ຕິໍ ອາປຊ຺ຊິຕ຺ວາ ນ ອິຈ຺ຉິສ຺ສຕິ ອາປຕ຺ຕິໍ ປສ຺ສິຕຸນ຺”ຕິ. ອຖ ໂຂ ເຕ ຠິກ຺ຂູ ຠຄວໂຕ ເອຕມຕ຺ຖໍ ອາໂຣເຈສຸໍ.


</Paragraph>

<Paragraph number="214" book="cula">
ອຖ ໂຂ ຠຄວາ ເອຕສ຺ມິໍ ນິທາເນ ເອຕສ຺ມິໍ ປກຣເຓ ຠິກ຺ຂຸສໍຆໍ ສນ຺ນິປາຕາເປຕ຺ວາ ຠິກ຺ຂູ ປຏິປຸຈ຺ຉິ—  “ສຈ຺ຈໍ ກິຣ, ຠິກ຺ຂເວ, ຉນ຺ໂນ ຠິກ຺ຂຸ ອາປຕ຺ຕິໍ ອາປຊ຺ຊິຕ຺ວາ ນ ອິຈ຺ຉຕິ ອາປຕ຺ຕິໍ ປສ຺ສິຕຸນ຺”ຕິ? “ສຈ຺ຈໍ, ຠຄວາ”ຕິ. ວິຄຣຫິ ພຸທ຺ໂຘ ຠຄວາ—  “ອນນຸຈ຺ຉວິກໍ …ເປ…  ກຖຎ຺ຫິ ນາມ ໂສ, ຠິກ຺ຂເວ, ໂມຆປຸຣິໂສ ອາປຕ຺ຕິໍ ອາປຊ຺ຊິຕ຺ວາ ນ ອິຈ຺ຉິສ຺ສຕິ ອາປຕ຺ຕິໍ ປສ຺ສິຕຸໍ? ເນຕໍ, ຠິກ຺ຂເວ, ອປ຺ປສນ຺ນານໍ ວາ ປສາທາຍ …ເປ…  ວິຄຣຫິຕ຺ວາ …ເປ…  ຘມ຺ມິໍ ກຖໍ ກຕ຺ວາ ຠິກ຺ຂູ ອາມນ຺ເຕສິ—  “ເຕນ ຫິ, ຠິກ຺ຂເວ, ສໍໂຆ ຉນ຺ນສ຺ສ ຠິກ຺ຂຸໂນ, ອາປຕ຺ຕິຍາ ອທສ຺ສເນ, ອຸກ຺ເຂປນີຍກມ຺ມໍ ກໂຣຕຸ—  ອສມ຺ໂຠຄໍ ສໍເຆນ. ເອວຎ຺ຈ ປນ, ຠິກ຺ຂເວ, ກາຕພ຺ພໍ—  ປຐມໍ ຉນ຺ໂນ ຠິກ຺ຂຸ ໂຈເທຕພ຺ໂພ, ໂຈເທຕ຺ວາ ສາເຣຕພ຺ໂພ, ສາເຣຕ຺ວາ ອາປຕ຺ຕິໍ ອາໂຣເປຕພ຺ໂພ, ອາປຕ຺ຕິໍ ອາໂຣເປຕ຺ວາ ພ຺ຍຕ຺ເຕນ ຠິກ຺ຂຸນາ ປຏິພເລນ ສໍໂຆ ຎາເປຕພ຺ໂພ—


</Paragraph>

</Division>
<Division number="47" book="cula">

<Paragraph number="215" book="cula">
‘ສຸຓາຕຸ ເມ, ຠນ຺ເຕ, ສໍໂຆ. ອຍໍ ຉນ຺ໂນ ຠິກ຺ຂຸ ອາປຕ຺ຕິໍ ອາປຊ຺ຊິຕ຺ວາ ນ ອິຈ຺ຉຕິ ອາປຕ຺ຕິໍ ປສ຺ສິຕຸໍ. ຍທິ ສໍຆສ຺ສ ປຕ຺ຕກລ຺ລໍ, ສໍໂຆ ຉນ຺ນສ຺ສ ຠິກ຺ຂຸໂນ, ອາປຕ຺ຕິຍາ ອທສ຺ສເນ, ອຸກ຺ເຂປນີຍກມ຺ມໍ ກເຣຍ຺ຍ—  ອສມ຺ໂຠຄໍ ສໍເຆນ. ເອສາ ຎຕ຺ຕິ.


</Paragraph>

<Paragraph number="216" book="cula">
ສຸຓາຕຸ ເມ, ຠນ຺ເຕ, ສໍໂຆ. ອຍໍ ຉນ຺ໂນ ຠິກ຺ຂຸ ອາປຕ຺ຕິໍ ອາປຊ຺ຊິຕ຺ວາ ນ ອິຈ຺ຉຕິ ອາປຕ຺ຕິໍ ປສ຺ສິຕຸໍ. ສໍໂຆ ຉນ຺ນສ຺ສ ຠິກ຺ຂຸໂນ, ອາປຕ຺ຕິຍາ ອທສ຺ສເນ, ອຸກ຺ເຂປນີຍກມ຺ມໍ ກໂຣຕິ—  ອສມ຺ໂຠຄໍ ສໍເຆນ. ຍສ຺ສາຍສ຺ມໂຕ ຂມຕິ ຉນ຺ນສ຺ສ ຠິກ຺ຂຸໂນ, ອາປຕ຺ຕິຍາ ອທສ຺ສເນ, ອຸກ຺ເຂປນີຍສ຺ສ ກມ຺ມສ຺ສ ກຣຓໍ—  ອສມ຺ໂຠຄໍ ສໍເຆນ, ໂສ ຕຸຓ຺ຫສ຺ສ; ຍສ຺ສ ນກ຺ຂມຕິ, ໂສ ຠາເສຍ຺ຍ.


</Paragraph>

<Paragraph number="217" book="cula">
ທຸຕິຍມ຺ປິ ເອຕມຕ຺ຖໍ ວທາມິ …ເປ…  ຕຕິຍມ຺ປິ ເອຕມຕ຺ຖໍ ວທາມິ—  ສຸຓາຕຸ ເມ, ຠນ຺ເຕ, ສໍໂຆ. ອຍໍ ຉນ຺ໂນ ຠິກ຺ຂຸ ອາປຕ຺ຕິໍ ອາປຊ຺ຊິຕ຺ວາ ນ ອິຈ຺ຉຕິ ອາປຕ຺ຕິໍ ປສ຺ສິຕຸໍ. ສໍໂຆ ຉນ຺ນສ຺ສ ຠິກ຺ຂຸໂນ, ອາປຕ຺ຕິຍາ ອທສ຺ສເນ, ອຸກ຺ເຂປນີຍກມ຺ມໍ ກໂຣຕິ—  ອສມ຺ໂຠຄໍ ສໍເຆນ. ຍສ຺ສາຍສ຺ມໂຕ ຂມຕິ ຉນ຺ນສ຺ສ ຠິກ຺ຂຸໂນ, ອາປຕ຺ຕິຍາ ອທສ຺ສເນ, ອຸກ຺ເຂປນີຍສ຺ສ ກມ຺ມສ຺ສ ກຣຓໍ—  ອສມ຺ໂຠຄໍ ສໍເຆນ, ໂສ ຕຸຓ຺ຫສ຺ສ; ຍສ຺ສ ນກ຺ຂມຕິ, ໂສ ຠາເສຍ຺ຍ.


</Paragraph>

<Paragraph number="218" book="cula">
ກຕໍ ສໍເຆນ ຉນ຺ນສ຺ສ ຠິກ຺ຂຸໂນ, ອາປຕ຺ຕິຍາ ອທສ຺ສເນ, ອຸກ຺ເຂປນີຍກມ຺ມໍ—  ອສມ຺ໂຠຄໍ ສໍເຆນ. ຂມຕິ ສໍຆສ຺ສ, ຕສ຺ມາ ຕຸຓ຺ຫີ, ເອວເມຕໍ ຘາຣຍາມີ’ຕິ.


</Paragraph>

<Paragraph number="219" book="cula">
ອາວາສປຣມ຺ປຣຎ຺ຈ, ຠິກ຺ຂເວ, ສໍສຖ—  ‘ຉນ຺ໂນ ຠິກ຺ຂຸ ສໍເຆນ ອາປຕ຺ຕິຍາ ອທສ຺ສເນ, ອຸກ຺ເຂປນີຍກມ຺ມກໂຕ—  ອສມ຺ໂຠຄໍ ສໍເຆນາ’ຕິ.

</Paragraph>
</Division>
