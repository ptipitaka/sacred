ᩈᩮᩣ ᨧᩥᨱ᩠ᨱᨾᩣᨶᨲ᩠ᨲᩮᩣ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩋᩣᩁᩮᩣᨧᩮᩈᩥ—  “ᩋᩉᩴ, ᩋᩣᩅᩩᩈᩮᩣ, ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ …ᨸᩮ…  ᩈᩮᩣᩉᩴ ᨧᩥᨱ᩠ᨱᨾᩣᨶᨲ᩠ᨲᩮᩣ. ᨠᨳᩴ ᨶᩩ ᨡᩮᩣ ᨾᨿᩣ ᨸᨭᩥᨸᨩ᩠ᨩᩥᨲᨻ᩠ᨻᨶ᩺”ᨲᩥ? ᨽᨣᩅᨲᩮᩣ ᩑᨲᨾᨲ᩠ᨳᩴ ᩋᩣᩁᩮᩣᨧᩮᩈᩩᩴ. “ᨲᩮᨶ ᩉᩥ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩴ ᨽᩥᨠ᩠ᨡᩩᩴ ᩋᨻ᩠ᨽᩮᨲᩩ. ᩑᩅᨬ᩠ᨧ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨻ᩠ᨽᩮᨲᨻ᩠ᨻᩮᩣ—
</Paragraph>

<Paragraph number="566" book="cula">
ᨲᩮᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩏᨴᩤᨿᩥᨶᩣ ᨽᩥᨠ᩠ᨡᩩᨶᩣ ᩈᩴᨥᩴ ᩏᨸᩈᩘᨠᨾᩥᨲ᩠ᩅᩣ …ᨸᩮ…  ᩑᩅᨾᩔ ᩅᨧᨶᩦᨿᩮᩣ—  ‘ᩋᩉᩴ, ᨽᨶ᩠ᨲᩮ, ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣᩉᩴ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨸᨬ᩠ᨧᩣᩉᨸᩁᩥᩅᩤᩈᩴ ᨿᩣᨧᩥᩴ. ᨲᩔ ᨾᩮ ᩈᩴᨥᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨸᨬ᩠ᨧᩣᩉᨸᩁᩥᩅᩤᩈᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣᩉᩴ ᨸᩁᩥᩅᩈᨶ᩠ᨲᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣᩉᩴ ᩈᩴᨥᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿᨸᨭᩥᨠᩔᨶᩴ ᨿᩣᨧᩥᩴ. ᨲᩴ ᨾᩴ ᩈᩴᨥᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿ ᨸᨭᩥᨠᩔᩥ. ᩈᩮᩣᩉᩴ ᨸᩁᩥᩅᩩᨲ᩠ᨳᨸᩁᩥᩅᩤᩈᩮᩣ ᨾᩣᨶᨲ᩠ᨲᩣᩁᩉᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣᩉᩴ ᩈᩴᨥᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿᨸᨭᩥᨠᩔᨶᩴ ᨿᩣᨧᩥᩴ. ᨲᩴ ᨾᩴ ᩈᩴᨥᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿ ᨸᨭᩥᨠᩔᩥ. ᩈᩮᩣᩉᩴ ᨸᩁᩥᩅᩩᨲ᩠ᨳᨸᩁᩥᩅᩤᩈᩮᩣ ᩈᩴᨥᩴ ᨲᩥᩔᨶ᩠ᨶᩴ ᩋᩣᨸᨲ᩠ᨲᩦᨶᩴ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᩥᩴ. ᨲᩔ ᨾᩮ ᩈᩴᨥᩮᩣ ᨲᩥᩔᨶ᩠ᨶᩴ ᩋᩣᨸᨲ᩠ᨲᩦᨶᩴ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣᩉᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨧᩁᨶ᩠ᨲᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣᩉᩴ ᩈᩴᨥᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿᨸᨭᩥᨠᩔᨶᩴ ᨿᩣᨧᩥᩴ. ᨲᩴ ᨾᩴ ᩈᩴᨥᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿ ᨸᨭᩥᨠᩔᩥ. ᩈᩮᩣᩉᩴ ᩈᩴᨥᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᩥᩴ. ᨲᩔ ᨾᩮ ᩈᩴᨥᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣᩉᩴ ᨧᩥᨱ᩠ᨱᨾᩣᨶᨲ᩠ᨲᩮᩣ ᩋᨻ᩠ᨽᩣᨶᩣᩁᩉᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣᩉᩴ ᩈᩴᨥᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿᨸᨭᩥᨠᩔᨶᩴ ᨿᩣᨧᩥᩴ. ᨲᩴ ᨾᩴ ᩈᩴᨥᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿ ᨸᨭᩥᨠᩔᩥ. ᩈᩮᩣᩉᩴ ᩈᩴᨥᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᩥᩴ. ᨲᩔ ᨾᩮ ᩈᩴᨥᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣᩉᩴ, ᨽᨶ᩠ᨲᩮ, ᨧᩥᨱ᩠ᨱᨾᩣᨶᨲ᩠ᨲᩮᩣ ᩈᩴᨥᩴ ᩋᨻ᩠ᨽᩣᨶᩴ ᨿᩣᨧᩣᨾᩦ’ᨲᩥ.
</Paragraph>

ᨴᩩᨲᩥᨿᨾ᩠ᨸᩥ ᨿᩣᨧᩥᨲᨻ᩠ᨻᩴ. ᨲᨲᩥᨿᨾ᩠ᨸᩥ ᨿᩣᨧᩥᨲᨻ᩠ᨻᩴ. ᨻ᩠ᨿᨲ᩠ᨲᩮᨶ ᨽᩥᨠ᩠ᨡᩩᨶᩣ ᨸᨭᩥᨻᩃᩮᨶ ᩈᩴᨥᩮᩣ ᨬᩣᨸᩮᨲᨻ᩠ᨻᩮᩣ—

‘ᩈᩩᨱᩣᨲᩩ ᨾᩮ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩮᩣ. ᩋᨿᩴ ᩏᨴᩤᨿᩦ ᨽᩥᨠ᩠ᨡᩩ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨸᨬ᩠ᨧᩣᩉᨸᩁᩥᩅᩤᩈᩴ ᨿᩣᨧᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨸᨬ᩠ᨧᩣᩉᨸᩁᩥᩅᩤᩈᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣ ᨸᩁᩥᩅᩈᨶ᩠ᨲᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿᨸᨭᩥᨠᩔᨶᩴ ᨿᩣᨧᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩴ ᨽᩥᨠ᩠ᨡᩩᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿ ᨸᨭᩥᨠᩔᩥ. ᩈᩮᩣ ᨸᩁᩥᩅᩩᨲ᩠ᨳᨸᩁᩥᩅᩤᩈᩮᩣ ᨾᩣᨶᨲ᩠ᨲᩣᩁᩉᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿᨸᨭᩥᨠᩔᨶᩴ ᨿᩣᨧᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩴ ᨽᩥᨠ᩠ᨡᩩᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿ ᨸᨭᩥᨠᩔᩥ. ᩈᩮᩣ ᨸᩁᩥᩅᩩᨲ᩠ᨳᨸᩁᩥᩅᩤᩈᩮᩣ ᩈᩴᨥᩴ ᨲᩥᩔᨶ᩠ᨶᩴ ᩋᩣᨸᨲ᩠ᨲᩦᨶᩴ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᨲᩥᩔᨶ᩠ᨶᩴ ᩋᩣᨸᨲ᩠ᨲᩦᨶᩴ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣ ᨾᩣᨶᨲ᩠ᨲᩴ ᨧᩁᨶ᩠ᨲᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿᨸᨭᩥᨠᩔᨶᩴ ᨿᩣᨧᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩴ ᨽᩥᨠ᩠ᨡᩩᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿ ᨸᨭᩥᨠᩔᩥ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣ ᨧᩥᨱ᩠ᨱᨾᩣᨶᨲ᩠ᨲᩮᩣ ᩋᨻ᩠ᨽᩣᨶᩣᩁᩉᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿᨸᨭᩥᨠᩔᨶᩴ ᨿᩣᨧᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩴ ᨽᩥᨠ᩠ᨡᩩᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿ ᨸᨭᩥᨠᩔᩥ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣ ᨧᩥᨱ᩠ᨱᨾᩣᨶᨲ᩠ᨲᩮᩣ ᩈᩴᨥᩴ ᩋᨻ᩠ᨽᩣᨶᩴ ᨿᩣᨧᨲᩥ. ᨿᨴᩥ ᩈᩴᨥᩔ ᨸᨲ᩠ᨲᨠᩃᩖᩴ, ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩴ ᨽᩥᨠ᩠ᨡᩩᩴ ᩋᨻ᩠ᨽᩮᨿ᩠ᨿ. ᩑᩈᩣ ᨬᨲ᩠ᨲᩥ.
</Paragraph>
