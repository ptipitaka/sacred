ᨲᩮᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩏᨴᩤᨿᩥᨶᩣ ᨽᩥᨠ᩠ᨡᩩᨶᩣ ᩈᩴᨥᩴ ᩏᨸᩈᩘᨠᨾᩥᨲ᩠ᩅᩣ ᩑᨠᩴᩈᩴ ᩏᨲ᩠ᨲᩁᩣᩈᩘᨣᩴ ᨠᩁᩥᨲ᩠ᩅᩣ ᩅᩩᨯ᩠ᨰᩣᨶᩴ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᨸᩣᨴᩮ ᩅᨶ᩠ᨴᩥᨲ᩠ᩅᩣ ᩏᨠ᩠ᨠᩩᨭᩥᨠᩴ ᨶᩥᩈᩦᨴᩥᨲ᩠ᩅᩣ ᩋᨬ᩠ᨩᩃᩥᩴ ᨸᨣ᩠ᨣᩉᩮᨲ᩠ᩅᩣ ᩑᩅᨾᩔ ᩅᨧᨶᩦᨿᩮᩣ—  ‘ᩋᩉᩴ, ᨽᨶ᩠ᨲᩮ, ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣᩉᩴ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᩣᨾᩥ. ᩋᩉᩴ, ᨽᨶ᩠ᨲᩮ, ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᨴᩩᨲᩥᨿᨾ᩠ᨸᩥ, ᩈᩮᩣᩉᩴ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᩣᨾᩥ. ᩋᩉᩴ, ᨽᨶ᩠ᨲᩮ, ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᨲᨲᩥᨿᨾ᩠ᨸᩥ ᩈᩮᩣᩉᩴ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᩣᨾᩦ’ᨲᩥ. ᨻ᩠ᨿᨲ᩠ᨲᩮᨶ ᨽᩥᨠ᩠ᨡᩩᨶᩣ ᨸᨭᩥᨻᩃᩮᨶ ᩈᩴᨥᩮᩣ ᨬᩣᨸᩮᨲᨻ᩠ᨻᩮᩣ—
</Paragraph>

</Division>

<Paragraph number="500" book="cula">
‘ᩈᩩᨱᩣᨲᩩ ᨾᩮ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩮᩣ. ᩋᨿᩴ ᩏᨴᩤᨿᩦ ᨽᩥᨠ᩠ᨡᩩ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᨲᩥ. ᨿᨴᩥ ᩈᩴᨥᩔ ᨸᨲ᩠ᨲᨠᩃᩖᩴ, ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨴᨴᩮᨿ᩠ᨿ. ᩑᩈᩣ ᨬᨲ᩠ᨲᩥ.
</Paragraph>

<Paragraph number="501" book="cula">
ᩈᩩᨱᩣᨲᩩ ᨾᩮ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩮᩣ. ᩋᨿᩴ ᩏᨴᩤᨿᩦ ᨽᩥᨠ᩠ᨡᩩ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᨲᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨴᩮᨲᩥ. ᨿᩔᩣᨿᩈ᩠ᨾᨲᩮᩣ ᨡᨾᨲᩥ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩔ ᨴᩤᨶᩴ, ᩈᩮᩣ ᨲᩩᨱ᩠ᩉᩔ; ᨿᩔ ᨶᨠ᩠ᨡᨾᨲᩥ, ᩈᩮᩣ ᨽᩣᩈᩮᨿ᩠ᨿ.
</Paragraph>

<Paragraph number="502" book="cula">
ᨴᩩᨲᩥᨿᨾ᩠ᨸᩥ ᩑᨲᨾᨲ᩠ᨳᩴ ᩅᨴᩤᨾᩥ—  ᩈᩩᨱᩣᨲᩩ ᨾᩮ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩮᩣ. ᩋᨿᩴ ᩏᨴᩤᨿᩦ ᨽᩥᨠ᩠ᨡᩩ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᨲᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨴᩮᨲᩥ. ᨿᩔᩣᨿᩈ᩠ᨾᨲᩮᩣ ᨡᨾᨲᩥ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩔ ᨴᩤᨶᩴ, ᩈᩮᩣ ᨲᩩᨱ᩠ᩉᩔ; ᨿᩔ ᨶᨠ᩠ᨡᨾᨲᩥ, ᩈᩮᩣ ᨽᩣᩈᩮᨿ᩠ᨿ.
</Paragraph>

<Paragraph number="503" book="cula">
ᨲᨲᩥᨿᨾ᩠ᨸᩥ ᩑᨲᨾᨲ᩠ᨳᩴ ᩅᨴᩤᨾᩥ—  ᩈᩩᨱᩣᨲᩩ ᨾᩮ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩮᩣ. ᩋᨿᩴ ᩏᨴᩤᨿᩦ ᨽᩥᨠ᩠ᨡᩩ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᨲᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨴᩮᨲᩥ. ᨿᩔᩣᨿᩈ᩠ᨾᨲᩮᩣ ᨡᨾᨲᩥ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩔ ᨴᩤᨶᩴ, ᩈᩮᩣ ᨲᩩᨱ᩠ᩉᩔ; ᨿᩔ ᨶᨠ᩠ᨡᨾᨲᩥ, ᩈᩮᩣ ᨽᩣᩈᩮᨿ᩠ᨿ.
</Paragraph>

<Paragraph number="504" book="cula">
ᨴᩥᨶ᩠ᨶᩴ ᩈᩴᨥᩮᨶ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ. ᨡᨾᨲᩥ ᩈᩴᨥᩔ, ᨲᩈ᩠ᨾᩣ ᨲᩩᨱ᩠ᩉᩦ, ᩑᩅᨾᩮᨲᩴ ᨵᩤᩁᨿᩣᨾᩦ’”ᨲᩥ.
</Paragraph>
</Division>
