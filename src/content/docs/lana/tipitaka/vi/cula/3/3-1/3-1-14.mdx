ᩈᩮᩣ ᨸᩁᩥᩅᩈᨶ᩠ᨲᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩋᩣᩁᩮᩣᨧᩮᩈᩥ—  “ᩋᩉᩴ, ᩋᩣᩅᩩᩈᩮᩣ, ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᨸᨠ᩠ᨡᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣᩉᩴ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨠ᩠ᨡᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨸᨠ᩠ᨡᨸᩁᩥᩅᩤᩈᩴ ᨿᩣᨧᩥᩴ. ᨲᩔ ᨾᩮ ᩈᩴᨥᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨠ᩠ᨡᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨸᨠ᩠ᨡᨸᩁᩥᩅᩤᩈᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣᩉᩴ ᨸᩁᩥᩅᩈᨶ᩠ᨲᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᨠᨳᩴ ᨶᩩ ᨡᩮᩣ ᨾᨿᩣ ᨸᨭᩥᨸᨩ᩠ᨩᩥᨲᨻ᩠ᨻᨶ᩺”ᨲᩥ. ᨽᨣᩅᨲᩮᩣ ᩑᨲᨾᨲ᩠ᨳᩴ ᩋᩣᩁᩮᩣᨧᩮᩈᩩᩴ. “ᨲᩮᨶ ᩉᩥ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩴ ᨽᩥᨠ᩠ᨡᩩᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿ ᨸᨭᩥᨠᩔᩥᨲ᩠ᩅᩣ ᨸᩩᩁᩥᨾᩣᨿ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨾᩮᩣᨵᩤᨶᨸᩁᩥᩅᩤᩈᩴ ᨴᩮᨲᩩ. ᩑᩅᨬ᩠ᨧ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨾᩪᩃᩣᨿ ᨸᨭᩥᨠᩔᩥᨲᨻ᩠ᨻᩮᩣ—
</Paragraph>

<Paragraph number="579" book="cula">
ᨲᩮᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩏᨴᩤᨿᩥᨶᩣ ᨽᩥᨠ᩠ᨡᩩᨶᩣ ᩈᩴᨥᩴ ᩏᨸᩈᩘᨠᨾᩥᨲ᩠ᩅᩣ …ᨸᩮ…  ᩑᩅᨾᩔ ᩅᨧᨶᩦᨿᩮᩣ—  ‘ᩋᩉᩴ, ᨽᨶ᩠ᨲᩮ, ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᨸᨠ᩠ᨡᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣᩉᩴ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨠ᩠ᨡᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨸᨠ᩠ᨡᨸᩁᩥᩅᩤᩈᩴ ᨿᩣᨧᩥᩴ. ᨲᩔ ᨾᩮ ᩈᩴᨥᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨠ᩠ᨡᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨸᨠ᩠ᨡᨸᩁᩥᩅᩤᩈᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣᩉᩴ ᨸᩁᩥᩅᩈᨶ᩠ᨲᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣᩉᩴ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿᨸᨭᩥᨠᩔᨶᩴ ᨿᩣᨧᩣᨾᩦ’ᨲᩥ. ᨴᩩᨲᩥᨿᨾ᩠ᨸᩥ ᨿᩣᨧᩥᨲᨻ᩠ᨻᩣ. ᨲᨲᩥᨿᨾ᩠ᨸᩥ ᨿᩣᨧᩥᨲᨻ᩠ᨻᩣ. ᨻ᩠ᨿᨲ᩠ᨲᩮᨶ ᨽᩥᨠ᩠ᨡᩩᨶᩣ ᨸᨭᩥᨻᩃᩮᨶ ᩈᩴᨥᩮᩣ ᨬᩣᨸᩮᨲᨻ᩠ᨻᩮᩣ—
</Paragraph>

</Division>

<Paragraph number="580" book="cula">
‘ᩈᩩᨱᩣᨲᩩ ᨾᩮ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩮᩣ. ᩋᨿᩴ ᩏᨴᩤᨿᩦ ᨽᩥᨠ᩠ᨡᩩ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᨸᨠ᩠ᨡᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨠ᩠ᨡᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨸᨠ᩠ᨡᨸᩁᩥᩅᩤᩈᩴ ᨿᩣᨧᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨠ᩠ᨡᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨸᨠ᩠ᨡᨸᩁᩥᩅᩤᩈᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣ ᨸᩁᩥᩅᩈᨶ᩠ᨲᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿᨸᨭᩥᨠᩔᨶᩴ ᨿᩣᨧᨲᩥ. ᨿᨴᩥ ᩈᩴᨥᩔ ᨸᨲ᩠ᨲᨠᩃᩖᩴ, ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩴ ᨽᩥᨠ᩠ᨡᩩᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿ ᨸᨭᩥᨠᩔᩮᨿ᩠ᨿ. ᩑᩈᩣ ᨬᨲ᩠ᨲᩥ.
</Paragraph>

<Paragraph number="581" book="cula">
ᩈᩩᨱᩣᨲᩩ ᨾᩮ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩮᩣ. ᩋᨿᩴ ᩏᨴᩤᨿᩦ ᨽᩥᨠ᩠ᨡᩩ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᨸᨠ᩠ᨡᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨠ᩠ᨡᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨸᨠ᩠ᨡᨸᩁᩥᩅᩤᩈᩴ ᨿᩣᨧᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨠ᩠ᨡᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨸᨠ᩠ᨡᨸᩁᩥᩅᩤᩈᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣ ᨸᩁᩥᩅᩈᨶ᩠ᨲᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿᨸᨭᩥᨠᩔᨶᩴ ᨿᩣᨧᨲᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩴ ᨽᩥᨠ᩠ᨡᩩᩴ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿ ᨸᨭᩥᨠᩔᨲᩥ. ᨿᩔᩣᨿᩈ᩠ᨾᨲᩮᩣ ᨡᨾᨲᩥ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᨸᨬ᩠ᨧᩣᩉᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨾᩪᩃᩣᨿᨸᨭᩥᨠᩔᨶᩣ, ᩈᩮᩣ ᨲᩩᨱ᩠ᩉᩔ; ᨿᩔ ᨶᨠ᩠ᨡᨾᨲᩥ, ᩈᩮᩣ ᨽᩣᩈᩮᨿ᩠ᨿ.
</Paragraph>
