ᩈᩮᩣ ᨧᩥᨱ᩠ᨱᨾᩣᨶᨲ᩠ᨲᩮᩣ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩋᩣᩁᩮᩣᨧᩮᩈᩥ—  “ᩋᩉᩴ, ᩋᩣᩅᩩᩈᩮᩣ, ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣᩉᩴ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᩥᩴ. ᨲᩔ ᨾᩮ ᩈᩴᨥᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣᩉᩴ ᨧᩥᨱ᩠ᨱᨾᩣᨶᨲ᩠ᨲᩮᩣ. ᨠᨳᩴ ᨶᩩ ᨡᩮᩣ ᨾᨿᩣ ᨸᨭᩥᨸᨩ᩠ᨩᩥᨲᨻ᩠ᨻᨶ᩺”ᨲᩥ? ᨽᨣᩅᨲᩮᩣ ᩑᨲᨾᨲ᩠ᨳᩴ ᩋᩣᩁᩮᩣᨧᩮᩈᩩᩴ. “ᨲᩮᨶ ᩉᩥ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩴ ᨽᩥᨠ᩠ᨡᩩᩴ ᩋᨻ᩠ᨽᩮᨲᩩ. ᩑᩅᨬ᩠ᨧ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨻ᩠ᨽᩮᨲᨻ᩠ᨻᩮᩣ—
</Paragraph>

<Paragraph number="506" book="cula">
ᨲᩮᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩏᨴᩤᨿᩥᨶᩣ ᨽᩥᨠ᩠ᨡᩩᨶᩣ ᩈᩴᨥᩴ ᩏᨸᩈᩘᨠᨾᩥᨲ᩠ᩅᩣ ᩑᨠᩴᩈᩴ ᩏᨲ᩠ᨲᩁᩣᩈᩘᨣᩴ ᨠᩁᩥᨲ᩠ᩅᩣ ᩅᩩᨯ᩠ᨰᩣᨶᩴ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᨸᩣᨴᩮ ᩅᨶ᩠ᨴᩥᨲ᩠ᩅᩣ ᩏᨠ᩠ᨠᩩᨭᩥᨠᩴ ᨶᩥᩈᩦᨴᩥᨲ᩠ᩅᩣ ᩋᨬ᩠ᨩᩃᩥᩴ ᨸᨣ᩠ᨣᩉᩮᨲ᩠ᩅᩣ ᩑᩅᨾᩔ ᩅᨧᨶᩦᨿᩮᩣ—  ‘ᩋᩉᩴ, ᨽᨶ᩠ᨲᩮ, ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ, ᩈᩮᩣᩉᩴ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᩥᩴ. ᨲᩔ ᨾᩮ ᩈᩴᨥᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣᩉᩴ, ᨽᨶ᩠ᨲᩮ, ᨧᩥᨱ᩠ᨱᨾᩣᨶᨲ᩠ᨲᩮᩣ ᩈᩴᨥᩴ ᩋᨻ᩠ᨽᩣᨶᩴ ᨿᩣᨧᩣᨾᩥ.
</Paragraph>

<Paragraph number="507" book="cula">
ᩋᩉᩴ, ᨽᨶ᩠ᨲᩮ, ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣᩉᩴ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᩥᩴ. ᨲᩔ ᨾᩮ ᩈᩴᨥᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣᩉᩴ ᨧᩥᨱ᩠ᨱᨾᩣᨶᨲ᩠ᨲᩮᩣ ᨴᩩᨲᩥᨿᨾ᩠ᨸᩥ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩴ ᩋᨻ᩠ᨽᩣᨶᩴ ᨿᩣᨧᩣᨾᩥ.
</Paragraph>

<Paragraph number="508" book="cula">
ᩋᩉᩴ, ᨽᨶ᩠ᨲᩮ, ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣᩉᩴ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᩥᩴ. ᨲᩔ ᨾᩮ ᩈᩴᨥᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣᩉᩴ ᨧᩥᨱ᩠ᨱᨾᩣᨶᨲ᩠ᨲᩮᩣ ᨲᨲᩥᨿᨾ᩠ᨸᩥ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩴ ᩋᨻ᩠ᨽᩣᨶᩴ ᨿᩣᨧᩣᨾᩦ’ᨲᩥ. ᨻ᩠ᨿᨲ᩠ᨲᩮᨶ ᨽᩥᨠ᩠ᨡᩩᨶᩣ ᨸᨭᩥᨻᩃᩮᨶ ᩈᩴᨥᩮᩣ ᨬᩣᨸᩮᨲᨻ᩠ᨻᩮᩣ—
</Paragraph>

<Paragraph number="509" book="cula">
‘ᩈᩩᨱᩣᨲᩩ ᨾᩮ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩮᩣ. ᩋᨿᩴ ᩏᨴᩤᨿᩦ ᨽᩥᨠ᩠ᨡᩩ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣ ᨧᩥᨱ᩠ᨱᨾᩣᨶᨲ᩠ᨲᩮᩣ ᩈᩴᨥᩴ ᩋᨻ᩠ᨽᩣᨶᩴ ᨿᩣᨧᨲᩥ. ᨿᨴᩥ ᩈᩴᨥᩔ ᨸᨲ᩠ᨲᨠᩃᩖᩴ, ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩴ ᨽᩥᨠ᩠ᨡᩩᩴ ᩋᨻ᩠ᨽᩮᨿ᩠ᨿ. ᩑᩈᩣ ᨬᨲ᩠ᨲᩥ.
</Paragraph>

<Paragraph number="510" book="cula">
ᩈᩩᨱᩣᨲᩩ ᨾᩮ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩮᩣ. ᩋᨿᩴ ᩏᨴᩤᨿᩦ ᨽᩥᨠ᩠ᨡᩩ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣ ᨧᩥᨱ᩠ᨱᨾᩣᨶᨲ᩠ᨲᩮᩣ ᩈᩴᨥᩴ ᩋᨻ᩠ᨽᩣᨶᩴ ᨿᩣᨧᨲᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩴ ᨽᩥᨠ᩠ᨡᩩᩴ ᩋᨻ᩠ᨽᩮᨲᩥ. ᨿᩔᩣᨿᩈ᩠ᨾᨲᩮᩣ ᨡᨾᨲᩥ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩋᨻ᩠ᨽᩣᨶᩴ, ᩈᩮᩣ ᨲᩩᨱ᩠ᩉᩔ; ᨿᩔ ᨶᨠ᩠ᨡᨾᨲᩥ, ᩈᩮᩣ ᨽᩣᩈᩮᨿ᩠ᨿ.
</Paragraph>

ᨴᩩᨲᩥᨿᨾ᩠ᨸᩥ ᩑᨲᨾᨲ᩠ᨳᩴ ᩅᨴᩤᨾᩥ—  ᩈᩩᨱᩣᨲᩩ ᨾᩮ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩮᩣ. ᩋᨿᩴ ᩏᨴᩤᨿᩦ ᨽᩥᨠ᩠ᨡᩩ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣ ᨧᩥᨱ᩠ᨱᨾᩣᨶᨲ᩠ᨲᩮᩣ ᩈᩴᨥᩴ ᩋᨻ᩠ᨽᩣᨶᩴ ᨿᩣᨧᨲᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩴ ᨽᩥᨠ᩠ᨡᩩᩴ ᩋᨻ᩠ᨽᩮᨲᩥ. ᨿᩔᩣᨿᩈ᩠ᨾᨲᩮᩣ ᨡᨾᨲᩥ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩋᨻ᩠ᨽᩣᨶᩴ, ᩈᩮᩣ ᨲᩩᨱ᩠ᩉᩔ; ᨿᩔ ᨶᨠ᩠ᨡᨾᨲᩥ, ᩈᩮᩣ ᨽᩣᩈᩮᨿ᩠ᨿ.
</Paragraph>

<Paragraph number="512" book="cula">
ᨲᨲᩥᨿᨾ᩠ᨸᩥ ᩑᨲᨾᨲ᩠ᨳᩴ ᩅᨴᩤᨾᩥ—  ᩈᩩᨱᩣᨲᩩ ᨾᩮ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩮᩣ. ᩋᨿᩴ ᩏᨴᩤᨿᩦ ᨽᩥᨠ᩠ᨡᩩ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᩈᩴᨥᩴ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨿᩣᨧᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᩋᨴᩤᩈᩥ. ᩈᩮᩣ ᨧᩥᨱ᩠ᨱᨾᩣᨶᨲ᩠ᨲᩮᩣ ᩈᩴᨥᩴ ᩋᨻ᩠ᨽᩣᨶᩴ ᨿᩣᨧᨲᩥ. ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩴ ᨽᩥᨠ᩠ᨡᩩᩴ ᩋᨻ᩠ᨽᩮᨲᩥ. ᨿᩔᩣᨿᩈ᩠ᨾᨲᩮᩣ ᨡᨾᨲᩥ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩋᨻ᩠ᨽᩣᨶᩴ, ᩈᩮᩣ ᨲᩩᨱ᩠ᩉᩔ; ᨿᩔ ᨶᨠ᩠ᨡᨾᨲᩥ, ᩈᩮᩣ ᨽᩣᩈᩮᨿ᩠ᨿ.
</Paragraph>

<Paragraph number="513" book="cula">
ᩋᨻ᩠ᨽᩥᨲᩮᩣ ᩈᩴᨥᩮᨶ ᩏᨴᩤᨿᩦ ᨽᩥᨠ᩠ᨡᩩ. ᨡᨾᨲᩥ ᩈᩴᨥᩔ, ᨲᩈ᩠ᨾᩣ ᨲᩩᨱ᩠ᩉᩦ, ᩑᩅᨾᩮᨲᩴ ᨵᩤᩁᨿᩣᨾᩦ’”ᨲᩥ.
