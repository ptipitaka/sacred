<Paragraph number="691" book="cula">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩩ ᨸᩁᩥᩅᩈᨶ᩠ᨲᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩈᩴᨥᩣᨴᩥᩈᩮᩈᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩮᩣ ᩋᩣᨸᨩ᩠ᨩᨲᩥ ᨸᩁᩥᨾᩣᨱᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣ. ᩈᩮᩣ ᨽᩥᨠ᩠ᨡᩩ ᨾᩪᩃᩣᨿ ᨸᨭᩥᨠᩔᩥᨲᨻ᩠ᨻᩮᩣ. (1)
</Paragraph>

<Paragraph number="692" book="cula">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩩ ᨸᩁᩥᩅᩈᨶ᩠ᨲᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩈᩴᨥᩣᨴᩥᩈᩮᩈᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩮᩣ ᩋᩣᨸᨩ᩠ᨩᨲᩥ ᨸᩁᩥᨾᩣᨱᩣ ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣ. ᩈᩮᩣ ᨽᩥᨠ᩠ᨡᩩ ᨾᩪᩃᩣᨿ ᨸᨭᩥᨠᩔᩥᨲᨻ᩠ᨻᩮᩣ. ᨿᨳᩣᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨶᨬ᩠ᨧᩔ ᩋᩣᨸᨲ᩠ᨲᩦᨶᩴ ᨸᩩᩁᩥᨾᩣᨿ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨾᩮᩣᨵᩤᨶᨸᩁᩥᩅᩤᩈᩮᩣ ᨴᩤᨲᨻ᩠ᨻᩮᩣ. (2)
</Paragraph>

<Paragraph number="693" book="cula">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩩ ᨸᩁᩥᩅᩈᨶ᩠ᨲᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩈᩴᨥᩣᨴᩥᩈᩮᩈᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩮᩣ ᩋᩣᨸᨩ᩠ᨩᨲᩥ ᨸᩁᩥᨾᩣᨱᩣ ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣᨸᩥ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣᨸᩥ. ᩈᩮᩣ ᨽᩥᨠ᩠ᨡᩩ ᨾᩪᩃᩣᨿ ᨸᨭᩥᨠᩔᩥᨲᨻ᩠ᨻᩮᩣ. ᨿᨳᩣᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨶᨬ᩠ᨧᩔ ᩋᩣᨸᨲ᩠ᨲᩦᨶᩴ ᨸᩩᩁᩥᨾᩣᨿ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨾᩮᩣᨵᩤᨶᨸᩁᩥᩅᩤᩈᩮᩣ ᨴᩤᨲᨻ᩠ᨻᩮᩣ. (3)
</Paragraph>

<Paragraph number="694" book="cula">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩩ ᨸᩁᩥᩅᩈᨶ᩠ᨲᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩈᩴᨥᩣᨴᩥᩈᩮᩈᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩮᩣ ᩋᩣᨸᨩ᩠ᨩᨲᩥ ᩋᨸᩁᩥᨾᩣᨱᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣ …ᨸᩮ…  ᩋᨸᩁᩥᨾᩣᨱᩣ ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣ …ᨸᩮ…  ᩋᨸᩁᩥᨾᩣᨱᩣ ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣᨸᩥ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣᨸᩥ …ᨸᩮ…  ᨸᩁᩥᨾᩣᨱᩣᨿᩮᩣᨸᩥ ᩋᨸᩁᩥᨾᩣᨱᩣᨿᩮᩣᨸᩥ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣ …ᨸᩮ…  ᨸᩁᩥᨾᩣᨱᩣᨿᩮᩣᨸᩥ ᩋᨸᩁᩥᨾᩣᨱᩣᨿᩮᩣᨸᩥ ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣ …ᨸᩮ…  ᨸᩁᩥᨾᩣᨱᩣᨿᩮᩣᨸᩥ ᩋᨸᩁᩥᨾᩣᨱᩣᨿᩮᩣᨸᩥ ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣᨸᩥ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣᨸᩥ. ᩈᩮᩣ ᨽᩥᨠ᩠ᨡᩩ ᨾᩪᩃᩣᨿ ᨸᨭᩥᨠᩔᩥᨲᨻ᩠ᨻᩮᩣ. ᨿᨳᩣᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨶᨬ᩠ᨧᩔ ᩋᩣᨸᨲ᩠ᨲᩦᨶᩴ ᨸᩩᩁᩥᨾᩣᨿ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨾᩮᩣᨵᩤᨶᨸᩁᩥᩅᩤᩈᩮᩣ ᨴᩤᨲᨻ᩠ᨻᩮᩣ. (9)
</Paragraph>

<Paragraph number="695" book="cula">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩩ ᨾᩣᨶᨲ᩠ᨲᩣᩁᩉᩮᩣ …ᨸᩮ…  ᨾᩣᨶᨲ᩠ᨲᩴ ᨧᩁᨶ᩠ᨲᩮᩣ …ᨸᩮ…  (ᨿᨳᩣᨸᩁᩥᩅᩤᩈᩴ ᨲᨳᩣ ᩅᩥᨲ᩠ᨳᩣᩁᩮᨲᨻ᩠ᨻᩴ) ᩋᨻ᩠ᨽᩣᨶᩣᩁᩉᩮᩣ ᩋᨶ᩠ᨲᩁᩣ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩈᩴᨥᩣᨴᩥᩈᩮᩈᩣ, ᩋᩣᨸᨲ᩠ᨲᩥᨿᩮᩣ ᩋᩣᨸᨩ᩠ᨩᨲᩥ ᨸᩁᩥᨾᩣᨱᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣ …ᨸᩮ…  ᨸᩁᩥᨾᩣᨱᩣ ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣ …ᨸᩮ…  ᨸᩁᩥᨾᩣᨱᩣ ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣᨸᩥ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣᨸᩥ …ᨸᩮ…  ᩋᨸᩁᩥᨾᩣᨱᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣ …ᨸᩮ…  ᩋᨸᩁᩥᨾᩣᨱᩣ ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣ …ᨸᩮ…  ᩋᨸᩁᩥᨾᩣᨱᩣ ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣᨸᩥ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣᨸᩥ …ᨸᩮ…  ᨸᩁᩥᨾᩣᨱᩣᨿᩮᩣᨸᩥ ᩋᨸᩁᩥᨾᩣᨱᩣᨿᩮᩣᨸᩥ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣ …ᨸᩮ…  ᨸᩁᩥᨾᩣᨱᩣᨿᩮᩣᨸᩥ ᩋᨸᩁᩥᨾᩣᨱᩣᨿᩮᩣᨸᩥ ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣ …ᨸᩮ…  ᨸᩁᩥᨾᩣᨱᩣᨿᩮᩣᨸᩥ ᩋᨸᩁᩥᨾᩣᨱᩣᨿᩮᩣᨸᩥ ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣᨸᩥ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿᩮᩣᨸᩥ. ᩈᩮᩣ ᨽᩥᨠ᩠ᨡᩩ ᨾᩪᩃᩣᨿ ᨸᨭᩥᨠᩔᩥᨲᨻ᩠ᨻᩮᩣ. ᨿᨳᩣᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨶᨬ᩠ᨧᩔ ᩋᩣᨸᨲ᩠ᨲᩦᨶᩴ ᨸᩩᩁᩥᨾᩣᨿ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨾᩮᩣᨵᩤᨶᨸᩁᩥᩅᩤᩈᩮᩣ ᨴᩤᨲᨻ᩠ᨻᩮᩣ. (36)

</Paragraph>
