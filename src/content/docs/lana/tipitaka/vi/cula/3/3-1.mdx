ᨲᩮᨶ ᩈᨾᨿᩮᨶ ᨻᩩᨴ᩠ᨵᩮᩤ ᨽᨣᩅᩤ ᩈᩣᩅᨲ᩠ᨳᩥᨿᩴ ᩅᩥᩉᩁᨲᩥ ᨩᩮᨲᩅᨶᩮ ᩋᨶᩣᨳᨸᩥᨱ᩠ᨯᩥᨠᩔ ᩋᩣᩁᩣᨾᩮ. ᨲᩮᨶ ᨡᩮᩣ ᨸᨶ ᩈᨾᨿᩮᨶ ᩋᩣᨿᩈ᩠ᨾᩣ ᩏᨴᩤᨿᩦ ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨶ᩠ᨶᩮᩣ ᩉᩮᩣᨲᩥ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᩈᩮᩣ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩋᩣᩁᩮᩣᨧᩮᩈᩥ—  “ᩋᩉᩴ, ᩋᩣᩅᩩᩈᩮᩣ, ᩑᨠᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᩋᩣᨸᨩ᩠ᨩᩥᩴ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩴ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᩴ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩴ. ᨠᨳᩴ ᨶᩩ ᨡᩮᩣ ᨾᨿᩣ ᨸᨭᩥᨸᨩ᩠ᨩᩥᨲᨻ᩠ᨻᨶ᩺”ᨲᩥ? ᨽᨣᩅᨲᩮᩣ ᩑᨲᨾᨲ᩠ᨳᩴ ᩋᩣᩁᩮᩣᨧᩮᩈᩩᩴ. “ᨲᩮᨶ ᩉᩥ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩈᩴᨥᩮᩣ ᩏᨴᩤᨿᩥᩔ ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣ ᩑᨠᩥᩔᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩈᨬ᩠ᨧᩮᨲᨶᩥᨠᩣᨿ ᩈᩩᨠ᩠ᨠᩅᩥᩔᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨧ᩠ᨨᨶ᩠ᨶᩣᨿ ᨨᩣᩁᨲ᩠ᨲᩴ ᨾᩣᨶᨲ᩠ᨲᩴ ᨴᩮᨲᩩ. ᩑᩅᨬ᩠ᨧ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨴᩤᨲᨻ᩠ᨻᩴ—

</Paragraph>
</Division>
