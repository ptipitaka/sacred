ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᩣᩅᩤᩈᩣ ᩋᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᩣᩅᩤᩈᩮᩣ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ. ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᩣᩅᩤᩈᩣ ᩋᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᨶᩣᩅᩤᩈᩮᩣ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ. ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᩣᩅᩤᩈᩣ ᩋᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᩣᩅᩤᩈᩮᩣ ᩅᩤ ᩋᨶᩣᩅᩤᩈᩮᩣ ᩅᩤ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ.
</Paragraph>

<Paragraph number="789" book="vi-maha">
ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᨶᩣᩅᩤᩈᩣ ᩋᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᩣᩅᩤᩈᩮᩣ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ. ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᨶᩣᩅᩤᩈᩣ ᩋᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᨶᩣᩅᩤᩈᩮᩣ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ. ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᨶᩣᩅᩤᩈᩣ ᩋᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᩣᩅᩤᩈᩮᩣ ᩅᩤ ᩋᨶᩣᩅᩤᩈᩮᩣ ᩅᩤ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ.
</Paragraph>

<Paragraph number="790" book="vi-maha">
ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᩣᩅᩤᩈᩣ ᩅᩤ ᩋᨶᩣᩅᩤᩈᩣ ᩅᩤ ᩋᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᩣᩅᩤᩈᩮᩣ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ. ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᩣᩅᩤᩈᩣ ᩅᩤ ᩋᨶᩣᩅᩤᩈᩣ ᩅᩤ ᩋᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᨶᩣᩅᩤᩈᩮᩣ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ. ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᩣᩅᩤᩈᩣ ᩅᩤ ᩋᨶᩣᩅᩤᩈᩣ ᩅᩤ ᩋᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᩣᩅᩤᩈᩮᩣ ᩅᩤ ᩋᨶᩣᩅᩤᩈᩮᩣ ᩅᩤ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ.
</Paragraph>

<Paragraph number="791" book="vi-maha">
ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᩣᩅᩤᩈᩣ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᩣᩅᩤᩈᩮᩣ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᨿᨲ᩠ᨳᩔᩩ ᨽᩥᨠ᩠ᨡᩪ ᨶᩣᨶᩣᩈᩴᩅᩤᩈᨠᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ. ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᩣᩅᩤᩈᩣ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᨶᩣᩅᩤᩈᩮᩣ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᨿᨲ᩠ᨳᩔᩩ ᨽᩥᨠ᩠ᨡᩪ ᨶᩣᨶᩣᩈᩴᩅᩤᩈᨠᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ. ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᩣᩅᩤᩈᩣ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᩣᩅᩤᩈᩮᩣ ᩅᩤ ᩋᨶᩣᩅᩤᩈᩮᩣ ᩅᩤ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᨿᨲ᩠ᨳᩔᩩ ᨽᩥᨠ᩠ᨡᩪ ᨶᩣᨶᩣᩈᩴᩅᩤᩈᨠᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ.
</Paragraph>

<Paragraph number="792" book="vi-maha">
ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᨶᩣᩅᩤᩈᩣ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᩣᩅᩤᩈᩮᩣ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᨿᨲ᩠ᨳᩔᩩ ᨽᩥᨠ᩠ᨡᩪ ᨶᩣᨶᩣᩈᩴᩅᩤᩈᨠᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ. ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᨶᩣᩅᩤᩈᩣ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᨶᩣᩅᩤᩈᩮᩣ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᨿᨲ᩠ᨳᩔᩩ ᨽᩥᨠ᩠ᨡᩪ ᨶᩣᨶᩣᩈᩴᩅᩤᩈᨠᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ. ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᨶᩣᩅᩤᩈᩣ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᩣᩅᩤᩈᩮᩣ ᩅᩤ ᩋᨶᩣᩅᩤᩈᩮᩣ ᩅᩤ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᨿᨲ᩠ᨳᩔᩩ ᨽᩥᨠ᩠ᨡᩪ ᨶᩣᨶᩣᩈᩴᩅᩤᩈᨠᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ.
</Paragraph>

<Paragraph number="793" book="vi-maha">
ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᩣᩅᩤᩈᩣ ᩅᩤ ᩋᨶᩣᩅᩤᩈᩣ ᩅᩤ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᩣᩅᩤᩈᩮᩣ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᨿᨲ᩠ᨳᩔᩩ ᨽᩥᨠ᩠ᨡᩪ ᨶᩣᨶᩣᩈᩴᩅᩤᩈᨠᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ. ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᩣᩅᩤᩈᩣ ᩅᩤ ᩋᨶᩣᩅᩤᩈᩣ ᩅᩤ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᨶᩣᩅᩤᩈᩮᩣ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᨿᨲ᩠ᨳᩔᩩ ᨽᩥᨠ᩠ᨡᩪ ᨶᩣᨶᩣᩈᩴᩅᩤᩈᨠᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ. ᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨲᨴᩉᩩᨸᩮᩣᩈᨳᩮ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩣ ᩋᩣᩅᩤᩈᩣ ᩅᩤ ᩋᨶᩣᩅᩤᩈᩣ ᩅᩤ ᩈᨽᩥᨠ᩠ᨡᩩᨠᩮᩣ ᩋᩣᩅᩤᩈᩮᩣ ᩅᩤ ᩋᨶᩣᩅᩤᩈᩮᩣ ᩅᩤ ᨣᨶ᩠ᨲᨻ᩠ᨻᩮᩣ, ᨿᨲ᩠ᨳᩔᩩ ᨽᩥᨠ᩠ᨡᩪ ᨶᩣᨶᩣᩈᩴᩅᩤᩈᨠᩣ, ᩋᨬ᩠ᨬᨲᩕ ᩈᩴᨥᩮᨶ ᩋᨬ᩠ᨬᨲᩕ ᩋᨶ᩠ᨲᩁᩣᨿᩣ.
</Paragraph>
