<Paragraph number="1075" book="vi-maha">
ᨲᩮᨶ ᨡᩮᩣ ᨸᨶ ᩈᨾᨿᩮᨶ ᨠᩮᩣᩈᩃᩮᩈᩩ ᨩᨶᨸᨴᩮ ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩈᩅᩁᨽᨿᩴ ᩋᩉᩮᩣᩈᩥ. ᨽᩥᨠ᩠ᨡᩪ ᨶᩣᩈᨠ᩠ᨡᩥᩴᩈᩩ ᨲᩮᩅᩤᨧᩥᨠᩴ ᨸᩅᩤᩁᩮᨲᩩᩴ. ᨽᨣᩅᨲᩮᩣ ᩑᨲᨾᨲ᩠ᨳᩴ ᩋᩣᩁᩮᩣᨧᩮᩈᩩᩴ. “ᩋᨶᩩᨩᩣᨶᩣᨾᩥ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨴ᩠ᩅᩮᩅᩤᨧᩥᨠᩴ ᨸᩅᩤᩁᩮᨲᩩᨶ᩺”ᨲᩥ.
</Paragraph>

<Paragraph number="1076" book="vi-maha">
ᨻᩣᩊ᩠ᩉᨲᩁᩴ ᩈᩅᩁᨽᨿᩴ ᩋᩉᩮᩣᩈᩥ. ᨽᩥᨠ᩠ᨡᩪ ᨶᩣᩈᨠ᩠ᨡᩥᩴᩈᩩ ᨴ᩠ᩅᩮᩅᩤᨧᩥᨠᩴ ᨸᩅᩤᩁᩮᨲᩩᩴ. ᨽᨣᩅᨲᩮᩣ ᩑᨲᨾᨲ᩠ᨳᩴ ᩋᩣᩁᩮᩣᨧᩮᩈᩩᩴ. “ᩋᨶᩩᨩᩣᨶᩣᨾᩥ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩑᨠᩅᩤᨧᩥᨠᩴ ᨸᩅᩤᩁᩮᨲᩩᨶ᩺”ᨲᩥ.
</Paragraph>

<Paragraph number="1077" book="vi-maha">
ᨻᩣᩊ᩠ᩉᨲᩁᩴ ᩈᩅᩁᨽᨿᩴ ᩋᩉᩮᩣᩈᩥ. ᨽᩥᨠ᩠ᨡᩪ ᨶᩣᩈᨠ᩠ᨡᩥᩴᩈᩩ ᩑᨠᩅᩤᨧᩥᨠᩴ ᨸᩅᩤᩁᩮᨲᩩᩴ. ᨽᨣᩅᨲᩮᩣ ᩑᨲᨾᨲ᩠ᨳᩴ ᩋᩣᩁᩮᩣᨧᩮᩈᩩᩴ. “ᩋᨶᩩᨩᩣᨶᩣᨾᩥ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩈᨾᩣᨶᩅᩔᩥᨠᩴ ᨸᩅᩤᩁᩮᨲᩩᨶ᩺”ᨲᩥ.
</Paragraph>

<Paragraph number="1078" book="vi-maha">
ᨲᩮᨶ ᨡᩮᩣ ᨸᨶ ᩈᨾᨿᩮᨶ ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᨾᨶᩩᩔᩮᩉᩥ ᨴᩤᨶᩴ ᨴᩮᨶ᩠ᨲᩮᩉᩥ ᨿᩮᨽᩩᨿ᩠ᨿᩮᨶ ᩁᨲ᩠ᨲᩥ ᨡᩮᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ. ᩋᨳ ᨡᩮᩣ ᨲᩮᩈᩴ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩑᨲᨴᩉᩮᩣᩈᩥ—  “ᨾᨶᩩᩔᩮᩉᩥ ᨴᩤᨶᩴ ᨴᩮᨶ᩠ᨲᩮᩉᩥ ᨿᩮᨽᩩᨿ᩠ᨿᩮᨶ ᩁᨲ᩠ᨲᩥ ᨡᩮᨸᩥᨲᩣ. ᩈᨧᩮ ᩈᩴᨥᩮᩣ ᨲᩮᩅᩤᨧᩥᨠᩴ ᨸᩅᩤᩁᩮᩔᨲᩥ, ᩋᨸ᩠ᨸᩅᩤᩁᩥᨲᩮᩣᩅ ᩈᩴᨥᩮᩣ ᨽᩅᩥᩔᨲᩥ, ᩋᨳᩣᨿᩴ ᩁᨲ᩠ᨲᩥ ᩅᩥᨽᩣᨿᩥᩔᨲᩥ. ᨠᨳᩴ ᨶᩩ ᨡᩮᩣ ᩋᨾ᩠ᩉᩮᩉᩥ ᨸᨭᩥᨸᨩ᩠ᨩᩥᨲᨻ᩠ᨻᨶ᩺”ᨲᩥ? ᨽᨣᩅᨲᩮᩣ ᩑᨲᨾᨲ᩠ᨳᩴ ᩋᩣᩁᩮᩣᨧᩮᩈᩩᩴ.
</Paragraph>

<Paragraph number="1079" book="vi-maha">
“ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᨾᨶᩩᩔᩮᩉᩥ ᨴᩤᨶᩴ ᨴᩮᨶ᩠ᨲᩮᩉᩥ ᨿᩮᨽᩩᨿ᩠ᨿᩮᨶ ᩁᨲ᩠ᨲᩥ ᨡᩮᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ. ᨲᨲᩕ ᨧᩮ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩑᩅᩴ ᩉᩮᩣᨲᩥ—  ‘ᨾᨶᩩᩔᩮᩉᩥ ᨴᩤᨶᩴ ᨴᩮᨶ᩠ᨲᩮᩉᩥ ᨿᩮᨽᩩᨿ᩠ᨿᩮᨶ ᩁᨲ᩠ᨲᩥ ᨡᩮᨸᩥᨲᩣ. ᩈᨧᩮ ᩈᩴᨥᩮᩣ ᨲᩮᩅᩤᨧᩥᨠᩴ ᨸᩅᩤᩁᩮᩔᨲᩥ, ᩋᨸ᩠ᨸᩅᩤᩁᩥᨲᩮᩣᩅ ᩈᩴᨥᩮᩣ ᨽᩅᩥᩔᨲᩥ, ᩋᨳᩣᨿᩴ ᩁᨲ᩠ᨲᩥ ᩅᩥᨽᩣᨿᩥᩔᨲᩦ’ᨲᩥ, ᨻ᩠ᨿᨲ᩠ᨲᩮᨶ ᨽᩥᨠ᩠ᨡᩩᨶᩣ ᨸᨭᩥᨻᩃᩮᨶ ᩈᩴᨥᩮᩣ ᨬᩣᨸᩮᨲᨻ᩠ᨻᩮᩣ—
</Paragraph>

<Paragraph number="1080" book="vi-maha">
‘ᩈᩩᨱᩣᨲᩩ ᨾᩮ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩮᩣ. ᨾᨶᩩᩔᩮᩉᩥ ᨴᩤᨶᩴ ᨴᩮᨶ᩠ᨲᩮᩉᩥ ᨿᩮᨽᩩᨿ᩠ᨿᩮᨶ ᩁᨲ᩠ᨲᩥ ᨡᩮᨸᩥᨲᩣ. ᩈᨧᩮ ᩈᩴᨥᩮᩣ ᨲᩮᩅᩤᨧᩥᨠᩴ ᨸᩅᩤᩁᩮᩔᨲᩥ, ᩋᨸ᩠ᨸᩅᩤᩁᩥᨲᩮᩣᩅ ᩈᩴᨥᩮᩣ ᨽᩅᩥᩔᨲᩥ, ᩋᨳᩣᨿᩴ ᩁᨲ᩠ᨲᩥ ᩅᩥᨽᩣᨿᩥᩔᨲᩥ. ᨿᨴᩥ ᩈᩴᨥᩔ ᨸᨲ᩠ᨲᨠᩃᩖᩴ, ᩈᩴᨥᩮᩣ ᨴ᩠ᩅᩮᩅᩤᨧᩥᨠᩴ, ᩑᨠᩅᩤᨧᩥᨠᩴ, ᩈᨾᩣᨶᩅᩔᩥᨠᩴ ᨸᩅᩤᩁᩮᨿ᩠ᨿᩣ’ᨲᩥ.
</Paragraph>

<Paragraph number="1081" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᨽᩥᨠ᩠ᨡᩪᩉᩥ ᨵᨾ᩠ᨾᩴ ᨽᨱᨶ᩠ᨲᩮᩉᩥ …ᨸᩮ…  ᩈᩩᨲ᩠ᨲᨶ᩠ᨲᩥᨠᩮᩉᩥ ᩈᩩᨲ᩠ᨲᨶ᩠ᨲᩴ ᩈᩘᨣᩤᨿᨶ᩠ᨲᩮᩉᩥ…  ᩅᩥᨶᨿᨵᩁᩮᩉᩥ ᩅᩥᨶᨿᩴ ᩅᩥᨶᩥᨧ᩠ᨨᩥᨶᨶ᩠ᨲᩮᩉᩥ…  ᨵᨾ᩠ᨾᨠᨳᩥᨠᩮᩉᩥ ᨵᨾ᩠ᨾᩴ ᩈᩣᨠᨧ᩠ᨨᨶ᩠ᨲᩮᩉᩥ…  ᨽᩥᨠ᩠ᨡᩪᩉᩥ ᨠᩃᩉᩴ ᨠᩁᩮᩣᨶ᩠ᨲᩮᩉᩥ ᨿᩮᨽᩩᨿ᩠ᨿᩮᨶ ᩁᨲ᩠ᨲᩥ ᨡᩮᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ. ᨲᨲᩕ ᨧᩮ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩑᩅᩴ ᩉᩮᩣᨲᩥ—  ‘ᨽᩥᨠ᩠ᨡᩪᩉᩥ ᨠᩃᩉᩴ ᨠᩁᩮᩣᨶ᩠ᨲᩮᩉᩥ ᨿᩮᨽᩩᨿ᩠ᨿᩮᨶ ᩁᨲ᩠ᨲᩥ ᨡᩮᨸᩥᨲᩣ. ᩈᨧᩮ ᩈᩴᨥᩮᩣ ᨲᩮᩅᩤᨧᩥᨠᩴ ᨸᩅᩤᩁᩮᩔᨲᩥ, ᩋᨸ᩠ᨸᩅᩤᩁᩥᨲᩮᩣᩅ ᩈᩴᨥᩮᩣ ᨽᩅᩥᩔᨲᩥ, ᩋᨳᩣᨿᩴ ᩁᨲ᩠ᨲᩥ ᩅᩥᨽᩣᨿᩥᩔᨲᩦ’ᨲᩥ, ᨻ᩠ᨿᨲ᩠ᨲᩮᨶ ᨽᩥᨠ᩠ᨡᩩᨶᩣ ᨸᨭᩥᨻᩃᩮᨶ ᩈᩴᨥᩮᩣ ᨬᩣᨸᩮᨲᨻ᩠ᨻᩮᩣ—
</Paragraph>

<Paragraph number="1082" book="vi-maha">
‘ᩈᩩᨱᩣᨲᩩ ᨾᩮ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩮᩣ. ᨽᩥᨠ᩠ᨡᩪᩉᩥ ᨠᩃᩉᩴ ᨠᩁᩮᩣᨶ᩠ᨲᩮᩉᩥ ᨿᩮᨽᩩᨿ᩠ᨿᩮᨶ ᩁᨲ᩠ᨲᩥ ᨡᩮᨸᩥᨲᩣ. ᩈᨧᩮ ᩈᩴᨥᩮᩣ ᨲᩮᩅᩤᨧᩥᨠᩴ ᨸᩅᩤᩁᩮᩔᨲᩥ, ᩋᨸ᩠ᨸᩅᩤᩁᩥᨲᩮᩣᩅ ᩈᩴᨥᩮᩣ ᨽᩅᩥᩔᨲᩥ, ᩋᨳᩣᨿᩴ ᩁᨲ᩠ᨲᩥ ᩅᩥᨽᩣᨿᩥᩔᨲᩥ. ᨿᨴᩥ ᩈᩴᨥᩔ ᨸᨲ᩠ᨲᨠᩃᩖᩴ, ᩈᩴᨥᩮᩣ ᨴ᩠ᩅᩮᩅᩤᨧᩥᨠᩴ, ᩑᨠᩅᩤᨧᩥᨠᩴ, ᩈᨾᩣᨶᩅᩔᩥᨠᩴ ᨸᩅᩤᩁᩮᨿ᩠ᨿᩣ’”ᨲᩥ.
</Paragraph>

<Paragraph number="1083" book="vi-maha">
ᨲᩮᨶ ᨡᩮᩣ ᨸᨶ ᩈᨾᨿᩮᨶ ᨠᩮᩣᩈᩃᩮᩈᩩ ᨩᨶᨸᨴᩮ ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᨾᩉᩣᨽᩥᨠ᩠ᨡᩩᩈᩴᨥᩮᩣ ᩈᨶ᩠ᨶᩥᨸᨲᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ, ᨸᩁᩥᨲ᩠ᨲᨬ᩠ᨧ ᩋᨶᩮᩣᩅᩔᩥᨠᩴ ᩉᩮᩣᨲᩥ, ᨾᩉᩣ ᨧ ᨾᩮᨥᩮᩣ ᩏᨣ᩠ᨣᨲᩮᩣ ᩉᩮᩣᨲᩥ. ᩋᨳ ᨡᩮᩣ ᨲᩮᩈᩴ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩑᨲᨴᩉᩮᩣᩈᩥ—  “ᩋᨿᩴ ᨡᩮᩣ ᨾᩉᩣᨽᩥᨠ᩠ᨡᩩᩈᩴᨥᩮᩣ ᩈᨶ᩠ᨶᩥᨸᨲᩥᨲᩮᩣ, ᨸᩁᩥᨲ᩠ᨲᨬ᩠ᨧ ᩋᨶᩮᩣᩅᩔᩥᨠᩴ, ᨾᩉᩣ ᨧ ᨾᩮᨥᩮᩣ ᩏᨣ᩠ᨣᨲᩮᩣ. ᩈᨧᩮ ᩈᩴᨥᩮᩣ ᨲᩮᩅᩤᨧᩥᨠᩴ ᨸᩅᩤᩁᩮᩔᨲᩥ, ᩋᨸ᩠ᨸᩅᩤᩁᩥᨲᩮᩣᩅ ᩈᩴᨥᩮᩣ ᨽᩅᩥᩔᨲᩥ, ᩋᨳᩣᨿᩴ ᨾᩮᨥᩮᩣ ᨸᩅᩔᩥᩔᨲᩥ. ᨠᨳᩴ ᨶᩩ ᨡᩮᩣ ᩋᨾ᩠ᩉᩮᩉᩥ ᨸᨭᩥᨸᨩ᩠ᨩᩥᨲᨻ᩠ᨻᨶ᩺”ᨲᩥ? ᨽᨣᩅᨲᩮᩣ ᩑᨲᨾᨲ᩠ᨳᩴ ᩋᩣᩁᩮᩣᨧᩮᩈᩩᩴ.
</Paragraph>

<Paragraph number="1084" book="vi-maha">
“ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᨾᩉᩣᨽᩥᨠ᩠ᨡᩩᩈᩴᨥᩮᩣ ᩈᨶ᩠ᨶᩥᨸᨲᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ, ᨸᩁᩥᨲ᩠ᨲᨬ᩠ᨧ ᩋᨶᩮᩣᩅᩔᩥᨠᩴ ᩉᩮᩣᨲᩥ, ᨾᩉᩣ ᨧ ᨾᩮᨥᩮᩣ ᩏᨣ᩠ᨣᨲᩮᩣ ᩉᩮᩣᨲᩥ. ᨲᨲᩕ ᨧᩮ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩑᩅᩴ ᩉᩮᩣᨲᩥ—  ‘ᩋᨿᩴ ᨡᩮᩣ ᨾᩉᩣᨽᩥᨠ᩠ᨡᩩᩈᩴᨥᩮᩣ ᩈᨶ᩠ᨶᩥᨸᨲᩥᨲᩮᩣ, ᨸᩁᩥᨲ᩠ᨲᨬ᩠ᨧ ᩋᨶᩮᩣᩅᩔᩥᨠᩴ, ᨾᩉᩣ ᨧ ᨾᩮᨥᩮᩣ ᩏᨣ᩠ᨣᨲᩮᩣ. ᩈᨧᩮ ᩈᩴᨥᩮᩣ ᨲᩮᩅᩤᨧᩥᨠᩴ ᨸᩅᩤᩁᩮᩔᨲᩥ, ᩋᨸ᩠ᨸᩅᩤᩁᩥᨲᩮᩣᩅ ᩈᩴᨥᩮᩣ ᨽᩅᩥᩔᨲᩥ, ᩋᨳᩣᨿᩴ ᨾᩮᨥᩮᩣ ᨸᩅᩔᩥᩔᨲᩦ’ᨲᩥ. ᨻ᩠ᨿᨲ᩠ᨲᩮᨶ ᨽᩥᨠ᩠ᨡᩩᨶᩣ ᨸᨭᩥᨻᩃᩮᨶ ᩈᩴᨥᩮᩣ ᨬᩣᨸᩮᨲᨻ᩠ᨻᩮᩣ—
</Paragraph>

<Paragraph number="1085" book="vi-maha">
‘ᩈᩩᨱᩣᨲᩩ ᨾᩮ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩮᩣ. ᩋᨿᩴ ᨾᩉᩣᨽᩥᨠ᩠ᨡᩩᩈᩴᨥᩮᩣ ᩈᨶ᩠ᨶᩥᨸᨲᩥᨲᩮᩣ, ᨸᩁᩥᨲ᩠ᨲᨬ᩠ᨧ ᩋᨶᩮᩣᩅᩔᩥᨠᩴ, ᨾᩉᩣ ᨧ ᨾᩮᨥᩮᩣ ᩏᨣ᩠ᨣᨲᩮᩣ. ᩈᨧᩮ ᩈᩴᨥᩮᩣ ᨲᩮᩅᩤᨧᩥᨠᩴ ᨸᩅᩤᩁᩮᩔᨲᩥ, ᩋᨸ᩠ᨸᩅᩤᩁᩥᨲᩮᩣᩅ ᩈᩴᨥᩮᩣ ᨽᩅᩥᩔᨲᩥ, ᩋᨳᩣᨿᩴ ᨾᩮᨥᩮᩣ ᨸᩅᩔᩥᩔᨲᩥ. ᨿᨴᩥ ᩈᩴᨥᩔ ᨸᨲ᩠ᨲᨠᩃᩖᩴ, ᩈᩴᨥᩮᩣ ᨴ᩠ᩅᩮᩅᩤᨧᩥᨠᩴ, ᩑᨠᩅᩤᨧᩥᨠᩴ, ᩈᨾᩣᨶᩅᩔᩥᨠᩴ ᨸᩅᩤᩁᩮᨿ᩠ᨿᩣ’ᨲᩥ.
</Paragraph>

<Paragraph number="1086" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩁᩣᨩᨶ᩠ᨲᩁᩣᨿᩮᩣ ᩉᩮᩣᨲᩥ …ᨸᩮ…  ᨧᩮᩣᩁᨶ᩠ᨲᩁᩣᨿᩮᩣ ᩉᩮᩣᨲᩥ…  ᩋᨣ᩠ᨿᨶ᩠ᨲᩁᩣᨿᩮᩣ ᩉᩮᩣᨲᩥ…  ᩏᨴᨠᨶ᩠ᨲᩁᩣᨿᩮᩣ ᩉᩮᩣᨲᩥ…  ᨾᨶᩩᩔᨶ᩠ᨲᩁᩣᨿᩮᩣ ᩉᩮᩣᨲᩥ…  ᩋᨾᨶᩩᩔᨶ᩠ᨲᩁᩣᨿᩮᩣ ᩉᩮᩣᨲᩥ…  ᩅᩤᩊᨶ᩠ᨲᩁᩣᨿᩮᩣ ᩉᩮᩣᨲᩥ…  ᩈᩁᩦᩈᨸᨶ᩠ᨲᩁᩣᨿᩮᩣ ᩉᩮᩣᨲᩥ…  ᨩᩦᩅᩥᨲᨶ᩠ᨲᩁᩣᨿᩮᩣ ᩉᩮᩣᨲᩥ…  ᨻᩕᩉ᩠ᨾᨧᩁᩥᨿᨶ᩠ᨲᩁᩣᨿᩮᩣ ᩉᩮᩣᨲᩥ. ᨲᨲᩕ ᨧᩮ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩑᩅᩴ ᩉᩮᩣᨲᩥ—  ‘ᩋᨿᩴ ᨡᩮᩣ ᨻᩕᩉ᩠ᨾᨧᩁᩥᨿᨶ᩠ᨲᩁᩣᨿᩮᩣ. ᩈᨧᩮ ᩈᩴᨥᩮᩣ ᨲᩮᩅᩤᨧᩥᨠᩴ ᨸᩅᩤᩁᩮᩔᨲᩥ, ᩋᨸ᩠ᨸᩅᩤᩁᩥᨲᩮᩣᩅ ᩈᩴᨥᩮᩣ ᨽᩅᩥᩔᨲᩥ, ᩋᨳᩣᨿᩴ ᨻᩕᩉ᩠ᨾᨧᩁᩥᨿᨶ᩠ᨲᩁᩣᨿᩮᩣ ᨽᩅᩥᩔᨲᩦ’ᨲᩥ, ᨻ᩠ᨿᨲ᩠ᨲᩮᨶ ᨽᩥᨠ᩠ᨡᩩᨶᩣ ᨸᨭᩥᨻᩃᩮᨶ ᩈᩴᨥᩮᩣ ᨬᩣᨸᩮᨲᨻ᩠ᨻᩮᩣ—
</Paragraph>

<Paragraph number="1087" book="vi-maha">
‘ᩈᩩᨱᩣᨲᩩ ᨾᩮ, ᨽᨶ᩠ᨲᩮ, ᩈᩴᨥᩮᩣ. ᩋᨿᩴ ᨻᩕᩉ᩠ᨾᨧᩁᩥᨿᨶ᩠ᨲᩁᩣᨿᩮᩣ. ᩈᨧᩮ ᩈᩴᨥᩮᩣ ᨲᩮᩅᩤᨧᩥᨠᩴ ᨸᩅᩤᩁᩮᩔᨲᩥ, ᩋᨸ᩠ᨸᩅᩤᩁᩥᨲᩮᩣᩅ ᩈᩴᨥᩮᩣ ᨽᩅᩥᩔᨲᩥ, ᩋᨳᩣᨿᩴ ᨻᩕᩉ᩠ᨾᨧᩁᩥᨿᨶ᩠ᨲᩁᩣᨿᩮᩣ ᨽᩅᩥᩔᨲᩥ. ᨿᨴᩥ ᩈᩴᨥᩔ ᨸᨲ᩠ᨲᨠᩃᩖᩴ, ᩈᩴᨥᩮᩣ ᨴ᩠ᩅᩮᩅᩤᨧᩥᨠᩴ, ᩑᨠᩅᩤᨧᩥᨠᩴ, ᩈᨾᩣᨶᩅᩔᩥᨠᩴ ᨸᩅᩤᩁᩮᨿ᩠ᨿᩣ’”ᨲᩥ.
</Paragraph>
