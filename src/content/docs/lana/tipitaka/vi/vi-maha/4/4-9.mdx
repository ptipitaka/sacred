<Paragraph number="999" book="vi-maha">
ᨲᩮᨶ ᨡᩮᩣ ᨸᨶ ᩈᨾᨿᩮᨶ ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩈᨶ᩠ᨶᩥᨸᨲᩥᩴᩈᩩ, ᨸᨬ᩠ᨧ ᩅᩤ ᩋᨲᩥᩁᩮᨠᩣ ᩅᩤ. ᨲᩮ ᨶ ᨩᩣᨶᩥᩴᩈᩩ—  “ᩋᨲ᩠ᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᨶᩣᨣᨲᩣ”ᨲᩥ. ᨲᩮ ᨵᨾ᩠ᨾᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᩥᨶᨿᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᨣ᩠ᨣᩤ ᩈᨾᨣ᩠ᨣᩈᨬ᩠ᨬᩥᨶᩮᩣ ᨸᩅᩤᩁᩮᩈᩩᩴ. ᨲᩮᩉᩥ ᨸᩅᩤᩁᩥᨿᨾᩣᨶᩮ ᩋᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᩣᨣᨧ᩠ᨨᩥᩴᩈᩩ ᨻᩉᩩᨲᩁᩣ. ᨽᨣᩅᨲᩮᩣ ᩑᨲᨾᨲ᩠ᨳᩴ ᩋᩣᩁᩮᩣᨧᩮᩈᩩᩴ.
</Paragraph>

<Paragraph number="1000" book="vi-maha">
“ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩈᨶ᩠ᨶᩥᨸᨲᨶ᩠ᨲᩥ, ᨸᨬ᩠ᨧ ᩅᩤ ᩋᨲᩥᩁᩮᨠᩣ ᩅᩤ. ᨲᩮ ᨶ ᨩᩣᨶᨶ᩠ᨲᩥ—  ‘ᩋᨲ᩠ᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᨶᩣᨣᨲᩣ’ᨲᩥ. ᨲᩮ ᨵᨾ᩠ᨾᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᩥᨶᨿᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᨣ᩠ᨣᩤ ᩈᨾᨣ᩠ᨣᩈᨬ᩠ᨬᩥᨶᩮᩣ ᨸᩅᩤᩁᩮᨶ᩠ᨲᩥ. ᨲᩮᩉᩥ ᨸᩅᩤᩁᩥᨿᨾᩣᨶᩮ ᩋᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩥ ᨻᩉᩩᨲᩁᩣ. ᨲᩮᩉᩥ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩪᩉᩥ ᨸᩩᨶ ᨸᩅᩤᩁᩮᨲᨻ᩠ᨻᩴ. ᨸᩅᩤᩁᩥᨲᩣᨶᩴ ᩋᨶᩣᨸᨲ᩠ᨲᩥ. (1)
</Paragraph>

<Paragraph number="1001" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩈᨶ᩠ᨶᩥᨸᨲᨶ᩠ᨲᩥ, ᨸᨬ᩠ᨧ ᩅᩤ ᩋᨲᩥᩁᩮᨠᩣ ᩅᩤ. ᨲᩮ ᨶ ᨩᩣᨶᨶ᩠ᨲᩥ—  ‘ᩋᨲ᩠ᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᨶᩣᨣᨲᩣ’ᨲᩥ. ᨲᩮ ᨵᨾ᩠ᨾᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᩥᨶᨿᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᨣ᩠ᨣᩤ ᩈᨾᨣ᩠ᨣᩈᨬ᩠ᨬᩥᨶᩮᩣ ᨸᩅᩤᩁᩮᨶ᩠ᨲᩥ. ᨲᩮᩉᩥ ᨸᩅᩤᩁᩥᨿᨾᩣᨶᩮ ᩋᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩥ ᩈᨾᩈᨾᩣ. ᨸᩅᩤᩁᩥᨲᩣ ᩈᩩᨸ᩠ᨸᩅᩤᩁᩥᨲᩣ, ᩋᩅᩈᩮᩈᩮᩉᩥ ᨸᩅᩤᩁᩮᨲᨻ᩠ᨻᩴ. ᨸᩅᩤᩁᩥᨲᩣᨶᩴ ᩋᨶᩣᨸᨲ᩠ᨲᩥ. (2)
</Paragraph>

<Paragraph number="1002" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩈᨶ᩠ᨶᩥᨸᨲᨶ᩠ᨲᩥ, ᨸᨬ᩠ᨧ ᩅᩤ ᩋᨲᩥᩁᩮᨠᩣ ᩅᩤ. ᨲᩮ ᨶ ᨩᩣᨶᨶ᩠ᨲᩥ—  ‘ᩋᨲ᩠ᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᨶᩣᨣᨲᩣ’ᨲᩥ. ᨲᩮ ᨵᨾ᩠ᨾᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᩥᨶᨿᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᨣ᩠ᨣᩤ ᩈᨾᨣ᩠ᨣᩈᨬ᩠ᨬᩥᨶᩮᩣ ᨸᩅᩤᩁᩮᨶ᩠ᨲᩥ. ᨲᩮᩉᩥ ᨸᩅᩤᩁᩥᨿᨾᩣᨶᩮ ᩋᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩥ ᨳᩮᩣᨠᨲᩁᩣ. ᨸᩅᩤᩁᩥᨲᩣ ᩈᩩᨸ᩠ᨸᩅᩤᩁᩥᨲᩣ, ᩋᩅᩈᩮᩈᩮᩉᩥ ᨸᩅᩤᩁᩮᨲᨻ᩠ᨻᩴ. ᨸᩅᩤᩁᩥᨲᩣᨶᩴ ᩋᨶᩣᨸᨲ᩠ᨲᩥ. (3)
</Paragraph>

<Paragraph number="1003" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩈᨶ᩠ᨶᩥᨸᨲᨶ᩠ᨲᩥ, ᨸᨬ᩠ᨧ ᩅᩤ ᩋᨲᩥᩁᩮᨠᩣ ᩅᩤ. ᨲᩮ ᨶ ᨩᩣᨶᨶ᩠ᨲᩥ—  ‘ᩋᨲ᩠ᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᨶᩣᨣᨲᩣ’ᨲᩥ. ᨲᩮ ᨵᨾ᩠ᨾᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᩥᨶᨿᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᨣ᩠ᨣᩤ ᩈᨾᨣ᩠ᨣᩈᨬ᩠ᨬᩥᨶᩮᩣ ᨸᩅᩤᩁᩮᨶ᩠ᨲᩥ. ᨲᩮᩉᩥ ᨸᩅᩤᩁᩥᨲᨾᨲ᩠ᨲᩮ ᩋᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩥ ᨻᩉᩩᨲᩁᩣ. ᨲᩮᩉᩥ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩪᩉᩥ ᨸᩩᨶ ᨸᩅᩤᩁᩮᨲᨻ᩠ᨻᩴ. ᨸᩅᩤᩁᩥᨲᩣᨶᩴ ᩋᨶᩣᨸᨲ᩠ᨲᩥ. (4)
</Paragraph>

<Paragraph number="1004" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩈᨶ᩠ᨶᩥᨸᨲᨶ᩠ᨲᩥ, ᨸᨬ᩠ᨧ ᩅᩤ ᩋᨲᩥᩁᩮᨠᩣ ᩅᩤ. ᨲᩮ ᨶ ᨩᩣᨶᨶ᩠ᨲᩥ—  ‘ᩋᨲ᩠ᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᨶᩣᨣᨲᩣ’ᨲᩥ. ᨲᩮ ᨵᨾ᩠ᨾᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᩥᨶᨿᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᨣ᩠ᨣᩤ ᩈᨾᨣ᩠ᨣᩈᨬ᩠ᨬᩥᨶᩮᩣ ᨸᩅᩤᩁᩮᨶ᩠ᨲᩥ. ᨲᩮᩉᩥ ᨸᩅᩤᩁᩥᨲᨾᨲ᩠ᨲᩮ ᩋᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩥ ᩈᨾᩈᨾᩣ. ᨸᩅᩤᩁᩥᨲᩣ ᩈᩩᨸ᩠ᨸᩅᩤᩁᩥᨲᩣ, ᨲᩮᩈᩴ ᩈᨶ᩠ᨲᩥᨠᩮ ᨸᩅᩤᩁᩮᨲᨻ᩠ᨻᩴ. ᨸᩅᩤᩁᩥᨲᩣᨶᩴ ᩋᨶᩣᨸᨲ᩠ᨲᩥ. (5)
</Paragraph>

<Paragraph number="1005" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩈᨶ᩠ᨶᩥᨸᨲᨶ᩠ᨲᩥ, ᨸᨬ᩠ᨧ ᩅᩤ ᩋᨲᩥᩁᩮᨠᩣ ᩅᩤ. ᨲᩮ ᨶ ᨩᩣᨶᨶ᩠ᨲᩥ—  ‘ᩋᨲ᩠ᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᨶᩣᨣᨲᩣ’ᨲᩥ. ᨲᩮ ᨵᨾ᩠ᨾᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᩥᨶᨿᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᨣ᩠ᨣᩤ ᩈᨾᨣ᩠ᨣᩈᨬ᩠ᨬᩥᨶᩮᩣ ᨸᩅᩤᩁᩮᨶ᩠ᨲᩥ. ᨲᩮᩉᩥ ᨸᩅᩤᩁᩥᨲᨾᨲ᩠ᨲᩮ ᩋᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩥ ᨳᩮᩣᨠᨲᩁᩣ. ᨸᩅᩤᩁᩥᨲᩣ ᩈᩩᨸ᩠ᨸᩅᩤᩁᩥᨲᩣ, ᨲᩮᩈᩴ ᩈᨶ᩠ᨲᩥᨠᩮ ᨸᩅᩤᩁᩮᨲᨻ᩠ᨻᩴ. ᨸᩅᩤᩁᩥᨲᩣᨶᩴ ᩋᨶᩣᨸᨲ᩠ᨲᩥ. (6)
</Paragraph>

<Paragraph number="1006" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩈᨶ᩠ᨶᩥᨸᨲᨶ᩠ᨲᩥ, ᨸᨬ᩠ᨧ ᩅᩤ ᩋᨲᩥᩁᩮᨠᩣ ᩅᩤ. ᨲᩮ ᨶ ᨩᩣᨶᨶ᩠ᨲᩥ—  ‘ᩋᨲ᩠ᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᨶᩣᨣᨲᩣ’ᨲᩥ. ᨲᩮ ᨵᨾ᩠ᨾᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᩥᨶᨿᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᨣ᩠ᨣᩤ ᩈᨾᨣ᩠ᨣᩈᨬ᩠ᨬᩥᨶᩮᩣ ᨸᩅᩤᩁᩮᨶ᩠ᨲᩥ. ᨲᩮᩉᩥ ᨸᩅᩤᩁᩥᨲᨾᨲ᩠ᨲᩮ, ᩋᩅᩩᨭ᩠ᨮᩥᨲᩣᨿ ᨸᩁᩥᩈᩣᨿ, ᩋᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩥ ᨻᩉᩩᨲᩁᩣ. ᨲᩮᩉᩥ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩪᩉᩥ ᨸᩩᨶ ᨸᩅᩤᩁᩮᨲᨻ᩠ᨻᩴ. ᨸᩅᩤᩁᩥᨲᩣᨶᩴ ᩋᨶᩣᨸᨲ᩠ᨲᩥ. (7)
</Paragraph>

<Paragraph number="1007" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩈᨶ᩠ᨶᩥᨸᨲᨶ᩠ᨲᩥ, ᨸᨬ᩠ᨧ ᩅᩤ ᩋᨲᩥᩁᩮᨠᩣ ᩅᩤ. ᨲᩮ ᨶ ᨩᩣᨶᨶ᩠ᨲᩥ—  ‘ᩋᨲ᩠ᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᨶᩣᨣᨲᩣ’ᨲᩥ. ᨲᩮ ᨵᨾ᩠ᨾᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᩥᨶᨿᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᨣ᩠ᨣᩤ ᩈᨾᨣ᩠ᨣᩈᨬ᩠ᨬᩥᨶᩮᩣ ᨸᩅᩤᩁᩮᨶ᩠ᨲᩥ. ᨲᩮᩉᩥ ᨸᩅᩤᩁᩥᨲᨾᨲ᩠ᨲᩮ, ᩋᩅᩩᨭ᩠ᨮᩥᨲᩣᨿ ᨸᩁᩥᩈᩣᨿ, ᩋᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩥ ᩈᨾᩈᨾᩣ. ᨸᩅᩤᩁᩥᨲᩣ ᩈᩩᨸ᩠ᨸᩅᩤᩁᩥᨲᩣ, ᨲᩮᩈᩴ ᩈᨶ᩠ᨲᩥᨠᩮ ᨸᩅᩤᩁᩮᨲᨻ᩠ᨻᩴ. ᨸᩅᩤᩁᩥᨲᩣᨶᩴ ᩋᨶᩣᨸᨲ᩠ᨲᩥ. (8)
</Paragraph>

<Paragraph number="1008" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩈᨶ᩠ᨶᩥᨸᨲᨶ᩠ᨲᩥ, ᨸᨬ᩠ᨧ ᩅᩤ ᩋᨲᩥᩁᩮᨠᩣ ᩅᩤ. ᨲᩮ ᨶ ᨩᩣᨶᨶ᩠ᨲᩥ—  ‘ᩋᨲ᩠ᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᨶᩣᨣᨲᩣ’ᨲᩥ. ᨲᩮ ᨵᨾ᩠ᨾᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᩥᨶᨿᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᨣ᩠ᨣᩤ ᩈᨾᨣ᩠ᨣᩈᨬ᩠ᨬᩥᨶᩮᩣ ᨸᩅᩤᩁᩮᨶ᩠ᨲᩥ. ᨲᩮᩉᩥ ᨸᩅᩤᩁᩥᨲᨾᨲ᩠ᨲᩮ, ᩋᩅᩩᨭ᩠ᨮᩥᨲᩣᨿ ᨸᩁᩥᩈᩣᨿ, ᩋᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩥ ᨳᩮᩣᨠᨲᩁᩣ. ᨸᩅᩤᩁᩥᨲᩣ ᩈᩩᨸ᩠ᨸᩅᩤᩁᩥᨲᩣ, ᨲᩮᩈᩴ ᩈᨶ᩠ᨲᩥᨠᩮ ᨸᩅᩤᩁᩮᨲᨻ᩠ᨻᩴ. ᨸᩅᩤᩁᩥᨲᩣᨶᩴ ᩋᨶᩣᨸᨲ᩠ᨲᩥ. (9)
</Paragraph>

<Paragraph number="1009" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩈᨶ᩠ᨶᩥᨸᨲᨶ᩠ᨲᩥ, ᨸᨬ᩠ᨧ ᩅᩤ ᩋᨲᩥᩁᩮᨠᩣ ᩅᩤ. ᨲᩮ ᨶ ᨩᩣᨶᨶ᩠ᨲᩥ—  ‘ᩋᨲ᩠ᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᨶᩣᨣᨲᩣ’ᨲᩥ. ᨲᩮ ᨵᨾ᩠ᨾᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᩥᨶᨿᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᨣ᩠ᨣᩤ ᩈᨾᨣ᩠ᨣᩈᨬ᩠ᨬᩥᨶᩮᩣ ᨸᩅᩤᩁᩮᨶ᩠ᨲᩥ. ᨲᩮᩉᩥ ᨸᩅᩤᩁᩥᨲᨾᨲ᩠ᨲᩮ, ᩑᨠᨧ᩠ᨧᩣᨿ ᩅᩩᨭ᩠ᨮᩥᨲᩣᨿ ᨸᩁᩥᩈᩣᨿ, ᩋᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩥ ᨻᩉᩩᨲᩁᩣ. ᨲᩮᩉᩥ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩪᩉᩥ ᨸᩩᨶ ᨸᩅᩤᩁᩮᨲᨻ᩠ᨻᩴ. ᨸᩅᩤᩁᩥᨲᩣᨶᩴ ᩋᨶᩣᨸᨲ᩠ᨲᩥ. (10)
</Paragraph>

<Paragraph number="1010" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩈᨶ᩠ᨶᩥᨸᨲᨶ᩠ᨲᩥ, ᨸᨬ᩠ᨧ ᩅᩤ ᩋᨲᩥᩁᩮᨠᩣ ᩅᩤ. ᨲᩮ ᨶ ᨩᩣᨶᨶ᩠ᨲᩥ—  ‘ᩋᨲ᩠ᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᨶᩣᨣᨲᩣ’ᨲᩥ. ᨲᩮ ᨵᨾ᩠ᨾᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᩥᨶᨿᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᨣ᩠ᨣᩤ ᩈᨾᨣ᩠ᨣᩈᨬ᩠ᨬᩥᨶᩮᩣ ᨸᩅᩤᩁᩮᨶ᩠ᨲᩥ. ᨲᩮᩉᩥ ᨸᩅᩤᩁᩥᨲᨾᨲ᩠ᨲᩮ, ᩑᨠᨧ᩠ᨧᩣᨿ ᩅᩩᨭ᩠ᨮᩥᨲᩣᨿ ᨸᩁᩥᩈᩣᨿ, ᩋᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩥ ᩈᨾᩈᨾᩣ. ᨸᩅᩤᩁᩥᨲᩣ ᩈᩩᨸ᩠ᨸᩅᩤᩁᩥᨲᩣ, ᨲᩮᩈᩴ ᩈᨶ᩠ᨲᩥᨠᩮ ᨸᩅᩤᩁᩮᨲᨻ᩠ᨻᩴ. ᨸᩅᩤᩁᩥᨲᩣᨶᩴ ᩋᨶᩣᨸᨲ᩠ᨲᩥ. (11)
</Paragraph>

<Paragraph number="1011" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩈᨶ᩠ᨶᩥᨸᨲᨶ᩠ᨲᩥ, ᨸᨬ᩠ᨧ ᩅᩤ ᩋᨲᩥᩁᩮᨠᩣ ᩅᩤ. ᨲᩮ ᨶ ᨩᩣᨶᨶ᩠ᨲᩥ—  ‘ᩋᨲ᩠ᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᨶᩣᨣᨲᩣ’ᨲᩥ. ᨲᩮ ᨵᨾ᩠ᨾᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᩥᨶᨿᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᨣ᩠ᨣᩤ ᩈᨾᨣ᩠ᨣᩈᨬ᩠ᨬᩥᨶᩮᩣ ᨸᩅᩤᩁᩮᨶ᩠ᨲᩥ. ᨲᩮᩉᩥ ᨸᩅᩤᩁᩥᨲᨾᨲ᩠ᨲᩮ, ᩑᨠᨧ᩠ᨧᩣᨿ ᩅᩩᨭ᩠ᨮᩥᨲᩣᨿ ᨸᩁᩥᩈᩣᨿ, ᩋᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩥ ᨳᩮᩣᨠᨲᩁᩣ. ᨸᩅᩤᩁᩥᨲᩣ ᩈᩩᨸ᩠ᨸᩅᩤᩁᩥᨲᩣ, ᨲᩮᩈᩴ ᩈᨶ᩠ᨲᩥᨠᩮ ᨸᩅᩤᩁᩮᨲᨻ᩠ᨻᩴ. ᨸᩅᩤᩁᩥᨲᩣᨶᩴ ᩋᨶᩣᨸᨲ᩠ᨲᩥ. (12)
</Paragraph>

<Paragraph number="1012" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩈᨶ᩠ᨶᩥᨸᨲᨶ᩠ᨲᩥ, ᨸᨬ᩠ᨧ ᩅᩤ ᩋᨲᩥᩁᩮᨠᩣ ᩅᩤ. ᨲᩮ ᨶ ᨩᩣᨶᨶ᩠ᨲᩥ—  ‘ᩋᨲ᩠ᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᨶᩣᨣᨲᩣ’ᨲᩥ. ᨲᩮ ᨵᨾ᩠ᨾᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᩥᨶᨿᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᨣ᩠ᨣᩤ ᩈᨾᨣ᩠ᨣᩈᨬ᩠ᨬᩥᨶᩮᩣ ᨸᩅᩤᩁᩮᨶ᩠ᨲᩥ. ᨲᩮᩉᩥ ᨸᩅᩤᩁᩥᨲᨾᨲ᩠ᨲᩮ, ᩈᨻ᩠ᨻᩣᨿ ᩅᩩᨭ᩠ᨮᩥᨲᩣᨿ ᨸᩁᩥᩈᩣᨿ, ᩋᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩥ ᨻᩉᩩᨲᩁᩣ. ᨲᩮᩉᩥ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩪᩉᩥ ᨸᩩᨶ ᨸᩅᩤᩁᩮᨲᨻ᩠ᨻᩴ. ᨸᩅᩤᩁᩥᨲᩣᨶᩴ ᩋᨶᩣᨸᨲ᩠ᨲᩥ. (13)
</Paragraph>

<Paragraph number="1013" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩈᨶ᩠ᨶᩥᨸᨲᨶ᩠ᨲᩥ, ᨸᨬ᩠ᨧ ᩅᩤ ᩋᨲᩥᩁᩮᨠᩣ ᩅᩤ. ᨲᩮ ᨶ ᨩᩣᨶᨶ᩠ᨲᩥ—  ‘ᩋᨲ᩠ᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᨶᩣᨣᨲᩣ’ᨲᩥ. ᨲᩮ ᨵᨾ᩠ᨾᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᩥᨶᨿᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᨣ᩠ᨣᩤ ᩈᨾᨣ᩠ᨣᩈᨬ᩠ᨬᩥᨶᩮᩣ ᨸᩅᩤᩁᩮᨶ᩠ᨲᩥ. ᨲᩮᩉᩥ ᨸᩅᩤᩁᩥᨲᨾᨲ᩠ᨲᩮ, ᩈᨻ᩠ᨻᩣᨿ ᩅᩩᨭ᩠ᨮᩥᨲᩣᨿ ᨸᩁᩥᩈᩣᨿ, ᩋᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩥ ᩈᨾᩈᨾᩣ. ᨸᩅᩤᩁᩥᨲᩣ ᩈᩩᨸ᩠ᨸᩅᩤᩁᩥᨲᩣ, ᨲᩮᩈᩴ ᩈᨶ᩠ᨲᩥᨠᩮ ᨸᩅᩤᩁᩮᨲᨻ᩠ᨻᩴ. ᨸᩅᩤᩁᩥᨲᩣᨶᩴ ᩋᨶᩣᨸᨲ᩠ᨲᩥ. (14)
</Paragraph>

<Paragraph number="1014" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩋᨬ᩠ᨬᨲᩁᩈ᩠ᨾᩥᩴ ᩋᩣᩅᩤᩈᩮ ᨲᨴᩉᩩ ᨸᩅᩤᩁᨱᩣᨿ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩈᨶ᩠ᨶᩥᨸᨲᨶ᩠ᨲᩥ, ᨸᨬ᩠ᨧ ᩅᩤ ᩋᨲᩥᩁᩮᨠᩣ ᩅᩤ. ᨲᩮ ᨶ ᨩᩣᨶᨶ᩠ᨲᩥ—  ‘ᩋᨲ᩠ᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᨶᩣᨣᨲᩣ’ᨲᩥ. ᨲᩮ ᨵᨾ᩠ᨾᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᩥᨶᨿᩈᨬ᩠ᨬᩥᨶᩮᩣ ᩅᨣ᩠ᨣᩤ ᩈᨾᨣ᩠ᨣᩈᨬ᩠ᨬᩥᨶᩮᩣ ᨸᩅᩤᩁᩮᨶ᩠ᨲᩥ. ᨲᩮᩉᩥ ᨸᩅᩤᩁᩥᨲᨾᨲ᩠ᨲᩮ, ᩈᨻ᩠ᨻᩣᨿ ᩅᩩᨭ᩠ᨮᩥᨲᩣᨿ ᨸᩁᩥᩈᩣᨿ, ᩋᨳᨬ᩠ᨬᩮ ᩋᩣᩅᩤᩈᩥᨠᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩥ ᨳᩮᩣᨠᨲᩁᩣ. ᨸᩅᩤᩁᩥᨲᩣ ᩈᩩᨸ᩠ᨸᩅᩤᩁᩥᨲᩣ, ᨲᩮᩈᩴ ᩈᨶ᩠ᨲᩥᨠᩮ ᨸᩅᩤᩁᩮᨲᨻ᩠ᨻᩴ. ᨸᩅᩤᩁᩥᨲᩣᨶᩴ ᩋᨶᩣᨸᨲ᩠ᨲᩥ. (15)

</Paragraph>
