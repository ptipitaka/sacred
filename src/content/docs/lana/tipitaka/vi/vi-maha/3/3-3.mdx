<Paragraph number="826" book="vi-maha">
ᩋᨳ ᨡᩮᩣ ᨽᨣᩅᩤ ᩁᩣᨩᨣᩉᩮ ᨿᨳᩣᨽᩥᩁᨶ᩠ᨲᩴ ᩅᩥᩉᩁᩥᨲ᩠ᩅᩣ ᨿᩮᨶ ᩈᩣᩅᨲ᩠ᨳᩥ ᨲᩮᨶ ᨧᩣᩁᩥᨠᩴ ᨸᨠ᩠ᨠᩣᨾᩥ. ᩋᨶᩩᨸᩩᨻ᩠ᨻᩮᨶ ᨧᩣᩁᩥᨠᩴ ᨧᩁᨾᩣᨶᩮᩣ ᨿᩮᨶ ᩈᩣᩅᨲ᩠ᨳᩥ ᨲᨴᩅᩈᩁᩥ. ᨲᨲᩕ ᩈᩩᨴᩴ ᨽᨣᩅᩤ ᩈᩣᩅᨲ᩠ᨳᩥᨿᩴ ᩅᩥᩉᩁᨲᩥ ᨩᩮᨲᩅᨶᩮ ᩋᨶᩣᨳᨸᩥᨱ᩠ᨯᩥᨠᩔ ᩋᩣᩁᩣᨾᩮ. ᨲᩮᨶ ᨡᩮᩣ ᨸᨶ ᩈᨾᨿᩮᨶ ᨠᩮᩣᩈᩃᩮᩈᩩ ᨩᨶᨸᨴᩮ ᩏᨴᩮᨶᩮᨶ ᩏᨸᩣᩈᨠᩮᨶ ᩈᩴᨥᩴ ᩏᨴ᩠ᨴᩥᩔ ᩅᩥᩉᩣᩁᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ. ᩈᩮᩣ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩈᨶ᩠ᨲᩥᨠᩮ ᨴᩪᨲᩴ ᨸᩣᩉᩮᩈᩥ—  “ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩩ ᨽᨴᨶ᩠ᨲᩣ, ᩍᨧ᩠ᨨᩣᨾᩥ ᨴᩤᨶᨬ᩠ᨧ ᨴᩤᨲᩩᩴ, ᨵᨾ᩠ᨾᨬ᩠ᨧ ᩈᩮᩣᨲᩩᩴ, ᨽᩥᨠ᩠ᨡᩪ ᨧ ᨸᩔᩥᨲᩩᨶ᩺”ᨲᩥ. ᨽᩥᨠ᩠ᨡᩪ ᩑᩅᨾᩣᩉᩴᩈᩩ—  “ᨽᨣᩅᨲᩣ, ᩋᩣᩅᩩᩈᩮᩣ, ᨸᨬ᩠ᨬᨲ᩠ᨲᩴ—  ‘ᨶ ᩅᩔᩴ ᩏᨸᨣᨶ᩠ᨲ᩠ᩅᩣ ᨸᩩᩁᩥᨾᩴ ᩅᩤ ᨲᩮᨾᩣᩈᩴ ᨸᨧ᩠ᨨᩥᨾᩴ ᩅᩤ ᨲᩮᨾᩣᩈᩴ ᩋᩅᩈᩥᨲ᩠ᩅᩣ ᨧᩣᩁᩥᨠᩣ ᨸᨠ᩠ᨠᨾᩥᨲᨻ᩠ᨻᩣ’ᨲᩥ. ᩋᩣᨣᨾᩮᨲᩩ ᩏᨴᩮᨶᩮᩣ ᩏᨸᩣᩈᨠᩮᩣ, ᨿᩣᩅ ᨽᩥᨠ᩠ᨡᩪ ᩅᩔᩴ ᩅᩈᨶ᩠ᨲᩥ. ᩅᩔᩴᩅᩩᨭ᩠ᨮᩣ ᩋᩣᨣᨾᩥᩔᨶ᩠ᨲᩥ. ᩈᨧᩮ ᨸᨶᩔ ᩋᨧ᩠ᨧᩣᨿᩥᨠᩴ ᨠᩁᨱᩦᨿᩴ, ᨲᨲ᩠ᨳᩮᩅ ᩋᩣᩅᩤᩈᩥᨠᩣᨶᩴ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩈᨶ᩠ᨲᩥᨠᩮ ᩅᩥᩉᩣᩁᩴ ᨸᨲᩥᨭ᩠ᨮᩣᨸᩮᨲᩪ”ᨲᩥ. ᩏᨴᩮᨶᩮᩣ ᩏᨸᩣᩈᨠᩮᩣ ᩏᨩ᩠ᨫᩣᨿᨲᩥ ᨡᩥᨿ᩠ᨿᨲᩥ ᩅᩥᨸᩣᨧᩮᨲᩥ—  “ᨠᨳᨬ᩠ᩉᩥ ᨶᩣᨾ ᨽᨴᨶ᩠ᨲᩣ ᨾᨿᩣ ᨸᩉᩥᨲᩮ ᨶ ᩋᩣᨣᨧ᩠ᨨᩥᩔᨶ᩠ᨲᩥ. ᩋᩉᨬ᩠ᩉᩥ ᨴᩤᨿᨠᩮᩣ ᨠᩣᩁᨠᩮᩣ ᩈᩴᨥᩩᨸᨭ᩠ᨮᩣᨠᩮᩣ”ᨲᩥ.
</Paragraph>

<Paragraph number="827" book="vi-maha">
ᩋᩔᩮᩣᩈᩩᩴ ᨡᩮᩣ ᨽᩥᨠ᩠ᨡᩪ ᩏᨴᩮᨶᩔ ᩏᨸᩣᩈᨠᩔ ᩏᨩ᩠ᨫᩣᨿᨶ᩠ᨲᩔ ᨡᩥᨿ᩠ᨿᨶ᩠ᨲᩔ ᩅᩥᨸᩣᨧᩮᨶ᩠ᨲᩔ. ᩋᨳ ᨡᩮᩣ ᨲᩮ ᨽᩥᨠ᩠ᨡᩪ ᨽᨣᩅᨲᩮᩣ ᩑᨲᨾᨲ᩠ᨳᩴ ᩋᩣᩁᩮᩣᨧᩮᩈᩩᩴ. ᩋᨳ ᨡᩮᩣ ᨽᨣᩅᩤ ᩑᨲᩈ᩠ᨾᩥᩴ ᨶᩥᨴᩤᨶᩮ ᩑᨲᩈ᩠ᨾᩥᩴ ᨸᨠᩁᨱᩮ ᨵᨾ᩠ᨾᩥᩴ ᨠᨳᩴ ᨠᨲ᩠ᩅᩣ ᨽᩥᨠ᩠ᨡᩪ ᩋᩣᨾᨶ᩠ᨲᩮᩈᩥ—  “ᩋᨶᩩᨩᩣᨶᩣᨾᩥ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩈᨲ᩠ᨲᨶ᩠ᨶᩴ ᩈᨲ᩠ᨲᩣᩉᨠᩁᨱᩦᨿᩮᨶ ᨸᩉᩥᨲᩮ ᨣᨶ᩠ᨲᩩᩴ, ᨶ ᨲ᩠ᩅᩮᩅ ᩋᨸ᩠ᨸᩉᩥᨲᩮ. ᨽᩥᨠ᩠ᨡᩩᩔ, ᨽᩥᨠ᩠ᨡᩩᨶᩥᨿᩣ, ᩈᩥᨠ᩠ᨡᨾᩣᨶᩣᨿ, ᩈᩣᨾᨱᩮᩁᩔ, ᩈᩣᨾᨱᩮᩁᩥᨿᩣ, ᩏᨸᩣᩈᨠᩔ, ᩏᨸᩣᩈᩥᨠᩣᨿ—  ᩋᨶᩩᨩᩣᨶᩣᨾᩥ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩍᨾᩮᩈᩴ ᩈᨲ᩠ᨲᨶ᩠ᨶᩴ ᩈᨲ᩠ᨲᩣᩉᨠᩁᨱᩦᨿᩮᨶ ᨸᩉᩥᨲᩮ ᨣᨶ᩠ᨲᩩᩴ, ᨶ ᨲ᩠ᩅᩮᩅ ᩋᨸ᩠ᨸᩉᩥᨲᩮ. ᩈᨲ᩠ᨲᩣᩉᩴ ᩈᨶ᩠ᨶᩥᩅᨲ᩠ᨲᩮᩣ ᨠᩣᨲᨻ᩠ᨻᩮᩣ.
</Paragraph>

</Division>

<Paragraph number="828" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩏᨸᩣᩈᨠᩮᨶ ᩈᩴᨥᩴ ᩏᨴ᩠ᨴᩥᩔ ᩅᩥᩉᩣᩁᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ. ᩈᩮᩣ ᨧᩮ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩈᨶ᩠ᨲᩥᨠᩮ ᨴᩪᨲᩴ ᨸᩉᩥᨱᩮᨿ᩠ᨿ—  ‘ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩩ ᨽᨴᨶ᩠ᨲᩣ, ᩍᨧ᩠ᨨᩣᨾᩥ ᨴᩤᨶᨬ᩠ᨧ ᨴᩤᨲᩩᩴ, ᨵᨾ᩠ᨾᨬ᩠ᨧ ᩈᩮᩣᨲᩩᩴ, ᨽᩥᨠ᩠ᨡᩪ ᨧ ᨸᩔᩥᨲᩩᨶ᩺’ᨲᩥ, ᨣᨶ᩠ᨲᨻ᩠ᨻᩴ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩈᨲ᩠ᨲᩣᩉᨠᩁᨱᩦᨿᩮᨶ, ᨸᩉᩥᨲᩮ, ᨶ ᨲ᩠ᩅᩮᩅ ᩋᨸ᩠ᨸᩉᩥᨲᩮ. ᩈᨲ᩠ᨲᩣᩉᩴ ᩈᨶ᩠ᨶᩥᩅᨲ᩠ᨲᩮᩣ ᨠᩣᨲᨻ᩠ᨻᩮᩣ.
</Paragraph>

<Paragraph number="829" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩏᨸᩣᩈᨠᩮᨶ ᩈᩴᨥᩴ ᩏᨴ᩠ᨴᩥᩔ ᩋᨯ᩠ᨰᨿᩮᩣᨣᩮᩤ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ …ᨸᩮ…  ᨸᩣᩈᩣᨴᩮᩤ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩉᨾ᩠ᨾᩥᨿᩴ ᨠᩣᩁᩣᨸᩥᨲᩴ ᩉᩮᩣᨲᩥ…  ᨣᩩᩉᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨸᩁᩥᩅᩮᨱᩴ ᨠᩣᩁᩣᨸᩥᨲᩴ ᩉᩮᩣᨲᩥ…  ᨠᩮᩣᨭ᩠ᨮᨠᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩏᨸᨭ᩠ᨮᩣᨶᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᩋᨣ᩠ᨣᩥᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨠᨸ᩠ᨸᩥᨿᨠᩩᨭᩥ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᩅᨧ᩠ᨧᨠᩩᨭᩥ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨧᩘᨠᨾᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᨧᩘᨠᨾᨶᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᩏᨴᨸᩣᨶᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩏᨴᨸᩣᨶᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨩᨶ᩠ᨲᩣᨥᩁᩴ ᨠᩣᩁᩣᨸᩥᨲᩴ ᩉᩮᩣᨲᩥ…  ᨩᨶ᩠ᨲᩣᨥᩁᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨸᩮᩣᨠ᩠ᨡᩁᨱᩦ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨾᨱ᩠ᨯᨸᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩋᩣᩁᩣᨾᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩋᩣᩁᩣᨾᩅᨲ᩠ᨳᩩ ᨠᩣᩁᩣᨸᩥᨲᩴ ᩉᩮᩣᨲᩥ. ᩈᩮᩣ ᨧᩮ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩈᨶ᩠ᨲᩥᨠᩮ ᨴᩪᨲᩴ ᨸᩉᩥᨱᩮᨿ᩠ᨿ—  ‘ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩩ ᨽᨴᨶ᩠ᨲᩣ, ᩍᨧ᩠ᨨᩣᨾᩥ ᨴᩤᨶᨬ᩠ᨧ ᨴᩤᨲᩩᩴ, ᨵᨾ᩠ᨾᨬ᩠ᨧ ᩈᩮᩣᨲᩩᩴ, ᨽᩥᨠ᩠ᨡᩪ ᨧ ᨸᩔᩥᨲᩩᨶ᩺’ᨲᩥ, ᨣᨶ᩠ᨲᨻ᩠ᨻᩴ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩈᨲ᩠ᨲᩣᩉᨠᩁᨱᩦᨿᩮᨶ, ᨸᩉᩥᨲᩮ, ᨶ ᨲ᩠ᩅᩮᩅ ᩋᨸ᩠ᨸᩉᩥᨲᩮ. ᩈᨲ᩠ᨲᩣᩉᩴ ᩈᨶ᩠ᨶᩥᩅᨲ᩠ᨲᩮᩣ ᨠᩣᨲᨻ᩠ᨻᩮᩣ.

ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩏᨸᩣᩈᩥᨠᩣᨿ ᩈᩴᨥᩴ ᩏᨴ᩠ᨴᩥᩔ ᩅᩥᩉᩣᩁᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ. ᩈᩣ ᨧᩮ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩈᨶ᩠ᨲᩥᨠᩮ ᨴᩪᨲᩴ ᨸᩉᩥᨱᩮᨿ᩠ᨿ—  ‘ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩩ ᩋᨿ᩠ᨿᩣ, ᩍᨧ᩠ᨨᩣᨾᩥ ᨴᩤᨶᨬ᩠ᨧ ᨴᩤᨲᩩᩴ, ᨵᨾ᩠ᨾᨬ᩠ᨧ ᩈᩮᩣᨲᩩᩴ, ᨽᩥᨠ᩠ᨡᩪ ᨧ ᨸᩔᩥᨲᩩᨶ᩺’ᨲᩥ, ᨣᨶ᩠ᨲᨻ᩠ᨻᩴ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩈᨲ᩠ᨲᩣᩉᨠᩁᨱᩦᨿᩮᨶ, ᨸᩉᩥᨲᩮ, ᨶ ᨲ᩠ᩅᩮᩅ ᩋᨸ᩠ᨸᩉᩥᨲᩮ. ᩈᨲ᩠ᨲᩣᩉᩴ ᩈᨶ᩠ᨶᩥᩅᨲ᩠ᨲᩮᩣ ᨠᩣᨲᨻ᩠ᨻᩮᩣ.
</Paragraph>

<Paragraph number="834" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩏᨸᩣᩈᩥᨠᩣᨿ ᩈᩴᨥᩴ ᩏᨴ᩠ᨴᩥᩔ ᩋᨯ᩠ᨰᨿᩮᩣᨣᩮᩤ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ …ᨸᩮ…  ᨸᩣᩈᩣᨴᩮᩤ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩉᨾ᩠ᨾᩥᨿᩴ ᨠᩣᩁᩣᨸᩥᨲᩴ ᩉᩮᩣᨲᩥ…  ᨣᩩᩉᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨸᩁᩥᩅᩮᨱᩴ ᨠᩣᩁᩣᨸᩥᨲᩴ ᩉᩮᩣᨲᩥ…  ᨠᩮᩣᨭ᩠ᨮᨠᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩏᨸᨭ᩠ᨮᩣᨶᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᩋᨣ᩠ᨣᩥᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨠᨸ᩠ᨸᩥᨿᨠᩩᨭᩥ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᩅᨧ᩠ᨧᨠᩩᨭᩥ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨧᩘᨠᨾᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᨧᩘᨠᨾᨶᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᩏᨴᨸᩣᨶᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩏᨴᨸᩣᨶᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨩᨶ᩠ᨲᩣᨥᩁᩴ ᨠᩣᩁᩣᨸᩥᨲᩴ ᩉᩮᩣᨲᩥ…  ᨩᨶ᩠ᨲᩣᨥᩁᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨸᩮᩣᨠ᩠ᨡᩁᨱᩦ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨾᨱ᩠ᨯᨸᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩋᩣᩁᩣᨾᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩋᩣᩁᩣᨾᩅᨲ᩠ᨳᩩ ᨠᩣᩁᩣᨸᩥᨲᩴ ᩉᩮᩣᨲᩥ. ᩈᩣ ᨧᩮ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩈᨶ᩠ᨲᩥᨠᩮ ᨴᩪᨲᩴ ᨸᩉᩥᨱᩮᨿ᩠ᨿ—  ‘ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩩ ᩋᨿ᩠ᨿᩣ, ᩍᨧ᩠ᨨᩣᨾᩥ ᨴᩤᨶᨬ᩠ᨧ ᨴᩤᨲᩩᩴ, ᨵᨾ᩠ᨾᨬ᩠ᨧ ᩈᩮᩣᨲᩩᩴ, ᨽᩥᨠ᩠ᨡᩪ ᨧ ᨸᩔᩥᨲᩩᨶ᩺’ᨲᩥ, ᨣᨶ᩠ᨲᨻ᩠ᨻᩴ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩈᨲ᩠ᨲᩣᩉᨠᩁᨱᩦᨿᩮᨶ, ᨸᩉᩥᨲᩮ, ᨶ ᨲ᩠ᩅᩮᩅ ᩋᨸ᩠ᨸᩉᩥᨲᩮ. ᩈᨲ᩠ᨲᩣᩉᩴ ᩈᨶ᩠ᨶᩥᩅᨲ᩠ᨲᩮᩣ ᨠᩣᨲᨻ᩠ᨻᩮᩣ.
</Paragraph>

ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩏᨸᩣᩈᩥᨠᩣᨿ ᩋᨲ᩠ᨲᨶᩮᩣ ᩋᨲ᩠ᨳᩣᨿ ᨶᩥᩅᩮᩈᨶᩴ ᨠᩣᩁᩣᨸᩥᨲᩴ ᩉᩮᩣᨲᩥ …ᨸᩮ…  ᩈᨿᨶᩥᨥᩁᩴ ᨠᩣᩁᩣᨸᩥᨲᩴ ᩉᩮᩣᨲᩥ…  ᩏᨴᩮᩤᩈᩥᨲᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩋᨭ᩠ᨭᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᨾᩣᩊᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩋᩣᨸᨱᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩋᩣᨸᨱᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨸᩣᩈᩣᨴᩮᩤ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩉᨾ᩠ᨾᩥᨿᩴ ᨠᩣᩁᩣᨸᩥᨲᩴ ᩉᩮᩣᨲᩥ…  ᨣᩩᩉᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨸᩁᩥᩅᩮᨱᩴ ᨠᩣᩁᩣᨸᩥᨲᩴ ᩉᩮᩣᨲᩥ…  ᨠᩮᩣᨭ᩠ᨮᨠᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩏᨸᨭ᩠ᨮᩣᨶᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᩋᨣ᩠ᨣᩥᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᩁᩈᩅᨲᩦ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨧᩘᨠᨾᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᨧᩘᨠᨾᨶᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᩏᨴᨸᩣᨶᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩏᨴᨸᩣᨶᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨸᩮᩣᨠ᩠ᨡᩁᨱᩦ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨾᨱ᩠ᨯᨸᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩋᩣᩁᩣᨾᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩋᩣᩁᩣᨾᩅᨲ᩠ᨳᩩ ᨠᩣᩁᩣᨸᩥᨲᩴ ᩉᩮᩣᨲᩥ…  ᨸᩩᨲ᩠ᨲᩔ ᩅᩤ ᩅᩤᩁᩮᨿ᩠ᨿᩴ ᩉᩮᩣᨲᩥ…  ᨵᩦᨲᩩᨿᩣ ᩅᩤ ᩅᩤᩁᩮᨿ᩠ᨿᩴ ᩉᩮᩣᨲᩥ…  ᨣᩥᩃᩣᨶᩣ ᩅᩤ ᩉᩮᩣᨲᩥ…  ᩋᨽᩥᨬ᩠ᨬᩣᨲᩴ ᩅᩤ ᩈᩩᨲ᩠ᨲᨶ᩠ᨲᩴ ᨽᨱᨲᩥ. ᩈᩣ ᨧᩮ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩈᨶ᩠ᨲᩥᨠᩮ ᨴᩪᨲᩴ ᨸᩉᩥᨱᩮᨿ᩠ᨿ—  ‘ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩩ ᩋᨿ᩠ᨿᩣ, ᩍᨾᩴ ᩈᩩᨲ᩠ᨲᨶ᩠ᨲᩴ ᨸᩁᩥᨿᩣᨸᩩᨱᩥᩔᨶ᩠ᨲᩥ, ᨸᩩᩁᩣᨿᩴ ᩈᩩᨲ᩠ᨲᨶ᩠ᨲᩮᩣ ᨸᩃᩩᨩ᩠ᨩᨲᩦ’ᨲᩥ. ᩋᨬ᩠ᨬᨲᩁᩴ ᩅᩤ ᨸᨶᩔᩣ ᨠᩥᨧ᩠ᨧᩴ ᩉᩮᩣᨲᩥ ᨠᩁᨱᩦᨿᩴ ᩅᩤ, ᩈᩣ ᨧᩮ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩈᨶ᩠ᨲᩥᨠᩮ ᨴᩪᨲᩴ ᨸᩉᩥᨱᩮᨿ᩠ᨿ—  ‘ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩩ ᩋᨿ᩠ᨿᩣ, ᩍᨧ᩠ᨨᩣᨾᩥ ᨴᩤᨶᨬ᩠ᨧ ᨴᩤᨲᩩᩴ, ᨵᨾ᩠ᨾᨬ᩠ᨧ ᩈᩮᩣᨲᩩᩴ, ᨽᩥᨠ᩠ᨡᩪ ᨧ ᨸᩔᩥᨲᩩᨶ᩺’ᨲᩥ, ᨣᨶ᩠ᨲᨻ᩠ᨻᩴ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩈᨲ᩠ᨲᩣᩉᨠᩁᨱᩦᨿᩮᨶ, ᨸᩉᩥᨲᩮ, ᨶ ᨲ᩠ᩅᩮᩅ ᩋᨸ᩠ᨸᩉᩥᨲᩮ. ᩈᨲ᩠ᨲᩣᩉᩴ ᩈᨶ᩠ᨶᩥᩅᨲ᩠ᨲᩮᩣ ᨠᩣᨲᨻ᩠ᨻᩮᩣ.

ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩩᨶᩣ ᩈᩴᨥᩴ ᩏᨴ᩠ᨴᩥᩔ …ᨸᩮ…  ᨽᩥᨠ᩠ᨡᩩᨶᩥᨿᩣ ᩈᩴᨥᩴ ᩏᨴ᩠ᨴᩥᩔ…  ᩈᩥᨠ᩠ᨡᨾᩣᨶᩣᨿ ᩈᩴᨥᩴ ᩏᨴ᩠ᨴᩥᩔ…  ᩈᩣᨾᨱᩮᩁᩮᨶ ᩈᩴᨥᩴ ᩏᨴ᩠ᨴᩥᩔ…  ᩈᩣᨾᨱᩮᩁᩥᨿᩣ ᩈᩴᨥᩴ ᩏᨴ᩠ᨴᩥᩔ…  ᩈᨾ᩠ᨻᩉᩩᩃᩮ ᨽᩥᨠ᩠ᨡᩪ ᩏᨴ᩠ᨴᩥᩔ…  ᩑᨠᩴ ᨽᩥᨠ᩠ᨡᩩᩴ ᩏᨴ᩠ᨴᩥᩔ…  ᨽᩥᨠ᩠ᨡᩩᨶᩥᩈᩴᨥᩴ ᩏᨴ᩠ᨴᩥᩔ…  ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᨽᩥᨠ᩠ᨡᩩᨶᩥᨿᩮᩣ ᩏᨴ᩠ᨴᩥᩔ…  ᩑᨠᩴ ᨽᩥᨠ᩠ᨡᩩᨶᩥᩴ ᩏᨴ᩠ᨴᩥᩔ…  ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩈᩥᨠ᩠ᨡᨾᩣᨶᩣᨿᩮᩣ ᩏᨴ᩠ᨴᩥᩔ…  ᩑᨠᩴ ᩈᩥᨠ᩠ᨡᨾᩣᨶᩴ ᩏᨴ᩠ᨴᩥᩔ…  ᩈᨾ᩠ᨻᩉᩩᩃᩮ ᩈᩣᨾᨱᩮᩁᩮ ᩏᨴ᩠ᨴᩥᩔ…  ᩑᨠᩴ ᩈᩣᨾᨱᩮᩁᩴ ᩏᨴ᩠ᨴᩥᩔ…  ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩈᩣᨾᨱᩮᩁᩥᨿᩮᩣ ᩏᨴ᩠ᨴᩥᩔ…  ᩑᨠᩴ ᩈᩣᨾᨱᩮᩁᩥᩴ ᩏᨴ᩠ᨴᩥᩔ…  ᩋᨲ᩠ᨲᨶᩮᩣ ᩋᨲ᩠ᨳᩣᨿ ᩅᩥᩉᩣᩁᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ …ᨸᩮ…  ᩋᨯ᩠ᨰᨿᩮᩣᨣᩮᩤ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᨸᩣᩈᩣᨴᩮᩤ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩉᨾ᩠ᨾᩥᨿᩴ ᨠᩣᩁᩣᨸᩥᨲᩴ ᩉᩮᩣᨲᩥ…  ᨣᩩᩉᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨸᩁᩥᩅᩮᨱᩴ ᨠᩣᩁᩣᨸᩥᨲᩴ ᩉᩮᩣᨲᩥ…  ᨠᩮᩣᨭ᩠ᨮᨠᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩏᨸᨭ᩠ᨮᩣᨶᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᩋᨣ᩠ᨣᩥᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨠᨸ᩠ᨸᩥᨿᨠᩩᨭᩥ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨧᩘᨠᨾᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᨧᩘᨠᨾᨶᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᩏᨴᨸᩣᨶᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩏᨴᨸᩣᨶᩈᩣᩃᩣ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨸᩮᩣᨠ᩠ᨡᩁᨱᩦ ᨠᩣᩁᩣᨸᩥᨲᩣ ᩉᩮᩣᨲᩥ…  ᨾᨱ᩠ᨯᨸᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩋᩣᩁᩣᨾᩮᩣ ᨠᩣᩁᩣᨸᩥᨲᩮᩣ ᩉᩮᩣᨲᩥ…  ᩋᩣᩁᩣᨾᩅᨲ᩠ᨳᩩ ᨠᩣᩁᩣᨸᩥᨲᩴ ᩉᩮᩣᨲᩥ. ᩈᩣ ᨧᩮ ᨽᩥᨠ᩠ᨡᩪᨶᩴ ᩈᨶ᩠ᨲᩥᨠᩮ ᨴᩪᨲᩴ ᨸᩉᩥᨱᩮᨿ᩠ᨿ…  ‘ᩋᩣᨣᨧ᩠ᨨᨶ᩠ᨲᩩ ᩋᨿ᩠ᨿᩣ, ᩍᨧ᩠ᨨᩣᨾᩥ ᨴᩤᨶᨬ᩠ᨧ ᨴᩤᨲᩩᩴ, ᨵᨾ᩠ᨾᨬ᩠ᨧ ᩈᩮᩣᨲᩩᩴ, ᨽᩥᨠ᩠ᨡᩪ ᨧ ᨸᩔᩥᨲᩩᨶ᩺’ᨲᩥ, ᨣᨶ᩠ᨲᨻ᩠ᨻᩴ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᩈᨲ᩠ᨲᩣᩉᨠᩁᨱᩦᨿᩮᨶ, ᨸᩉᩥᨲᩮ, ᨶ ᨲ᩠ᩅᩮᩅ ᩋᨸ᩠ᨸᩉᩥᨲᩮ. ᩈᨲ᩠ᨲᩣᩉᩴ ᩈᨶ᩠ᨶᩥᩅᨲ᩠ᨲᩮᩣ ᨠᩣᨲᨻ᩠ᨻᩮᩣ”ᨲᩥ.
