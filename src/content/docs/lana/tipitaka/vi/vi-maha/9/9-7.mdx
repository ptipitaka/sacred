<Paragraph number="1814" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩩᩔ ᨶ ᩉᩮᩣᨲᩥ ᩋᩣᨸᨲ᩠ᨲᩥ ᨴᨭ᩠ᨮᨻ᩠ᨻᩣ. ᨲᨾᩮᨶᩴ ᨧᩮᩣᨴᩮᨲᩥ ᩈᩴᨥᩮᩣ ᩅᩤ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩅᩤ ᩑᨠᨸᩩᨣ᩠ᨣᩃᩮᩣ ᩅᩤ—  ‘ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᨲ᩠ᩅᩴ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨶ᩠ᨶᩮᩣ, ᨸᩔᩈᩮᨲᩴ ᩋᩣᨸᨲ᩠ᨲᩥᨶ᩺’ᨲᩥ? ᩈᩮᩣ ᩑᩅᩴ ᩅᨴᩮᨲᩥ—  ‘ᨶᨲ᩠ᨳᩥ ᨾᩮ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨲ᩠ᨲᩥ, ᨿᨾᩉᩴ ᨸᩔᩮᨿ᩠ᨿᨶ᩺’ᨲᩥ. ᨲᩴ ᩈᩴᨥᩮᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩋᨴᩔᨶᩮ ᩏᨠ᩠ᨡᩥᨸᨲᩥ—  ᩋᨵᨾ᩠ᨾᨠᨾ᩠ᨾᩴ.
</Paragraph>

<Paragraph number="1815" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩩᩔ ᨶ ᩉᩮᩣᨲᩥ ᩋᩣᨸᨲ᩠ᨲᩥ ᨸᨭᩥᨠᩣᨲᨻ᩠ᨻᩣ. ᨲᨾᩮᨶᩴ ᨧᩮᩣᨴᩮᨲᩥ ᩈᩴᨥᩮᩣ ᩅᩤ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩅᩤ ᩑᨠᨸᩩᨣ᩠ᨣᩃᩮᩣ ᩅᩤ—  ‘ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᨲ᩠ᩅᩴ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨶ᩠ᨶᩮᩣ, ᨸᨭᩥᨠᩁᩮᩣᩉᩥ ᨲᩴ ᩋᩣᨸᨲ᩠ᨲᩥᨶ᩺’ᨲᩥ. ᩈᩮᩣ ᩑᩅᩴ ᩅᨴᩮᨲᩥ—  ‘ᨶᨲ᩠ᨳᩥ ᨾᩮ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨲ᩠ᨲᩥ, ᨿᨾᩉᩴ ᨸᨭᩥᨠᩁᩮᨿ᩠ᨿᨶ᩺’ᨲᩥ. ᨲᩴ ᩈᩴᨥᩮᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨠᨾ᩠ᨾᩮ ᩏᨠ᩠ᨡᩥᨸᨲᩥ—  ᩋᨵᨾ᩠ᨾᨠᨾ᩠ᨾᩴ.
</Paragraph>

<Paragraph number="1816" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩩᩔ ᨶ ᩉᩮᩣᨲᩥ ᨸᩣᨸᩥᨠᩣ ᨴᩥᨭ᩠ᨮᩥ ᨸᨭᩥᨶᩥᩔᨩ᩠ᨩᩮᨲᩣ. ᨲᨾᩮᨶᩴ ᨧᩮᩣᨴᩮᨲᩥ ᩈᩴᨥᩮᩣ ᩅᩤ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩅᩤ ᩑᨠᨸᩩᨣ᩠ᨣᩃᩮᩣ ᩅᩤ—  ‘ᨸᩣᨸᩥᨠᩣ ᨲᩮ, ᩋᩣᩅᩩᩈᩮᩣ, ᨴᩥᨭ᩠ᨮᩥ, ᨸᨭᩥᨶᩥᩔᨩ᩠ᨩᩮᨲᩴ ᨸᩣᨸᩥᨠᩴ ᨴᩥᨭ᩠ᨮᩥᨶ᩺’ᨲᩥ. ᩈᩮᩣ ᩑᩅᩴ ᩅᨴᩮᨲᩥ—  ‘ᨶᨲ᩠ᨳᩥ ᨾᩮ, ᩋᩣᩅᩩᩈᩮᩣ, ᨸᩣᨸᩥᨠᩣ ᨴᩥᨭ᩠ᨮᩥ, ᨿᨾᩉᩴ ᨸᨭᩥᨶᩥᩔᨩ᩠ᨩᩮᨿ᩠ᨿᨶ᩺’ᨲᩥ. ᨲᩴ ᩈᩴᨥᩮᩣ ᨸᩣᨸᩥᨠᩣᨿ ᨴᩥᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨶᩥᩔᨣ᩠ᨣᩮ ᩏᨠ᩠ᨡᩥᨸᨲᩥ—  ᩋᨵᨾ᩠ᨾᨠᨾ᩠ᨾᩴ.
</Paragraph>

<Paragraph number="1817" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩩᩔ ᨶ ᩉᩮᩣᨲᩥ ᩋᩣᨸᨲ᩠ᨲᩥ ᨴᨭ᩠ᨮᨻ᩠ᨻᩣ, ᨶ ᩉᩮᩣᨲᩥ ᩋᩣᨸᨲ᩠ᨲᩥ ᨸᨭᩥᨠᩣᨲᨻ᩠ᨻᩣ. ᨲᨾᩮᨶᩴ ᨧᩮᩣᨴᩮᨲᩥ ᩈᩴᨥᩮᩣ ᩅᩤ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩅᩤ ᩑᨠᨸᩩᨣ᩠ᨣᩃᩮᩣ ᩅᩤ—  ‘ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᨲ᩠ᩅᩴ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨶ᩠ᨶᩮᩣ, ᨸᩔᩈᩮᨲᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ? ᨸᨭᩥᨠᩁᩮᩣᩉᩥ ᨲᩴ ᩋᩣᨸᨲ᩠ᨲᩥᨶ᩺’ᨲᩥ. ᩈᩮᩣ ᩑᩅᩴ ᩅᨴᩮᨲᩥ—  ‘ᨶᨲ᩠ᨳᩥ ᨾᩮ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨲ᩠ᨲᩥ, ᨿᨾᩉᩴ ᨸᩔᩮᨿ᩠ᨿᩴ. ᨶᨲ᩠ᨳᩥ ᨾᩮ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨲ᩠ᨲᩥ, ᨿᨾᩉᩴ ᨸᨭᩥᨠᩁᩮᨿ᩠ᨿᨶ᩺’ᨲᩥ. ᨲᩴ ᩈᩴᨥᩮᩣ ᩋᨴᩔᨶᩮ ᩅᩤ ᩋᨸ᩠ᨸᨭᩥᨠᨾ᩠ᨾᩮ ᩅᩤ ᩏᨠ᩠ᨡᩥᨸᨲᩥ—  ᩋᨵᨾ᩠ᨾᨠᨾ᩠ᨾᩴ.
</Paragraph>

<Paragraph number="1818" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩩᩔ ᨶ ᩉᩮᩣᨲᩥ ᩋᩣᨸᨲ᩠ᨲᩥ ᨴᨭ᩠ᨮᨻ᩠ᨻᩣ, ᨶ ᩉᩮᩣᨲᩥ ᨸᩣᨸᩥᨠᩣ ᨴᩥᨭ᩠ᨮᩥ ᨸᨭᩥᨶᩥᩔᨩ᩠ᨩᩮᨲᩣ. ᨲᨾᩮᨶᩴ ᨧᩮᩣᨴᩮᨲᩥ ᩈᩴᨥᩮᩣ ᩅᩤ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩅᩤ ᩑᨠᨸᩩᨣ᩠ᨣᩃᩮᩣ ᩅᩤ—  ‘ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᨲ᩠ᩅᩴ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨶ᩠ᨶᩮᩣ, ᨸᩔᩈᩮᨲᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ? ᨸᩣᨸᩥᨠᩣ ᨲᩮ ᨴᩥᨭ᩠ᨮᩥ, ᨸᨭᩥᨶᩥᩔᨩ᩠ᨩᩮᨲᩴ ᨸᩣᨸᩥᨠᩴ ᨴᩥᨭ᩠ᨮᩥᨶ᩺’ᨲᩥ. ᩈᩮᩣ ᩑᩅᩴ ᩅᨴᩮᨲᩥ—  ‘ᨶᨲ᩠ᨳᩥ ᨾᩮ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨲ᩠ᨲᩥ, ᨿᨾᩉᩴ ᨸᩔᩮᨿ᩠ᨿᩴ; ᨶᨲ᩠ᨳᩥ ᨾᩮ, ᩋᩣᩅᩩᩈᩮᩣ, ᨸᩣᨸᩥᨠᩣ ᨴᩥᨭ᩠ᨮᩥ, ᨿᨾᩉᩴ ᨸᨭᩥᨶᩥᩔᨩ᩠ᨩᩮᨿ᩠ᨿᨶ᩺’ᨲᩥ. ᨲᩴ ᩈᩴᨥᩮᩣ ᩋᨴᩔᨶᩮ ᩅᩤ ᩋᨸ᩠ᨸᨭᩥᨶᩥᩔᨣ᩠ᨣᩮ ᩅᩤ ᩏᨠ᩠ᨡᩥᨸᨲᩥ—  ᩋᨵᨾ᩠ᨾᨠᨾ᩠ᨾᩴ.
</Paragraph>

<Paragraph number="1819" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩩᩔ ᨶ ᩉᩮᩣᨲᩥ ᩋᩣᨸᨲ᩠ᨲᩥ ᨸᨭᩥᨠᩣᨲᨻ᩠ᨻᩣ, ᨶ ᩉᩮᩣᨲᩥ ᨸᩣᨸᩥᨠᩣ ᨴᩥᨭ᩠ᨮᩥ ᨸᨭᩥᨶᩥᩔᨩ᩠ᨩᩮᨲᩣ. ᨲᨾᩮᨶᩴ ᨧᩮᩣᨴᩮᨲᩥ ᩈᩴᨥᩮᩣ ᩅᩤ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩅᩤ ᩑᨠᨸᩩᨣ᩠ᨣᩃᩮᩣ ᩅᩤ—  ‘ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᨲ᩠ᩅᩴ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨶ᩠ᨶᩮᩣ, ᨸᨭᩥᨠᩁᩮᩣᩉᩥ ᨲᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ; ᨸᩣᨸᩥᨠᩣ ᨲᩮ ᨴᩥᨭ᩠ᨮᩥ, ᨸᨭᩥᨶᩥᩔᨩ᩠ᨩᩮᨲᩴ ᨸᩣᨸᩥᨠᩴ ᨴᩥᨭ᩠ᨮᩥᨶ᩺’ᨲᩥ. ᩈᩮᩣ ᩑᩅᩴ ᩅᨴᩮᨲᩥ—  ‘ᨶᨲ᩠ᨳᩥ ᨾᩮ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨲ᩠ᨲᩥ, ᨿᨾᩉᩴ ᨸᨭᩥᨠᩁᩮᨿ᩠ᨿᩴ. ᨶᨲ᩠ᨳᩥ ᨾᩮ, ᩋᩣᩅᩩᩈᩮᩣ, ᨸᩣᨸᩥᨠᩣ ᨴᩥᨭ᩠ᨮᩥ, ᨿᨾᩉᩴ ᨸᨭᩥᨶᩥᩔᨩ᩠ᨩᩮᨿ᩠ᨿᨶ᩺’ᨲᩥ. ᨲᩴ ᩈᩴᨥᩮᩣ ᩋᨸ᩠ᨸᨭᩥᨠᨾ᩠ᨾᩮ ᩅᩤ ᩋᨸ᩠ᨸᨭᩥᨶᩥᩔᨣ᩠ᨣᩮ ᩅᩤ ᩏᨠ᩠ᨡᩥᨸᨲᩥ—  ᩋᨵᨾ᩠ᨾᨠᨾ᩠ᨾᩴ.
</Paragraph>

<Paragraph number="1820" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩩᩔ ᨶ ᩉᩮᩣᨲᩥ ᩋᩣᨸᨲ᩠ᨲᩥ ᨴᨭ᩠ᨮᨻ᩠ᨻᩣ, ᨶ ᩉᩮᩣᨲᩥ ᩋᩣᨸᨲ᩠ᨲᩥ ᨸᨭᩥᨠᩣᨲᨻ᩠ᨻᩣ, ᨶ ᩉᩮᩣᨲᩥ ᨸᩣᨸᩥᨠᩣ ᨴᩥᨭ᩠ᨮᩥ ᨸᨭᩥᨶᩥᩔᨩ᩠ᨩᩮᨲᩣ. ᨲᨾᩮᨶᩴ ᨧᩮᩣᨴᩮᨲᩥ ᩈᩴᨥᩮᩣ ᩅᩤ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩅᩤ ᩑᨠᨸᩩᨣ᩠ᨣᩃᩮᩣ ᩅᩤ—  ‘ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᨲ᩠ᩅᩴ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨶ᩠ᨶᩮᩣ, ᨸᩔᩈᩮᨲᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ? ᨸᨭᩥᨠᩁᩮᩣᩉᩥ ᨲᩴ ᩋᩣᨸᨲ᩠ᨲᩥᩴ; ᨸᩣᨸᩥᨠᩣ ᨲᩮ ᨴᩥᨭ᩠ᨮᩥ, ᨸᨭᩥᨶᩥᩔᨩ᩠ᨩᩮᨲᩴ ᨸᩣᨸᩥᨠᩴ ᨴᩥᨭ᩠ᨮᩥᨶ᩺’ᨲᩥ. ᩈᩮᩣ ᩑᩅᩴ ᩅᨴᩮᨲᩥ—  ‘ᨶᨲ᩠ᨳᩥ ᨾᩮ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨲ᩠ᨲᩥ, ᨿᨾᩉᩴ ᨸᩔᩮᨿ᩠ᨿᩴ. ᨶᨲ᩠ᨳᩥ ᨾᩮ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨲ᩠ᨲᩥ, ᨿᨾᩉᩴ ᨸᨭᩥᨠᩁᩮᨿ᩠ᨿᩴ. ᨶᨲ᩠ᨳᩥ ᨾᩮ, ᩋᩣᩅᩩᩈᩮᩣ, ᨸᩣᨸᩥᨠᩣ ᨴᩥᨭ᩠ᨮᩥ, ᨿᨾᩉᩴ ᨸᨭᩥᨶᩥᩔᨩ᩠ᨩᩮᨿ᩠ᨿᨶ᩺’ᨲᩥ. ᨲᩴ ᩈᩴᨥᩮᩣ ᩋᨴᩔᨶᩮ ᩅᩤ ᩋᨸ᩠ᨸᨭᩥᨠᨾ᩠ᨾᩮ ᩅᩤ ᩋᨸ᩠ᨸᨭᩥᨶᩥᩔᨣ᩠ᨣᩮ ᩅᩤ ᩏᨠ᩠ᨡᩥᨸᨲᩥ—  ᩋᨵᨾ᩠ᨾᨠᨾ᩠ᨾᩴ.
</Paragraph>

<Paragraph number="1821" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩩᩔ ᩉᩮᩣᨲᩥ ᩋᩣᨸᨲ᩠ᨲᩥ ᨴᨭ᩠ᨮᨻ᩠ᨻᩣ. ᨲᨾᩮᨶᩴ ᨧᩮᩣᨴᩮᨲᩥ. ᩈᩴᨥᩮᩣ ᩅᩤ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩅᩤ ᩑᨠᨸᩩᨣ᩠ᨣᩃᩮᩣ ᩅᩤ—  ‘ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᨲ᩠ᩅᩴ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨶ᩠ᨶᩮᩣ, ᨸᩔᩈᩮᨲᩴ ᩋᩣᨸᨲ᩠ᨲᩥᨶ᩺’ᨲᩥ? ᩈᩮᩣ ᩑᩅᩴ ᩅᨴᩮᨲᩥ—  ‘ᩋᩣᨾᩣᩅᩩᩈᩮᩣ, ᨸᩔᩣᨾᩦ’ᨲᩥ. ᨲᩴ ᩈᩴᨥᩮᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩋᨴᩔᨶᩮ ᩏᨠ᩠ᨡᩥᨸᨲᩥ—  ᩋᨵᨾ᩠ᨾᨠᨾ᩠ᨾᩴ.
</Paragraph>

<Paragraph number="1823" book="vi-maha">

ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩩᩔ ᩉᩮᩣᨲᩥ ᩋᩣᨸᨲ᩠ᨲᩥ ᨴᨭ᩠ᨮᨻ᩠ᨻᩣ. ᨲᨾᩮᨶᩴ ᨧᩮᩣᨴᩮᨲᩥ ᩈᩴᨥᩮᩣ ᩅᩤ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩅᩤ ᩑᨠᨸᩩᨣ᩠ᨣᩃᩮᩣ ᩅᩤ—  ‘ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᨲ᩠ᩅᩴ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨶ᩠ᨶᩮᩣ, ᨸᩔᩈᩮᨲᩴ ᩋᩣᨸᨲ᩠ᨲᩥᨶ᩺’ᨲᩥ? ᩈᩮᩣ ᩑᩅᩴ ᩅᨴᩮᨲᩥ—  ‘ᨶᨲ᩠ᨳᩥ ᨾᩮ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨲ᩠ᨲᩥ, ᨿᨾᩉᩴ ᨸᩔᩮᨿ᩠ᨿᨶ᩺’ᨲᩥ. ᨲᩴ ᩈᩴᨥᩮᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩋᨴᩔᨶᩮ ᩏᨠ᩠ᨡᩥᨸᨲᩥ—  ᨵᨾ᩠ᨾᨠᨾ᩠ᨾᩴ.

<Paragraph number="1826" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩩᩔ ᩉᩮᩣᨲᩥ ᩋᩣᨸᨲ᩠ᨲᩥ ᨸᨭᩥᨠᩣᨲᨻ᩠ᨻᩣ. ᨲᨾᩮᨶᩴ ᨧᩮᩣᨴᩮᨲᩥ ᩈᩴᨥᩮᩣ ᩅᩤ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩅᩤ ᩑᨠᨸᩩᨣ᩠ᨣᩃᩮᩣ ᩅᩤ—  ‘ᩋᩣᨸᨲ᩠ᨲᩥᩴ ᨲ᩠ᩅᩴ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨶ᩠ᨶᩮᩣ, ᨸᨭᩥᨠᩁᩮᩣᩉᩥ ᨲᩴ ᩋᩣᨸᨲ᩠ᨲᩥᨶ᩺’ᨲᩥ. ᩈᩮᩣ ᩑᩅᩴ ᩅᨴᩮᨲᩥ—  ‘ᨶᨲ᩠ᨳᩥ ᨾᩮ, ᩋᩣᩅᩩᩈᩮᩣ, ᩋᩣᨸᨲ᩠ᨲᩥ, ᨿᨾᩉᩴ ᨸᨭᩥᨠᩁᩮᨿ᩠ᨿᨶ᩺’ᨲᩥ. ᨲᩴ ᩈᩴᨥᩮᩣ ᩋᩣᨸᨲ᩠ᨲᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨠᨾ᩠ᨾᩮ ᩏᨠ᩠ᨡᩥᨸᨲᩥ—  ᨵᨾ᩠ᨾᨠᨾ᩠ᨾᩴ.

<Paragraph number="1827" book="vi-maha">
ᩍᨵ ᨸᨶ, ᨽᩥᨠ᩠ᨡᩅᩮ, ᨽᩥᨠ᩠ᨡᩩᩔ ᩉᩮᩣᨲᩥ ᨸᩣᨸᩥᨠᩣ ᨴᩥᨭ᩠ᨮᩥ ᨸᨭᩥᨶᩥᩔᨩ᩠ᨩᩮᨲᩣ. ᨲᨾᩮᨶᩴ ᨧᩮᩣᨴᩮᨲᩥ ᩈᩴᨥᩮᩣ ᩅᩤ ᩈᨾ᩠ᨻᩉᩩᩃᩣ ᩅᩤ ᩑᨠᨸᩩᨣ᩠ᨣᩃᩮᩣ ᩅᩤ—  ‘ᨸᩣᨸᩥᨠᩣ ᨲᩮ, ᩋᩣᩅᩩᩈᩮᩣ, ᨴᩥᨭ᩠ᨮᩥ, ᨸᨭᩥᨶᩥᩔᨩ᩠ᨩᩮᨲᩴ ᨸᩣᨸᩥᨠᩴ ᨴᩥᨭ᩠ᨮᩥᨶ᩺’ᨲᩥ. ᩈᩮᩣ ᩑᩅᩴ ᩅᨴᩮᨲᩥ—  ‘ᨶᨲ᩠ᨳᩥ ᨾᩮ, ᩋᩣᩅᩩᩈᩮᩣ, ᨸᩣᨸᩥᨠᩣ ᨴᩥᨭ᩠ᨮᩥ, ᨿᨾᩉᩴ ᨸᨭᩥᨶᩥᩔᨩ᩠ᨩᩮᨿ᩠ᨿᨶ᩺’ᨲᩥ. ᨲᩴ ᩈᩴᨥᩮᩣ ᨸᩣᨸᩥᨠᩣᨿ ᨴᩥᨭ᩠ᨮᩥᨿᩣ ᩋᨸ᩠ᨸᨭᩥᨶᩥᩔᨣ᩠ᨣᩮ ᩏᨠ᩠ᨡᩥᨸᨲᩥ—  ᨵᨾ᩠ᨾᨠᨾ᩠ᨾᩴ.
