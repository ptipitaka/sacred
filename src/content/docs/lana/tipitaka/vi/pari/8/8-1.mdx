ᩏᨸᩮᩣᩈᨳᨠᨾ᩠ᨾᩔ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᨸᩅᩤᩁᨱᩣᨠᨾ᩠ᨾᩔ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᨲᨩ᩠ᨩᨶᩦᨿᨠᨾ᩠ᨾᩔ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᨶᩥᨿᩔᨠᨾ᩠ᨾᩔ …ᨸᩮ…  ᨸᨻ᩠ᨻᩣᨩᨶᩦᨿᨠᨾ᩠ᨾᩔ …ᨸᩮ…  ᨸᨭᩥᩈᩣᩁᨱᩦᨿᨠᨾ᩠ᨾᩔ …ᨸᩮ…  ᩏᨠ᩠ᨡᩮᨸᨶᩦᨿᨠᨾ᩠ᨾᩔ …ᨸᩮ…  ᨸᩁᩥᩅᩤᩈᨴᩤᨶᩔ …ᨸᩮ…  ᨾᩪᩃᩣᨿᨸᨭᩥᨠᩔᨶᩣᨿ …ᨸᩮ…  ᨾᩣᨶᨲ᩠ᨲᨴᩤᨶᩔ …ᨸᩮ…  ᩋᨻ᩠ᨽᩣᨶᩔ …ᨸᩮ…  ᩏᨸᩈᨾ᩠ᨸᨴᩤᨠᨾ᩠ᨾᩔ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᨲᨩ᩠ᨩᨶᩦᨿᨠᨾ᩠ᨾᩔ ᨸᨭᩥᨸ᩠ᨸᩔᨴ᩠ᨵᩥᨿᩣ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᨶᩥᨿᩔᨠᨾ᩠ᨾᩔ ᨸᨭᩥᨸ᩠ᨸᩔᨴ᩠ᨵᩥᨿᩣ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᨸᨻ᩠ᨻᩣᨩᨶᩦᨿᨠᨾ᩠ᨾᩔ ᨸᨭᩥᨸ᩠ᨸᩔᨴ᩠ᨵᩥᨿᩣ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᨸᨭᩥᩈᩣᩁᨱᩦᨿᨠᨾ᩠ᨾᩔ ᨸᨭᩥᨸ᩠ᨸᩔᨴ᩠ᨵᩥᨿᩣ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᩏᨠ᩠ᨡᩮᨸᨶᩦᨿᨠᨾ᩠ᨾᩔ ᨸᨭᩥᨸ᩠ᨸᩔᨴ᩠ᨵᩥᨿᩣ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᩈᨲᩥᩅᩥᨶᨿᩔ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨩᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᩋᨾᩪᩊ᩠ᩉᩅᩥᨶᨿᩔ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᨲᩔᨸᩣᨸᩥᨿᩈᩥᨠᩣᨿ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᨲᩥᨱᩅᨲ᩠ᨳᩣᩁᨠᩔ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᨽᩥᨠ᩠ᨡᩩᨶᩮᩣᩅᩤᨴᨠᩈᨾ᩠ᨾᩩᨲᩥᨿᩣ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᨲᩥᨧᩦᩅᩁᩮᨶ ᩋᩅᩥᨸ᩠ᨸᩅᩤᩈᩈᨾ᩠ᨾᩩᨲᩥᨿᩣ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᩈᨶ᩠ᨳᨲᩈᨾ᩠ᨾᩩᨲᩥᨿᩣ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᩁᩪᨸᩥᨿᨨᨯ᩠ᨯᨠᩈᨾ᩠ᨾᩩᨲᩥᨿᩣ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᩈᩣᨭᩥᨿᨣ᩠ᨣᩤᩉᩣᨸᨠᩈᨾ᩠ᨾᩩᨲᩥᨿᩮᩣ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᨸᨲ᩠ᨲᨣ᩠ᨣᩤᩉᩣᨸᨠᩈᨾ᩠ᨾᩩᨲᩥᨿᩣ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᨴᨱ᩠ᨯᩈᨾ᩠ᨾᩩᨲᩥᨿᩣ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᩈᩥᨠ᩠ᨠᩣᩈᨾ᩠ᨾᩩᨲᩥᨿᩣ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ? ᨴᨱ᩠ᨯᩈᩥᨠ᩠ᨠᩣᩈᨾ᩠ᨾᩩᨲᩥᨿᩣ ᨠᩮᩣ ᩋᩣᨴᩥ, ᨠᩥᩴ ᨾᨩ᩠ᨫᩮ, ᨠᩥᩴ ᨸᩁᩥᨿᩮᩣᩈᩣᨶᩴ?
</Paragraph>
