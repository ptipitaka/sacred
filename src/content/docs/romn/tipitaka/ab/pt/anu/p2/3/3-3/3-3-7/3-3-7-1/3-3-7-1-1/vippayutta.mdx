<Division number="97">

<Paragraph number="1229">
Āsavasampayutto dhammo āsavavippayuttassa dhammassa vippayuttapaccayena paccayo—  sahajātaṃ, pacchājātaṃ. <Emphasis>Sahajātā</Emphasis>—  āsavasampayuttā khandhā cittasamuṭṭhānānaṃ rūpānaṃ vippayuttapaccayena paccayo. <Emphasis>Pacchājātā</Emphasis>—  āsavasampayuttā khandhā purejātassa imassa kāyassa vippayuttapaccayena paccayo. (1)
</Paragraph>

<Paragraph number="1230">
Āsavavippayutto dhammo āsavavippayuttassa dhammassa vippayuttapaccayena paccayo—  sahajātaṃ, purejātaṃ, pacchājātaṃ. (Saṃkhittaṃ. Vitthāretabbaṃ.) (1)
</Paragraph>

</Division>
<Division number="98">

<Paragraph number="1231">
Āsavavippayutto dhammo āsavasampayuttassa dhammassa vippayuttapaccayena paccayo. <Emphasis>Purejātaṃ</Emphasis>—  vatthu āsavasampayuttakānaṃ khandhānaṃ vippayuttapaccayena paccayo. (1)
</Paragraph>

<Paragraph number="1232">
Āsavavippayutto dhammo āsavasampayuttassa ca āsavavippayuttassa ca dhammassa vippayuttapaccayena paccayo. <Emphasis>Purejātaṃ</Emphasis>—  vatthu domanassasahagatānaṃ vicikicchāsahagatānaṃ uddhaccasahagatānaṃ khandhānaṃ mohassa ca vippayuttapaccayena paccayo. (2)
</Paragraph>

<Paragraph number="1233">
Āsavasampayutto ca āsavavippayutto ca dhammā āsavavippayuttassa dhammassa vippayuttapaccayena paccayo—  sahajātaṃ, pacchājātaṃ. <Emphasis>Sahajātā</Emphasis>—  domanassasahagatā vicikicchāsahagatā uddhaccasahagatā khandhā ca moho ca cittasamuṭṭhānānaṃ rūpānaṃ vippayuttapaccayena paccayo. <Emphasis>Pacchājātā</Emphasis>—  domanassasahagatā vicikicchāsahagatā uddhaccasahagatā khandhā ca moho ca purejātassa imassa kāyassa vippayuttapaccayena paccayo. (1)
</Paragraph>
</Division>
