<Division number="89">

<Paragraph number="1212">
Āsavavippayutto dhammo āsavavippayuttassa dhammassa purejātapaccayena paccayo—  ārammaṇapurejātaṃ, vatthupurejātaṃ. <Emphasis>Ārammaṇapurejātaṃ</Emphasis>—  cakkhuṃ …pe…  vatthuṃ aniccato dukkhato anattato vipassati; dibbena cakkhunā rūpaṃ passati, dibbāya sotadhātuyā saddaṃ suṇāti. Rūpāyatanaṃ cakkhuviññāṇassa …pe…  phoṭṭhabbāyatanaṃ kāyaviññāṇassa purejātapaccayena paccayo. <Emphasis>Vatthupurejātaṃ</Emphasis>—  cakkhāyatanaṃ cakkhuviññāṇassa …pe…  kāyāyatanaṃ kāyaviññāṇassa …pe…  vatthu āsavavippayuttānaṃ khandhānaṃ mohassa ca purejātapaccayena paccayo. (1)
</Paragraph>

<Paragraph number="1213">
Āsavavippayutto dhammo āsavasampayuttassa dhammassa purejātapaccayena paccayo—  ārammaṇapurejātaṃ, vatthupurejātaṃ. <Emphasis>Ārammaṇapurejātaṃ</Emphasis>—  cakkhuṃ …pe…  vatthuṃ assādeti abhinandati, taṃ ārabbha rāgo uppajjati …pe…  domanassaṃ uppajjati. <Emphasis>Vatthupurejātaṃ</Emphasis>—  vatthu āsavasampayuttakānaṃ khandhānaṃ purejātapaccayena paccayo. (2)
</Paragraph>

<Paragraph number="1214">
Āsavavippayutto dhammo āsavasampayuttassa ca āsavavippayuttassa ca dhammassa purejātapaccayena paccayo—  ārammaṇapurejātaṃ, vatthupurejātaṃ. <Emphasis>Ārammaṇapurejātaṃ</Emphasis>—  cakkhuṃ …pe…  vatthuṃ ārabbha domanassasahagatā vicikicchāsahagatā uddhaccasahagatā khandhā ca moho ca uppajjanti. <Emphasis>Vatthupurejātaṃ</Emphasis>—  vatthu domanassasahagatānaṃ vicikicchāsahagatānaṃ uddhaccasahagatānaṃ khandhānaṃ mohassa ca purejātapaccayena paccayo. (3)
</Paragraph>
</Division>
