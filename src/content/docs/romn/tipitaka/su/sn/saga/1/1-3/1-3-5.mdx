_“Yo hoti bhikkhu arahaṃ katāvī,_  
_Khīṇāsavo antimadehadhārī;_  
_Ahaṃ vadāmītipi so vadeyya,_  
_Mamaṃ vadantītipi so vadeyyā”ti._  

</Paragraph>

<Paragraph number="108" type="verses" book="saga">
_“Yo hoti bhikkhu arahaṃ katāvī,_  
_Khīṇāsavo antimadehadhārī;_  
_Ahaṃ vadāmītipi so vadeyya,_  
_Mamaṃ vadantītipi so vadeyya;_  
_Loke samaññaṃ kusalo viditvā,_  
_Vohāramattena so vohareyyā”ti._  

</Paragraph>

<Paragraph number="109" type="verses" book="saga">
_“Yo hoti bhikkhu arahaṃ katāvī,_  
_Khīṇāsavo antimadehadhārī;_  
_Mānaṃ nu kho so upagamma bhikkhu,_  
_Ahaṃ vadāmītipi so vadeyya;_  
_Mamaṃ vadantītipi so vadeyyā”ti._  

</Paragraph>

<Paragraph number="110" type="verses" book="saga">
_“Pahīnamānassa na santi ganthā,_  
_Vidhūpitā mānaganthassa sabbe;_  
_Sa vītivatto maññataṃ sumedho,_  
_Ahaṃ vadāmītipi so vadeyya._  

</Paragraph>

<Paragraph number="111" type="verses" book="saga">
_Mamaṃ vadantītipi so vadeyya,_  
_Loke samaññaṃ kusalo viditvā;_  
_Vohāramattena so vohareyyā”ti._
</Paragraph>
</Division>
