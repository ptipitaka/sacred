_kākanīḷasmimacchati;_

_yassa cetaṃ kulāvakaṃ”._

_dija sāmākabhojana;_  
_Akatvā vacanaṃ tuyhaṃ,_  
_passa lūnosmi āgato”._  

</Paragraph>

_sīlañhi tava tādisaṃ;_  
_Na hi mānusakā bhogā,_  
_subhuñjā honti pakkhinā”ti._  


</Paragraph>
