<Paragraph number="786" type="verses" book="ja1">
_“Yathā māṇavako panthe,_  
_siṅgāliṃ vanagocariṃ;_  
_Atthakāmaṃ pavedentiṃ,_  
_anatthakāmāti maññati;_  
_Anatthakāmaṃ satapattaṃ,_  
_atthakāmoti maññati._  

</Paragraph>

</Division>
<Division number="86" book="ja1" bookNo="22" page="213">

<Paragraph number="787" type="verses" book="ja1">
_Evameva idhekacco,_  
_puggalo hoti tādiso;_  
_Hitehi vacanaṃ vutto,_  
_paṭigaṇhāti vāmato._  

</Paragraph>

</Division>
<Division number="87" book="ja1" bookNo="22" page="213">

<Paragraph number="788" type="verses" book="ja1">
_Ye ca kho naṃ pasaṃsanti,_  
_bhayā ukkaṃsayanti vā;_  
_Tañhi so maññate mittaṃ,_  
_satapattaṃva māṇavo”ti._  


</Paragraph>
</Division>

<Paragraph number="789" type="center" book="ja1">
Satapattajātakaṃ navamaṃ.
</Paragraph>
